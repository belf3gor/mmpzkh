Настройка параметров учета

Форма предназначена для установки параметров ведения учета, которые являются общими для всех организаций информационной базы.

Параметры учета сгруппированы по закладкам:

  • Налог на прибыль,
  • НДС,
  • Банк и касса,
  • Расчеты с контрагентами,
  • Розничная торговля,
  • Запасы,
  • Комиссионная торговля,
  • Производство,
  • Сотрудники и зарплата.
Налог на прибыль

Флажок  Применяются разные ставки налога на прибыль позволяет настроить величину ставки налога на прибыль, уплачиваемой в бюджет субъекта РФ для каждой организации отдельно. Если флажок не установлен, то используется единая величина ставки налога на прибыль в бюджет субъекта РФ для всех организаций.
Указать ставки налога на прибыль в федеральный бюджет и бюджеты субъектов РФ можно по гиперссылке Ставки налога на прибыль

Выбирается один из способов учета в стоимости имущества и услуг, предварительно оплаченных по договору в валюте:

  • Поступления или реализации имущества и услуг – стоимость имущества в рублях определяется стоимостью имущества в валюте, пересчитанной в рубли на дату перехода права собственности,
  • Получения или выдачи аванса – стоимость имущества в рублях формируется с учетом величин перечисленных авансов в рублях на дату перечисления авансов.

При выборе второго варианта указывается дата, с которой он применяется. До этой даты стоимость имущества будет определяться по первому варианту.

НДС

Устанавливается дата, начиная с которой организация ведет учет  в соответствии с постановлением правительства РФ от 26 декабря 2011 года № 1137. До этой даты применяется порядок, установленный постановлением правительства РФ от 2 декабря 2000г. № 914

Банк и касса

Организация может вести учет расчетов с контрагентами в валюте и у.е. В этом случае, в настройках параметров учета нужно установить флажок Ведется учета расчетов в валюте и у.е. Выбранный вариант влияет на интерфейс программы.
Из формы по ссылке можно перейти к справочнику валют и на форму регистра сведений счетов с особым порядком переоценки валют

Настраивается аналитический учет по счетам 50 и 51. Учет на счете 51 по расчетным счетам ведется всегда. Дополнительно установкой соответствующих флажков можно включить ведение аналитического учета по статьям движения денежных средств.

Обороты по статьям движения денежных средств используются при формировании бухгалтерской отчетности для автоматического заполнения Формы №4 "Отчет о движении денежных средств".
Если нет необходимости в автоматическом заполнении Формы №4, то учет по статьям движения денежных средств можно не вести. Это упрощает работу с банковскими и кассовыми документами - при вводе документов в этом случае не нужно указывать статью движения денежных средств.

Примечание
По статьям движения денежных средств можно анализировать только обороты, остатки денежных средств в разрезе статей движения не хранятся.

Настройка самих статей ДДС производится в форме справочника по ссылке Статьи движения денежных средств.

Расчеты с контрагентами

Настраиваются срок, по истечении которого долг покупателя и долг продавца соответственно, считается просроченным, если другой порядок не установлен в договоре соответственно в полях Срок оплаты покупателей и Срок оплаты поставщикам.

Розничная торговля

Если хотя бы в одной из организаций ведется розничная торговля флажок Ведется розничная торговля должен быть установлен.

Настраивается аналитический учет по счету 41.12 "Товары в розничной торговле (в НТТ по продажной стоимости)". Этот счет предназначен для учета товаров в рознице по продажным ценам (с использованием счета 42 "Торговая наценка") для складов, являющихся неавтоматизированными торговыми точками (НТТ).

Учет на счете 41.12 всегда ведется по складам (торговым точкам). Дополнительно установкой соответствующих флажков можно включить ведение аналитики по номенклатуре или по ставкам НДС.

Примечание
По счету 41.12 в разрезе номенклатуры можно анализировать только обороты, остатки в разрезе номенклатуры не хранятся.

Запасы

Настраиваются параметры учета материалов, товаров, готовой продукции и других видов материально-производственных запасов.

Аналитический учет запасов на счетах учета всегда ведется по номенклатурным позициям (наименованиям товаров, материалов, продукции). Кроме того, дополнительно может быть установлен аналитический учет запасов по партиям и по складам. При этом устанавливаются единые настройки для бухгалтерского учета и для налогового учета.
Если в учетной политике хотя бы одной из организаций задан способ оценки стоимости материально-производственных запасов ФИФО, то необходимо установить флажок Ведется учет по партиям (документам поступления).

Необходимо выбрать вариант ведения складского учета.

  • Если не ведется учет запасов по складам, то необходимо снять флажок соответствующий субконто По складам (местам хранения).
  • Если складской учет ведется, то можно вести только количественный учет. Для этого необходимо при установленном флажке субконто По складам (местам хранения) выбрать вариант По количеству. Это позволит контролировать наличие запасов на разных складах в количественном выражении, а цена списания запасов будет определяться путем деления общей стоимости данной номенклатурной позиции на всех складах на общее количество.
  • Предусмотрена возможность вести количественный и суммовой складской учет. Для этого необходимо при установленном флажке субконто По складам (местам хранения) выбрать вариант По количеству и сумме. В этом случае цена списания запасов будет определяться по каждому складу отдельно.

При проведении документов выбытия материально-производственных запасов, контролируется наличие списываемых запасов. При их отсутствии проведение документа отменяется и выдается сообщение об ошибке. Иногда такой контроль не нужен, например, если сначала в информационную базу вводится информация о списании запасов и только потом о поступлении. В этом случае необходимо установить флажок Разрешается списание запасов при отсутствии остатков по данным учета.

Если хотя бы в одной организации ведется учет оборота возвратной тары, то необходимо установить флажок Ведется учет возвратной тары. Тогда в документах движения запасов добавляется закладка "Тара", на которой указывается информация о поступлении и списании возвратной тары.

Если хотя бы в одной организации ведется учет импортных товаров в разрезе ГТД и стран происхождения, то необходимо установить флажок Ведется учет импортных товаров. В этом случае в документах поступления, реализации и списания товаров добавляется графы "Номер ГТД" и "Страна происхождения", в карточке номенклатуры становится доступна информация о номере ГТД и стране происхождения, а в командном интерфейсе становится доступным справочник "Номера ГТД" и документ "ГТД Импорт".

Также на закладке устанавливается режим вывода колонок «Код» или «Артикул» при формировании печатных форм.

Комиссионная торговля

Если хотя бы в одной из организаций предполагается деятельность связанная с комиссионной торговлей и агентскими услугами, должен быть установлен флажок Ведется деятельность, связанная с комиссионной торговлей и агентскими услугами. Установка флажка изменяет интерфейс программы: в меню появляется отчет комиссионера и отчет комитенту, в договорах контрагента появляются реквизиты способ расчета и % комиссионного вознаграждения, в документах Отчет о розничных продажах и Реализация (акт, накладная) появляется табличная часть "Агентские услуги", в документе Перемещение товаров - табличная часть "Товары на комиссии".

Производство

Флаг Ведется производственная деятельность предоставляет доступ к документам учета производства.

Заполнение Типа плановых цен позволяет автоматически заполнять плановую стоимость в документах учета производства.

 

На счетах 90.02 "Себестоимость продаж", 90.07 "Коммерческие расходы", 90.08 "Управленческие расходы" может использоваться дополнительная аналитика учета расходов - по элементам и статьям затрат.

Детализация формирования расходов настраивается в учетной политике организации.

На закладке "Производство" предоставлена возможность отключить использование дополнительной аналитики для информационной базы в целом. Для этого используются флаги По элементам затрат и По статьям затрат.

Возможность формировать расходы по элементам и статьям затрат доступна в версии КОРП

Сотрудники и зарплата

Организация может вести учет расчетов с персоналом по заработной плате и кадровый учет в текущей информационной базе или во внешней программе. Выбор варианта учета расчетов необходимо указать в настройках параметров учета. Выбранный вариант влияет на интерфейс программы и набор документов, которые используются при отражении расчетов с персоналом.

  • Если хотя бы одной из организаций необходимо вести учет по заработной плате по каждому сотруднику, то необходимо установить переключатель По каждому работнику. В этом случае к счету 70 "Расчеты с персоналом по оплате труда" устанавливается субконто "Работники организаций".

  • Если все организации планируют вносить данные по заработной плате в информационную базу сводными проводками по всем работникам, то необходимо установить переключатель Сводно по всем работникам.
