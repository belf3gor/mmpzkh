Список отчетов для банков

Список «Отчеты для банков» содержит список всех введенных отчетов. Список открывается по кнопке «Отчеты для банков» из раздела «Банк и касса».

При открытии ранее введенных отчетов сразу открывается форма отчета, без вызова «стартовой» формы.

Состояния отчетов обновляются автоматически при открытой форме с периодичностью 5 минут.

Для принудительного обновления состояния отчетов необходимо выполнить команду "Обновить состояния".

Финальным состоянием отчета является значение "Доставлено".
