Печать листовок (ЖКХ)

Форма предназначена для настройки, формирования и печати листовок ЖКХ для жильцов.

Форма позволяет установить следующие настройки:

Вид листовки

Настройки заполнения

Настройки печати персональных листовок

Вид листовки

В данном разделе настроек расположена таблица, которая предназначена для предпросмотра листовки и которая содержит наименование каждого варианта листовки.

Для выбора варианта листовки необходимо установить строку с вариантом листовки в качестве активной строки.

Настройки заполнения

В данном разделе настроек расположены следующие элементы:

  • Организация - наименование организации. Наименование вводится как вручную, так и автоматически при выборе организации из справочника "Организации" (при этом в данном поле выводится сокращенное наименование организации).
  • Сайт - адрес сайта организации. При выборе организации из справочника "Организации" значение данного поля заполняется автоматически из поля "Адрес сайта компании" Учетной политики (ЖКХ) выбранной организации.
  • Адрес - юридический или фактический адрес организации. При выборе организации из справочника "Организации" значение данного поля заполняется автоматически из поля "Юридический адрес" выбранной организации.
  • Телефон - контактный телефон организации. При выборе организации из справочника "Организации" значение данного поля заполняется автоматически из поля "Контактный телефон" выбранной организации.
  • Эл. почта - электронная почта организации. При выборе организации из справочника "Организации" значение данного поля заполняется автоматически из поля "Email" выбранной организации.
  • Дополнительный текст - дополнительная информация для вывода в листовке.

Заполненные настройки выводятся в листовке автоматически в соответствующих полях.

Настройки печати персональных листовок

Данный раздел доступен для листовок о доступе в личный кабинет ЖКХ с помощью мобильного приложения.

В данном разделе настроек расположены следующие элементы:

  • Печатать Общие / Персональные листовки. Настройка позволяет выбирать вариант печати листовки. Если выбрано значение "Общие", то информация в листовках содержит общую информацию. Если выбрано значение "Персональные", то информация в листовках содержит информацию по выбранным лицевым счетам.
  • Печатать по Всем / Выбранным лицевым счетам. Настройка доступна при выбранном значении "Персональные" настройки "Печатать Общие / Персональные листовки". Настройка позволяет устанавливать отбор по лицевым счетам, по которым необходимо печатать листовки. Если выбрано значение "Всем", то будут сформированы листовки по всем лицевым счетам, по которым имеется идентификатор для использования мобильного приложения. Если выбрано значение "Выбранным", то будут сформированы листовки только по выбранным лицевым счетам, по которым имеется идентификатор для использования мобильного приложения.
  • Таблица с лицевыми счетами. Данная таблица доступна при выбранном значении "Выбранным" настройки "Печатать по Всем / Выбранным лицевым счетам". В таблице можно выбрать лицевые счета, по которым необходимо распечатать листовки о доступе в личный кабинет ЖКХ с помощью мобильного приложения.
  • Кнопка "Заполнить". По нажатию данной кнопки открывается форма подбора лицевых счетов для заполнения таблицы с лицевыми счетами.
