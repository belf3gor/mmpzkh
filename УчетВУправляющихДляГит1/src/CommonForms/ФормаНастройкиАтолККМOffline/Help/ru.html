Форма настройки драйвера подключаемого оборудования "АТОЛ: ККМ offline"

Форма предназначена для настройки параметров обмена со смежным программным обеспечением в режиме «ККМ Offline» с помощью файлового обмена.

Параметр Формат обмена.

Параметр Формат обмена служит для указания используемого формата обмена АТОЛ экземпляром подключаемого оборудования.

Содержание выгружаемых файлов и процесс загрузки файлов будет меняться соотвественно выбранному значению параметра Формат обмена. Каждое значение параметра Формат обмена соответствует формату одного или нескольких программных продуктов АТОЛ:

  • "Frontol 5" - формат обмена с "Frontol 5";
  • "Frontol Win32 v.4.x" - формат обмена с "Frontol Win32" версии v.4;
  • "Frontol WinCE v.2.x" - формат обмена с "Frontol WinCE" версии v.2;
  • "АТОЛ:РМК" - формат обмена с "АТОЛ: Рабочее место кассира";
  • "Frontol xPOS 1.6.x" - формат обмена "Frontol xPOS" версии 1.6;
  • "Frontol xPOS 1.5.x" - формат обмена с "Frontol xPOS" версии 1.5.

 

После изменения значений параметров следует нажать кнопку "Записать и закрыть" для того, чтобы сохранить внесённые изменения, или кнопку "Отмена" для того, чтобы отказаться от их сохранения.
