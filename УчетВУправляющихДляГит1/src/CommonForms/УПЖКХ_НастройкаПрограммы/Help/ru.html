Идентификаторы

Задаются форматы идентификаторов лицевых счетов и приборов учета. Идентификаторы могут быть использованы при обмене данными с информационными системами, такими как "Система Город" и "Сайт ЖКХ".

Префикс – префикс для идентификатора лицевого счета и счетчика.

Можно вручную определить формат идентификаторов. Для этого необходимо установить флажок Определить формат идентификатора вручную. После установки флажка становятся доступными параметры идентификатора:

  • состав уникальной части: цифры, символы верхнего регистра, символы нижнего регистра;
  • длина уникальной части.

WEB-Интеграция

Здаются настройки формирования логинов и паролей для лицевых счетов, предназначенных для доступа на сайт ЖКХ.

  • Устанавливается длина пароля доступа (количество символов).
  • Настраивается состав пароля доступа, он может состоять из символов верхнего регистра, нижнего регистра, цифр, знаков пунктуации и любых их сочетаний.

Права доступа

Использовать ограничение доступа по объектам учета - расширенная настройка, позволяющая настроить ограничение прав доступа пользователей к объектам учета. Если настройка установлена, то пользователю будут доступны только те объекты учета, которые назначены ему в «Настройках доступа пользователей к объектам учета».

Примечание:
Ограничение доступа по объектам учета используется для пользователей информационной базы, настройки ролей которых удовлетовряют следующим требованиям:
- пользователю назначена одна или несколько ролей подсистемы ЖКХ;
- пользователь не обладает полными правами, т.е. роль "Полные права" пользователю не должна быть назначена.

Функциональность программы

Задаются настройки использования расширенного функционала:

Использовать функционал расчета составных услуг с несколькими нормативными ограничениями - если установить флажок, то появляется возможность рассчитывать составные услуги в зависимости от потребленного объема по тарифу каждой из составных услуг в пределах заданного ограничения по каждой из услуг. Данная настройка позволяет выполнить расчет услуги в соответствии с:

  • требованиями постановлений правительства Нижегородской области №594 от 30.08.2012 и №310 от 28.05.2012;
  • приказом Государственного комитета по ценам и тарифам Республики Крым от 21.12.2016 №53/1;
  • приказом Департамента городского хозяйства г.Севастополя от 23.01.2017 №15-ОД.

При установке флажка в справочнике "Услуги" для услуг со способом расчета "С ограничением по нормам" и "По показаниям счетчика и норме потребления" появляется возможность задавать размеры ограничений непосредственно при настройке составной услуги, а именно: в таблице "Составные услуги" становится видимой колонка "Размер ограничения", где по каждой составной услуге можно задавать свою норму ограничения.

Примечание:
Размеры ограничений при такой настройке задаются только в самой услуге, их не нужно устанавливать на лицевые счета.

Пример:

Необходимо рассчитать услугу Электроэнергия по трем тарифам:
   2,42 руб.- за объем, потребленный до 150 кВт∙ч;
   2,96 руб.- за объем, потребленный от 150 до 600 кВт∙ч;
   5,40 руб.- за объем, потребленный сверх 600 кВт∙ч.
Общий объем потребления по услуге составил 700 Квт.
Начисление = 150 * 2,42 + 450 * 2,96 + 100 * 5,40 = 2235 руб.

Вести учет владельцев единственного жилья* - если установить флажок, то становится доступным функционал ведения учета владельцев единственного жилья (для выполнения требований постановления Правительства Москвы №848-ПП от 13.12.2016 при расчете начислений услуг по содержанию жилого помещения).

При установке флажка в документах "Открытие лицевого счета" и "Изменение лицевого счета" для каждого жильца становится доступным признак "Владелец (наниматель) единственного жилья" (ВЕЖ), а в справочнике "Нормы потребления услуг" для норм на лицевой счет составных услуг становится доступным признак "Норма для владельцев (нанимателей) единственного жилья".

Примечание:
Признак ВЕЖ жильцу возможно установить:
- если жилец является собственником помещения для частного вида собственности;
- если жилец является ответственным собственником, нанимателем для всех видов собственности.

При начислении услуги с нормой, в которой установлен признак "Норма для владельцев (нанимателей) единственного жилья", расчет производится следующим образом:
1. Выполняется поиск зарегистрированных жильцов лицевого счета, для которых установлен признак "Владелец (наниматель) единственного жилья";
2. Если такие жильцы найдены, то в качестве основания нормы берется количество зарегистрированных жильцов на лицевом счете;
3. При расчете объема норматива учитывается доля собственности жильцов с признаком ВЕЖ.

Отладка программы

Режим отладки предназначен для использования дополнительного функционала, позволяющего упростить анализ работы программы.

Внимание!
Данный режим может привести к незапланированным изменениям в учетных данных, поэтому его включение рекомендуется производить только разработчикам или опытным пользователям.

Включить режим отладки - если установить флажок, то становятся доступны следующие возможности:

  1. Повторная выгрузка и загрузка файлов обмена, имеющихся в журнале обмена механизма "Интеграция с сайтом ЖКХ".
  2. Повторная выгрузка и загрузка файлов обмена, имеющихся в журнале обмена механизма "Обмен данными с мобильным приложением "ЖКХ: Личный кабинет".
