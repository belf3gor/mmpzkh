Настройка параметров рассылки квитанций

На форме настройки параметров рассылки выделены:

  • Основные параметры;
  • Параметры тестовой рассылки.

К основным параметрам относятся:

  • «Учетная запись» –  учетная запись электронной почты, которая будет использована в качестве отправителя писем;
  • «Вариант сохранения квитанций» – для рассылки квитанций требуется их сохранить, для выбора доступны следующие варианты:
  • Сохранить в каталог временных файлов - при рассылке квитанции сохраняются во временных каталог в виде файлов в формате "pdf";
  • Сохранить в пользовательский каталог - при рассылке квитанции сохраняются в каталог, указанный пользователем, в формате "pdf".

    • Примечание:
    Если информационная база работает в клиент-серверном режиме, то необходимо указывать каталог на сервере 1С:Предприятия или доступный сетевой каталог.

    • «Задержка при отправке» –  задержка перед отправкой каждого следующего письма в секундах. Установка задержки может потребоваться для того, чтобы почтовый сервер не блокировал массовую отправку писем.

      Например, если задержка при отправке установлена в значение 5 секунд, то каждое последующее письмо будет отправляться через 5 секунд после предыдущего письма.

    Параметры содержания электронного письма:

    • «Отправлять в письме сведения об организации» – признак того, что при формировании электронного письма будут выводиться сведения об организации:
      • скоращенное наименование;
      • контактный телефон;
      • фактический адрес.

    К параметрам тестовой рассылки относятся:

    • «E-mail получателя» – адрес электронной почты, на которой будет производится отправка писем при тестовой рассылке;
    • «Количество тестовых писем» – количество тестовых писем, отправляемых при тестовой рассылке.
