Отчет «Сведения о задолженностях по лицевым счетам»

Отчет предназначен для вывода данных о просроченных задолженностях по лицевым счетам на указанную дату, с возможностью отбора задолженности по услугам.

Перед формированием отчета необходимо указать настройки отчета:

  • «Период» – указывается дата, на которую формируется отчет. При этом в отчете учитывается настройка "Срок оплаты начислений" из Учетной политики ЖКХ:
    • Если дата, выбранная в поле "Период" меньше даты срока оплаты, то начисления за предыдущий месяц НЕ выводятся в отчете;
    • Если дата, выбранная в поле "Период" больше даты срока оплаты, то начисления за предыдущий месяц выводятся в отчете.
  • «Организация»– из справочника «Организации» выбирается организация, предоставляющая услуги.
  • «Объект»– указывается объект, по которому производится формирование отчета.
  • «Минимальная сумма» – указывается минимальная сумма задолженности. В отчет будут попадать только те должники, сумма задолженности которых не меньше указанной.
  • «Мин. кол-во месяцев» – указывается минимальная количество месяцев задолженности. В отчет будут попадать только те должники, количество месяцев задолженности которых не меньше указанной.
  • «Вид собственности» – указывается вид собственности. В отчет будут попадать только должники с выбранным видом собственности.
  • «Вариант отбора услуг» - вариант отбора услуг, с учетом которого будет сформирован отчет. 
     Доступны следующие значения:
    1. "формировать отчет по всем услугам" - при выборе данного значения отчет будет сформирован без отбора по услугам,
    2. "формировать отчет по указанным услугам" - при выборе данного значения отчет будет сформирован только по тем услугам, которые выбраны в таблице "Отбор услуг (только для варианта "формировать отчет по указанным услугам")",
    3. "формировать отчет по услугам, отличным от кап. ремонта" - при выборе данного значения отчет будет сформирован только по услугам, отличным от услуг капитального ремонта, установленным для Отчетной организации,
    4. "формировать отчет по услугам, отличным от кап. ремонта" - при выборе данного значения отчет будет сформирован только по услугам капитального ремонта, установленным для Отчетной организации.  

 

Отчет вызывается из «Работа с должниками / Отчеты / Сведения о задолженностях по лицевым счетам».
