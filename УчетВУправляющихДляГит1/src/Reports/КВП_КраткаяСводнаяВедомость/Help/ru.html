Отчет «Краткая сводная ведомость»

Отчет предназначен для формирования краткой формы сводной ведомости по начислению и оплате за указанный период, с возможностью отбора начислений и оплат по услугам.

В отчет выводятся данные:

  • «Квартира» – номер квартиры, владелец и номер л/с;
  • «Площадь» – размер общей площади, приходящейся на лицевой счет;
  • «Кол-во проживающих» – количество человек по данному лицевому счету или группе лицевых счетов;
  • «Сальдо на начало месяца» – общая задолженность на начало указанного периода;
  • «Итого начислено» – итоговая сумма начислений по услугам;
  • «Итого оплачено» – итоговая сумма оплаты;
  • «Сальдо на конец месяца» – общая задолженность на конец указанного периода.

Перед формированием отчета необходимо указать:

  • «Период с … по …» – период формирования отчета (далее - Отчетный период);
  • «Организация» - организация, от имени которой формируется отчет (далее - Отчетная организация);
  • «Объект» – указывается здание, подъезд или лицевой счет, по которому производится формирование сводной ведомости;
  • «Приватизация» – вид приватизации помещения, в котором открыт лицевой счет.
     Доступны следующие значения:
     1. "Все" - отчет будет сформирован по всем помещениям, без отбора приватизации;
     2. "Приватизированные" - отчет будет сформирован только по приватизированным помещениям;
     3. "Не приватизированные" - отчет будет сформирован только по не приватизированным помещениям.
  • «Вариант отбора услуг» - вариант отбора услуг, с учетом которого будет сформирован отчет. 
     Доступны следующие значения:
    1. "формировать отчет по всем услугам" - при выборе данного значения отчет будет сформирован без отбора по услугам,
    2. "формировать отчет по указанным услугам" - при выборе данного значения отчет будет сформирован только по тем услугам, которые выбраны в таблице "Отбор услуг (только для варианта "формировать отчет по указанным услугам")",
    3. "формировать отчет по услугам, отличным от кап. ремонта" - при выборе данного значения отчет будет сформирован только по услугам, отличным от услуг капитального ремонта, установленным для Отчетной организации,
    4. "формировать отчет по услугам, отличным от кап. ремонта" - при выборе данного значения отчет будет сформирован только по услугам капитального ремонта, установленным для Отчетной организации.

Для того, чтобы сформировать отчет, необходимо нажать на кнопку «Сформировать отчет».

Отчет открывается через меню «Начисление услуг / Отчеты / Краткая сводная ведомость».
