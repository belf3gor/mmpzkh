Отчет «Карточка расчетов»

Отчет предназначен для формирования карточки расчетов по выбранному лицевому счету за указанный период, с возможностью отбора расчетов по услугам.

Перед формированием отчета указывается:

  • «Период» - период формирования отчета (далее - Отчетный период);
  • «Организация» - организация, от имени которой формируется отчет (далее - Отчетная организация); 
  • «Лицевой счет» – указывается лицевой счет, по которому производится формирование карточки расчетов;
  • «Вариант отбора услуг» - вариант отбора услуг, с учетом которого будет сформирован отчет. 
     Доступны следующие значения:
    1. "формировать отчет по всем услугам" - при выборе данного значения отчет будет сформирован без отбора по услугам,
    2. "формировать отчет по указанным услугам" - при выборе данного значения отчет будет сформирован только по тем услугам, которые выбраны в таблице "Отбор услуг (только для варианта "формировать отчет по указанным услугам")",
    3. "формировать отчет по услугам, отличным от кап. ремонта" - при выборе данного значения отчет будет сформирован только по услугам, отличным от услуг капитального ремонта, установленным для Отчетной организации,
    4. "формировать отчет по услугам, отличным от кап. ремонта" - при выборе данного значения отчет будет сформирован только по услугам капитального ремонта, установленным для Отчетной организации.
  • «Вариант вывода сведений о задолженности» - вариант вывода сведений о задолженности с учетом которого будет сформирован отчет.
     Доступны следующие значения:
    1. "в основной таблице" - при выборе данного варианта сведения о задолженности выводятся в основной таблице начислений.
         При этом:
           а. В первой строке выводятся сведения о сальдо по выбранному лицевому счету по состоянию на начало выбранного периода (далее - Начальное сальдо).
           б. Во всех строках, в которых содержатся сведениях об операциях по начислениям и оплатам, в колонке"Задолженность" выводится промежуточное сальдо (далее - Операции по начислениям и оплаты).
           в. В последней строке выводятся сведения о сальдо, которое рассчитывается с учетом Начального сальдо и Операций по начислениям и оплаты.
    2. "в дополнительной таблице" - при выборе данного варианта сведения о задолженности выводятся в дополнительной таблице (далее - Таблица задолженности), расположенной под основной таблицей начислений.
         При этом:
           а. В строке "Начальное сальдо" выводятся сведения о сальдо по выбранному лицевому счету по состоянию на начало выбранного периода (в разрезе операций расчету по услуг, пени и рассрочке).
           б. В строке "Начислено за период" выводятся итоговые сведения об операциях по начислениям, произведенных в выбранном периоде на выбранный лицевой счет (в разрезе операций по расчету услуг, пени и рассрочке).
           в. В строке "Оплачено за период" выводятся итоговые сведения об оплатах, поступивших в выбранном периоде от выбранного лицевого счета (в разрезе операций по расчету услуг, пени и рассрочке).
           г. В строке "Итого за период" выводятся итоговые сведения по всем операциям по начислениям и оплатам в выбранном периоде по выбранному лицевому счету (в разрезе операций по расчету услуг, пени и рассрочке).
           д. В строке "Конечное сальдо" выводятся сведения о сальдо по выбранному лицевому счету по состоянию на конец выбранного периода (в разрезе операций расчету по услуг, пени и рассрочке).
  •  «Раздел учета» – раздел учета, по которому произведены начисления в течении Отчетного периода.
      Доступны следующие значения:
      1. "Расчет по услугам" - при выборе данного значения отчет будет сформирован только по ежемесячным расчетам по услугам, 
      2. "Льготы" - при выборе данного значения отчет будет сформирован только по начисленным льготам,      
      3. "Пени" - при выборе данного значения отчет будет сформирован только по начисленным пеням,
      4. "Комиссия банка" - при выборе данного значения отчет будет сформирован только по начисленным комиссиям банка,
      5. "Рассрочка" - при выборе данного значения отчет будет сформирован только по начислениям на ранее предоставленные рассрочки платежей,
      6. "Добровольное страхование" - при выборе данного значения отчет будет сформирован только по начислениям по добровольному страхованию.
  • «Вид начисления» – вид начисления, по которому произведены начисления в течении Отчетного периода.
     Доступны следующие значения:
     1. "Ввод начального сальдо" - при выборе данного значения отчет будет сформирован только по операциям, связанным с вводом начального сальдо (остатки по ежемесячным начислениям, остатки по пени и т.д.),
     2. "Начисление" - при выборе данного значения отчет будет сформирован только по операциям, связанным с начислениями (ежемесячные расчеты по услугам, льготы, пени и т.д.),
     3. "Корректировка" - при выборе данного значения отчет будет сформирован только по операциям, связанным с корректировками (корректировка ежемесячных расчетов по услугам, корректировка пени и т.д.),
     4. "Оплата" - при выборе данного значения отчет будет сформирован только по операциям, связанным с оплатами,
     5. "Закрытие авансов" - при выборе данного значения отчет будет сформирован только по операциям, связанным с закрытием месяца начисления.

Для того, чтобы сформировать отчет, необходимо нажать на кнопку «Сформировать отчет».

Отчет открывается через меню «Начисление услуг / Отчеты / Карточка расчетов».
