Отчет "Сведения по потреблениям приборов учета"

Отчет предоставляет возможность сформировать сведения о потреблении приборов учета, установленных на объекты учета.
По умолчанию возможны два варианта формирования отчета: по видам услуг, по месяцам.

Возможные показатели отчета

  • Здание
  • Объект учета
  • Вид услуги
  • Счетчик
  • Дата показаний, предшествующих дате начала периода
  • Дата текущих показаний
  • Расход показаний
  • Учетное количество (объем показаний, с учетом расхода и коэффициента трансформации счетчика)
  • Месяц расхода показаний
