Отчет «Справка на субсидию»

Отчет предназначен для формирования справки «Справка на субсидию». 

Перед формированием отчета необходимо заполнить следующие поля:
   1. Порядковый номер справки (заполняется автоматически),
   2. Дата формирования справки (по умолчанию устанавливается текущая системная дата ),
   3. Получатель справки (жилец из базы, который является получателем справки),
   4. Период начисления ЖКУ (период, за который в справке необходимо вывести сведения о начислениях ЖКУ),
   5. Организация (от имени которой формируется справка).

В разделе «Дополнительные настройки» доступны следующие настройки:
1. «Льгота на жильца» - признак, который регулирует вывод в отчет сведений о начисленных льготах только по выбранному проживающему или по лицевому счету в целом.

2. «Выводить по периодам» - признак, который позволяет группировать сведения о начислениях в разрезе различных месяцев.

3. «Вид задолженности» - способ получения сведений о задолженности лицевого счета, по которому проживает получатель справка. Доступны 2 значения:

Просроченная задолженность - это задолженность по взаиморасчетам лицевого счета, в котором проживает получатель справки, с учетом настройки "Срок оплаты" (далее - Срок оплаты), установленной в Учетной политике ЖКХ для выбранной организации по состоянию на дату формирования справки (далее - Дата справки). При этом:
1.1. Если Дата справки больше даты Срока оплаты, то взаиморасчеты месяца (далее - Текущий месяц), в котором содержится Дата справки, не учитываются при расчете задолженности.
1.2. Если Дата справки меньше или равна дате Срока оплаты, то взаиморасчеты Текущего и предыдущего месяца не учитываются при расчете задолженности.

Общая задолженность - это задолженность по взаиморасчетам выбранного лицевого счета по состоянию на Дату справки без учета даты Срока оплаты.

Для того чтобы сформировать отчет необходимо нажать на кнопку «Сформировать отчет».

Отчет доступен в разделе «Управление лицевыми счетами / Отчеты / Справка на субсидию».
