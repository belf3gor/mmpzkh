Отчет «Декларация по ЕНВД»

Отчет "Декларация по ЕНВД" предназначен для подготовки декларации по единому налогу на вмененный доход для отдельных видов деятельности по форме, утверждённой приказом ФНС РФ от 4 июля 2014 г. № ММВ-7-3/353@.

Порядок заполнения декларации

Рекомендуется соблюдать такую последовательность подготовки листов:

1. Титульный лист.

2. Раздел 2 "Расчет единого налога на вмененный доход для отдельных видов деятельности".

3. Раздел 3 "Расчет суммы единого налога на вмененный доход за налоговый период".

4. Раздел 1 "Сумма единого налога на вмененный доход для отдельных видов деятельности, подлежащая уплате в бюджет, по данным налогоплательщика".

Особенности заполнения некоторых листов декларации

Титульный лист

Следует проверить все автоматически заполненные сведения об организации и должностных лицах. Автоматически заполненные сведения, при необходимости, можно отредактировать в соответствующих справочниках и регистрах информационной базы. Нажатие кнопки «Обновить» вызовет повторное заполнение титульного и других листов, где отображаются сведения об организации и ответственных лицах.

Необходимо указать код места представления декларации, выбрав его из списка, выданного на экран после нажатия клавиши Enter в этом поле или двойного щелчка мыши.

Раздел 2 "Расчет единого налога на вмененный доход для отдельных видов деятельности"

Раздел 2 заполняется отдельно по каждому виду осуществляемой предпринимательской деятельности и каждому месту осуществления такой деятельности. Для добавления дополнительных экземпляров листа Раздела 2 необходимо воспользоваться механизмом управления дополнительными страницами.

Для управления дополнительными страницами (добавление/удаление) необходимо использовать кнопки, расположенные вверху страницы, либо воспользоваться контекстным меню в панели навигации по разделам отчета, выделив нужную страницу.

Код вида деятельности указывается в строке 010 путем выбора из списка, выданного на экран после нажатия клавиши Enter в этом поле или двойного щелчка мыши.

После выбора кода вида деятельности, значение базовой доходности для выбранного вида деятельности будет подставлено в строку 040 автоматически.

Строка 030 "Код ОКТМО" должна быть обязательно заполнена.

В строке 050 указывается значение коэффициента-дефлятора, ежегодно устанавливаемого Минэкономразвития России. При создании новой декларации, значение коэффициента К1 на отчетный год подставляется автоматически.

В строке 060 указывается коэффициент базовой доходности, устанавливаемый местными представительными органами.

Расчет Налоговой базы в строке 100 производится автоматически в общем случае как произведение  базовой доходности (строка 040), скорректированной на коэффициенты К1 (строка 050) и К2 (строка 060) и исчисленной за налоговый период, и величины физического показателя, характеризующего данный вид деятельности (сумма строк 070, 080 и 090) с учетом фактически отработанных календарных дней в месяце постановки на учет\снятия с учета в качестве налогоплательщика.

Для определения количества календарных дней осуществления деятельности в Помощнике расчета необходимо ввести даты постановки на учет и\или снятия с учета. Эти данные указаны в Уведомлениях о постановке на учет\снятии с учета.

В таком случае налоговая база вмененного дохода за текущий налоговый квартал будет скорректирована в зависимости от фактического количества дней, когда велась "вмененная" деятельность.

«Помощник расчета календарных дней..» на печать не выводится и не выгружается.

Раздел 3 "Расчет суммы единого налога на вмененный доход за налоговый период"

Раздел 3 может быть заполнен частично автоматически, если предварительно был заполнен раздел 2 декларации. Для заполнения раздела необходимо нажать кнопку "Заполнить Раздел 3 по данным Раздела 2".

Значение в строке 040 вычисляется в зависимости от признака налогоплательщика, установленного в строке 005.

Для признака налогоплательщика равного «1» строка 040 равна разнице значений строк 010 и 020, при этом, если значение меньше 50% от суммы исчисленного налога по строке 010, значение по строке 040 автоматически корректируется, т.е. увеличивается до величины 50% суммы исчисленного налога по строке 010.

Для признака налогоплательщика равного «2» строка 040 равна разнице значений строк 010 и 030.

В соответствии с федеральным законом № 237-ФЗ от 23.07.2013 начиная с 2014 года сумма страховых взносов зависит от дохода Индивидуального предпринимателя.

Раздел 1 "Сумма единого налога на вмененный доход для отдельных видов деятельности, подлежащая уплате в бюджет, по данным налогоплательщика"

Раздел 1 может быть заполнен автоматически, если предварительно были заполнены разделы 2 и 3 декларации. Для заполнения раздела необходимо нажать кнопку "Заполнить Раздел 1 по данным Раздела 2 и Раздела 3". Раздел будет заполнен кодами ОКТМО, указанными в разделе 2, и соответствующими каждому коду ОКТМО суммами налога. При необходимости, будет автоматически создано требуемое количество групп строк Раздела 1.

Раздел 1 также можно заполнить вручную. Сумму налога по каждому коду ОКТМО в этом случае также необходимо рассчитать самостоятельно.

Для добавления дополнительных групп строк Раздела 1 необходимо воспользоваться механизмом управления дополнительными страницами.

Для управления дополнительными страницами (добавление/удаление) необходимо использовать кнопки, расположенные вверху страницы, либо воспользоваться контекстным меню в панели навигации по разделам отчета, выделив нужную страницу.
