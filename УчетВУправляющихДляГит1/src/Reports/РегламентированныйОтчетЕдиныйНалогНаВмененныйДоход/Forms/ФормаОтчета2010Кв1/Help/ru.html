Отчет «Декларация по ЕНВД»

Отчет "Декларация по ЕНВД" предназначен для подготовки декларации по единому налогу на вмененный доход для отдельных видов деятельности по форме, утвержденной приказом Министерства финансов Российской Федерации от 08.12.2008 № 137н (в редакции приказа Министерства финансов Российской Федерации от 19.11.2009 № 119н).

Помимо подготовки в печатном виде, отчет имеет возможность выгрузки в электронном виде в формате версии 5.02, утвержденном приказом ФНС России от 12.01.2010 № ММ-7-6/4@.

Порядок заполнения декларации

Рекомендуется соблюдать такую последовательность подготовки листов:

1. Титульный лист.

2. Раздел 2 "Расчет единого налога на вмененный доход для отдельных видов деятельности".

3. Раздел 3 "Расчет суммы единого налога на вмененный доход за налоговый период".

4. Раздел 1 "Сумма единого налога на вмененный доход для отдельных видов деятельности, подлежащая уплате в бюджет, по данным налогоплательщика".

Особенности заполнения некоторых листов декларации

Титульный лист

Следует проверить все автоматически заполненные сведения об организации и должностных лицах. Автоматически заполненные сведения, при необходимости, можно отредактировать в соответствующих справочниках и регистрах информационной базы. Нажатие кнопки  «Обновить» вызовет повторное заполнение титульного и других листов, где отображаются сведения об организации и ответственных лицах.

Необходимо указать код места представления декларации, выбрав его из списка, выданного на экран после нажатия клавиши Enter в этом поле или двойного щелчка мыши.

Раздел 2 "Расчет единого налога на вмененный доход для отдельных видов деятельности"

Раздел 2 заполняется отдельно по каждому виду осуществляемой предпринимательской деятельности и каждому месту осуществления такой деятельности. Для добавления дополнительных экземпляров листа Раздела 2 необходимо воспользоваться механизмом управления дополнительными страницами.

Для управления дополнительными страницами (добавление/удаление) необходимо использовать кнопки, расположенные вверху страницы, либо воспользоваться контекстным меню в панели навигации по разделам отчета, выделив нужную страницу.

Код вида деятельности указывается в строке 010 путем выбора из списка, выданного на экран после нажатия клавиши Enter в этом поле или двойного щелчка мыши.

После выбора кода вида деятельности, значение базовой доходности для выбранного вида деятельности будет подставлено в строку 040 автоматически.

Строка 030 "Код ОКАТО" должна быть обязательно заполнена.

В строке 080 указывается значение коэффициента-дефлятора, ежегодно устанавливаемого Минэкономразвития России. При создании новой декларации, значение коэффициента К1 на отчетный год подставляется автоматически.

В строке 090 указывается коэффициент базовой доходности, устанавливаемый местными представительными органами.

Раздел 3 "Расчет суммы единого налога на вмененный доход за налоговый период"

Значение в строке 050 вычисляется автоматически как сумма значений строк 030 и 040, при этом, в случае, если сумма значение строк 030 и 040 превышает 50% от суммы исчисленного налога по строке 020, значение по строке 050 автоматически уменьшается до величины, не превышающей 50% суммы исчисленного налога по строке 020.

Раздел 1 "Сумма единого налога на вмененный доход для отдельных видов деятельности, подлежащая уплате в бюджет, по данным налогоплательщика"

Раздел 1 может быть заполнен автоматически, если предварительно были заполнены разделы 2 и 3 декларации. Для заполнения раздела необходимо нажать кнопку "Заполнить Раздел 1 по данным Раздела 2 и Раздела 3". Раздел будет заполнен кодами ОКАТО, указанными в разделе 2, и соответствующими каждому коду ОКАТО суммами налога. При необходимости, будет автоматически создано требуемое количество экземпляров листов Раздела 1.

Раздел 1 также можно заполнить вручную. Сумму налога по каждому коду ОКАТО в этом случае также необходимо рассчитать самостоятельно.

Для добавления дополнительных экземпляров листа Раздела 1 необходимо воспользоваться механизмом управления дополнительными страницами.

Для управления дополнительными страницами (добавление/удаление) необходимо использовать кнопки, расположенные вверху страницы, либо воспользоваться контекстным меню в панели навигации по разделам отчета, выделив нужную страницу.
