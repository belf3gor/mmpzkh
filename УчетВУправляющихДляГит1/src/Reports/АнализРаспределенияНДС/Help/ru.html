Отчет по распределению НДС с косвенных расходов по вариантам реализации

Покупка ® Ведение книги покупок ® Отчет о распределении НДС косвенных расходов

Отчет предназначен для отражения распределения НДС по приобретенным ценностям, относящегося к косвенным расходам.

Распределение НДС по ценностям, списанным на расходы, по операциям, облагаемым НДС, не облагаемым НДС и (или) облагаемым по ставке 0%, осуществляется документом Распределение НДС косвенных расходов.

В отчете отражается база распределения косвенных расходов (объем выручки) и результаты распределения сумм НДС по каждому счету-фактуре поставщика.

Для отражения суммы выручки по определенному документу, нужно установить флажок Выводить выручку по данным документа и указать документ.
