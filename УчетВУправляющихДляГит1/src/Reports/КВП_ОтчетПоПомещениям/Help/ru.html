Отчет «Отчет по помещениям»

Отчет предназначен для формирования списка помещений в зданиях. Позволяет выводить общую информацию, информацию о площадях всех помещений здания, о параметрах благоустройства, о собственниках помещений.

На форме отчета можно задать группировку помещений, путём установки следующих флажков:

  • «Группировать по видам помещений» - помещения группируются по видам помещений;
  • «Группировать по подъездам» - помещения группируются по подъездам;
  • «Выводить данные по комнатам» - для каждой квартиры будет сформирован список комнат;
  • «Выводить данные по собственникам» - для каждой квартиры будет сформирован список собственников.


При этом вывод данных по комнатам и собственникам одновременно недоступен.

Для формирования отчета необходимо указать:

  • «На дату» – дата формирования отчета.

В верхней части диалогового окна отчета расположены кнопки:

  • «Сформировать отчет» – формируется отчет;
  • «Печать» – осуществляет печать отчета;
  • «Сохранить как» - сохраняет отчет в табличный документ, либо в формате Excel;
  • «Настройка» – вызываются настройки параметров формирования отчета;
  • «История» – формирует выпадающий список ранее сформированных отчетов;
  • «Загрузить настройки отчета» – открывает диалоговое окно выбора файла ранее сохраненных настроек;
  • «Сохранить настройки отчета» – открывает диалоговое окно выбора файла для сохранения настроек;
  • «Открыть справку» – открывает справку по использованию отчета.

Отчет доступен из раздела «Объекты учета / Отчеты/ Отчет по помещениям».
