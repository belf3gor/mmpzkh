Отчет «Список зарегистрированных и проживающих»

Отчет предназначен для формирования списка зарегистрированных и проживающих.

Отчет имеет четыре варианта вывода информации:

  • Данные о регистрации / проживании - выводятся данные о датах регистрации / снятия с регистрации, прибытия / выбытия жильцов на указанную дату;
  • Зарегистрированные / прибывшие за период - выводятся данные о вновь зарегистрированных / прибывших жильцах за указанный период;
  • Состояние жильцов - в отчет попадают все жильцы и их статусы, содержащиеся в информационной базе.
  • Паспортные данные жильцов - в отчет попадают проживающие или зарегистрированные жильцы с паспортными данными.

В отчет выводятся данные об адресе, лицевом счете, списке проживающих, их статусе регистрации и проживании, родственной связи и прочих личных данных. Данные, отображаемые в отчете, определяются при настройке параметров формирования отчета.

Поля, доступные для отбора:

  • «Организация» - организация, по которой отбираются лицевые счета, выводимые в отчет: если здание обслуживается указанной организацией, то лицевые счета данного здания включаются в отчет. Для вывода лицевых счетов без отбора по организации необходимо очистить поле ввода организации. Сведения по организациям, обслуживающим объекты учета, устанавливаются на вкладке «Обслуживающие организации» формы элемента справочника «Здания и сооружения»;
  • «Здание» - фактический адрес здания;
  • «Лицевой счет» - лицевой счет, по которому числится проживающий;
  • «Проживающий» - ФИО проживающего;
  • «Помещение» - номер квартиры, на которой числится проживающий;
  • «Дата прибытия» - дата заселения;
  • «Дата регистрации» - дата регистрации. 

Отчет вызывается из раздела «Управление лицевыми счетами» «Отчеты» - «Список зарегистрированных и проживающих».
