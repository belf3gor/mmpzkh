Отчет "Продажи по месяцам"


Информация для руководителя


В отчете Продажи по месяцам отражается сумма продаж по основным видам деятельности организации по месяцам за разные годы. Для анализа данных по отдельной номенклатурной группе, номенклатуре или контрагенту можно установить соответствующий отбор.

В сумму продаж включается выручка по отгруженным товарам или предоставленным услугам, независимо от того, поступила оплата от покупателей или нет.

Визуально отчет представлен в виде графика.

Информация для бухгалтера


В сумму продаж включаются только продажи по основным видам деятельности (оборот по кредиту счета 90.01), а прочие доходы (оборот по кредиту счета 91) не включаются.

Показатели отчета могут быть расшифрованы отчетом Продажи в разрезе номенклатурных групп.
