Отчет «Формы ТСЖ»

Отчет предназначен для формирования бланков заявлений о вступлении в ТСЖ, о выходе из ТСЖ, об исключении из членов ТСЖ.

Перед формированием отчета указываются:

  • «Дата» – дата формирования отчета;
  • «Организация» – указывается организация, по которой будет формироваться статистическая информация;
  • «Номер формы» – указывается наименование бланка заявления.  

В верхней части диалогового окна формы расположены кнопки:

  • «Действия» – открывает список выбора действий, которые можно произвести с данным отчетом;
  • «Сформировать» – при нажатии по данной кнопке будет сформирован соответствующий бланк.

Отчет открывается через меню «Учет в ЖКХ / Смета ТСЖ / Вступление/Выход из ТСЖ / Формы ТСЖ».
