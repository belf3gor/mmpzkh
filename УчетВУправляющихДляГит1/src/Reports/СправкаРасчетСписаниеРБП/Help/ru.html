Справка-расчет списания расходов будущих периодов

Отчет предназначен для отражения списания расходов будущих периодов.

Печатная форма отчета является учетным документом, утверждающим списание расходов будущих периодов на расходы в месяце формирования отчета.

Примечание
Отчет нужно формировать только после выполнения регламентной операции закрытия месяца "Закрытие счета 97 "Расходы будущих периодов".

Помощник закрытия месяца позволяет сформировать справки-расчеты, объясняющие расчеты и отражающие результаты выполнения регламентных операций, по кнопке Сформировать справку-расчет. При выборе этой команды из меню выполненной операции формируются те справки-расчеты, которые соответствуют выбранной операции.

Параметры формирования отчета можно задать по кнопке Настройка:

  • выводить данные бухгалтерского учета, налогового учета или бухгалтерского учета с постоянными и временными разницами,
  • отбор по расходам будущих периодов.

В отчет выводятся следующие данные:

  • Счет списания – счет учета текущих расходов, на который списываются расходы будущих периодов.
  • Аналитика списания - аналитика учета текущих расходов, на которую списываются расходы будущих периодов.
  • Дата начала / окончания списания – дата начала и дата окончания списания расходов будущих периодов.
  • Остаточное кол-во списания – количество месяцев (дней), оставшееся с начала текущего месяца до даты окончания списания.
  • Кол-во в текущем периоде – количество месяцев (дней) в текущем месяце. Актуально только для расходов будущих периодов, для которых установлен порядок списания По календарным дням.
  • Остаток на начало – сальдо, числящееся по указанной статье расходов будущих периодов, на начало текущего месяца.
  • Остаток на конец – сальдо, числящееся по указанной статье расходов будущих периодов, на конец текущего месяца.
  • Сумма списания – сумма расходов будущих периодов, списанная на расходы текущего месяца при выполнении регламентной операции.

Вывод в печатную форму заголовка и подписей настраивается флажками Заголовок и Подписи.
