Отчет «Выписка по лицевому счету»

Данный отчет выводит общую информацию по лицевому счету, список собственников и взаиморасчеты по услугам за указанный период.

Перед формированием отчета указывается:

  • «Период» - период формирования отчета (далее - Отчетный период);
  • «Организация» - организация, от имени которой формируется отчет (далее - Отчетная организация); 
  • «Лицевой счет» – указывается лицевой счет, по которому производится формирование выписки по лицевому счету (далее - Отчетный лицевой счет);
  • «Вариант отбора услуг» - вариант отбора услуг, с учетом которого будет сформирован отчет.
     Доступны следующие значения:
    1. "формировать отчет по всем услугам" - при выборе данного значения отчет будет сформирован без отбора по услугам,
    2. "формировать отчет по указанным услугам" - при выборе данного значения отчет будет сформирован только по тем услугам, которые выбраны в таблице "Отбор услуг (доступен только для вариантов "формировать отчет по указанным услугам" и "формировать отчет, кроме указанных услуг")",
    3. "формировать отчет, кроме указанных услуг" - при выборе данного значения отчет будет сформирован только по тем услугам, которые не выбраны в таблице "Отбор услуг (доступен только для вариантов "формировать отчет по указанным услугам" и "формировать отчет, кроме указанных услуг")",
    3. "формировать отчет по услугам, отличным от кап. ремонта" - при выборе данного значения отчет будет сформирован только по услугам, отличным от услуг капитального ремонта, установленным для Отчетной организации,
    4. "формировать отчет по услугам, отличным от кап. ремонта" - при выборе данного значения отчет будет сформирован только по услугам капитального ремонта, установленным для Отчетной организации.
  • «Формировать задолженность по данным» - указывается момент времени, по состоянию на которую будут получены сведения о задолженности по Отчетному лицевому счету. Для выбора доступны 3 варианта: "На начало последнего месяца периода", "На конец последнего месяца периода" и "На дату выдачи".
  • «Дата выдачи» – дата, по состоянию на которую выдается отчет.
  • «Выводить сведения по месяцам» - сведения о начислениях по каждому месяцу начисления, входящему в Отчетный период, будут выводиться в отдельный таблицах. При этом, перед каждой таблицей будет выводится информация о месяце начисления в формате "ММММ ГГГГ" (н-р, "Январь 2019").
  • «Детализировать сведения по пеням» – в таблицу с начислениями будет добавлена детализация сведений по пени.
     

Для того, чтобы сформировать отчет, необходимо нажать на кнопку «Сформировать отчет».

Отчет открывается через меню «Управление лицевыми счетами / Отчеты / Выписка по лицевому счету».
