Отчет «Сведения о количестве объектов водных биологических ресурсов, подлежащих изъятию из среды их обитания в качестве разрешенного прилова, на основании разрешения на добычу (вылов) водных биологических ресурсов и суммах сбора, подлежащих уплате в виде единовременного взноса»

Настоящий отчет предназначен для подготовки сведений о количестве объектов водных биологических ресурсов, подлежащих изъятию из среды их обитания в качестве разрешенного прилова, на основании разрешения на добычу (вылов) водных биологических ресурсов и суммах сбора, подлежащих уплате в виде единовременного взноса, в соответствии с приказом ФНС России от 07.07.2010 № ММВ-7-3/320@ (в ред. приказа ФНС России от 14 ноября 2013 г. № ММВ-7-3/501@).

Особенности заполнения листов отчета

Титульный лист

Автоматически заполненные сведения на титульном листе отчета необходимо проверить и, при необходимости, отредактировать.

Обязательно следует заполнить показатель «по месту нахождения (учета) (код)».

Раздел «Сведения о количестве объектов водных биологических ресурсов, подлежащих изъятию из среды их обитания в качестве разрешенного прилова, на основании разрешения на добычу (вылов) водных биологических ресурсов»

Раздел заполняется плательщиком отдельно в отношении каждого кода бюджетной классификации, а также по каждому коду по ОКТМО муниципального образования, на территории которого происходит уплата сбора.

Раздел заполняется отдельно по каждому разрешению на добычу (вылов), для которого разрешен прилов.

Для ввода сведений о новом коде бюджетной классификации, коду по ОКТМО или  сведении о разрешений необходимо добавить новый экземпляр раздела. Для этого следует воспользоваться управлением дополнительными страницами (добавление/удаление) необходимо использовать кнопки, расположенные вверху страницы, либо воспользоваться контекстным меню в панели навигации по разделам отчета, выделив нужную страницу.

В графе 1 указывается код наименования объекта водных биологических ресурсов, разрешенного к добыче в соответствии с разрешением, сведения о котором представлены в строках 050, 060 и 070 раздела. Показатели в графе заполняются путем выбора из списка, который вызывается нажатием клавиши Enter.

В графе 2 указывается разрешенный объем прилова (в тоннах) по наименованию объектов водных биологических ресурсов, код которых указан в графе 1 соответствующей строки.

В графе 3 указывается исчисленная сумма единовременного взноса (в рублях), подлежащая уплате по коду объекта водных биологических ресурсов, указанному в графе 1 соответствующей строки.

Если строк, отведенных в экранной форме для указания сведений об объеме разрешенного прилова, недостаточно, необходимо добавлять новые строки.

Для добавления новой строки необходимо использовать кнопки "+" или "Добавить строку", расположенную непосредственно под многострочной частью отчета, либо воспользоваться контекстным меню, выделив любую ячейку добавляемой строки.

Для удаления строк необходимо использовать кнопки "х", расположенные слева от строк многострочных частей отчета, либо воспользоваться контекстным меню, выделив любую ячейку удаляемой строки.
