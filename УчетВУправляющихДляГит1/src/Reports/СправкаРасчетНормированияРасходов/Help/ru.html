Справка-расчет нормирования расходов

Отчет предназначен для отражения включения в состав расходов для целей налогообложения прибыли, затрат, порядок признания которых предусматривает ограничения.

Печатная форма отчета является учетным документом, утверждающим порядок включения в состав расходов для целей налогообложения прибыли, затрат, порядок признания которых предусматривает ограничения, в месяце формирования отчета.

Примечание
Отчет нужно формировать только после выполнения регламентных операций закрытия месяца "Расчет долей списания косвенных расходов".

Помощник закрытия месяца позволяет сформировать справки-расчеты, объясняющие расчеты и отражающие результаты выполнения регламентных операций, по кнопке Сформировать справку-расчет. При выборе этой команды из меню выполненной операции формируются те справки-расчеты, которые соответствуют выбранной операции.

Отчет раскрывает порядок нормирования для следующих видов косвенных расходов:

  • расходы на рекламу,
  • представительские расходы,
  • расходы на добровольное страхование жизни работников,
  • расходы на добровольное медицинское страхование,
  • расходы на страхование на случай смерти и утраты трудоспособности,
  • расходы на возмещение затрат работников по уплате процентов.

Отбор по виду косвенных расходов можно задать по кнопке Настройка.

В отчет выводятся следующие данные:

  • Сумма – сумма затрат, признанная на основании данных первичных документов, за период с начала года по месяц формирования отчета.
  • Порядок ограничения – номер статьи НК РФ, база ограничения, ее величина, процент ограничения, предельный размер признаваемых расходов.
  • Расходы, принятые в целях налогообложения в состав расходов, уменьшающих базу налога на прибыль - сумма расходов, принятых ранее нарастающим итогом с начала года, и сумма расходов, подлежащая признанию в месяце формирования отчета.
  • Постоянные разницы в оценке расходов, принятых в состав расходов, уменьшающих базу налога на прибыль, по данным бухгалтерского и налогового учета - разница между величиной расходов, признаваемых для целей налогообложения прибыли, и величиной признанных затрат по данным первичной документации.

Вывод в печатную форму заголовка и подписей настраивается флажками Заголовок и Подписи.

При необходимости получения детализации по статьям затрат рекомендуется формировать Справку-расчет "Списание косвенных расходов".
