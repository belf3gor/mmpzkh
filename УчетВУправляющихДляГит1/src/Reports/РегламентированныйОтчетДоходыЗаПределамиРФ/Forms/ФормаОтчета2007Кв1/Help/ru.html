Отчет «Доходы за пределами РФ»

Отчет «Доходы за пределами РФ» предназначен для подготовки декларации о доходах, полученных российской организацией от источников за пределами Российской Федерации, в соответствии с Приложением №1 к приказу МНС России от 23.12.2003 № БГ-3-23/709.

После вызова формы декларации из списка «Регламентированные отчеты», в диалоге управления отчетом следует, в первую очередь, указать организацию, для которой будет заполняться отчет.

В качестве даты подписи отчета при первом открытии отчета ставится рабочая дата.

Порядок заполнения

При заполнении декларации рекомендуется соблюдать такую последовательность подготовки листов:

1. Титульный лист.

2. Сведения о физическом лице (вторая страница титульного листа).

3. Раздел I. «Доходы, не связанные с деятельностью обособленного подразделения российской организации в иностранном государстве».

4. Раздел II. «Доходы, связанные с деятельностью в иностранном государстве через постоянное представительство».

5. Раздел III. «Расходы постоянного представительства».
6. Раздел IV. «Сведения об учетной политике для целей налогообложения».

Примечание 1. Разделы I - IV могут заполняться в произвольном порядке.

Особенности заполнения некоторых листов декларации

Титульный лист

Следует указать сведения об организации и сведения о должностных лицах, которые не были заполнены автоматически. Автоматически заполненные сведения необходимо проверить и, при необходимости, отредактировать.

Сведения о физическом лице (вторая страница титульного листа)

Заполняется только в отношении должностных лиц и уполномоченного представителя, у которых ИНН отсутствует или не указан на титульном листе.

Раздел I «Доходы, не связанные с деятельностью обособленного подразделения российской организации в иностранном государстве»

Заполняется отдельно по каждой иностранной организации или по каждому иностранному физическому лицу - источнику выплаты дохода.

Для заполнения раздела отдельно по каждому источнику выплаты дохода необходимо воспользоваться механизмом управления дополнительными страницами (добавление/удаление), необходимо использовать кнопки, расположенные вверху страницы, либо воспользоваться контекстным меню в панели навигации по разделам отчета, выделив нужную страницу.

Тот же механизм используется для заполнения подраздела «Сведения о доходах» по каждому виду дохода.

В строке 010 указывается наименование иностранной организации так, как оно указано в договоре (контракте), заключенном между российской организацией и иностранной организацией и являющемся основанием для выплаты дохода.

В строке 020 указывается полное имя иностранного физического лица - источника выплаты дохода, как оно указано в договоре (контракте), заключенном между российской организацией и иностранным физическим лицом и являющемся основанием для выплаты дохода.

Раздел II «Доходы, связанные с деятельностью в иностранном государстве через постоянное представительство».

При заполнении раздела дополнительные листы (варианты заполнения) могут потребоваться, в частности, в следующих случаях:

  • наличия нескольких постоянных представительств;
  • ведения постоянным представительством российской организации нескольких видов деятельности, по которым порядок расчета налоговой базы в Российской Федерации различается;
  • ведения постоянным представительством российской организации нескольких видов деятельности, доход от каждого из которых облагается в иностранном государстве и (или) в Российской Федерации по различным налоговым ставкам.

В строке 010 указывается наименование обособленного подразделения российской организации, под которым оно состоит на налоговом учете в иностранном государстве.

В строке 020 указывается ИНН обособленного подразделения или его аналог, присвоенный в иностранном государстве.

Для заполнения реквизита «Налоговый (отчетный) период, в котором доход включен в декларацию по налогу на прибыль в Российской Федерации (код строки 060)» необходимо дважды щелкнуть мышью в ячейках для отображения даты, и в выданные на экран запросы ввести дату начала периода и дату конца периода.

Раздел III «Расходы постоянного представительства»

Заполняется по каждому постоянному представительству в иностранном государстве.

Для заполнения раздела по нескольким постоянным представительствам необходимо воспользоваться механизмом управления дополнительными страницами (добавление/удаление), необходимо использовать кнопки, расположенные вверху страницы, либо воспользоваться контекстным меню в панели навигации по разделам отчета, выделив нужную страницу.

Расходы каждого постоянного представительства раскрываются по видам расходов.

Если в таблице отчета не хватает строк для раскрытия требуемых видов расходов, то необходимо использовать механизм создания дополнительных строк.

Для добавления новой строки необходимо использовать кнопки "+" или "Добавить строку", расположенную непосредственно под многострочной частью отчета, либо воспользоваться контекстным меню, выделив любую ячейку добавляемой строки.

Для удаления строк необходимо использовать кнопки "х", расположенные слева от строк многострочных частей отчета, либо воспользоваться контекстным меню, выделив любую ячейку удаляемой строки.
