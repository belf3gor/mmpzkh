Отчет «Отчет по лицевым счетам»

Отчет предназначен для формирования списка действующих лицевых счетов. Может содержать следующие поля:
  • «Здание» - наименование здания;
  • «Помещение» – наименование квартиры;
  • «Лицевой счет» – наименование лицевого счета;
  • «Ответственный владелец» - наименование ответственного владельца лицевого счета;
  • «Дата открытия лицевого счета» - дата, когда был открыт лицевой счет;
  • «Дата закрытия лицевого счета» – дата, когда был закрыт лицевой счет;
  • «Доля лицевого счета» – доля лицевого счета от площади квартиры;
  • «Номер помещения» – номер квартиры;
  • «Льгота» – количество зарегистрированных льготников;
  • «Количество зарегистрированных» - количество человек, зарегистрированных по данному лицевому счету;
  • «Количество проживающих» – количество человек, проживающих по данному лицевому счету;
  • «Общая площадь» – общая площадь помещения с учетом доли лицевого счета;
  • «Жилая площадь» –жилая площадь помещения с учетом доли лицевого счета.


Также возможно отображение информации о площадях, установленных для помещения.

Форма отчета содержит флажок «Раскрывать услуги», установка которого позволяет выводить информацию о действующих услугах, содержащую следующие поля:
  • «Услуга» - наименование услуги;
  • «Дата начала действия услуги» - дата назначения услуги;
  • «Дата окончания действия услуги» - дата окончания действия услуги;
  • «Способ расчета» - способ расчета начислений по услуге;
  • «Счетчик» - наименование счетчика услуги;
  • «Норма» - размер нормы потребления услуги.

Для формирования отчета необходимо указать:
  • «На дату» – дата формирования отчета;
  • «Организация» – указывается текущая организация.

В верхней части диалогового окна отчета расположены кнопки:
  • «Сформировать отчет» – формируется отчет;
  • «Печать» – осуществляет печать отчета;
  • «Сохранить как» - сохраняет отчет в табличный документ, либо в формате Excel;
  • «Настройка» – вызываются настройки параметров формирования отчета;
  • «История» – формирует выпадающий список ранее сформированных отчетов;
  • «Загрузить настройки отчета» – открывает диалоговое окно выбора файла ранее сохраненных настроек;
  • «Сохранить настройки отчета» – открывает диалоговое окно выбора файла для сохранения настроек;
  • «Открыть справку» – открывает справку по использованию отчета.

Отчет открывается через меню «Учет в ЖКХ / Лицевые счета / Отчет по лицевым счетам».
