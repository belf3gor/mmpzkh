Отчет «Отчет по пользованию услугами»

Отчет предназначен для формирования отчета по пользованию услугами из списка за указанный период.

В верхней части диалогового окна отчета расположены кнопки:

  • «Действия» – выводит список действий, которые можно произвести с данным отчетом;
  • «Сформировать» – при нажатии будет сформирован отчет.

Отчет открывается через меню «Учет в ЖКХ / Отчеты / Отчет по пользованию услугами».
