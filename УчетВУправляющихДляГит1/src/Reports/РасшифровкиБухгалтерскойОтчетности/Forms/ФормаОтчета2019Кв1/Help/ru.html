Отчет "Бухгалтерская отчетность для Сбербанка"

Отчет "Бухгалтерская отчетность для Сбербанка" предназначен для подготовки форм как годовой, так и промежуточной бухгалтерской отчетности организаций для представления непосредственно в отделение Сбербанка.

Состав отчетности:  

  • Титульный лист
  • Бухгалтерский баланс (квартал, год);
  • Отчет о финансовых результатах (квартал, год);
  • Отчет об изменениях капитала (год);
  • Отчет о движении денежных средств (год);
  • Отчет о целевом использовании средств (год);
  • Дополнительные показатели (квартал, год);
  • Расшифровки обеспечений (квартал, год):
    • полученные;
    • выданные;
  • Задолженность по кредитам и займам (квартал, год):
    • краткосрочная;
    • долгосрочная;
  • Расходы по лизингу (квартал, год);
  • Денежные средства (месяц, квартал, год):
    • касса;
    • расчетные счета;
    • валютные счета;
    • специальные счета в банках;
  • ОСВ (Оборотно-Сальдовая ведомость) (месяц, квартал, год);
  • ОСВ по счету (Оборотно-Сальдовая ведомость по счету) (при необходимости);
  • Анализ счета (при необходимости).

 

В Настройках (кнопка «Отчеты») при необходимости можно изменить отображение отчетности:

- на вкладке Отчеты – состав;

- на вкладке Общие – единицу измерения и количество знаков после запятой.

Для перехода к нужному отчету необходимо выбрать соответствующую закладку в форме отчета.

Для каждого раздела отчетности есть возможность отдельного заполнения (кнопка «Заполнить – Все отчеты/Текущий отчет»).

При нажатии на кнопку «Проверка» проверяется правильность заполнения справочной информации, а также отдельных показателей отчетности.

В автозаполняемых разделах отчетности данные ячеек можно расшифровать (кнопка «Расшифровать»).

Представление отчетности в электронном виде производится при нажатии на кнопку «Отправить».

Возможность печати Бухгалтерской отчетности для Сбербанка отсутствует. 

Особенности заполнения.

Бухгалтерский баланс, Отчет о финансовых результатах, Отчет об изменениях капитала, Отчет о движении денежных средств, Отчет о целевом использовании средств

Разделы формируются в соответствии с образцами форм, утвержденными приказом Минфина России от 02.07.2010 г. № 66н (в редакции приказа Минфина России от 06.04.2015 N 57н).

Дополнительные показатели

В разделе указываются сведения за последние три года по:

  • накопленной амортизации;
  • незавершенному строительству и незаконченным операциям по приобретению, модернизации и и т.п. основных средств;
  • долгосрочной дебиторской задолженности;
  • краткосрочной  дебиторской задолженности;
  • просроченной дебиторской задолженности;
  • просроченной кредиторской задолженности;
  • задолженности учредителей по взносам в уставный (складочный) капитал;
  • финансовому лизингу.

Раздел заполняется автоматически по данным бухгалтерского учета. Исключение: данные по финансовому лизингу указываются вручную.

Расшифровки обеспечений.

Подраздел «Расшифровки обеспечений полученные» заполняется по данным стандартного отчета «Оборотно-сальдовая ведомость по счету» по счету 008. В графе «Срок окончания» выводится срок действия договора из Справочника «Договоры».

Подраздел «Расшифровки обеспечений выданные» заполняется по данным стандартного отчета «Оборотно-сальдовая ведомость по счету» по счету 009. В графе «Срок окончания» выводится срок действия договора из Справочника «Договоры».

Задолженность по кредитам и займам.

Раздел заполняется на основании данных «Оборотно-сальдовой ведомости по счету» по счетам 66 и 67.

Раздел заполняется автоматически при нажатии кнопки «Заполнить». После заполнения раздела необходимо вручную указать сведения о кредитном продукте, по которому имеется задолженность, и ввести график платежей.

Внимание! При повторном нажатии кнопки «Заполнить» сведения, введенные вручную, будут очищены.

Расходы по финансовому лизингу.

Раздел заполняется вручную отдельно по каждой компании.

Денежные средства

Раздел заполняется автоматически при нажатии на кнопку «Заполнить» на основании стандартных отчетов «Анализ счета» по счетам 50, 51, 52 и 55 и вручную не корректируются.

Для внесения изменений следует использовать документы расчетов в конфигурации или ввести ручные операции в информационную базу.

Оборотно-сальдовая ведомость.

Раздел идентичен стандартному отчету конфигурации «Оборотно-сальдовая ведомость».

Сведения по соответствующим счетам показываются развернуто.

Сведения о подписанте - уполномоченном лице извлекаются из справочника "Регистрации в налоговом органе". Сведения об уполномоченном лице включаются в файл отправляемого электронного представления.
