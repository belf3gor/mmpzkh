Отчет «Декларация об объеме собранного винограда для производства винодельческой продукции»

Декларация об объеме собранного винограда для производства винодельческой продукции заполняется сельскохозяйственными товаропроизводителями, организациями, не признаваемыми сельскохозяйственными товаропроизводителями, гражданами, ведущими личное подсобное хозяйство, осуществляющими выращивание винограда для производства винодельческой продукции.

Особенности заполнения:

 

Титульный лист.


Необходимо проверить сведения об организации и сведения о должностных лицах, которые были заполнены автоматически, и при необходимости отредактировать.

Для удобства редактирования данных организации на форме в разделе "Титульный лист" все используемые реквизиты отображены в виде гиперссылок, по нажатию на которые Вам открывается форма редактирования организации.


Раздел Декларация.


В разделе Декларация необходимо ввести сведения об объеме собранного винограда и количествах отгруженного винограда получателям.

Установочные данные по обособленному подразделению вносятся в форме заполнения обособленного подразделения, вызываемой по клику на гиперссылке поля  Обособленное подразделение (с указанием Наименования, КПП и адреса) или Сведения по месту нахождения виноградника (с указанием Наименования и адреса), если декларант индивидуальный предприниматель.

Если организация не имеет обособленных подразделений, то необходимо заполнить сведения по организации.

В случае, если обособленные подразделения есть, необходимо заполнить страницу по организации (головному офису), где указать данные только по головному офису (т.е. не в целом общие данные по всей организации, а только данные по головному офису), а также  добавить дополнительные страницы, по одной для каждого обособленного подразделения (месту нахождения виноградника,  если декларант индивидуальный предприниматель).

Для удобства заполнения реквизитов ОП по головному офису организации, на форме имеется переключатель (радиобатон) с положениями

  • По месту нахождения ОП (по умолчанию)
  • По месту нахождения головной организации, 

    или, если декларант индивидуальный предприниматель,
     
  • По месту нахождения виноградника (по умолчанию)
  • По месту нахождения ИП. 

    При выборе переключателя в  положение "По месту нахождения головной организации" ("По месту нахождения ИП") данные обособленного подразделения автоматически заполняются данными организации-декларанта.

    На странице раздела с выбранным положением переключателя "По месту нахождения головной организации" ("По месту нахождения ИП") становится недоступной гиперссылка поля Обособленное подразделение или Сведения по месту нахождения виноградника, поскольку данные ОП в этом случае всегда совпадают с данными организации-декларанта.
    При изменении данных организации-декларанта и обновлении данных Титульного листа, данные полей Обособленное подразделение или Сведения по месту нахождения виноградника с установленным переключателем "По месту нахождения головной организации" ("По месту нахождения ИП")  также обновятся. 

Для управления дополнительными страницами (добавление/удаление) необходимо использовать кнопки, расположенные вверху страницы, либо воспользоваться контекстным меню в панели навигации по разделам отчета, выделив нужную страницу.

Таблица ввода в разделе Декларация предназначена для ввода сведений о получателях винограда.


В начало

Особенности работы с таблицами ввода данных.

 

Добавление новой строки, Копирование и Удаление активной строки таблицы ввода осуществляется кнопками над таблицей ввода, или пунктами контекстного меню таблицы ввода.

Удаление (а также Отмена удаления) текущей активной строки осуществляется путем:

 

  • выбора Удалить в меню таблицы;
  • по нажатию на клавишу DEL;
  • двойному клику в крайней левой колонке таблицы (со значком  в заголовке).

    При этом в крайней левой колонке строки отобразится значок  , а сама строка окрашивается в цвет             (окраска видна, когда строка не является текущей).

    Таким образом, строка физически не удаляется, а лишь помечается как неактивная, "помечается на удаление".
    Неактивные строки не выводятся на печать и не выгружаются в файл выгрузки, данные по неактивным строкам не участвуют в Итогах таблицы.

    Неактивные строки можно сделать активными, для чего необходимо неактивную строку "удалить" еще раз.

При Копировании существующей строки таблицы или Добавлении новой открывается отдельная форма ввода данных.

Редактирование информации существующей строки таблицы осуществляется в отдельной форме ввода, открываемой при нажатии клавиши Enter или по двойному клику мышкой на любом поле строки, кроме крайней левой колонки.

Меню Сервис отчета

Восстановление удаленных страниц отчета.


В процессе работы с отчетом, из него могли многократно удалять некоторые страницы разделов.
Данные этих ранее удаленных страниц при этом физически из регистров сведений не удаляются, поскольку это требует времени.
Эта особенность позволяет восстанавливать удаленные страницы многострочных разделов.
Для этого в меню Сервис отчета добавлена функция Восстановить удаленные страницы.

Удаленные страницы раздела "Сведения о лицензиях" не восстанавливаются, поскольку эта информация не выгружается и нужна только для печати.
Кроме того, в случае если деятельность декларанта не требует лицензирования, страниц в разделе "Сведения о лицензиях" не может быть больше одной, и значит, восстановленные страницы все равно в этом случае нельзя будет показать в отчете.

Кроме того, следует иметь в виду, что можно восстановить лишь те страницы отчета, которые были удалены после последнего применения процедуры Очистить регистры от ненужной информации из меню Сервис.

Восстанавливаются все доступные для восстановления страницы отчета, они добавляются в конец списка страниц соответствующего раздела отчета.

Реквизиты обособленных подразделений на восстановленных страницах будут не заполнены. Восстанавливаются только данные таблиц.

После восстановления удаленных страниц, отчет автоматически сохранится. Ненужные страницы из числа восстановленных можно будет повторно удалить.

Рекомендуется перед восстановлением сделать копию отчета, т.е. скопировать отчет в форме Регламентированные отчеты (1С-Отчетность), выбрав Скопировать в контекстном меню списка отчетов или нажав F9, или сохранить отчет с новым номером корректировки, при сохранении на вопрос "Создать новый отчет?" необходимо выбрать "Да, создать...".
В скопированном отчете удаленных страниц не будет.
После восстановления удаленных страниц в исходном отчете, может быть полезно иметь под рукой исходный вариант без восстановленных страниц.

В начало

Очистка регистров сведений отчета от ненужной информации.


В процессе работы с отчетом, из него могли многократно удалять некоторые страницы разделов.
Данные этих ранее удаленных страниц при этом физически из регистров сведений не удаляются, поскольку это требует времени.
Кроме того, помеченные на удаление, неактивные, строки остаются в отчете.

Поскольку иногда размер информационной базы достигает определенного критического предела, после которого необходимо удалять ненужную информацию, в меню Сервис отчета добавлена функция Очистить регистры от ненужной информации.

Очистка не влияет на работоспособность отчета, она необходима лишь в целях уменьшения размера информационной базы, особенно если база файловая.

Для файловой ИБ возможна выдача ошибок о превышении максимального размера внутреннего файла 1Cv8.1CD в Журнале регистрации при операциях Заполнения, Копирования отчета или при интерактивном добавлении строк табличной части.
Если это произошло - необходимо полностью удалить ненужные отчеты (не только пометить на удаление), а в тех отчетах, которые необходимы, выполнить данную процедуру очистки.

При выборе в меню отчета Сервис функции Очистить регистры от ненужной информации, открывается окно диалога с выбором варианта очистки Частичная или Полная. 

Частичная очистка - очистка регистров от данных удаленных страниц.
Может выполняться длительное время, если удаленные страницы содержали большое количество строк.

Полная очистка - очистка регистров от данных удаленных страниц и дополнительно удаляются строки, помеченные на удаление в активных (не удаленных) страницах.
Выполняется дольше частичной очистки. 

Если помеченных на удаление строк немного, рекомендуется частичная очистка.

Если Отчет изменен, перед очисткой изменения будут сохранены.
