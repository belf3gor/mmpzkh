Отчет «Отчет по ежемесячным оплатам»

Отчет предназначен для формирования списка внесенных за указанный период оплат и начислений по лицевым счетам с разделением по месяцам.

Для формирования квитанции необходимо указать:

  • «Период с … по …» – период формирования отчета.

В верхней части диалогового окна отчета расположены кнопки:

  • «Действия» – выводит список действий, которые можно совершить с данным отчетом;
  • «Сформировать» – формируется отчет.
  • «Настройка» – вызывается форму настройки параметров формирования отчета.
