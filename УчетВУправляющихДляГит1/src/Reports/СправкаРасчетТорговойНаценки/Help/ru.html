Справка-расчет реализованной торговой наценки

Отчет предназначен для пояснения хода расчета реализованной торговой наценки документом "Регламентная операция".

Печатная форма отчета является документом, поясняющим ход расчета реализованной торговой наценки в месяце формирования отчета.

Примечание
Отчет нужно формировать только после выполнения регламентной операции закрытия месяца "Расчет торговой наценки по проданным товарам".

Помощник закрытия месяца позволяет сформировать справки-расчеты, объясняющие расчеты и отражающие результаты выполнения регламентных операций, по кнопке Сформировать справку-расчет. При выборе этой команды из меню выполненной операции формируются те справки-расчеты, которые соответствуют выбранной операции.

Параметры формирования отчета можно задать по кнопке Настройка:

  • выводить данные бухгалтерского учета, налогового учета или бухгалтерского учета с постоянными и временными разницами,
  • отбирать данные по конкретным складам и счетам учета (для вывода отдельно отчета только по автоматизированной или неавтоматизированной торговой точке).

В отчет выводятся следующие данные:

  • По неавтоматизированной торговой точке:
    • Склад – склад по которому производится расчет реализованной торговой наценки.
    • Ставка НДС – ставка НДС по которой производится расчет реализованной торговой наценки.
    • Остаток товаров по продажной стоимости на конец периода – суммовой остаток  по счету учета остатков на неавтоматизированной торговой точке.
    • Сумма реализации в продажных ценах — сумма реализации в продажных ценах по неавтоматизированной торговой точке.
    • Остаток торговой наценки на конец периода – остаток торговой наценки на конец периода на счете учета торговой наценки по неавтоматизированной торговой точке.
    • Процент скидки (накидки) на остаток непроданных товаров – средний процент скидки накидки который рассчитывается как отношение остатка торговой наценки к сумме реализации и остатка на конец периода в продажных ценах.
    • Торговая наценка приходящаяся на реализованный товар – сумма списания торговой наценки на реализованный товар рассчитываемая как произведение процента скидки (накидки) на сумму реализации.
  • По автоматизированной торговой точке
    • Склад– склад по которому производится расчет реализованной торговой наценки.
    • Номенклатура – наименование номенклатуры, по которой производится расчет реализованной торговой наценки.
    • Партия – представление партии, по которой производится расчет реализованной торговой наценки.
    • Остаток товаров по продажной стоимости на конец периода — суммовой остаток по счету учета остатков на автоматизированной торговой точке.
    • Сумма реализации в продажных ценах – сумма реализации в продажных ценах по автоматизированной торговой точке.
    • Остаток торговой наценки на конец периода – остаток торговой наценки на конец периода на счете учета торговой наценки по неавтоматизированной торговой точке.
    • Процент скидки (накидки) на остаток непроданных товаров – средний процент скидки накидки который рассчитывается как отношение остатка торговой наценки к сумме реализации и остатка на конец периода в продажных ценах.
    • Торговая наценка приходящаяся на реализованный товар – сумма списания торговой наценки на реализованный товар рассчитываемая как произведение процента скидки (накидки) на сумму реализации.
