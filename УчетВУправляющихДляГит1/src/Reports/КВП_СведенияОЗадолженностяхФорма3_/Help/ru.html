Отчет «Отчет о задолженностях, Форма 3»

Отчет предназначен для формирования карточки расчетов по выбранному лицевому счету за указанный период, с возможностью отбора расчетов по услуге и формирования по жилым/нежилым/всем типам помещений.

Перед формированием отчета указывается:

  • «Период» формирования отчета;
  • «Организация»– из справочника «Организации» выбирается организация, предоставляющая услуги;
  • «Исполнитель» – из справочника «Пользователи» выбирается исполнитель; указанный пользователь будет выведен в подвале отчета.
  • «Формировать на» - устанавливается отбор помещений, по которым будет формироваться задолженность. Настройка представляет собой список выбора, состоящий из следующих значений:
      • «Все помещения». При выборе данного значения в отчет попадут задолженности по всем помещениям без отбора.
      • «Жилые помещения». При выборе данного значения в отчет попадут задолженности по всем жилым помещениям. Под жилыми помещениями понимаются все помещения, для которых указан тип помещения «Квартира».
      • «Нежилые помещения». При выборе данного значения в отчет попадут задолженности по всем нежилым помещениям.

 

Вверху формы расположены кнопки:

  • «Действия» – при щелчке вызывается список, где предлагается выбрать одно из стандартных действий;
  • «Сформировать» – при нажатии будет сформирован отчет.

 

Отчет вызывается из «Учет в ЖКХ / Отчеты / Сведения о задолженностях, Форма 3».
