Отчет "Расшифровка показателей НДС"

Отчет предназначен для представления детальных сведений по показателям отчетности по НДС.

Данные можно вывести с группировкой по виду ценностей, ставке НДС, документам отгрузки, оплаты и другим показателям, отражаемым в отчетности по НДС. По этим же полям можно установить отборы.

Отчет формируется автоматически из регламентированной отчетности по НДС и отчета "Анализ налогового учета по НДС" с установленными настройками для расшифровки интересующего показателя.
