Остатки алкогольной продукции

Отчет предназначен для анализа информации об остатках в ЕГАИС, зафиксированных на момент запроса отчета. При формировании отчета по регистру №1 выводятся данные в разрезе справок 2. При формировании отчета по регистру №2 выводятся данные в разрезе алкогольной продукции.

При двойном нажатии на ячейку с кодом алкогольной продукции будет доступна форма исходящего запроса в ЕГАИС на получение данных классификатора по выбранному коду. Аналогичным образом можно запросить данные справки 2, щелкнув по ячейке с ее номером.
