Отчет «Сводный отчет по объекту»

Отчет предназначен для вывода сводной информации по элементам или группам справочников лицевых счетов, подъездов, зданий с разбивкой по месяцам о начисленных суммах, суммах оплат и сальдо на начало и конец месяца.

Перед формированием отчета необходимо указать:

  • «Период с … по …» – период формирования отчета;
  • «Объект» – объект, по которому собираются данные. Если указать группу справочника, то информация будет собрана по всем элементам из данной группы;
  • «Организация» – организация, предоставляющая услуги;
  • «Вариант отбора услуг» - вариант отбора услуг, с учетом которого будет сформирован отчет. 
     Доступны следующие значения:
    1. "формировать отчет по всем услугам" - при выборе данного значения отчет будет сформирован без отбора по услугам,
    2. "формировать отчет по указанным услугам" - при выборе данного значения отчет будет сформирован только по тем услугам, которые выбраны в таблице "Отбор услуг (только для варианта "формировать отчет по указанным услугам")",
    3. "формировать отчет по услугам, отличным от кап. ремонта" - при выборе данного значения отчет будет сформирован только по услугам, отличным от услуг капитального ремонта, установленным для Отчетной организации,
    4. "формировать отчет по услугам кап. ремонта" - при выборе данного значения отчет будет сформирован только по услугам капитального ремонта, установленным для Отчетной организации. 

Вверху формы расположены кнопки:

  • «Действия» – выводит список действий, которые можно произвести с данным отчетом;
  • «Сформировать» – сформировать отчет.

Отчет вызывается из «Начисления услуг / Отчеты / Сводный отчет по объекту».
