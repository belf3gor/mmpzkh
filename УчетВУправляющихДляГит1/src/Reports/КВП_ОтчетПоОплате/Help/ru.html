Отчет «Отчет по оплате»

Отчет предназначен для формирования списка внесенных за указанный период оплат по лицевым счетам с разделением по услугам.

Для формирования квитанции необходимо указать:

  • «Период с … по …» – период формирования отчета.

В верхней части диалогового окна отчета расположены кнопки:

  • «Действия» – выводит список действий, которые можно совершить с данным отчетом;
  • «Сформировать» – формируется отчет.
  • «Настройка» – вызывается форму настройки параметров формирования отчета.

 

Необходимо обратить внимание на корректность работы с отчетом в случае наличия в базе нескольких услуг с одинаковым видом услуг.

Отчет может быть сформирован с группировкой данных по оплатам как по услугам, так и по видам услуг (соответствующая группировка устанавливается в настройках структуры отчета).

При этом если отчет формируется с группировкой по видам услуг, то отбор корректнее выполнять по полю "Вид услуги".

Соответственно, если отчет формируется с группировкой по услугам, то отбор корректнее выполнять по полю "Услуга".

В противном случае при группировке данных в отчете по виду услуги и выполненном отборе по услуге часть данных - по другим услугам с тем же видом - в отчет не вводится.

 

Отчет открывается через меню «Учет в ЖКХ / Отчеты / Отчет по оплате».
