Отчет «Расчет по коллективным приборам учета»

Отчет предназначен для формирования данных по распределению коллективных показаний приборов учета по лицевым счетам. 

В отчете выводятся две таблицы на каждый объект (здание, подъезд). Верхняя таблица выводит сведения о показаниях приборов учета: общедомовым (счетчики, установленные на здания или подъезды), индивидуальным (счетчики, установленные на лицевые счета) и начисления по норме (в случае отсутствия показания счетчика, но установленной нормы на услугу лицевого счета). Расчет коэффициента производиться в зависимости от настроек услуги:

  • Если в настройках распределения услуги указано значение пропорционально "площади", "количеству помещений в квартире" или "количеству жильцов", то берется суммарный расчет данных по лицевому счету, у которых есть общедомовое распределение.
  • Если в настройках распределения услуги указано распределять пропорционально "количеству расхода по услуге", то брать суммарный расход по индивидуальным приборам учета и суммарный расход по нормативу (расход по нормативу будет браться, если установлен флаг "Включать начисления по нормативу в суммарный индивидуальный расход").
  • Если в настройках распределения услуги указано распределять "поровну", то будет браться суммарное количество лицевых счетов, у которых есть распределение по общедомовым приборам учета.

В нижней таблице выводятся данные расчета по общедомовым приборам учета на каждый лицевой счет. В таблице рассчитывается коэффициент распределения на лицевой счет. Коэффициент рассчитывается делением начисленного объема на лицевой счет по общедомовым приборам учета и полученным данным о потерях в таблице сведений о показаниях приборов учета.

В верхней части диалогового окна отчета расположены кнопки:

  • «Сформировать отчет» – при нажатии на кнопку будет сформирован отчет;
  • «Печать» – выводит на печать сформированный отчет.

Для корректного вывода данных в отчете необходимо произвести предварительную настройку:

  • «Период» – указывается месяц, за который было произведено ввод показаний и начисления по приборам учета;
  • «Организация» – организация, оказывающая услуги;
  • «Услуга» – услуга, по которым необходимо сформировать отчет;
  • «Объект» – указывается здание, по которому необходимо вывести данные в отчет. Параметр является не обязательным.  

Отчет открывается через меню «Приборы учета / Отчеты / Расчет по коллективным ПУ».
