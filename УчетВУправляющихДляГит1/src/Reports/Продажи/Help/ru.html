Отчет "Продажи"


Отражаются данные о продажах по основным видам деятельности организации в разрезе номенклатурных групп, номенклатуры или контрагентов. На основании данных программы можно проанализировать изменение продаж за период. Для анализа данных по отдельной номенклатурной группе, номенклатуре или контрагенту можно установить соответствующий отбор. Формируется визуальное представление данных в виде гистограммы.

Продажи по номенклатуре, Продажи по контрагентам, Продажи по номенклатурным группам 
  • В сумму продаж включается сумма отгруженных товаров или предоставленных услуг, независимо от того, поступила оплата от покупателей или нет
  • Возможно получение данных о продажах в количественном выражении (в единицах измерения номенклатуры). Для этого, на закладке "Показатели" в настройках отчета следует установить флаг "Количество"
  • Если выбраны показатели "Сумма" и "Количество" или только "Сумма", гистограмма будет сформирована по сумме. В случае, если выбран только показатель "Количество", гистограмма будет сформирована по количеству
  • Информация для бухгалтера
    • В сумму продаж включаются только продажи по основным видам деятельности (оборот по кредиту счета 90.01), а прочие доходы (оборот по кредиту счета 91) не включаются
    • Сумма вознаграждения комиссионера по продаже попадает в отчет по организации - комитенту
    • Сумма выручки комитента по продаже не попадает в отчет по организации - комиссионеру

Продажи по контрагентам (по оплате)
  • В сумму продаж включаются суммы оплат, поступивших от покупателей на расчетные счета, в кассы организации, а так же оплаты платежными картами
  • Данные суммы продаж выводятся в разрезе контрагентов
  • Сумма продаж не зависит от того, производилась ли отгрузка товаров или предоставлены ли услуги в счет поступивших оплат
  • Информация для бухгалтера
    • Сумма оплаты за комиссионный товар попадает в отчет по организации - комиссионеру
    • При оплате платежной картой, моментом оплаты считается момент возникновения долга эквайера
    • Сумма вознаграждения комиссионера по продаже не попадает в отчет по организации - комитенту
    • Данные о взаимозачетах в отчет не попадают
    • Сумма вознаграждения, полученного от комитента за услуги по закупке не выводится в отчет по организации - комиссионеру
