Обработка «Печать адресных листков»

Данная обработка позволяет печатать формы регистрационного учета №2 «Адресный листок прибытия» и №7 «Адресный листок убытия» для произвольного списка проживающих.

В группе «Отбор для списка проживающих» можно указать правила отбора проживающих по следующим признакам:

  • «Проживающий» - ссылка на проживающего;
  • «Лицевой счет» - лицевой счет, к которому относится проживающий;
  • «Пол» - пол проживающего;
  • «Дата рождения» - дата рождения проживающего;
  • «Дата регистрации права собственности» - дата регистрации права собственности проживающего.

При нажатии кнопки «Отобрать» заполняется список проживающих согласно установленному отбору.

При выполнении обработки для выбранных проживающих формируются адресные листки согласно указанному виду адреса. Если нажата кнопка «Печатать», то сформированные листки сразу будут посланы на принтер, установленный по умолчанию.
