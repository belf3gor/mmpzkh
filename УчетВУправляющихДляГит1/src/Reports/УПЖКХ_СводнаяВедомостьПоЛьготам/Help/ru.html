Форма отчета

Вверху отчета в центре расположена панель вариантов отчета. На ней отображаются кнопки выбора варианта отчета, доступные текущему пользователю. Текущая (выбранная) кнопка варианта отчета имеет контекстное меню, совпадающее с действиями кнопки «Изменить вариант». При наведении мыши на кнопку варианта появляется подсказка – описание варианта отчета.

Справа вверху на форме располагаются кнопки: «Список вариантов» - используется для выбора варианта отчета из списка, для изменения Наименования, Описания и состава пользователей варианта отчета, «Сохранить» - содержащая два пункта – «Сохранить» - сохраняет текущий вариант отчета, если пользователь имеет право изменять с сохранением или вызывается действие «Сохранить как…» в противном случае, «Сохранить как..» - пользователю предлагается ввести название нового варианта отчета или сохранить текущий вариант под другим вариантом. При этом пользователю доступно для сохранения как другой вариант только те варианты, которые он имеет право изменять с сохранением. При создании нового варианта в списке пользователей отчета создается текущий пользователь с признаком права изменения.

Рядом с кнопкой «Сформировать» располагаются три кнопки: «Печать сразу» - вывод сразу на принтер, «Действия с результатом», «Выводить заголовок отчета» - отображать ли заголовок отчета. «Действия с результатом» содержит следующие пункты: «Вывести сформированный отчет в табличный документ», «Сохранить сформированный отчет для сравнения» и «Сравнить сформированный отчет с сохраненным». Сохранение текущего сформированного отчета происходит для каждой пары Пользователь – Вариант отчета.

Справа в отчете располагается панель пользователя. Все настройки панели пользователя сохраняются автоматически для каждого пользователя.

Отчет открывается с помощью пункта меню «Учет в ЖКХ / Подомовые затраты / Анализ подомовых затрат».
