Отчет «Декларация по налогу, уплачиваемому в связи с применением упрощенной системы налогообложения»

Данная форма открывается при создании нового регламентированного отчета.

Стартовая форма предназначена для:

  •  выбора организации;
  •  выбора периода формирования регламентированного отчета;
  •  выбора редакции формы регламентированного отчета либо автоматического определения редакции формы, соответствующей выбранному отчетному периоду.

«Создать» - создает новый регламентированный отчет. Если для выбранного отчетного периода нет доступных редакций форм регламентированного отчета, кнопка становится недоступной.

«Организация» - организация, от имени которой создается новый регламентированный отчет.

«Стрелка вправо» / «Стрелка влево» - кнопки предназначены для выбора отчетного периода. Нажатие кнопок изменяет период, за который может составляться данный регламентированный отчет.

Кнопка выбора из списка («Стрелка вниз») - кнопка используется для выбора отчетного периода с нужной периодичностью в случае, если регламентированный отчет может представляться с различной периодичностью - ежемесячно или ежеквартально.

«Редакция формы» - в случае, если в выбранном отчетном периоде действует несколько редакций формы регламентированного отчета, пользователю предоставляется возможность выбрать необходимую.

«Изменения законодательства» - ссылка открывает Веб-страницу мониторинга отражения изменений в законодательстве, связанных с формами регламентированной отчетности. Ссылка в стартовой форме отображается для отчетных периодов, начиная с 2013 года.
