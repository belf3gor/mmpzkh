Отчет «Формы регистрационного учета»

Данная форма предназначена для формирования форм документов по регистрационному учету граждан.

Имеются следующие формы паспортного стола:

  • Ф1 - Заявление о регистрации по месту пребывания.
  • Ф1П - Заявление о выдаче или замене паспорта.
  • Ф2 - Адресный листок прибытия.
  • Ф3 - Свидетельство о регистрации по месту пребывания.
  • Ф5 - Анкета.
  • Ф6 - Заявление о регистрации / о снятии с регистрационного учета по месту жительства.
  • Ф6р - Заявление о регистрации по месту жительства.
  • Ф6с - Заявление о снятии с регистрационного учета по месту жительства.
  • Ф7 - Адресный листок убытия.
  • Ф8 - Свидетельство о регистрации по месту жительства.
  • Ф9 - Карточка регистрации граждан.
  • Ф10 - Поквартирная карточка.
  • Ф11 - Домовая (поквартирная) книга.
  • Ф12П - Листок статистического учета прибытия (к документам о регистрации по новому месту жительства).
  • Ф12В - Листок статистического учета выбытия (к документам о снятии с регистрационного учета).
  • Ф14 - Уведомление в орган регистрационного учета.
  • Ф15 - Уведомление собственнику (нанимателю) жилого помещения.
  • Ф17 - Выписка из домовой книги.
  • Ф7_СПБ - Характеристика жилого помещения (СПБ).
  • Ф3_МСК - Сведения о зарегистрированных по месту жительства в связи с рождением (МСК).
  • З_С - Заявление собственников для регистрации.
  • РФЛ1 - Сведения о регистрации гражданина РФ по месту жительства.
  • ППВУ - Список подлежащих первоначальной постановке на воинский учет.

Имеются следующие формы справок паспортного стола:
  • Ф16 - Справка о составе семьи.
  • Ф9_СПБ - Справка о регистрации (СПБ).
  • С_С - Справка в суд.
  • С_Р - Справка о регистрации по месту жительства.
  • С_ЗП - Справка об отсутствии паспорта в связи с заменой.
  • С_ПР - Справка об отсутствии паспорта (на регистрации).

Перед формированием отчета указываются:

  • «Дата отчета» – дата формирования отчета;
  • «Организация» – указывается текущая организация;
  • «Лицевой счет» – указывается лицевой счет;
  • «Проживающий» – указывается проживающий лицевого счета, для которого будет формироваться выбранная форма;
  • «Номер формы» – указывается номер формы паспортного стола;

В верхней части диалогового окна формы расположены кнопки:

  • «Действия» – открывает список выбора действий, которые можно произвести с данным отчетом;
  • «Сформировать» – при нажатии по данной кнопке будет сформирована выбранная форма паспортного стола.
  • «Настройка» – при нажатии по данной кнопке будет вызвано диалоговое окно для настройки отчета. В настройках можно указать следующие параметры:
    • «Размер бумаги» - задается формат бумаги (А4 или А5). Данная настройка доступна только для следующих форм: Ф2, Ф5, Ф7, Ф8, Ф9, Ф10, Ф15, С_Р, С_ЗП, С_ПР, З_С.
    • «Выводить» - данная настройка позволяет выводить (данная настройка действует только для форм Ф10, Ф11, Ф15, Ф16 и Ф9_СПБ):
      • «Жильцов из лицевого счета» - в отчет будут выводиться только те жильцы, которые принадлежат лицевому счету, указанному в поле отчета «Лицевой счет»;
      • «Близких родственников» - в отчет будут выводиться жильцы для помещения лицевого счета указанного в поле отчета «Лицевой счет».


        Пример:

        Пусть на жилое помещение «Кв. 1» открыто два лицевых счета «Л/с 1» и «Л/с 2». По лицевому счету «Л/с 1» зарегистрированы жильцы:

        «Иванов Иван» - ответственный собственник, наниматель лицевого счета «Л/с 1»;

        «Иванов Федор» - является сыном (близким родственником) для «Иванова Ивана».

        Также по «Л/с 1» проживает «Иванова Мария», которая является матерью(близким родственником) для «Иванова Ивана».

        По лицевому счету «Л/с 2» зарегистрированы два жильца:

        «Иванова Эльвира» - является ответственным собственником, нанимателем лицевого счета «Л/с 2»;

        «Иванова Юлия» - является дочерью (близким родственником) для «Ивановой Эльвиры».

        Также в документе «Смена степени родства физических лиц» задана связь, что «Иванова Юлия» является также дочерью (близким родственником) для «Иванова Ивана». Все данные можно увидеть наглядно в таблице:

        Для примера будем формировать отчет «Ф16 - Справка о составе семьи». В поле «Лицевой счет» формы отчета укажем лицевой счет «Л/с 1» и в поле «Жилец» укажем жильца «Иванов Иван».

        Рассмотрим два случая:

        • Если в настройках отчета в поле «Выводить» будет указано значение «Жильцов из лицевого счета», то при формировании формы Ф16 для лицевого счета «Л/с 1» в отчет попадут только зарегистрированные жильцы лицевого счета «Л/с 1»: «Иванов Иван» и «Иванов Федор». «Иванова Мария» не попадет в отчет, т.к. она не зарегистрирована по лицевому счету на момент формирования отчета. Жильцы «Иванова Эльвира» и «Иванова Юлия» также не попадут в отчет, потому что они зарегистрированы по другому лицевому счету.
        • Если в настройках отчета в поле «Выводить» будет указано значение «Близких родственников», то при формировании формы Ф16 для лицевого счета «Л/с 1» в отчет попадут зарегистрированные жильцы как лицевого счета «Л/с 1», так и «Л/с 2», при этом из «Л/с 2» попадут только те жильцы, которые имеют близкое родство по отношению к ответственному собственнику, нанимателю «Л/с 1». Т.е. в этом случае в отчет попадут следующие жильцы: «Иванов Иван», «Иванов Федор», «Иванова Мария», «Иванова Юлия».

Ф1 - Заявление о регистрации по месту пребывания.

Данная форма предназначена для заполнения заявления о регистрации по месту пребывания по «форме № 1» для жильца, который указан в поле «Жилец». Заявление можно формировать на жильцов лицевого счета, независимо являются ли они зарегистрированными, проживающими или собственниками помещения на дату формирования отчета.

Заявление о регистрации по месту жительства также можно сформировать из документа «Заявление о регистрации по месту пребывания».

Ф1П - Заявление о выдаче или замене паспорта.

Данная форма предназначена для заполнения заявления о получении нового паспорта гражданина РФ, либо при замене старого по «форме № 1П». Используется при выдаче паспортов нового образца. Отчет формы заполняется на жильца, указанного в поле «Жилец». Заявление будет формироваться на жильцов лицевого счета, независимо являются ли они зарегистрированными, проживающими или собственниками помещения на дату формирования отчета.

Ф2 - Адресный листок прибытия.

Данная форма предназначена для формирования адресных листков прибытия по «форме № 2». Отчет формы будет заполняться на жильца, указанного в поле «Жилец». Адресный листок прибытия будет формироваться на жильцов лицевого счета, независимо являются ли они зарегистрированными, проживающими или собственниками помещения на дату формирования отчета.

Пункт №13 "Переменил(а) ФИО и прочие сведения с:" заполняется предыдущими ФИО только при формировании отчета в день изменения ФИО.

Адресный листок прибытия также можно сформировать из документа «Заявление о регистрации по месту пребывания» или из документа «Заявление о регистрации по месту жительства».

Ф3 - Свидетельство о регистрации по месту пребывания.

Данная форма предназначена для формирования документов, подтверждающим регистрацию по месту пребывания по «форме № 3» для жильца, указанного в поле «Жилец». Свидетельство будет формироваться на жильцов лицевого счета, независимо являются ли они зарегистрированными, проживающими или собственниками помещения на дату формирования отчета.

Ф5 - Анкета.

Данная форма предназначена для формирования анкетных данных по «форме № 5». Анкета будет формироваться на жильца, указанного в поле «Жилец», независимо являются ли они зарегистрированными, проживающими или собственниками помещения на дату формирования отчета.

Ф6 - Заявление о регистрации / о снятии с регистрационного учета по месту жительства.

Данная форма предназначена для формирования заявления о регистрации или о снятии с регистрационного учета по месту жительства по «форме № 6» для жильца, указанного в поле «Жилец». Заявление может формироваться на жильцов лицевого счета, независимо являются ли они зарегистрированными, проживающими или собственниками помещения на дату формирования отчета.

Заявление о регистрации или о снятии с регистрационного учета по месту жительства также можно сформировать из документа «Заявление о регистрации по месту жительства» с соответствующим видом операции «Регистрация» или «Снятие с регистрации».

Ф6р - Заявление о регистрации по месту жительства.

Данная форма предназначена для формирования заявления регистрации по месту жительства по «форме № 6 - Регистрация по месту жительства». Отчет формы будет заполняться данными для жильца, указанного в поле «Жилец». Заявление будет формироваться на жильцов лицевого счета, независимо являются ли они зарегистрированными, проживающими или собственниками помещения на дату формирования отчета.

Заявление о регистрации по месту жительства также можно сформировать из документа «Заявление о регистрации по месту жительства» с видом операции «Регистрация».

Ф6с - Заявление о снятии с регистрационного учета по месту жительства.

Данная форма предназначена для формирования заявления о снятии с регистрационного учета по «форме № 6 - Снятие с регистрационного учета по месту жительства». Отчет формы будет формироваться для жильца, указанного в поле «Жилец». Заявление будет формироваться на жильцов лицевого счета, независимо являются ли они зарегистрированными, проживающими или собственниками помещения на дату формирования отчета.

Заявление о снятии с регистрационного учета по месту жительства также можно сформировать из документа «Заявление о регистрации по месту жительства» с видом операции «Снятие с регистрация».

Ф7 - Адресный листок убытия.

Данная форма предназначена для формирования адресного листка убытия для граждан считающихся снятыми с регистрационного учета по месту пребывания в жилых помещениях по «форме № 7». Адресный листок убытия формируется на жильца, указанного в поле «Жилец». В качестве жильца лицевого счета можно указать, независимо является ли он зарегистрированным, проживающим или собственником помещения на дату формирования отчета.

Пункт №14 "Переменил(а) ФИО и прочие сведения на:" заполняется новыми ФИО только при формировании отчета в день изменения ФИО. Пункты 1-3 при этом заполняются предыдущими ФИО.

Отчет также можно сформировать из документа «Заявление о регистрации по месту жительства» с видом операции «Снятие с регистрация».

Ф8 - Свидетельство о регистрации по месту жительства.

Данная форма предназначена для документов подтверждающих регистрацию по месту жительства «форме № 8» граждан, которые по последнему месту жительства были зарегистрированы по иным (кроме паспорта) документам. Свидетельство будет формироваться на жильца, указанного в поле «Жилец» лицевого счета, независимо является ли они зарегистрированным, проживающим или собственником помещения на дату формирования отчета.

Ф9 - Карточка регистрации граждан.

Данная форма отчета предназначена для формирования карточки регистрации по «форме № 9». Отчет оформляется, когда гражданин встает на учет по месту регистрации.

Карточка будет формироваться на жильца, указанного в поле формы «Жилец», независимо является ли они зарегистрированным, проживающим или собственником помещения на дату формирования отчета.

Ф10 - Поквартирная карточка.

Данная форма предназначена для формирования поквартирной карточки по «форме № 10». Форма оформляется, когда проживающий встает на учет по месту регистрации. В табличную часть отчета будут попадать только зарегистрированные жильцы или жильцы, которые были сняты с регистрации на дату формирования отчета.

Ф11 - Домовая (поквартирная) книга.

Данная форма предназначена для формирования домовой (поквартирной) книги по «форме № 11». Домовые (поквартирные) книги оформляются при регистрации граждан в жилых помещениях, принадлежащих гражданам на праве собственности, которые хранятся у владельцев домов (квартир). В табличную часть «Раздел 1» будут попадать только собственники указанного лицевого счета, а в табличную часть «Раздел 2» будут попадать все зарегистрированные жильцы лицевого счета.

Книга формируется на зарегистрированных проживающих на дату формирования отчета и имеющим право пользования жилым помещением, принадлежащим на праве собственности физическим и юридическим лицам.

Ф12П - Листок статистического учета прибытия (к документам о регистрации по новому месту жительства).

Данная форма предназначена для формирования листков статистического учета прибытия по «форме № 12 П». Форма формируется на граждан, которым регистрация по месту пребывания осуществляется на срок один год и более. Отчет формы будет заполняться для жильца, указанного в поле «Жилец». Листок статистического учета прибытия будет формироваться на жильцов лицевого счета, независимо являются ли они зарегистрированными, проживающими или собственниками помещения на дату формирования отчета.

Отчет также можно сформировать из документа «Заявление о регистрации по месту жительства» с видом операции «Регистрация».

Ф12В - Листок статистического учета выбытия (к документам о снятии с регистрационного учета).

Данная форма предназначена для формирования листков статистического учета выбытия по «форме № 12 В». Листки статистического учета выбытия составляются только на граждан, подлежащих статистическому наблюдению. Отчет формы будет заполняться для жильца, указанного в поле «Жилец». Листок статистического учета выбытия будет формироваться на жильцов лицевого счета, независимо являются ли они зарегистрированными, проживающими или собственниками помещения на дату формирования отчета.

Отчет также можно сформировать из документа «Заявление о регистрации по месту жительства» с видом операции «Снятие с регистрации».

Ф14 - Уведомление в орган регистрационного учета.

Данная форма предназначена для формирования документа уведомления о снятии с регистрационного учета жильца по «форме № 14». Отчет формы будет заполняться данными на жильца, указанного в поле «Жилец».

Уведомление будет формироваться на жильца лицевого счета, независимо являются ли они зарегистрированными, проживающими или собственниками помещения на дату формирования отчета.

Ф15 - Уведомление собственнику (нанимателю) жилого помещения.

Данная форма предназначена для формирования уведомления собственнику (нанимателю) жилого помещения по «форме № 15». Карточка будет формироваться на жильца, указанного в поле «Жилец», независимо является ли они зарегистрированным, проживающим или собственником помещения на дату формирования отчета.

Ф16 - Справка о составе семьи.

Данная форма предназначена для формирования справки о составе семьи по «форме № 16». Табличная часть справки о составе семьи будет заполняться только на зарегистрированных проживающих на дату формирования отчета.

Ф17 - Выписка из домовой книги.

Данная форма предназначена для формирования выписки из домой книги по «форме № 17». Табличная часть выписки заполняется жильцами указанного лицевого счета, независимо от того, зарегистрированы ли они, проживают или являются собственниками помещения по состоянию на дату формирования отчета.

Ф7_СПБ - Характеристика жилого помещения (СПБ).

Данная форма предназначена для формирования характеристики жилого помещения по «форме № 7» (только для Петербурга и Ленинградской области). Характеристика жилого помещения будет формироваться на жильца, указанного в поле «Жилец», независимо является ли он зарегистрированным, проживающим или собственником помещения на дату формирования отчета. В табличную часть «Сведения о проживающих» попадают жильцы являющимися собственниками помещения на дату формирования отчета.

Ф9_СПБ - Справка о регистрации (СПБ).

Данная форма предназначена для формирования справки о регистрации по «форме № 9» (только для Петербурга и Ленинградской области). Отчет будет формироваться на жильца, указанного в поле «Жилец». Справка формируется только на жильцов, которые зарегистрированы на дату формирования отчета.

Ф3_МСК - Сведения о зарегистрированных по месту жительства в связи с рождением (МСК).

Данная форма предназначена для передачи сведений о зарегистрированных по месту жительства гражданах в связи с рождением (только для Москвы и Московской области) главам управ районов города Москвы в административном округе города Москвы, включающем в себя районы, префекту административного округа города Москвы в административном округе города Москвы, включающем в себя поселения. Сведения предоставляются по состоянию на первое число каждого месяца не позднее 7 числа каждого месяца.

С_С - Справка в суд.

Данная форма предназначена для формирования справки в суд. Табличная часть отчета будет заполняться зарегистрированными жильцами или жильцами, снятыми с регистрации на дату формирования отчета.

С_Р - Справка о регистрации по месту жительства.

Данная форма предназначена для формирования справки о регистрации по месту жительства. Отчет формы будет формироваться на жильца, указанного в поле «Жилец». Справка формируется на жильцов, которые зарегистрированы на дату формирования отчета.

С_ЗП - Справка об отсутствии паспорта в связи с заменой.

Данная форма предназначена для формирования справки об отсутствии паспорта в связи с его заменой у жильца, указанного в поле «Жилец». Отчет будет формироваться независимо от того, является ли жилец зарегистрированным, проживающим или собственником помещения на момент формирования отчета.

С_ПР - Справка об отсутствии паспорта (на регистрации).

Данная форма предназначена для формирования справки об отсутствии паспорта у жильца, указанного в поле «Жилец», независимо является ли он зарегистрированным, проживающим или собственником помещения на момент формирования отчета.

З_С - Заявление собственников для регистрации.

Данная форма предназначена для формирования заявления от собственников помещения о том, что они не возражают против регистрации на территории их собственности указанного лица. Жилец, указанный в поле «Жилец» - лицо, которое планируется зарегистрировать по адресу лицевого счета. Заявление будет формироваться на каждого жильца, являющегося собственником на дату формирования отчета.

РФЛ1 - Сведения о регистрации гражданина РФ по месту жительства.

Данная форма предназначена для формирования сведений о регистрации гражданина по месту его жительства. Отчет будет заполняться на жильца, указанного в поле «Жилец». Сведения будут формироваться независимо от того, является ли жилец зарегистрированным, проживающим или собственником помещения на момент формирования отчета.

ППВУ - Список подлежащих первоначальной постановке на воинский учет.

Данная форма предназначена для учета граждан, подлежащих первоначальной постановке на воинский учет. Орган местного самоуправления, на который возложено ведение первичного воинского учета граждан, организация, осуществляющая эксплуатацию жилых помещений формируют и представляют список в военный комиссариат на бумажном носителе и в электронном виде.

Отчет открывается через меню «Учет в ЖКХ / Отчеты / Формы регистрационного учета /».
