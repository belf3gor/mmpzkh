Отчет «Декларация 3-НДФЛ» (форма выбора периода и варианта отчета)

Данная форма открывается при нажатии кнопки «Открыть» в форме списка справочника Регламентированные отчеты или при открытии регламентированного отчета через меню «Операции» главного меню программы.

Стартовая форма предназначена для:

  • выбора периода формирования регламентированного отчета;
  • автоматического определения формы регламентированного отчета, соответствующей выбранному отчетному периоду.

«Стрелка вправо» / «Стрелка влево» - кнопки предназначены для выбора отчетного периода. Нажатие кнопок «прокручивает» периоды, за которым может составляться данный регламентированный отчет.

Выбор периодичности  представления регламентированного отчета производится в выпадающем списке поля между «Стрелка вправо» и  «Стрелка влево» , в котором отображаются периоды. Кнопка выбора из списка становится доступной только если в выбранном периоде имеется возможность выбора разной периодичности представления регламентированного отчета. В зависимости от выбранной периодичности, выбор отчетного периода будет выполняться либо по кварталам, либо по месяцам.

«Срок сдачи» - информация по срокам сдачи отчетности.

«Кто сдает отчет» - информация по типам (видам) налогоплательщиков, условиям налогаемым на осуществляемую хозяйственную деятельность, при удовлетворении которым налогоплательщик обязан сдавать в соответствующий орган данный вид регламентированного отчета.

«Редакция формы» - реквизит становится видимым и доступным, если за выбранный период регламентированный отчет может быть составлен по разным формам. В этом случае у пользователя появляется  возможность выбора редакции формы регламентированного отчета. При выборе формы в нижней части диалога выдается краткое описание выбранной формы, например, реквизиты документа, которым она была утверждена.

«Создать» - открывает выбранную форму регламентированного отчета. Если для выбранного отчетного периода нет доступных форм регламентированного отчета, становится недоступной.

«Изменения законодательства» - ссылка открывает web страницу мониторинга законодательства начиная с 2013 года, связанного с отчетом. Для периодов ранее 2013 года ссылка не показывается.
