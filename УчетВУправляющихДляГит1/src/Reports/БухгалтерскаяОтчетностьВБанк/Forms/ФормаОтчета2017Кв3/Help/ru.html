Отчет "Бухгалтерская отчетность для Сбербанка"

Отчет "Бухгалтерская отчетность для Сбербанка" предназначен для подготовки форм как годовой, так и промежуточной бухгалтерской отчетности организаций для представления непосредственно в отделение Сбербанка.

Состав отчетности:  

  • Титульный лист
  • Бухгалтерский баланс (квартал, год);
  • Отчет о финансовых результатах (квартал, год);
  • Отчет об изменениях капитала (год);
  • Отчет о движении денежных средств (год);
  • Отчет о целевом использовании средств (год);
  • Дополнительные показатели (квартал, год);
  • Расшифровки обеспечений (квартал, год):
    • полученные;
    • выданные;
  • Задолженность по кредитам и займам (квартал, год):
    • краткосрочная;
    • долгосрочная;
  • Расходы по лизингу (квартал, год);
  • Денежные средства (месяц, квартал, год):
    • касса;
    • расчетные счета;
    • валютные счета;
    • специальные счета в банках;
  • ОСВ (Оборотно-Сальдовая ведомость) (месяц, квартал, год);
  • ОСВ по счету (Оборотно-Сальдовая ведомость по счету) (при необходимости);
  • Анализ счета (при необходимости);
  • Операции по банковским счетам (при необходимости);
  • Дебиторская и кредиторская задолженность (квартал, год);
    • Дебиторская
    • Кредиторская
  • Прочие доходы и расходы (квартал, год);
  • Финансовые вложения (квартал, год);
    • Краткосрочные
    • Долгосрочные
  • Справка о структуре выручки и прочих поступлениях (при необходимости);
  • Справка о прочих показателях деятельности (при необходимости);
  • Дополнительные файлы (при необходимости);
  • Отчетность принятая в ФНС (при необходимости).

 

В Настройках (кнопка «Отчеты») при необходимости можно изменить отображение отчетности:

- на вкладке Отчеты – состав;

- на вкладке Общие – единицу измерения и количество знаков после запятой.

Для перехода к нужному отчету необходимо выбрать соответствующую закладку в форме отчета.

Для каждого раздела отчетности есть возможность отдельного заполнения (кнопка «Заполнить – Все отчеты/Текущий отчет»).

При нажатии на кнопку «Проверка» проверяется правильность заполнения справочной информации, а также отдельных показателей отчетности.

В автозаполняемых разделах отчетности данные ячеек можно расшифровать (кнопка «Расшифровать»).

Представление отчетности в электронном виде производится при нажатии на кнопку «Отправить». Перед отправкой в банк отчетность можно сохранить на диск или открыть для просмотра.

Возможность печати Бухгалтерской отчетности для Сбербанка отсутствует. 

Особенности заполнения.

Бухгалтерский баланс, Отчет о финансовых результатах, Отчет об изменениях капитала, Отчет о движении денежных средств, Отчет о целевом использовании средств

Отчеты формируются в соответствии с образцами форм, утвержденными приказом Минфина России от 02.07.2010 г. № 66н (в редакции приказа Минфина России от 06.04.2015 N 57н).

Дополнительные показатели

В отчете указываются сведения за последние три года по:

  • накопленной амортизации;
  • незавершенному строительству и незаконченным операциям по приобретению, модернизации и т.п. основных средств;
  • долгосрочной дебиторской задолженности;
  • краткосрочной  дебиторской задолженности;
  • просроченной дебиторской задолженности;
  • просроченной кредиторской задолженности;
  • задолженности учредителей по взносам в уставный (складочный) капитал;
  • финансовому лизингу.

Отчет заполняется автоматически по данным бухгалтерского учета. Исключение: данные по финансовому лизингу указываются вручную.

Расшифровки обеспечений.

Подраздел "Расшифровка полученных обеспечений обязательств и платежей" заполняется по сведениям счета 008.

Подраздел "Расшифровка выданных обеспечений обязательств и платежей" заполняется по сведениям счета 009.

Информация об участниках сделки (кто предоставил, в чью пользу и за кого предоставлено обеспечение), вид обеспечения (аваль, залог, поручительство, гарантия) и сроки действия заполняются из договора.

Задолженность по кредитам и займам.

Отчет заполняется на основании данных «Оборотно-сальдовой ведомости по счету» по счетам 66 и 67.

Отчет заполняется автоматически при нажатии кнопки «Заполнить». После заполнения раздела необходимо вручную указать сведения о кредитном продукте, по которому имеется задолженность, и ввести график платежей.

Внимание! При повторном нажатии кнопки «Заполнить» сведения, введенные вручную, будут очищены.

Расходы по финансовому лизингу.

Отчет заполняется вручную отдельно по каждой компании.

Денежные средства

Отчет заполняется автоматически при нажатии на кнопку «Заполнить» на основании стандартных отчетов «Анализ счета» по счетам 50, 51, 52 и 55 и вручную не корректируются.

Для внесения изменений следует использовать документы расчетов в конфигурации или ввести ручные операции в информационную базу.

Оборотно-сальдовая ведомость.

Операции по банковским счетам

Отчет заполняется по сведениям счетов 51, 52 и 55.

В отчет включены детальные сведения об операциях по всем банковским счетам.

Дебиторская и кредиторская задолженность

Отчет заполняется по алгоритмам заполнения пояснений 5 к бухгалтерскому балансу и отчету о финансовых результатах.

При необходимости в отчете указывается планируемая дата погашения просроченной задолженности и комментарий к ней.

Прочие доходы и расходы

Отчет заполняется по алгоритмам расчета значений строк 2340 и 2350 отчета о финансовых результатах.

Финансовые вложения

Отчет заполняется по алгоритмам расчета значений строк 1190 и 1240 бухгалтерского баланса.

Справка о структуре выручки и прочих поступлениях

Отчет заполняется вручную.

Справка о прочих показателях деятельности

Отчет заполняется вручную.

Дополнительные файлы

К отчетности, отправляемой в банк, прикладываются дополнительные отчеты и справки, запрошенные банком:

  • Письмо-заверение об отсутствии изменений в учредительных документах общества и о фактах деятельности общества, имеющих юридическое значение
  • Расшифровка производных финансовых инструментов
  • Отчет о ходе реализации проекта
  • Отчет о реализации квартир
  • Производственная программа
  • Справка об отсутствии лизинговых обязательств
  • Справка об отсутствии дебиторской задолженности
  • Справка об отсутствии кредиторской задолженности
  • Справка об отсутствии финансовых вложений
  • Прочие файлы

Каждый вид прикладываемого файла ограничен по формату и размеру.

Отчетность принятая в ФНС

К отчетности, отправляемой в банк, прикладываются отчеты, представленные в налоговые органы через сервис 1С-Отчетность:

  • Бухгалтерская отчетность
  • Декларация по налогу на прибыль
  • Декларация по НДС
  • Справка об исполнении обязанностей по уплате налогов

Формат учитывает возможность руководителя организации передать свое полномочие по подписанию бухгалтерской (финансовой) отчетности организации иному лицу на основании доверенности.

Сведения о подписанте - уполномоченном лице извлекаются из справочника "Регистрации в налоговом органе". Сведения об уполномоченном лице включаются в файл отправляемого электронного представления.
