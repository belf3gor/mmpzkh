Отчет «Доходы, выплаченные иностранным организациям»

Отчет «Доходы, выплаченные иностранным организациям» предназначен для подготовки налогового расчета о суммах выплаченных иностранным организациям доходов и удержанных налогов в соответствии с приказом МНС России от 14.04.2004 г. №САЭ-3-23/286@.

Порядок заполнения

При заполнении отчета рекомендуется соблюдать такую последовательность подготовки листов:

1. Титульный лист.

2. Сведения о физическом лице (вторая страница титульного листа).

3. Раздел 1.

4. Раздел 2.

Особенности заполнения некоторых листов отчета

Титульный лист

Следует указать сведения об организации и сведения о должностных лицах, которые не были заполнены автоматически. Автоматически заполненные сведения необходимо проверить и, при необходимости, отредактировать.

Сведения о физическом лице (вторая страница титульного листа)

Заполняется только в отношении должностных лиц и уполномоченного представителя, у которых ИНН отсутствует или не указан на титульном листе. При вводе новой декларации листы со сведениями о физических лицах формируются автоматически.

Раздел 1

Раздел заполняется отдельно в отношении выплат в пользу каждой иностранной организации, за исключением банков.

В случае, если количество произведенных в соответствующем периоде выплат превышает количество отведенных для их отражения в подразделе 1.2 строк, необходимо заполнить дополнительные бланки по разделу 1.

В случае, если налоговым агентом в течение отчетного (налогового) периода удерживался налог, которому соответствуют различные коды бюджетной классификации (КБК), в отношении каждого КБК необходимо заполнить отдельные бланки по форме раздела 1.

Для создания дополнительных бланков раздела 1 необходимо воспользоваться механизмом управления дополнительными страницами.

Для управления дополнительными страницами (добавление/удаление) необходимо использовать кнопки, расположенные вверху страницы, либо воспользоваться контекстным меню в панели навигации по разделам отчета, выделив нужную страницу.

Заполнять строки таблицы необходимо строго в соответствии с инструкцией по заполнению отчета.

Раздел 2

Раздел заполняется налогоплательщиком отдельно в отношении выплат в пользу каждого конкретного иностранного банка.

В случае, если количество произведенных в соответствующем периоде выплат превышает количество отведенных для их отражения в подразделе 2.2 строк, необходимо заполнить дополнительные бланки по разделу 2. Для создания дополнительных бланков раздела 2 необходимо воспользоваться механизмом управления дополнительными страницами.

Для управления дополнительными страницами (добавление/удаление) необходимо использовать кнопки, расположенные вверху страницы, либо воспользоваться контекстным меню в панели навигации по разделам отчета, выделив нужную страницу.

Заполнять строки таблицы необходимо строго в соответствии с инструкцией по заполнению отчета.
