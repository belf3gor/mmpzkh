Отчет «Декларация по акцизам на нефтепродукты»

 

Перед началом работы по составлению регламентированных отчетов рекомендуется ознакомиться с описанием работы с регламентированными отчетами.

Ниже приведен порядок заполнения налоговой декларации по акцизам на нефтепродукты (Приложение №5 к Приказу Минфина РФ от 03.03.2005 г. №32н, далее - Декларация) с учетом рекомендаций налоговых органов и требований программы.

Декларация реализована в виде отдельных листов, каждый из которых может готовиться независимо от других. Однако для правильного автоматического переноса данных между листами Декларации, рекомендуется соблюдать такую последовательность подготовки листов:

 

1. Титульный лист.
2. Сведения о физическом лице (вторая страница титульного листа).
3. Приложения №№ 1 - 4 к Декларации (в произвольной последовательности).
4. Раздел 2.1 «Расчет суммы акциза».
5. Раздел 2.2 «Расчет суммы акциза».
6. Раздел 2.3 «Расчет суммы акциза» («Расчет суммы акциза, подлежащей уплате по месту нахождения обособленного подразделения налогоплательщика».
7. Раздел 1 «Сумма акциза на нефтепродукты,  подлежащая уплате в бюджет, по данным налогоплательщика».
8. Приложение №5 к Декларации.


ОСОБЕННОСТИ ЗАПОЛНЕНИЯ  НЕКОТОРЫХ ЛИСТОВ ДЕКЛАРАЦИИ:

Титульный лист

Необходимо указать дату подписания отчета.

 

Следует указать сведения об организации и сведения о должностных лицах, которые не были заполнены автоматически. Автоматически заполненные сведения необходимо проверить и, при необходимости, отредактировать.

Приложение  №1

Заполняется отдельно по каждому поставщику.

В верхней части приложения указываются сведения о поставщике. При наличии нескольких поставщиков необходимо воспользоваться управлением дополнительными страницами (добавление/удаление) необходимо использовать кнопки, расположенные вверху страницы, либо воспользоваться контекстным меню в панели навигации по разделам отчета, выделив нужную страницу.

При наличии по одному виду нефтепродуктов нескольких счетов-фактур, по каждому счету-фактуре заполняется отдельная строка. Для добавления в отчет требуемого количества строк следует воспользоваться механизмом создания дополнительных строк.

Приложение  №3

Заполняются отдельно по каждому покупателю. В верхней части приложения указываются сведения о покупателе.

При наличии нескольких покупателей необходимо заполнить приложения отдельно по каждому из них. Для создания дополнительных бланков по приложению следует воспользоваться управлением дополнительными страницами (добавление/удаление) необходимо использовать кнопки, расположенные вверху страницы, либо воспользоваться контекстным меню в панели навигации по разделам отчета, выделив нужную страницу.

При наличии нескольких видов подакцизных товаров, по каждому виду заполняется отдельная строка. Для добавления в отчет требуемого количества строк следует воспользоваться механизмом создания дополнительных строк, органы управления которого расположены в диалоге управления формированием отчета.

Приложения №№2, 4

В приложениях указываются объемы нефтепродуктов, полученных (поставленных) в целом за налоговый период. Каждое приложение заполняются в одном экземпляре. 

Приложение  №5 «Сведения о суммах акциза, уплачиваемых по месту нахождения налогоплательщика и его обособленных подразделений»

Заполняется в случае уплаты налогоплательщиком сумм акциза по месту своего нахождения и по месту нахождения каждого своего обособленного подразделения.


Раздел 2.2  «Расчет суммы акциза»


В графу 5 по строкам 010 - 050 переносится налоговая база, рассчитанная в строках по коду 460 - 500 таблицы 2.1 «Расчет налоговой базы».


Раздел 2.3 «Расчет суммы акциза, подлежащей уплате по месту нахождения обособленного подразделения налогоплательщика».

Объем продукции, реализованной на территории РФ по организации в целом переносится в графу 4 по строкам 030 - 070 из строк 510 - 550 таблицы 2.1 «Расчет налоговой базы»

Суммы акциза, подлежащие уплате в бюджет в целом по организации, переносятся в графу 6 по строкам 030 - 070 из строк 360 - 400 таблицы 2.2 «Расчет суммы акциза».

Сумма превышения налоговых вычетов над исчисленной суммой акциза переносятся в графу 6 по строкам 030 - 070 из строк 410 - 450 таблицы 2.2 «Расчет суммы акциза».


Раздел 1  «Сумма акциза на нефтепродукты,  подлежащая уплате в бюджет, по данным налогоплательщика"

В Разделе 1 отражаются суммы акциза, подлежащие уплате в бюджет, с разбивкой по кодам бюджетной классификации (КБК), на которые подлежат зачислению суммы акциза, рассчитанные в декларации. По каждому коду бюджетной классификации необходимо заполнять отдельно показатели по строкам, имеющим коды 010 - 030.
Строки 030 заполняются на основании расчета, произведенного в пункте 16 таблицы 2.2 раздела 2.

Если несколько сумм акциза, рассчитанных по разным кодам вида подакцизного товара, подлежат зачислению на один код бюджетной классификации, следует сложить эти суммы и результат внести в строку по коду 030.
