Отчет «Платежный документ»

Отчет предназначен для формирования списка квитанций и извещений на оплату коммунальных услуг.

Платежный документ разработан в соответствии с предложенной Минрегионом РФ формой платежного документа для внесения платы за содержание и ремонт жилого помещения и предоставление коммунальных услуг, разработанного в соответствии с "Правилами предоставления коммунальных услуг собственникам и пользователям помещений в многоквартирных домах и жилых домов", утвержденными Постановлением Правительства РФ от 06.05.2011 N 354.

Для печати квитанций имеется несколько форм. Платежный документ может быть выбран в качестве основной формы квитанции и извещения на оплату коммунальных услуг в настройках Учетной политики ЖКХ на закладке «Квитанции».

Для формирования квитанции необходимо указать:

  • «Организация» – организация, оказывающая услуги;
  • «Объекты» – лицевые счета, по которым будет формироваться квитанция;
    Они могут быть выбраны одним из следующих двух способов:
    • Из справочника «Лицевые счета». Для этого необходимо выбрать пункт «Выбор из справочника» выпадающего меню в поле «Объекты»;
    • Посредством формы подбора объектов. Для этого необходимо выбрать пункт «Подбор объектов» выпадающего меню в поле «Объекты».
  • «Настройка сбора данных» – настройка сбора данных для формирования квитанций, выбранная из справочника «Настройки сбора данных для квитанций и обменов»;
  • «Период» – месяц формирования квитанции;
  • Настройки ведения учета по услугам капитального ремонта для выбранной организации с помощью соответствующей гиперссылки.

В верхней части диалогового окна отчета расположены кнопки:

  • «Действия» – выводит список действий, которые можно совершить с данным отчетом;
  • «Сформировать отчет» – формируется отчет;
  • «Сменить квитанцию» – пердоставляет выбор формы квитанции для открытия. При открытии выбранной квитанции, отличной от текущей, окно текущей квитанции закрывается;
  • «Сравнить макет с типовым» – выполняет сравнение измененного макета (пользовательского) квитанции со стандартным. Данная кнопка доступна, только когда используется измененный макет квитанции.

Рассылка квитанций на электронную почту лицевых счетов

Доступна возможность рассылки сформированных платежных документов на электронную почту лицевых счетов. Включить/отключить данную возможность можно в разделе "Главное - Настройки ЖКХ - Функциональность - Платежные документы".

Для рассылки квитанций необходимо:

  1. Сформировать квитанции;
  2. Нажать на кнопку «Отправить по электронной почте».

Причем есть возможность тестовой рассылки для того, чтобы проверить корректность работы механизма и посмотреть на вид получаемых писем.

Для тестовой отправки квитанций необходимо:

  1. Установить параметры рассылки квитанций «E-mail получателя» и «Количество тестовых писем»;
  2. Сформировать квитанции;
  3. Нажать на кнопку «Настройка рассылки - Отправить на тестовый e-mail».
