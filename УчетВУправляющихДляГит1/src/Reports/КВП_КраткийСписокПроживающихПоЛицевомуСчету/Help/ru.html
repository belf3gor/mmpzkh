Отчет «Краткий список проживающих по лицевому счету»

Отчет предназначен для отображения списка жильцов по выбранному лицевому счету.

По каждому из людей указывается, проживает ли он, зарегистрирован ли по данному адресу, является ли собственником. Также по каждому из этих трех показателей выводится итоговое количество людей.

Отчет вызывается из формы списка справочника «Лицевые счета» нажатием кнопки «Проживающие» или переходом на закладку «Список проживающих».
