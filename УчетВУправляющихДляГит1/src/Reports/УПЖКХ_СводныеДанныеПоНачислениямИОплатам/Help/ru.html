Отчет "Сводные данные по начислениям и оплатам"

Отчет предоставляет возможность сформировать сведения о начислениях, оплатах, задолженностях по организациям.
По умолчанию возможны два варианта формирования отчета: "Начисления и оплаты", "Динамика задолженности".

Возможные показатели отчета

  • Организация
  • Начальное сальдо
  • Начисления
    • по начислениям
    • по пени
    • по корректировкам
    • по льготам
  • Оплаты
    • по пени
    • по начислениям
  • Конечное сальдо
  • Изменение задолженности
