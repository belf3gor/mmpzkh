Управление сохраненными буферами обмена

Данные, скопированные в специальный "буфер обмена" можно сохранить для дальнейшего использования с помощью этой формы. Данные надо вначале скопировать в буфер обмена с помощью специальной команды. Потом, с помощью этой формы, скопированным данным надо задать имя (в поле Короткое наименование) и нажать кнопку "Сохранить".

После сохранения, эти данные можно вставлять с помощью специального меню вставки.

Если сохраненные данные буфера обмена более не актуальны, их можно удалить из списка сохраненных - выберите ненужные даные и нажмите кнопку "Удалить из сохраненных".

Для целей обмена между разными пользователями или компьютерами, можно выгружать список сохраненных буферов обмена в файл. Также данные можно загружать из файла.
