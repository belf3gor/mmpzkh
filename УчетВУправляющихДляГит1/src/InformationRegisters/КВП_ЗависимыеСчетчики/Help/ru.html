Регистр сведений «Зависимые приборы учета»

Данный регистр предназначен для настройки и хранения зависимостей приборов учета. Такая зависимость означает, что при вводе показаний по прибору учета документом "Ввод показаний счетчика" автоматически происходит определение "входных" приборов учета, связанных с текущим, и уменьшение расхода по ним на величину расхода текущего прибора учета.

 

Регистр независимый, непериодический.

В регистре используются измерения:

  • «"Выходной" прибор учета» - ссылка на справочник "Счетчики" - прибор учета, на расход которого будет уменьшен расход "входного" прибора учета;

  • «"Входной" прибор учета» - ссылка на справочник "Счетчики" - прибор учета, расход по которому будет уменьшен. В качестве "входного" прибора учета не может быть выбран счетчик, у которого установлен признак "Используется в качестве "выходного" прибора учета".

 

Тарифность "входного" и "выходного" приборов учета обязательно должна совпадать.

Кроме того, по умолчанию нельзя настроить зависимость приборов учета с разными услугами. Если в каких-либо ситуациях необходимо задать зависимость приборов учета с разными услугами, необходимо установить флажок "Отключить проверку выбора приборов учета с одинаковыми услугами", выбрать нужные приборы учета и записать изменения.
