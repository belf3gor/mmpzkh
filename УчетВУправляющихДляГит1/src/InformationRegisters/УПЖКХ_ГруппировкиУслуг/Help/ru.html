Регистр сведений «Группировки услуг»

Данный регистр предназначен для хранения данных по группировкам услуг. Данная информация необходима для определения услуг, относящихся к различным группам услуг и видам групп услуг.

В регистре указываются поля:

  • «Вид групп услуг» - содержит ссылку на справочник "Виды групп услуг". Выбираются только группы справочника 1-го уровня иерархии, соответствующие видам групп услуг.
  • «Группа услуг» - содержит ссылку на справочник "Виды групп услуг". Выбираются только элементы справочника 2-го уровня иерархии, соответствующие группам услуг.
  • «Услуга» - содержит ссылку на справочник "Услуга".

    Важно: одна услуга может быть отнесена только к одной группе услуг внутри каждого вида групп услуг.


Регистр независимый, непериодический.
