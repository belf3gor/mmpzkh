Регистр сведений «Услуги для расчета льгот по другим тарифам»

Данный регистр предназначен для хранения услуг, для которых необходимо рассчитывать льготу по льготному тарифу, отличному от тарифа услуги.

Регистр содержит следующие поля:

  • «Организация» - указывается организация, по которой производится расчет льгот;
  • «Услуга» - указывается услуга, для которой необходимо использовать льготный тариф. Льготный тариф задается в документе "Установка тарифов".

Регистр независимый, не периодический.
