Регистр сведений «Настройка индивидуальных ограничений для расчета льгот»

Данный регистр предназначен для установки индивидуальных размеров ограничений по объему услуги для расчета льгот.

Если заданы индивидуальные размеры ограничений, то они будут использоваться при расчете суммы льготы в документе "Расчет льгот".

Если индивидульные размеры ограничений не заданы, то для всех лицевых счетов будет использоваться ограничение по объему услуги, заданное в документе "Установка скидок".


В регистре указываются поля:

  • «Лицевой счет» - содержит ссылку на справочник "Лицевые счета".
  • «Льготная категория» - содержит ссылку на справочник "Льготные категории".
  • «Услуга» - содержит ссылку на справочник "Услуги".
  • «Ограничение по объему услуги» - содержит ссылку на справочник "Нормы потребления услуг".

Регистр независимый, непериодический.
