Регистр сведений «Зависимости услуг»

Данный регистр предназначен для хранения услуг, зависящих от других услуг. Такая зависимость означает, что при начислении суммы N по услуге-основанию документом «Начисление услуг» автоматически происходит начисление по зависимой услуге на сумму (N * коэффициент). Регистр периодический, периодичность регистра 1 месяц.

  • «Действует с» – начало периода действия зависимости услуг;
  • «Действует» – признак определяет, действует ли данная зависимость услуг;
  • «Объект» – ссылка на элемент справочника «Здания». Определяет объект учета, для которого действует данная зависимость услуг. Если объект учета не указан, то данная зависимость будет действовать на все здания, для которых индивидуально не указана зависимость услуг. Поле доступно для выбора, если в Настройках параметров учета (ЖКХ) на закладке «Учет начислений» установлена настройка «Указывать зависимость услуг в разрезе зданий»;
  • «Услуга-основание» – указывается услуга-основание, по которой производится начисление с помощью документов назначения и начисления;
  • «Услуга» – зависимая услуга, начисления по которой происходят автоматически;
  • «Коэффициент» – отношение требуемого размера начислений по зависимой услуге к размеру начислений по услуге-основанию (если по услуге-основанию начислено 1000, а по зависимой услуге должно быть 800, то коэффициент = 800 / 1000 = 0,8);
  • «Начислять при распределении показаний коллективных ПУ» – признак расчета начислений по зависимой услуге при распределении показаний коллективных приборов учета;
  • «Разрешить начисление при распределении отрицательного расхода ОДН» – признак определяет, разрешено ли начисление зависимой услуги при распределении отрицательного расхода ОДН коллективных приборов учета. Использование данного признака доступно только в случае, если установлена настройка «Начислять при распределении показаний коллективных ПУ».

Регистр можно открыть через меню «Учет в ЖКХ / Справочники / Услуги / Зависимости услуг».
