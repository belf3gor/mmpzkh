Протокол обмена с ЕГАИС

Регистр сведений Протокол обмена с ЕГАИС содержит информацию о каждой сессии обмена с УТМ. Для сессии обмена сохраняется Идентификатор запроса, присвоенный в УТМ, Вид документа, ссылка на Документ-основание.

Также есть возможность просмотра полученного или отправленного XML-файла.
