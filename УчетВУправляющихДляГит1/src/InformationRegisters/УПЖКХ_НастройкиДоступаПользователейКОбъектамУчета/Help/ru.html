Регистр сведений «Настройки доступа пользователей к объектам учета»

Регистр предназначен для хранения настроек ограничения прав доступа пользователей к объектам учета.

Регистр содержит следующие поля:

  • «Пользователь» - пользователь, для которого устанавливается настройка доступа;
  • «Объект учета» - объект учета, для которого устанавливается настройка доступа.

Если установлен флажок «Все объекты учета», то пользователю предоставляется доступ сразу на все объекты учета.
