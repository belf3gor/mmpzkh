Справочник «Сведения о материалах, используемых при проведении работ с оборудованием»

Регистр используется для хранения информации о материалах, которые могут использоваться при выполнении определенных видов работ с оборудованием.

В регистре используются измерения:

  • «Оборудование» - является ссылкой на элемент справочника «Прочие объекты учета, оборудование» - оборудование, для которого определено использование материала по виду работ;
  • «Вид материала» - является ссылкой на элемент справочника «Виды материалов» - вид материала, к которому относится используемый материал;
  • «Номенклатура» - является ссылкой на элемент справочника «Номенклатура» - материал, используемый при выполнении видов работ с оборудованием.
