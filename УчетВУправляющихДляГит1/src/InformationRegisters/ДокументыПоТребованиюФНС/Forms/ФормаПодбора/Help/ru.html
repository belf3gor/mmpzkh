Форма предназначена для подбора и формирования данных, отправляемых налогоплательщиком в ФНС в ответ на требование о представлении документов (информации).

В верхнем списке отображаются документы, данные которых можно отправить в ответ на требование.
В нижнем списке отображаются документы, отобранные из верхнего списка, для последующего переноса в ответ на требование о представлении документов (информации).

В зависимости от вида затребованного документа, его данные можно представить в ФНС либо в виде набора файлов выгрузки и файлов электронных подписей сторон, либо в виде сканированных листов документа с набором сопроводительных реквизитов.

Файлы выгрузки и файлы подписей сторон автоматически формируются при использовании механизма электронного документооборота между контрагентами.
Эти данные готовы к отправке, дополнительных действий по их подготовке выполнять не требуется.

Данные других документов требуют перед отправкой предварительной подготовки.
Подготовка данных документа к отправке заключается в добавлении файлов изображений сканированных листов, а также в заполнении реквизитов документа.
Если в программе есть затребованный документ, он отображается в списке и его можно использовать в качестве источника данных при подготовке. Для этого нужно нажать на ссылку "Подготовить" в строке списка отобранных документов.
При этом данные для отправки по возможности будут заполнены автоматически.
Подготовку документов с использованием документов-источников можно производить не только по отдельности, но и массово. Для этого можно использовать ссылку "Подготовить все" (ссылка появляется в командной панели списка отобранных документов при попадании в список документов, неподготовленных к отправке)

Если затребованного документа нет в списке, следует вручную подготовить его данные к отправке . Для этого нужно нажать кнопку "Добавить" командной панели списка отобранных документов.

Если все данные документа для отправки заполнены, он отображается в списке отобранных документов со статусом "Готов к отправке".
Если все документы в нижнем списке "Отобранные документы" готовы к отправке, можно закончить подбор, нажав кнопку "Перенести в ответ на требование".

Краткая инструкция по работе с формой

1. Выберите в верхнем списке документы, запрашиваемые в одном пункте требования
2. Переместите их в нижний список "Отобранные документы", используя кнопки переноса
3. Подготовьте их к отправке. Для массовой подготовки можно использовать ссылку "Подготовить все"
(ссылка появляется в командной панели списка отобранных документов при попадании в список документов, неподготовленных к отправке)
4. Нажмите кнопку "Перенести в ответ на требование"
