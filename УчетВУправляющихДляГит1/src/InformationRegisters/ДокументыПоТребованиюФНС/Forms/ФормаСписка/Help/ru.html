Форма предназначена для предварительного формирования данных, отправляемых налогоплательщиком в ФНС в ответ на требование о представлении документов (информации).

В зависимости от вида затребованного документа, его данные можно представить в ФНС либо в виде набора файлов выгрузки и файлов электронных подписей сторон, либо в виде сканированных листов документа с набором сопроводительных реквизитов.

Файлы выгрузки и файлы подписей сторон автоматически формируются при использовании механизма электронного документооборота между контрагентами.
Эти данные готовы к отправке, дополнительных действий по их подготовке выполнять не требуется.

Данные других документов требуют перед отправкой предварительной подготовки.
Подготовка данных документа к отправке заключается в добавлении файлов изображений сканированных листов, а также в заполнении реквизитов документа.
Если в программе есть затребованный документ, он отображается в списке и его можно использовать в качестве источника данных при подготовке. Для этого нужно нажать на ссылку "Подготовить" в строке списка.
При этом данные для отправки по возможности будут заполнены автоматически.
Подготовку документов с использованием документов-источников можно производить не только по отдельности, но и массово. Для этого нужно выделить строки списка с затребованными документами и нажать кнопку "Подготовить выделенные".

Если затребованного документа нет в списке, следует вручную подготовить его данные к отправке . Для этого нужно нажать кнопку "Добавить" командной панели списка.

Если все данные документа для отправки заполнены, он отображается в списке со статусом "Готов к отправке".

Отправка осуществляется из формы ответа на требование о представлении документов.
Создать ответ на требование можно на закладке "Письма" формы "1С-Отчетность".
