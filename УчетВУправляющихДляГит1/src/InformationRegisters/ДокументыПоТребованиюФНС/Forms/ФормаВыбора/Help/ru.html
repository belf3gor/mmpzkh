Форма предназначена для выбора документа, который будет служить источником для заполнения данных, отправляемых налогоплательщиком в ФНС в ответ на требование о представлении документов (информации).

Отправляемые данные документа представляют собой набор изображений сканированных листов и сопроводительных реквизитов документа.
Если в программе есть затребованный документ, он отображается в списке и его можно использовать в качестве источника данных при подготовке. В результате выбора данные для отправки по возможности будут заполнены автоматически.
