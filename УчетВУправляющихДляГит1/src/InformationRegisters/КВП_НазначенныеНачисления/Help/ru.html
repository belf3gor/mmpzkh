Регистр сведений «Назначенные начисления»

Данный регистр предназначен для хранения списка услуг, назначенных для начисления на лицевые счета.

Регистр периодичен по позиции регистратора, подчинен документам «Назначение услуг», «Закрытие лицевого счета».

В регистре используются измерения:

  • «Объект» - является ссылкой на элемент справочника «Лицевые счета»;
  • «Организация» - является ссылкой на элемент справочника «Организациии»;
  • «Услуга» - является ссылкой на элемент справочника «Услуги».

В регистре используются ресурсы:

  • «Действует» - признак нужно ли начислять услугу по объекту (Да/Нет). Установливается при назначении услуги, снимается при отмене/приостановке;
  • «Дата изменения» - дата, с которой начинает действовать изменение по текущему назначению.
