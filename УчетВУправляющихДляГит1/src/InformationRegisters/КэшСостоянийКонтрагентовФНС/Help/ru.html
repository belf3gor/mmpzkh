Регистр сведений предназначен для хранения результатов проверки состояний контрагентов сервисом ФНС сервисом.

  • Кэширование используется для оптимизации обращений к сервису. Перед выполнением запроса будет выполнена проверка наличия данных в кэше и в случае если проверка уже выполнялась, результаты будет получены из регистра сведений;
  • Информация в кэше будет храниться 4 месяца, т.к. данные состояний проверки контрагентов используются при формировании квартальной отчетности;
  • Информация о состояниях загружается из ЕГРН один раз в 6 дней, поэтому получить окончательное состояние можно только через 7 дней, поэтому повторную проверку состояния выполняет регламентное задание.
