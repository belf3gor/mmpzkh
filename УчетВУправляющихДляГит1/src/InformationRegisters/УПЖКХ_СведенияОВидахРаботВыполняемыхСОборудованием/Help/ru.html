Регистр сведений «Сведения о видах работ, выполняемых с оборудованием»

Данный регистр предназначен для хранения сведений о видах работ, которые могут выполняться с оборудованием. Регистр непериодический, независимый.

В регистре используются измерения:

  • «Вид оборудования» - является ссылкой на элемент справочника «Прочие объекты учета, оборудование» - оборудование, с которым может выполняться указанный вид работ;
  • «Вид работ» - является ссылкой на элемент справочника «Виды работ» - выполняемый с оборудованием вид работ.

В регистре используются ресурсы: 

  • «Плановый» - признак, определяющий, выполняется ли данный вид работ по плану.
  • «Периодичность» - периодичность выполнения вида работ по плану, может принимать значения «День», «Месяц», «Год».
  • «Количество периодов» - количество периодов с заданной периодичностью, определяющих частоту выполнения работ по плану.
