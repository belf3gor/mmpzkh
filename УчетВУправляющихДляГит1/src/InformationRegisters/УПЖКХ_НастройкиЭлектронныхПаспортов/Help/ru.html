Регистр сведений «Настройки электронных паспортов»

Данный регистр предназначен для настройки формирования электронных паспортов зданий. Регистр непериодический и независимый.

Настройки задаются для каждой организации отдельно. Для этого в поле «Организация» необходимо выбрать организацию из справочника «Организации»;

 

На закладке «Общие настройки» задаются основные настройки:

  • «Каталог сохранения паспортов» – выбирается каталог, в который будут сохраняться сформированные электронные паспорта (XML-файлы).
  • «Отправлять паспорта на свой сайт» – при установке флага станет доступной возможность отправлять сформированные электронные паспорта на свой сайт.
  • «Настройка доступа к сайту» – выбирается настройка обмена данными с сайтом из справочника «Настройки доступа к сайту». Настройка доступна только при установленном флаге «Отправлять паспорта на свой сайт».
  • «Подписывать и отправлять паспорта в орган местного самоуправления» – при установке флага станет доступной возможность отправлять сформированные электронные паспорта в орган местного самоуправления. Для того, чтобы воспользоваться этой возможностью необходимо в программе настроить электронный документооборот.
  • «Настройка обмена с органом местного самоуправления» – выбирается настройка обмена данными с контрагентом из справочника «Настройки ЭДО». Настройка доступна только при установленном флаге «Подписывать и отправлять паспорта в орган местного самоуправления».
  • «Сопроводительная записка к письму» – указывается текст, который будет отправлен в орган местного самоуправления вместе с электронным паспортом. Настройка доступна только при установленном флаге «Подписывать и отправлять паспорта в орган местного самоуправления».

 

На закладке «Услуги содержания и ремонта» хранится список элементов справочника «Виды услуг», которые относятся к услугам содержания и ремонта имущества собственников помещений.

На закладке «Коммунальные ресурсы» хранится соответствие между предопределенными коммунальными ресурсами и элементами справочника «Виды услуг».

На закладке «Коммунальные услуги» хранится соответствие между предопределенными коммунальными услугами и элементами справочника «Виды услуг».

На закладке «Услуги по управлению» хранится список элементов справочника «Виды услуг», которые относятся к услугам управления зданием.
