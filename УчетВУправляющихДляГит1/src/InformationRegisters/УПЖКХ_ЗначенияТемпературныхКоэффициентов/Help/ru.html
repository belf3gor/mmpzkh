Регистр сведений «Значения температурных коэффициентов»

Данный регистр предназначен для хранения температурных коэффициентов, значения которых будут учитываться при начислении услуг по показаниям приборов учета без температурной компенсации, установленными вне помещений.

В регистре указываются поля:

  • «Участок» - содержит ссылку на справочник "Участки". Выбирается участок или группа участков, к которым относятся здания, где установлены индивидуальные или домовые приборы учета без температурной компенсации.
  • «Вид услуги» - содержит ссылку на справочник "Виды услуг". Выбирается вид услуги, для которого задаются температурные коэффициенты (должен совпадать с видом услуги приборов учета без температурной компенсации).
  • «Коэффициент» - размер температурного коэффициента.

Регистр независимый, имеет периодичность в пределах месяца.
