Регистр сведений «Значения повышающих коэффициентов для начисления по нормативу»

Данный регистр предназначен для хранения значений повышающих коэффициентов, которые используются при начислении услуг по нормативу потребления.

Регистр имеет периодичность в пределах месяца.

При этом:

  • если в здании присутствует возможность установки прибора учета услуги, но в помещении прибор учета не установлен, то начисление услуги по нормативу потребления производится с учетом повышающего коэффициента, заданного для услуги;
  • если в здании отсутствует возможность установки прибора учета услуги или в помещении установлен прибор учета, то повышающий коэффициент при начислении услуги не используется.

В регистре используются измерения:

  • «Услуга» - услуга, для которой устанавливаются значение повышающего коэффициента - является ссылкой на элемент справочника «Услуги».

В регистре используются ресурсы:

  • «Повышающий коэффициент» - числовое значение повышающего коэффициента по заданной услуге.

Примечание
Если для услуги значение повышающего коэффициента не задано, то начисление производится по нормативу потребления с коэффициентом, равным "1".
