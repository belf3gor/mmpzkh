Регистр сведений «Счета учета расчетов по пеням»

Данный регистр предназначен для ручной настройки счетов учета для отражения пеней в регламентированном учете. Регистр не периодичен.

В регистре используются измерения:

  • «Организация» - выбирается организация (ссылка на элемент справочника «Организации»), по которой производятся начисления пени;
  • «Вид взаиморасчетов» - выбирается вид взаиморасчетов договоров контрагентов (ссылка на элемент справочника «Виды взаиморасчетов»).

В регистре используются ресурсы (ссылки на элемент планов счетов «Хозрасчетный»):

  • «Счет учета расчетов»;
  • «Счет учета авансов»;
  • «Счет учета доходов»;
  • «Счет учета расходов».

При этом для счета учета доходов могут быть указаны виды субконто (не обязательны для заполнения).

Для открытия регистра можно воспользоваться гиперссылкой "Счета учета расчетов по пеням" или "Счета учета расчетов по пеням за услуги капитального ремонта", если ведется раздельный учет услуг капитального ремонта, на закладке "Регламентированный учет" в «Учетной политике (ЖКХ)».
