Регистр сведений "Первоначальные сведения ОС (бухгалтерский учет)"

Содержит первоначальные сведения об основных средствах организаций  для целей бухгалтерского учета:

  • Инвентарный номер;
  • Способ поступления;
  • Способ начисления амортизации;
  • Первоначальная стоимость;
  • Параметр выработки;
  • Признак списания на затраты при принятии к учету.

Данные из регистра используются для расчета амортизации основных средств. 

По данным регистра определяется организация, в которой учитывается основное средство, в определенный момент времени.

Регистр периодический, изменяется документами:

  • "Ввод начальных остатков";
  • "Принятие к учету ОС".
