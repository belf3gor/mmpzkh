Регистр сведений «Сведения о возможности установки приборов учета в зданиях и помещениях»

Данный регистр предназначен для хранения сведений о возможности установки коллективных (общедомовых), индивидуальных или общих (квартирных) приборов учета коммунальных услуг в зданиях и помещениях. Можно указать возможность установки приборов учета для здания (сразу для всех его помещений) и отдельно для каждого помещения этого здания, при этом для помещения будет использоваться:

  • настройка для здания, если есть сведения только по зданию;
  • настройка для помещения, если есть сведения только по помещению или как по зданию, так и по помещению.

Сведения о возможности установки приборов учета используются при начислении услуг с учетом повышающих коэффициентов, при этом:

  • если в помещении присутствует возможность установки прибора учета услуги, но прибор учета не установлен, то начисление услуги по нормативу потребления производится с учетом повышающего коэффициента, заданного для услуги;
  • если в помещении отсутствует возможность установки прибора учета услуги или в помещении установлен прибор учета, то повышающий коэффициент при начислении услуги не используется.

Примечание
Если сведения о возможности установки приборов учета не заданы ни для помещения, ни для здания, то считается, что возможность установки приборов учета в данном помещении отсутствует, а повышающий коэффициент при расчете начислений не учитывается.

Для указания возможности установки приборов учета сразу для всех помещений в здании нужно измерение регистра «Помещение» оставить не заполненным.

На командной панели списка сведений расположена кнопка «Групповое заполнение сведений», при нажатии на которую открывается обработка «Заполнение сведений о возможности установки приборов учёта», предназначенная для группового заполнения сведений о возможности установки приборов учёта в зданиях и помещениях.

В регистре используются измерения:

  • «Здание» - здание, для которого устанавливаются сведения о возможности установки приборов учета - является ссылкой на элемент справочника «Здания, сооружения»;
  • «Помещение» - помещение, для которого устанавливаются сведения о возможности установки приборов учета - является ссылкой на элемент справочника «Помещения»;
  • «Услуга» - услуга, рассчитываемая по показаниям приборов учета, по которой устанавливаются сведения о возможности установки приборов учета - является ссылкой на элемент справочника «Услуги».

В регистре используются ресурсы:

  • «Техническая возможность присутствует» - признак, определяющий возможность установки приборов учета в указанном здании или помещении здания по заданной услуге.
