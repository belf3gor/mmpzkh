Регистр сведений "Состояние нематериального актива организации (бух. учет)"

Данный регистр является хранилищем состояния в организации объектов НМА и положительных результатов НИОКР.

Состояния являются для каждого объекта взаимоисключающимися, то есть в один момент времени каждый объект может находиться в одной организации в одном состоянии.

Эти состояния общие для бухгалтерского и налогового учета (если объект принят к налоговому учету).

Состояние устанавливается документами "Поступление НМА организаций", "Принятие к учету НМА организаций", "Передача НМА", "Списание НМА".
