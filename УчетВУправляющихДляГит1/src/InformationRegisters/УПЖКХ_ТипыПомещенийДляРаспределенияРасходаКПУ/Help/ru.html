Регистр сведений «Типы помещений для распределения расхода коллективных приборов учета»

Данный регистр предназначен для хранения типов помещений, между площадями которых будет выполняться распределение расхода коллективных приборов учета в случае их установки на несколько объектов учета.

В регистре указываются поля:

  • «Тип помещения» - содержит ссылку тип помещения. Выбираются только типы помещений 1-го уровня.

Регистр независимый, непериодический.

Для открытия регистра можно воспользоваться ссылкой "Список типов помещений для распределения показаний" на закладке "Приборы учета" / "Распределение показаний" в "Настройках параметров учета (ЖКХ)", доступной при выбранном способе распределения показаний прибора учета, установленного на несколько подъездов / зданий "Пропорционально площадям помещений".
