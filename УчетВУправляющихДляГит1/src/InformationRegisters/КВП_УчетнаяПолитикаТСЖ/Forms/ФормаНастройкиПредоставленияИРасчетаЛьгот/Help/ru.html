Настройки предоставления и расчета льгот

Форма предназначена для редактирования настроек, используемых в механизме расчета льгот. Настройки выполняются в разрезе групп услуг.

Настройки устанавливаются в строках таблицы, где необходимо заполнить следующие поля:

  • «Группы услуг» - указывается группа услуг, для которой задаются настройки предоставления и расчета льгот.

    Указанные для группы услуг настройки будут применяться ко всем услугам, входящим в группу.

    Поле «Группа услуг» может быть не заполнено. В этом случае настройки будут применяться для всех остальных групп услуг, для которых настройки явно не заданы, и для услуг, не входящих ни в одну из определенных групп.

  • «Льготы предоставлять» - тип жильцов (проживающие, зарегистрированные или собственники), которым предоставляются льготы на услуги текущей группы услуг.
  • «Льготный объем рассчитывать, исходя из» - способ распределения объема начисленной услуги между получателями льгот в рамках одного лицевого счета. Объем услуги, на который предоставляется льгота, в общем случае рассчитывается по следующей формуле: [Объем потребления услуги] * [Количество получателей льготы] / [Количество жильцов], где [Количество жильцов] рассчитывается, исходя из значения, определенного текущей настройкой. Настройка может принимать следующие значения:
    • делить на количество всех жильцов;
    • делить на количество всех жильцов, пользующихся льготой;
    • делить на количество льготников, получающих льготу по услуге.
  • «Размер ограничения услуги рассчитывать, исходя из» - способ распределения объема ограничения услуги между получателями льгот в рамках одного лицевого счета. Размер ограничения, который используется при начислении льготы, в общем случае рассчитывается по следующей формуле: [Размер ограничения] * [Количество получателей льготы] / [Количество жильцов], где [Количество жильцов] рассчитывается, исходя из значения, определенного текущей настройкой. Настройка может принимать следующие значения:
    • делить на количество всех жильцов;
    • делить на количество всех жильцов, пользующихся льготой;
    • делить на количество жильцов, пользующихся льготой текущего льготника.
Примечание

Настройки для расчета льгот по каждой услуге определяются в следующем порядке приоритетности:

  1. В первую очередь используется настройка, установленная для группы услуг, в которую входит текущая услуга.
  2. Во вторую очередь используется настройка, установленная для группы "Все группы".
  3. Если настройка не определена в соответствии с п.1 и п.2, то используются значения настроек по умолчанию.
