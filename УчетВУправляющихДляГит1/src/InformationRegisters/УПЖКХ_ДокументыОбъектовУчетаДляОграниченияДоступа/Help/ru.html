Регистр сведений «Документы объектов учета для ограничения доступа»

Регистр предназначен для хранения информации об отношении документов подсистемы ЖКХ к объектам учета, которая используется для ограничения прав доступа пользователей.

Регистр содержит следующие поля:

  • «Документ» - документ подсистемы ЖКХ;
  • «Объект учета» - объект учета, к которому относится документ.
