Регистр сведений «Услуги-исключения для начисления пени»

Данный регистр предназначен для хранения услуг, по которым не нужно производить начислений пени.

Регистр содержит следующие ресурсы:

  • «Организация» - указывается организация, для которой нужно ввести услугу-исключение;
  • «Услуга» - указывается услуга, начисление пени по которой не будет производиться. 

Регистр независимый, не периодический.
