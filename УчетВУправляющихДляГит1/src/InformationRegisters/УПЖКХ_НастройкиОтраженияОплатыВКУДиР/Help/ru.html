Регистр сведений «Настройки отражения оплаты в КУДиР»

Регистр предназначен для хранения услуг, по которым необходимо производить оплаты в КУДиР.

Регистр содержит следующие поля:

  • «Организация» - указывается организация, по которой необходимо отражать оплаты;
  • «Услуга» - указывается услуга, по которой необходимо отражать оплаты;
  • «Доходы всего» - указывается метод учета доходов: включать в КУДиР или обнулять;
  • «в т.ч. принимаемые» - указывается метод учета принимаемых доходов: включать в КУДиР или обнулять.

При отражении оплат в "Книге учета доходов и расходов" документом "Регистрация оплаты" настройки учитываются по следующим приоритетам:

  1. Настройки, указанные для конкретной услуги;
  2. Настройки, указанные для группы услуг;
  3. Настройки, у которых поле «Услуга» не заполнено.

Регистр независимый, не периодический.
