Регистр сведений «Ключевая ставка ЦБ»

Данный регистр предназначен для хранения для хранения ключевой ставки ЦБ РФ. Ключевая ставка ЦБ используется при расчете пени в качестве ставки рефинансирования ЦБ, так как по новым правилам, действующим с 1 января 2016 года, доля ставки рефинансирования ЦБ равна ключевой ставке ЦБ.

Регистр доступен в разделе командного интерфейса "Начисления услуг" - "См. также".

В форме списка записей регистра можно настроить автоматическое обновление ключевой ставки ЦБ по заданному расписанию. Для обновления ключевой ставки с сайта ЦБРФ требуется подключение к сети интернет.

Техническая информация:

Автоматическое обновление ключевой ставки ЦБ производится с веб-сервиса http://www.cbr.ru/DailyInfoWebServ/DailyInfo.asmx?WSDL сайта http://www.cbr.ru/.

Подробное описание веб-сервиса, его методов и параметров доступно по ссылке: http://www.cbr.ru/scripts/Root.asp?Prtid=DWS.
