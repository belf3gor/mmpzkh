Справочник «Виды услуг»

Данный справочник предназначен для хранения списка видов услуг.

С его помощью можно обобщить введенные услуги в группы для отображения в отчетах.

Например, если у нас есть услуги найма, тариф которых зависит от типа здания, то в справочнике услуг необходимо завести несколько элементов, каждый со своим тарифом: «Найм (кирпичный)», «Найм (панельный)», «Найм (деревянный)». Если же в отчетах необходимо эти услуги обобщить, не разделяя по тарифам, то во всех трех услугах нужно указать вид услуги «Найм», и три элемента справочника услуг в отчетах сгруппируются в одну услугу «Найм».

Ввод нового элемента справочника и редактирование уже существующего элемента справочника производится в форме элемента. При вводе нового элемента необходимо заполнить реквизиты:

  • «Наименование» - наименование вида услуг;
  • «Краткое наименование» - краткое наименование, которое используется при автоматическом формировании имен счетчиков по данному виду услуг;
  • «Единица измерения» – единица измерения, в которой измеряется данная услуга;
  • «Код» - код элемента, который устанавливается автоматически и, при необходимости, может быть отредактирован вручную.
  • «Способ вывода в квитанциях» - Настройка позволяет организовать свёртку услуг, принадлежащих одному виду услуги, в одну строку в квитанциях и платежных документах. При этом:
    • «Не сворачивать» – услуги будут выводиться отдельными строками;
    • «Тариф сворачивать, количество суммировать» – услуги будут свёрнуты в одну строку. При этом количество потребления будет просуммировано, а тариф – не изменится. Данный способ свёртки подойдёт для услуг с одинаковым тарифом (например, для коммунальных услуг);
    • «Количество сворачивать, тариф суммировать» – услуги будут свёрнуты в одну строку. При этом тариф будет просуммирован, а количество потребления – не изменится. Данный способ свёртки подойдёт для услуг с одинаковым объемом потребления (например, для услуг, которые рассчитываются по площади или по количеству жильцов).

Сведения для ГИС ЖКХ:

  • Реестровый номер справочника - код справочника, в котором хранится услуга ГИС ЖКХ.
  • Реестровый номер позиции - строковый код услуги.
  • Реестровый номер позиции услуги для СОИ: если вид услуги относится к одному из коммунальных ресурсов "Холодная вода", "Горячая вода", "Электрическая энергия", "Сточные воды", то в данном поле указывается номер позиции услуги для СОИ (потребление при содержании общего имущества, реестровый номер справочника "2"). При выгрузке информации о платежных документов в файл данный вид услуги будет разделен на 2 строки: индивидуальное начисление и потребление при содержании общего имущества (в соответствии с Постановлением Правительства РФ №1498 от 26.12.2016).

Блок полей «Реестровые номера позиций тарифов (для услуг по счетчикам)»:

Для многотарифных услуг (услуг по приборам учета) в ГИС ЖКХ необходимо создавать дополнительные услуги, подчиненные основной услуге, по одной на каждый из видов тарифов. Т.е. для отражения разных видов тарифов (например, для трехтарифной услуги "Электроснабжение") в ГИС ЖКХ необходимо создать 3 услуги, подчиненные основной услуге, добавив в наименование для удобства суффикс, определяющий вид тарифа ("Эл энергия по тарифу Дневной", "Эл энергия по тарифу Ночной", "Эл энергия по тарифу Пиковый"). Для связи основной услуги с подчиненными услугами в ГИС ЖКХ необходимо заполнить соответствующие поля реестровых номеров позиций тарифов:

    • Реестровый номер позиции для дневного вида тарифа (в нашем примере это будет реестровый номер позиции услуги "Эл энергия по тарифу Дневной" в ГИС ЖКХ);
    • Реестровый номер позиции для ночного вида тарифа (в нашем примере это будет реестровый номер позиции услуги "Эл энергия по тарифу Ночной" в ГИС ЖКХ);
    • Реестровый номер позиции для пикового вида тарифа (в нашем примере это будет реестровый номер позиции услуги "Эл энергия по тарифу Пиковый" в ГИС ЖКХ).

Данные номера могут быть получены в файле шаблона платежного документа, выгруженного из ГИС ЖКХ, на листе «Услуги исполнителя».

Справочник доступен в командном интерфейсе в разделе «Начисления услуг» - «См. также» - «Виды услуг».
