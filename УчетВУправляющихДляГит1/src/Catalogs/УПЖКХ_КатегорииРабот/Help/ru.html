Справочник "Категории работ"

Данный справочник используется для хранения категорий работ, которые могут осуществляться по заявкам на ремонт жилого и нежилого фонда.

При вводе нового элемента справочника указываются:

  • «Код»: код элемента справочника. Устанавливается автоматически. Может быть изменен вручную;
  • «Наименование»: указывается наименование категории работ.
