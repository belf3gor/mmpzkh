Хозяйствующие субъекты ВетИС

Справочник предназначен для хранения сведений об участниках хозяйственной деятельности (юридическое лицо, физическое лицо, индивидуальный предприниматель), зарегистрированных в системе ВетИС, в частности производителях и продавцах продукции, подлежащей ветеринарному контролю, участвующих в информационном обмене с ВетИС. Справочник ведется на федеральном уровне. Новые элементы в справочник могут добавляться следующим способом:

  • Загружаться автоматически при поступлении документов из ВетИС
  • Загружаться интерактивно на закладке Поиск в онлайн-классификаторе.

В справочник вносятся собственные организации и контрагенты (производители, поставщики). В зависимости от значения переключателя В информационной базе соответствует каждый элемент справочника сопоставляется с элементом справочника контрагентов или с организациями предприятия.

Для всех хозяйствующих субъектов указывается список связанных с ним предприятий – мест осуществления деятельности, где вырабатывается, хранится или реализуется продукция (ферма, завод, магазин). С одним хозяйствующим субъектом может быть связано несколько предприятий.

Для хозяйствующих субъектов, соответствующих собственной организации, в списке можно создавать новые предприятия, устанавливать и разрывать связь с существующими предприятиями.

Для хозяйствующих субъектов с настроенным подключением к Ветис.API, указывается список пользователей информационной базы, уполномоченных работать с ВитИС. Для каждого пользователя указывается учетная запись ВетИС, набор прав и зона ответственности.

Из списка пользователей можно выполнить следующие действия:

  • связать пользователя ВетИС с пользователем информационной базы,
  • создать нового пользователя ВетИС,
  • изменить права и зону ответственности пользователя ВетИС,
  • удалить связь пользователя ВетИС с этим хозяйствующим субъектом.

Также возможно выполнить синхронизацию связанных с этим хозяйствующим субъектом пользователей ВетИС и их прав с данными системы Ветис.Паспорт.

Для хозяйствующего субъекта, который соответствует собственной организации, выполнение перечисленных действий доступно пользователю информационной базы, который сопоставлен с пользователем ВетИС с ролью «Администратор» этого хозяйствующего субъекта в системе Ветис.Паспорт.

Элемент справочника хозяйствующих субъектов сопоставляется с элементом справочника организаций или справочника контрагентов. При сопоставлении из документов полученных из ВетИС производится поиск по ИНН / КПП / GLN.
