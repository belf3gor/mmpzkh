Справочник «Услуги»

Данный справочник предназначен для хранения списка жилищно–коммунальных и прочих услуг, оказываемых организацией. Ввод нового и редактирование уже записанного элемента справочника осуществляется через форму элемента.

Форма элемента справочника состоит из закладок: «Основная», «Тарифы», «Группы услуг»:

  1. На закладке «Основная» указываются:
    • «Код» – код элемента справочника. Устанавливается автоматически;
    • «Наименование» – указывается наименование услуги;
    • «Вид услуги» – указывается один из видов услуг, к которому принадлежит данная услуга (см. справочник «Виды услуг»);
    • «Коммунальный ресурс» – для коммунальных услуг указывается коммунальный ресурс;
    • «Способ расчета» – из списка выбирается способ расчета стоимости коммунальных услуг;
    • «Вариант начисления услуги» – указывается вариант начисления услуги.
  2. На закладке «Тарифы» в таблице отображены актуальные данные о стоимости услуги: «Вид тарифа» и размер «Тарифа».
  3. На закладке «Группы услуг» можно установить принадлежность услуги какой-либо группе услуг. Для этого отображаются названия всех видов групп услуг и краткое описание, для чего данный вид групп используется. В поле, напротив названия каждого вида групп можно указать, к какой группе услуг данного вида следует отнести услугу. Одна услуга может быть отнесена только к одной группе услуг внутри каждого вида групп услуг.

    Способы расчета

    Расчет «По показаниям счетчика»

      Данный расчет необходим для начисления по показаниям приборов учета (далее - ПУ):

      • начисление по индивидуальным ПУ,
      • распределение показаний по коллективным ПУ.

     Настройка формул расчета по коллективным приборам учета (далее - КПУ) выполняется на закладке «Формулы расчета»:

     Описание формул расчета см. на странице «Формулы расчета» или на сайте.

      Настройка начислений по индивидуальным приборам учета (далее - ИПУ) выполняется на закладке «Настройки начисления по ИПУ»:

      • «Расчет начислений при отсутствии действующего счетчика» - может принимать следующие значения:
        • «Не производить» - расчет начислений производиться не будет.
        • «Производить по норме потребления» - расчет начислений будет производиться по норме потребления услуги, указанной в справочнике «Лицевые счета».

        Повышающие коэффициенты:
        При начислении услуги по нормативу потребления в случае, когда на лицевом счете отсутствует действующий прибор учета, а техническая возможность установки приборов учета в здании (помещении здания) присутствует, можно учитывать повышающие коэффициенты, согласно Постановлению №344 Правительства РФ. Для этого необходимо задать размеры повышающего коэффициента по услуге, а также ввести сведения о возможности установки приборов учета данной услуги по зданиям или по помещениям здания, ввод которых доступен через панель навигации форм здания и помещения.

        • «Производить по среднему значению» - расчет начислений будет производиться по среднемесячному объему потребления коммунального ресурса, определенному по показаниям индивидуального или общего (квартирного) прибора учета за указанное в поле «за период» количество месяцев. При этом если период работы прибора учета составил меньше минимального указанного количества месяцев или средний расход оказался отрицательным, то расчет будет производиться по нормативу потребления. В случае, если минимальное количество месяцев не указано (значение = 0), то ограничение на период работы прибора учета для расчета среднего использоваться не будет.
          Если на одну услугу установлено несколько приборов учета, то тогда имеется возможность выбрать, в каком случае будет считаться, что действующий счетчик отсутствует. Возможные варианты:
          • Отключен один из приборов учета.
          • Отключены все приборы учета.
      • «Расчет начислений при отсутствии показаний» - может принимать следующие значения:
        • «Не производить» - расчет начислений производиться не будет.
        • «Производить по норме потребления» - расчет начислений будет производиться по норме потребления услуги, указанной в справочнике «Лицевые счета».
        • «Производить по среднему значению» - расчет начислений будет производиться по среднемесячному объему потребления коммунального ресурса, определенному по показаниям индивидуального или общего (квартирного) прибора учета за указанное в поле «за период» количество месяцев. При этом если период работы прибора учета составил меньше минимального указанного количества месяцев или средний расход оказался отрицательным, то расчет будет производиться по нормативу потребления (в соответствии с п.59 постановления Правительства РФ № 354, в ред. постановления Правительства РФ № 344). В случае, если минимальное количество месяцев не указано (значение = 0), то ограничение на период работы прибора учета для расчета среднего использоваться не будет.

          Кроме этого, имеется возможность выбрать продолжительность начисления по среднему значению при отсутствии показаний:

          • «В течение указанного периода» - в течение указанного количества месяцев расчет будет производиться по среднемесячному объему потребления. Если данные для расчета среднемесячного объема потребления за прошедший период отсутствуют, то начисление производиться не будет. По истечении данного периода расчет будет производиться по нормам потребления (в соответствии с п.60 постановления Правительства РФ № 354, в ред. постановления Правительства РФ № 344). Если период работы прибора учета составляет меньше минимального указанного количества месяцев, расчет по среднему производится с первого месяца действия прибора учета, но так как данные для расчета по среднему еще отсутствуют, то расчет производится по нормативу.
          • «При наличии данных» - если данные для расчета среднемесячного объема за прошедший период отсутствуют, начисление всегда будет производиться по нормам потребления.

          Пример:

          Расчет начислений при отсутствии показаний производится по среднему значению за период 6 месяцев. Если время работы счетчика меньше 6 месяцев, то за фактическое время, но не менее 3 месяцев.
          Начисления по среднему производятся в течение 3 месяцев.

          Таким образом, если после установки прибора учета показания ни разу не передавались, то расчет по среднему при отсутствии показаний производится с первого месяца действия прибора учета. Данный расчет не противоречит постановлению Правительства РФ №354, но при этом позволяет избежать ситуации, когда начисления будут равны 0. То есть, если бы в данном примере расчет по среднему производился с мая (месяца превышающего установленный в настройках услуги минимальный период работы прибора учета), то в мае, июне и июле начисления были бы равны 0, так как нет данных для расчета по среднему.

      Если на одну услугу установлено несколько приборов учета, то тогда имеется возможность выбрать, в каком случае будет считаться, что показания отсутствуют. Возможные варианты:
      • Не введены показания хотя бы по одному прибору учета.
      • Не введены показания по всем действующим приборам учета.

      Настройки распределения показаний коллективных приборов учета (далее - КПУ) выполняются на закладке «Настройки начисления по КПУ».

      Распределение показаний коллективных приборов учета осуществляется по следующему алгоритму:

      1. Определяется суммарный индивидуальный расход.

      • «В суммарный индивидуальный расход включать» - настройка определяет, какие начисления входят в состав суммарного индивидуального расхода, который вычитается из расхода КПУ при расчете ОДН. В суммарном индивидуальном расходе помимо начислений по показаниям, норме потребления и среднему расходу также учитываются объемы сторно при условии, что начисление при отсутствии показаний ИПУ участвовало в расчете ОДН в соответствии с данной настройкой.
        Настройка позволяется выбрать один из следующих вариантов определения индивидуального расхода:
        • «Только расход по приборам учета» - в суммарный индивидуальный расход входит только расход по индивидуальным приборам учета;
        • «Расход по приборам учета и начисления по нормативу» - в суммарный индивидуальный расход включаются расход по индивидуальным приборам учета и начисления по нормативу;
        • «Расход по приборам учета, начисления по нормативу и по среднему» - в суммарный индивидуальный расход включаются расход по индивидуальным приборам учета, начисления по нормативу и начисления по среднему расходу.

          Примечание:
          При расчете ОДН по услугам общедомового счетчика к суммарному индивидуальному расходу прибавляется индивидуальный расход услуг-баз, которые начисляются в отдельных документах начисления услуг.

      2. Определяется расход по коллективному прибору учета.

        Объем общедомовых нужд (далее - ОДН) – это определенный исходя из показаний коллективного (общедомового) прибора учета объем коммунального ресурса, подлежащий к распределению между потребителями соответствующего вида коммунальной услуги.


      3. Разница между расходом коллективного прибора учета и суммарным индивидуальным расходом распределяется на лицевые счета.

      • «Вариант начисления» - определяет по какому правилу и между какими лицевыми счетами будет выполняться распределение разницы между расходом коллективного прибора учета и суммарным индивидуальным расходом. При выборе из списка конкретного постановления на форме будет представлено описание в каких случаях будет применяться та или иная формула выбранного постановления. Если ни одна из формул постановления не подходит, то распределение будет выполняться по ручным настройкам пользователя для данной услуги.

        Возможные варианты:

        • «по формулам постановления Правительства РФ №307 от 23 мая 2006 г.»
        • «по формулам постановления Правительства РФ №354 от 06 мая 2011 г.»
        • «по формулам постановления Правительства РФ №344 от 16 апреля 2013 г.»
        • «по формулам постановления Правительства РФ №1708 от 28 декабря 2018 г.»
        • «ручная настройка» - если выбрана ручная настройка, то на форме будет отображаться список формул и для расчета по определенной формуле нужно ее отметить флажком. При этом нельзя в одной услуге одновременно выбрать противоречащие друг другу формулы из разных постановлений.

         

      • «Вариант распределения расхода по тарифам» - данная настройка определяет, каким образом необходимо распределять разницу между расходом коллективного прибора учета и суммарным индивидуальным расходом по тарифам и может принимать следующие значения:
        • «Суммарный расход» - на лицевые счета распределяется суммарный расход по всем видам тарифа коллективного прибора учета;

При данном варианте распределения объем расхода для распределения определяется как разница суммарного расхода коллективного прибора учета и суммарного индивидуального расхода, где суммарный расход - сумма расхода по всем видам тарифа прибора учета в соответствии с его тарифностью.

Пример:

Расход коллективного прибора учета составил 250 кВт.ч и 100 кВт.ч дневного и ночного вида тарифов соответственно.
Расход индивидуальных приборов учета составил 200 кВт.ч и 70 кВт.ч дневного и ночного вида тарифов соответственно.
Тогда суммарный расход для распределения на лицевые счета будет определен следующим образом:

(250 кВт.ч + 100 кВт.ч) - (200 кВт.ч + 70 кВт.ч) = 80 кВт.ч.

Далее, полученный суммарный расход распределяется на лицевые счета в соответствии с настройками распределения расхода на лицевые счета.

        • «По виду тарифа» - распределение на лицевые счета происходит по каждому виду тарифа коллективного прибора учета отдельно. Данный вариант распределения будет использоваться только в случаях, когда тарифности коллективного прибора учета и индивидуальных приборов учета совпадают, иначе будет выполнено распределение по суммарному расходу.

При данном варианте распределения объем расхода для распределения определяется как разница расхода коллективного прибора учета и суммарного индивидуального расхода по каждому виду тарифа отдельно. Распределение полученных разниц происходит строго по видам тарифа.

Пример:

Расход коллективного прибора учета составил 250 кВт.ч и 100 кВт.ч дневного и ночного вида тарифов соответственно.
Расход индивидуальных приборов учета составил 200 кВт.ч и 70 кВт.ч дневного и ночного вида тарифов соответственно.
Тогда определение разниц для распределения на лицевые счета будет выполнено следующим образом:

250 кВт.ч - 200 кВт.ч = 50 кВт.ч по дневному виду тарифа;

100 кВт.ч - 70 кВт.ч = 30 кВт.ч по ночному виду тарифа.

Далее, распределение расхода производится отдельно по каждому виду тарифа на лицевые счета в соответствии с настройками распределения расхода на лицевые счета.

          • «При отсутствии начислений по ИПУ начисление производить» - данная настройка определяет, по каким тарифам производить начисление по показаниям коллективных приборов учета, если показания индивидуального прибора учета отсутствуют или отсутствует прибор учета. Имеет следующие варианты:
              • «По общему тарифу» - начисление будет производиться по общему тарифу;
              • «По тарифам, соответствующим тарифности коллективного ПУ» - начисление будет производиться по тарифам, соответствующим тарифам коллективного прибора учета. ВАЖНО: Это справедливо лишь в случае совпадения тарифности коллективного прибора учета и индивидуальных приборов учета. В противном случае поведение аналогично варианту «По общему тарифу».

Пример:

Имеется двухтарифный коллективный прибор учета. У одного из лицевых счетов в периоде начисления отсутствуют показания индивидуального прибора учета, тогда:

              • Если установлена настройка «По общему тарифу», то сумма начисления по показаниям коллективного прибора учета (при распределении) будет рассчитана по общему тарифу;

        • «Распределять на лицевые счета» - данная настройка определяет между какими лицевыми счетами будет выполнено распределение. Имеет следующие варианты: 
          • «С индивидуальными приборами учета» - распределение будет производиться только на лицевые счета, у которых установлены индивидуальные приборы учета;
          • «Без индивидуальных приборов учета» - распределение будет производиться только на лицевые счета, у которых нет действующих приборов учета;
          • «Все» - будут учитываться все лицевые счета.

            • «При отсутствии показаний считать, что счетчик не установлен» - данная настройка становится доступной, если выбран вариант распределения на лицевые счета «С индивидуальными приборами учета» или «Без индивидуальных приборов учета».

    Если установить данную настройку, то при отсутствии по лицевому счету показаний индивидуального прибора учета в месяце расчета и выбранном варианте распределения на лицевые счета «С индивидуальными приборами учета» распределение ОДН на данный лицевой счет выполняться не будет. Аналогично, если индивидуальный прибор учета установлен, но в месяце расчета отсутствуют показания и выбран вариант распределения на лицевые счета «Без индивидуальных приборов учета»,  то на данный лицевой счет будет выполнено распределение ОДН.

          • «Распределять пропорционально» - данная настройка определяет правило, по которому будет выполнено распределение. Доступны следующие варианты распределения:
            • «Площади» - распределение производится по площади помещений. Вид площади можно выбрать в соответствующем поле;
            • «Количеству помещений в квартире» - распределение будет производиться по количеству комнат в помещении;
            • «Количеству расхода по услуге» - распределение будет производиться по количеству услуги, потребленной в месяце расчета;
            • «Количеству жильцов» - распределение будет производиться по количеству жильцов. Вид жильцов можно выбрать в соответствующем поле;
            • «Поровну» - распределение будет производиться поровну между всеми лицевыми счетами.

        • «Разрешить распределение отрицательного расхода ОДН». Объем ОДН может быть как положительным, так и отрицательным (в случае, например, если фактическое потребление в помещениях, не оборудованных индивидуальными приборами учета, меньше норматива).

    Если флажок «Разрешить распределение отрицательного расхода ОДН» не установлен, между потребителями услуги распределяется только положительный объем ОДН.

    Если флажок установлен, распределяется также и отрицательный объем ОДН.

    Положительный объем ОДН увеличивает начисление, отрицательный объем, наоборот, уменьшает начисление по услуге.

    В случае если для какого-либо потребителя приходящаяся на него в результате распределения доля ОДН превышает индивидуальный объем коммунального ресурса, то размер начисления за соответствующий вид коммунальной услуги по данному потребителю будет равен нулю. Излишек коммунального ресурса на следующий расчетный период не переносится и при расчете размера платы в следующем расчетном периоде не учитывается.

          • «Распределение отрицательного расхода ОДН будет производится пропорционально» - данная настройка определяет правило, по которому будет выполнено распределение отрицательного объема ОДН. Имеет следующие варианты:
            • «Площади» - распределение производится по площади помещений. Вид площади можно выбрать в соответствующем поле;
            • «Количеству жильцов» - распределение будет производиться по количеству жильцов. Вид жильцов можно выбрать в соответствующем поле.

      Если установлен флажок «Распределять только на жилые помещения», то объем образовавшейся экономии будет распределяться только между жилыми помещениями (в соответствии с п.47 постановления №354 от 06 мая 2011 г.). Если флажок не установлен, то распределение отрицательного объема ОДН будет выполняться без учета типа фонда помещения жилое/нежилое. К жилым относятся помещения с типами "Квартира" (для многоквартирных домов) и "Дом" (для немногоквартирных домов).

        • «Использовать формулу №9 постановления №307» - расчет производится по формуле №9 согласно постановлению Правительства РФ №307 от 23 мая 2006г. При установке данного флажка остальные настройки устанавливаются по умолчанию и становятся недоступными для изменения.
        • «Использовать формулы №3 и №14 постановления №354 при наличии коллективного прибора учета и отсутствии индивидуальных».

    Данный флажок позволяет настроить расчет начислений за потребление коммунального ресурса для многоквартирного дома, который оборудован коллективным прибором учета и в котором во всех жилых и нежилых помещениях отсутствуют индивидуальные приборы учета:

          • Если флажок установлен, то индивидуальное потребление коммунального ресурса и доля ОДН в жилом или нежилом помещении будут рассчитываться по формулам №3 и №14 постановления №354 от 06 мая 2011г.
          • Если флажок не установлен, то описанная ситуация будет отрабатываться обычным образом.
        • «Использовать формулу №15 постановления №344 при отсутствии коллективного прибора учета».

    Данный флажок позволяет настроить расчет начислений за потребление коммунального ресурса, предоставленного за расчетный период на общедомовые нужды в многоквартирном доме, не оборудованном коллективным (общедомовым) прибором учета:

          • Если флажок установлен, то доля ОДН в жилом или нежилом помещении будет рассчитываться по формуле №15 постановления №354 от 06 мая 2011г.
          • Если флажок не установлен, то описанная ситуация будет отрабатываться обычным образом.
        • «Использовать формулу №3 постановления №344 при наличии коллективного прибора учета и частичном оборудовании помещений индивидуальными приборами учета».

    Данный флажок позволяет настроить расчет начислений за потребление коммунального ресурса, предоставленного за расчетный период на общедомовые нужды в многоквартирном доме, который оборудован коллективным (общедомовым) прибором учета и в котором не все жилые и нежилые помещения оборудованы индивидуальными приборами учета:

          • Если флажок установлен, то доля ОДН в жилом или нежилом помещении будет рассчитываться по формуле №3 постановления №344 от 16 апреля 2013 г. При этом ОДН будет всегда распределяться между помещениями, не оборудованными индивидуальными приборами учета, пропорционально их общей площади
          • Если флажок не установлен, то описанная ситуация будет отрабатываться обычным образом.
        • «Использовать формулу №3(1) постановления №344 при наличии коллективного прибора учета и оборудовании всех помещений индивидуальными приборами учета».

    Данный флажок позволяет настроить расчет начислений за потребление коммунального ресурса, предоставленного за расчетный период на общедомовые нужды в многоквартирном доме, который оборудован коллективным (общедомовым) прибором учета и в котором все жилые и нежилые помещения оборудованы индивидуальными приборами учета:

          • Если флажок установлен, то доля ОДН в жилом или нежилом помещении будет рассчитываться по формуле №3(1) постановления №344 от 16 апреля 2013 г. При этом ОДН будет всегда распределяться между помещениями, оборудованными индивидуальными приборами учета, пропорционально их общей площади.
          • Если флажок не установлен, то описанная ситуация будет отрабатываться обычным образом.
        • «Ограничить распределяемый расход на ОДН нормативом потребления».

    Данный флажок позволяет ограничить распределяемый между потребителями объем коммунальной услуги, предоставленный на общедомовые нужды, нормативом потребления услуги в соответствии с изменениями п.44 постановления №354 от 06 мая 2011 г., описанными в постановлении №344 от 16 апреля 2013 г.

    Для ограничения используется норматив потребления по текущей услуге, указанный в справочнике "Здания, сооружения".

    При этом:

          • Если выбран вариант распределения расхода по тарифам «Суммарный расход», то нормативом потребления ограничивается суммарный расход на ОДН по всем видам тарифа коллективного (общедомового) прибора учета.
          • Если выбран вариант распределения расхода по тарифам «По виду тарифа», то определяется норматив потребления по каждому виду тарифа и ограничение расхода на ОДН происходит по каждому виду тарифа коллективного (общедомового) прибора учета отдельно. Для определения норматива потребления по текущему виду тарифа используется настройка «Способ распределения нормативов потребления по видам тарифов»:
            • «поровну с ограничением по фактическому расходу» – размер норматива потребления делится поровну на тарифность коллективного (общедомового) прибора учета, после чего проверяется, есть ли такой вид тарифа, где фактический расход меньше полученного норматива на один вид тарифа; если такой вид тарифа есть, то по нему берется фактический расход, а разница передается другому виду тарифа, но не более фактического расхода;
            • «поровну без ограничения по фактическому расходу» – размер норматива потребления делится поровну на тарифность коллективного (общедомового) прибора учета;
            • «пропорционально фактическому расходу» – размер норматива потребления распределяется по видам тарифа пропорционально фактическому расходу коллективного (общедомового) прибора учета.
        • «Ограничить распределяемый расход на ОДН процентом от суммарного индивидуального расхода».

    Данный флажок позволяет ограничить распределяемый между потребителями объем коммунальной услуги, предоставленный на общедомовые нужды, процентом от суммарного индивидуального расхода, определенного в соответствии с настройкой состава суммарного индивидуального расхода. Для ввода значений размеров процентов необходимо перейти по гиперссылке «Размеры процентов для ограничения».

    Расчет «По количеству жильцов»

      Данный расчет необходим для начисления услуг, которые зависят от количества жильцов. Расчет будет производиться по правилу: Тариф х Количество жильцов. При этом необходимо выбрать вид жильцов в соответствующем поле.
      Для данного способа расчета возможно дополнительно установить настройку «Начислять независимо от периода действия услуги». При установке данного флага расчет начислений будет производиться за полный месяц вне зависимости от периода действия услуги в месяце начисления. То есть начисление будет выполнено одинаково как при действии услуги в течение всего месяца начисления, так и при действии услуги в течение только одного дня в месяце начисления. При этом данные о количестве жильцов при начислении услуги будут браться по состоянию на последний день действия услуги в месяце начисления.

    Расчет «По площади»

      Данный расчет необходим для начисления услуг, которые зависят от площади помещения. Расчет будет производиться по правилу: Тариф х Размер площади. При этом необходимо выбрать вид площади в соответствующем поле.
      Для данного способа расчета возможно дополнительно установить настройку «Начислять независимо от периода действия услуги». При установке данного флага расчет начислений будет производиться за полный месяц вне зависимости от периода действия услуги в месяце начисления. То есть начисление будет выполнено одинаково как при действии услуги в течение всего месяца начисления, так и при действии услуги в течение только одного дня в месяце начисления. При этом данные о площади при начислении услуги будут браться по состоянию на последний день действия услуги в месяце начисления.

    Расчет «По нормам потребления»

      Данный расчет необходим для начисления услуг, которые зависят от норм потребления. Расчет будет производиться по правилу: Тариф х Размер нормы потребления. Нормы потребления для лицевых счетов задаются на форме лицевого счета, на закладке «Нормы потребления услуг».
      Для данного способа расчета возможно дополнительно установить настройку «Начислять независимо от периода действия услуги». При установке данного флага расчет начислений будет производиться за полный месяц вне зависимости от периода действия услуги в месяце начисления. То есть начисление будет выполнено одинаково как при действии услуги в течение всего месяца начисления, так и при действии услуги в течение только одного дня в месяце начисления. При этом данные о норме потребления при начислении услуги будут браться по состоянию на последний день действия услуги в месяце начисления.

    Расчет «Фиксированное начисление»

      Данный способ расчета используется для услуг, начисление которых производится в фиксированном размере. Для данного способа расчета дополнительно указывается настройка, каким образом производить начисление, если на одно помещение открыто несколько лицевых счетов:
      • «Полностью на каждый лицевой счет» - услуга будет начислена в полной мере на каждый лицевой счет;
      • «Пропорционально долям лицевых счетов» - сумма услуги будет распределена между лицевыми счетами пропорционально их долям;
      • «Пропорционально количеству лицевых счетов в помещении» - сумма услуги будет распределена между лицевыми счетами поровну.
      Для данного способа расчета возможно дополнительно установить настройку «Начислять независимо от периода действия услуги». При установке данного флага расчет начислений будет производиться за полный месяц вне зависимости от периода действия услуги в месяце начисления. То есть начисление будет выполнено одинаково как при действии услуги в течение всего месяца начисления, так и при действии услуги в течение всего только одного дня в месяце начисления.

    Расчет «С ограничением по нормам», «По показаниям счетчика и норме потребления»

      Данные способы расчета необходимы для случаев, когда тариф по услуге меняется в зависимости от объема данной услуги. Например, на объем потребления услуги, находящийся в пределах норматива потребления, расчет производится по тарифу Т1, а на объем, превышающий норматив потребления – по тарифу Т2. Поскольку программа предусматривает назначение на услугу только одного тарифа, то подобные расчеты реализованы следующим образом: для услуги указывается, что она состоит из нескольких подуслуг, при этом для каждой подуслуги указывается ограничение объема потребления, в рамках которого она действует. Эта информация и указывается в табличной части «Составляющие услуги» (*). Добавляемые в табличную часть услуги должны иметь одинаковый способ расчета: «По количеству проживающих» или «По площади» (для услуги-основания со способом расчета «С ограничением по норме»), «По показаниям счетчика» (для услуги-основания со способом расчета «По показаниям счетчика и норме потребления»).

      Для способа расчета «С ограничением по нормам» возможно дополнительно установить настройку «Начислять независимо от периода действия услуги». При установке данного флага расчет начислений будет производиться за полный месяц вне зависимости от периода действия услуги в месяце начисления. То есть начисление будет выполнено одинаково как при действии услуги в течение всего месяца начисления, так и при действии услуги в течение только одного дня в месяце начисления.


      Пример:

      Услуга «Электроэнергия», расчет производится в пределах нормы потребления по тарифу 1,47 руб./кВт.ч , сверх нормы - 2,65 руб./кВт.ч. Норма потребления - 50 кВт.ч. на одного человека (устанавливается для лицевого счета в  регистре сведений «Нормы потребления услуг лицевых счетов / зданий»). Для установки норм потребления можно использовать обработку «Заполнение норм потребления услуг».

      Для выполнения расчета по услуге «Электроэнергия» необходимо создать в справочнике «Услуги» 2 услуги, например: «Электроэнергия (по норме)» (способ расчета - «По показаниям счетчика», тариф - 1,47 руб./кВт.ч) и «Электроэнергия (сверх нормы)» (способ расчета - «По показаниям счетчика», тариф - 2,65 руб./кВт.ч), а также услугу «Электроэнергия» со способом расчета «По показаниям счетчика и норме потребления». В появившейся табличной части настроек услуги «Электроэнергия» необходимо добавить 2 строки:

      1. «Электроэнергия (по норме)», вид ограничения - Норма потребления.

      2. «Электроэнергия (сверх нормы)», вид ограничения - Без ограничения.


      (*) Имеется возможность более гибкой настройки размеров нормативных ограничений. Для этого в настройки программы ЖКХ добавлен флаг "Использовать функционал расчета составных услуг с несколькими нормативными ограничениями ", при установке которого в табличной части «Составные услуги» становится видимой колонка "Размер ограничения", где можно указать норматив для ограничения расхода по нормативному тарифу. Это может быть необходимо в случаях:

      - если норматив ограничения отличен от норматива потребления, используемого для расчета услуги (в соответствии с постановлениями правительства Нижегородской области №594 от 30.08.2012, № 310 от 28.05.2012);

      - если расчет услуги выполняется в пределах нескольких нормативных ограничений (в соответствии с приказами Государственного комитета по ценам и тарифам Республики Крым от 21.12.2016 №53/1 и Департамента городского хозяйства г.Севастополя от 23.01.2017 №15-ОД).

      Примечание:
      Если ограничения задаются непосредственно в самой услуге, то нормативы, установленные на лицевые счета, при расчете составной услуги учитываться не будут.

      Пример:

      Необходимо рассчитать услугу «Электроэнергия»по трем тарифам: 2,42 руб./кВт.ч - за объем, потребленный до 150 кВт∙ч; 2,96 руб./кВт.ч - за объем, потребленный от 150 до 600 кВт∙ч; 5,40 руб./кВт.ч - за объем, потребленный сверх 600 кВт∙ч. Общий объем потребления по услуге составил 700 Квт.

      Для выполнения расчета по услуге «Электроэнергия» необходимо создать в справочнике «Услуги» 3 услуги, например: «Электроэнергия (в пределах 150 кВт∙ч)» (способ расчета - «По показаниям счетчика», тариф - 2,42 руб./кВт.ч), «Электроэнергия (в пределах 600 кВт∙ч)» (способ расчета - «По показаниям счетчика», тариф - 2,96 руб./кВт.ч) и «Электроэнергия (сверх нормы)» (способ расчета - «По показаниям счетчика», тариф - 5,40 руб./кВт.ч), а также услугу «Электроэнергия» со способом расчета «По показаниям счетчика и норме потребления». В появившейся табличной части настроек услуги «Электроэнергия» необходимо добавить 3 строки:

      1. «Электроэнергия (в пределах 150 кВт∙ч)», вид ограничения - Норма потребления, размер ограничения - норматив в размере 150 кВт∙ч.

      2. «Электроэнергия (в пределах 600 кВт∙ч)», вид ограничения - Норма потребления, размер ограничения - норматив в размере 450 кВт∙ч (600 - 150).

      3. «Электроэнергия (сверх нормы)», вид ограничения - Без ограничения.


      Начисление по услуге составит 150 * 2,42 + 450 * 2,96 + 100 * 5,40 = 2235 руб.


      Кроме того, для услуг со способом расчета «По показаниям счетчика и норме потребления» необходимо указать способ распределения указанных ограничений по видам тарифов приборов учета (дневной, ночной, пиковый):

      • «поровну с ограничением по фактическому расходу» – размер ограничения делится поровну на тарифность прибора учета, после чего проверяется, есть ли такой вид тарифа, где фактический расход меньше полученного ограничения; если такой вид тарифа есть, то по нему берется фактический расход, а разница передается другому виду тарифа по этому же ограничению;
      • «поровну без ограничения по фактическому расходу» – размер ограничения делится поровну на тарифность прибора учета;
      • «пропорционально фактическому расходу» – размер ограничения распределяется по видам тарифа пропорционально фактическому расходу прибора учета по данному виду тарифа.


      Пример:

      Услуга «Электроэнергия» имеет способ расчета «По показаниям счетчика и норме потребления», для данной услуги определены составные услуги «Электроэнергия (по норме)» и «Электроэнергия (сверх нормы)». По услуге «Электроэнергия» был введен расход по показаниям приборов учета: дневной расход - 80 кВт.ч, ночной расход - 40 кВт.ч. Норма потребления - 100 кВт.ч на всю семью. В зависимости от выбранного способа распределение ограничения по видам тарифов будет происходить следующим образом:

      1. «поровну с ограничением по фактическому расходу»:

        • «Электроэнергия (по норме)». Дневной расход - 60 кВт.ч, ночной расход - 40 кВт.ч.

        • «Электроэнергия (сверх нормы)». Дневной расход - 20 кВт.ч.

      2. «поровну без ограничения по фактическому расходу»:

        • «Электроэнергия (по норме)». Дневной расход - 50 кВт.ч, ночной расход - 40 кВт.ч.

        • «Электроэнергия (сверх нормы)». Дневной расход - 30 кВт.ч.

      3. «пропорционально фактическому расходу»:

        • «Электроэнергия (по норме)». Дневной расход - 66,667 кВт.ч, ночной расход - 33,333 кВт.ч.

        • «Электроэнергия (сверх нормы)». Дневной расход - 13,333 кВт.ч, ночной расход - 6,667 кВт.ч.

    Также для услуг со способом расчета «По показаниям счетчика и норме потребления» можно задать настройку «Предоставлять объем по нормативу за месяцы отсутствия показаний». Данная настройка действует в ситуациях, когда жилец передал показания по индивидуальному прибору учёта после одного или нескольких месяцев непредоставления. Если флажок не установлен, то в месяце предоставления показаний ограничения (нормативы) составной услуги рассчитываются в однократном размере. Если же флажок установлен, то размер ограничений (нормативов) составной услуги берется кратный количеству месяцев отсутствия показаний.

     

    Пример:

    Услуга «Электроэнергия» имеет способ расчета «По показаниям счетчика и норме потребления», для данной услуги определены составные услуги «Электроэнергия (по норме)» и «Электроэнергия (сверх нормы)». Норма потребления - 60 кВт.ч на всю семью. По услуге «Электроэнергия» в январе и феврале показания не были введены, и начисление произведено по норме потребления: за январь по норме - 60 кВт.ч, за февраль по норме - 60 кВт.ч.  В марте по услуге «Электроэнергия» введены показания в объеме 200 кВт.ч. В зависимости от значения настройки предоставления объема по нормативу за месяцы отсутствия показаний начисление услуги будет происходить следующим образом:

    1. Флажок «Предоставлять объем по нормативу за месяцы отсутствия показаний» не установлен:

        • «Электроэнергия (по норме)». Дневной расход - 60 кВт.ч.

        • «Электроэнергия (сверх нормы)». Дневной расход - 140 кВт.ч.

    2. Флажок «Предоставлять объем по нормативу за месяцы отсутствия показаний» установлен:

        • «Электроэнергия (по норме)». Дневной расход - 60 (за январь) + 60 (за февраль) + 60 (за март) = 180 кВт.ч.
        • «Электроэнергия (сверх нормы)». Дневной расход - 20 кВт.ч.

    Расчет «Комиссия банка»

      Данный расчет предназначен для начисления комиссии банка. Для данных услуг дополнительно указывается настройка с правилами расчета комиссии:
      • «Сверху от суммы начисления за месяц» - начисление комиссии банка будет производится по формуле: Комиссия = S * St/100, где S - сумма начислений за месяц, St - процент банка;
      • «В сумме от начислений за месяц» - начисление комиссии банка будет производится по формуле: Комиссия = S * St/(100-St), где S - сумма начислений за месяц, St - процент банка.

    Вверху формы расположены кнопки:

    • «Записать и закрыть» – записать элемент и закрыть форму справочника;
    • «Записать объект» – записать элемента справочника.

    Справочник можно открыть из «Учет в ЖКХ / Начисление / Оплата / Услуги».

     

    Техническая информация

    Учет отрицательного индивидуального расхода при определении объема ОДН к распределению.

    Отрицательный индивидуальный расход всегда учитывается при определении объема ОДН к распределению.

    Например. Объем расхода коллективного прибора учета составил 17 кВт. Индивидуальный расход по лицевым счетам л/с 0001, л/с 0002, л/с 0003 составил соответственно 2, -1 и 6 кВт. Тогда объем к распределению ОДН определяется следующим образом.
    Объем к распределению ОДН = (Объем расхода коллективного прибора учета) - (инд. расход л/с 0001) - (инд. расход л/с 0002) - (инд. расход л/с 0003) = 17 кВт - 2 кВт - (-1) кВт - 6 кВт = 10 кВт.

    Порядок распределения расхода ОДН на лицевые счета.

    Порядок распределения зависит от знака объема ОДН к распределению следующим образом:

    Объем расхода ОДН к распределению положительный (+):

    • Если распределение производится пропорционально «Количеству расхода по услуге», то распределение производится только на те лицевые счета, индивидуальное потребление которых положительное.

      Например. Объем ОДН к распределению составил 10 кВт. Индивидуальный расход по лицевым счетам л/с 0001, л/с 0002, л/с 0003 составил соответственно 2, -1 и 6 кВт.  Тогда распределение ОДН пропорционально количеству расхода по услуге составит:
      л/с 0001 - (2 / 8) * 10 = 2,5 кВт;
      л/с 0002 - 0 кВт;
      л/с 0003 - (6 / 8) * 10 = 7,5 кВт.

    • Если распределение производится пропорционально «Площади», Количеству помещений в квартире», «Количеству жильцов» или «Поровну», то распределение производится на все лицевые счета, вне зависимости от количества индивидуального расхода по услуге.

      Например. Объем ОДН к распределению составил 10 кВт. Индивидуальный расход по лицевым счетам л/с 0001, л/с 0002, л/с 0003 составил соответственно 2, -1 и 6 кВт, общие площади помещений составляют соответственно 20, 30 и 40 кв.м. Тогда распределение ОДН пропорционально общей площади помещений составит:
      л/с 0001 - (20 / 90) * 10 = 2,22 кВт;
      л/с 0002 - (30 / 90) * 10 = 3,33 кВт;
      л/с 0003 - (40 / 90) * 10 = 4,44 кВт.

    Объем расхода ОДН к распределению отрицательный (-). Распределение происходит в 2 этапа:

    1. Распределение на лицевые счета пропрционально «Площади» или «Количеству жильцов».

    2. Отнесение распределенной суммы на лицевые счета с ограничением по индивидуальному расходу вплоть до нуля: индивидуальный расход в сумме с отрицательным распределением не должен быть отрицательным.

    Например. Объем ОДН к распределению составил -10 кВт. Индивидуальный расход по лицевым счетам л/с 0001, л/с 0002, л/с 0003 составил соответственно 2, -1 и 6 кВт, общие площади помещений составляют соответственно 20, 30 и 40 кв.м.

    1. Распределение ОДН пропорционально общей площади помещений составит:

    л/с 0001 - (20 / 90) * 10 = 2,22 кВт;
    л/с 0002 - (30 / 90) * 10 = 3,33 кВт;
    л/с 0003 - (40 / 90) * 10 = 4,44 кВт.

    2. Распределенные объемы относятся на лицевые счета с ограничением по индивидуальному расходу вплоть до нуля следующим образом:

    л/с 0001 - МАКСИМУМ(-2 - индивидуальный расход; -2,22 - распределение) = -2 кВт. Суммарный расход с учетом распределения 2 - 2 = 0 кВт.

    л/с 0002 - Индивидуальный расход < 0, поэтому распределение отрицательного расхода невозможно. Суммарный расход с учетом распределения (-1) кВт;

    л/с 0003 - МАКСИМУМ(-6 - индивидуальный расход; -4,44 - распределение) = -4,44 кВт. Суммарный расход с учетом распределения 6 - 4,44 = 1,56 кВт.

    Добровольное страхование.
    Для услуги, которая относится к добровольному страхованию (указана в соответствующем поле в «Настройках параметров учета (ЖКХ)»), можно указать только следующие способы расчета: «Фиксированное начисление», «По площади».
