Справочник "Значения дополнительных реквизитов"

Справочник предназначен для хранения списка дополнительных реквизитов, созданных для справочников и документов.

Например, для справочника Номенклатура можно ввести дополнительный реквизит – Вид товара. Для этого дополнительного реквизита выбрать Тип значения - Произвольный список. Тогда, в справочник необходимо ввести значения, соответствующие виду товаров, например, конфеты, печенье, пряники и т.д.

Значения дополнительных реквизитов могут использоваться для группировки и отбора данных в различных отчетах, например, в стандартных отчетах.
