Справочник предназначен для хранения списка валют, используемых в организации.

Список валют можно редактировать, вводить новые записи или удалять имеющиеся.

Новую валюту можно добавить в справочник:

  • Из общероссийского классификатора валют по кнопке Подбор из классификатора.
  • Вручную по кнопке Создать.

Для того чтобы в печатных формах правильно указывалось наименование валюты прописью, необходимо задать параметры прописи валюты. Для валюты задаются параметры прописи целой и дробной части валюты. В случае если дробная часть в печатных формах отображается числом, то можно установить нужное количество разрядов при выводе печатных форм документов. Например, если установлено количество разрядов 1 знак, то сумма по документу будет отображена, как 150,2 евро, а если 2 знака, то – 150,20 евро.
