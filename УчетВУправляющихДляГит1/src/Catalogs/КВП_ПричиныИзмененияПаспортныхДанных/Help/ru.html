Справочник «Причины изменения паспортных данных»

Данный справочник предназначен для хранения списка причин изменения паспортных данных. Ввод нового элемента справочника и редактирование уже существующего элемента справочника производится в форме списка. При вводе нового элемента указываются следующие реквизиты:

  • «Код» — код элемента справочника устанавливается автоматически;
  • «Наименование» — наименование причины изменения паспортных данных.

Справочник содержит предопределенные элементы: «Потеря документа», «Изменение фамилии», «Изменение имени», «Изменение отчества».

Справочник можно открыть через меню «Учет в ЖКХ / Справочники / Прочие / Причины изменения паспортных данных».
