Справочник «Настройки доступа к сайту»

Данный справочник предназначен для хранения параметров, в соответствии с которыми будет осуществляться подключение к сайту управляющей организации.

На форме справочника располагаются следующие элементы:

  • «Код» — код элемента справочника устанавливается автоматически;
  • «Наименование» — наименование настройки;
  • «Организация» — организация, выбранная из справочника «Организации»;
  • «Адрес сайта» — адрес сайта управляющей организации (указывается в формате «demo.vdgb-soft.ru»);
  • «Логин» — логин пользователя сайта, имеющего права на выполнение обмена с учетной программой;
  • «Пароль» — пароль пользователя сайта, имеющего права на выполнение обмена с учетной программой;  
  • «Хранить пароль» — флаг, при снятии которого введенный пароль не будет сохраняться в базе данных;
  • «Использовать безопасное соединение (SSL)» — флаг, необходимо устанавливать, если для соединения с сайтом необходимо использовать безопасное соединение;
  • «Проверить подключение» — кнопка, при нажатии на которую осуществляется пробное подключение к сайту по параметрам, указанным в текущей настройке доступа;
  • «Число окончания проведения начислений» — число месяца, по состоянию на которое все начисления по услугам проведены. Данная опция необходима для корректного отображения квитанций на сайте;
  • «Не выгружать персональные данные на сайт» — флаг, при установке которого на сайт не будут выгружаться персональные данные собственников лицевых счетов. А именно: ФИО, электронная почта и сведения о помещении собственника лицевого счета;
  • «Учетная запись электронной почты» — учетная запись электронной почты, выбранная из справочника «Учетные записи электронной почты», которая будет использоваться при использовании рассылки памяток доступа на электронные почтовые ящики лицевых счетов.
