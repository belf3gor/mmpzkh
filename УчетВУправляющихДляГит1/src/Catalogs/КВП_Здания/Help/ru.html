Справочник «Здания, сооружения»

Данный справочник предназначен для хранения списка зданий, сооружений.

Форма списка элементов

На командной панели диалогового окна списка зданий расположены следующие кнопки:

  • «Отчеты» - по нажатию кнопки выводится меню кнопок: «Действующие начисления», «Отчет по зданиям», «Отчет по помещениям». При нажатии кнопок открывается отчет, соответствующий наименованию кнопки.
  • «Заполнить нормы потребления» - по нажатию кнопки открывается обработка «Заполнение норм потребления услуг».
  • «Открыть ЛС» - по нажатию кнопки открывается обработка «Помощник открытия лицевого счета».
  • «Настройки графиков» - по нажатию кнопки открывается окно «Настройки графиков».

Внизу диалогового окна списка зданий расположено поле, состоящее из закладок: «Помещения», «Подъезды», «Действующие начисления», «Счетчики», «Взаиморасчеты»:

  • на закладке «Помещения» расположен список квартир, офисов и др. помещений текущего здания;
  • на закладке «Подъезды» расположен список подъездов текущего здания;
  • на закладке «Счетчики» расположены следующие элементы:
    • список коллективных (общедомовых) приборов учета, закрепленных за текущим зданием;
    • графики расходов по услугам приборов учета, установленных на текущем здании. Графики доступны, если установлен флаг «Выводить графики» на форме «Настройки графиков».

При переходе на строки списка коллективных приборов учета график расхода по текущей услуге прибора учета выводится первым.

  • закладка «Взаиморасчеты» доступна, когда установлен флаг «Выводить графики» на форме «Настройки графиков». На закладке расположены графики, содержащие следующие данные: начисления и оплаты, процент собираемости оплат, начисления и оплаты пени, процент собираемости оплат по пени, задолженность по месяцам. Графики формируются за выбранный период в поле "Период". Отбор сведений по организации в графиках осуществляется по выбранной организации в поле «Организация», при этом, если поле «Организация» не заполнена, то данные графиков выводятся по всем организациям.

Форма редактирования элемента

Ввод нового элемента справочника и редактирование уже существующего осуществляется через диалоговое окно ввода элемента справочника.

Форма элемента справочника «Здания» состоит из закладок:

Закладка «Общие»

На данной закладке указываются основные данные элемента справочника:

  • «Тип сооружения» – указывается тип здания. Доступные значения: Здание, Паркинг;
  • «Код» – код элемента справочника. Устанавливается автоматически;
  • «Этажей» – указывается количество этажей в здании;
  • «Подъездов» – указывается количество подъездов в здании;
  • «Многоквартирное» – указывает, является ли здание многоквартирным;
  • «Вид собственности» – вид собственности здания. Определяется по виду собственности помещений данного здания.
    • Если все помещения здания муниципальные – здание муниципальное.
    • Если  все помещения здания государственные – здание государственное.
    • В противном случае здание общее.
  • «Вид сооружения» – указывается вид здания из справочника «Виды зданий, сооружений».
  • «Тип жилого фонда» – указывается тип жилого фонда здания;
  • «ОВД» – указывается орган внутренних дел из справочника «ОВД».
  • «Номенклатурная группа» - указывается элемент из справочника «Номенклатурные группы».
  • «Участок» – указывается участок из справочника «Участки».

Закладка «Адрес и контакты»

На данной закладке расположены реквизиты:

  • «Почтовый адрес» - указывается почтовый адрес здания;
  • «Строительный адрес» - указывается строительный адрес здания.

Закладка «Площади»

На данной закладке расположены следующие элементы:

  • поля ввода общих площадей здания:
    • площадь всех помещений в здании;
    • площадь всех жилых и нежилых помещений в здании;
    • площадь помещений, входящих в состав общего имущества здания.
  • таблица для указания размеров площадей здания. Таблица состоит из следующих колонок:
    • «Тип площади» – выбирается тип площади здания из справочника «Типы площадей здания»;
    • «Площадь» – указывается размер площади.

Закладка «Технические характеристики»

На данной закладке указывается техническая информация по зданию:

  • «Серия дома» – указывается серия здания;
  • «Год постройки» – указывается год постройки здания;
  • «Год последнего капитального ремонта» – указывается год последнего капитального ремонта здания;
  • «Процент износа» – указывается процент износа здания;
  • «Средняя внутренняя высота помещений» – указывается средняя внутренняя высота помещений в здании;
  • «Объем» – указывается объем здания;
  • «Материал стен» – указывается материал изготовления стен здания из справочника «Материал стен зданий»;
  • «Вид кровли» – указывается вид кровли здания из справочника «Вид кровли»;
  • «Дата ввода в эксплуатацию» – указывается дата ввода здания в эксплуатацию.

Закладка «Благоустройство»

На данной закладке указывается вид благоустройства здания в поле «Благоустройство». Доступные значения для выбора: «Благоустроенное жилье», «Жилье с частичным благоустройством».

Также расположена таблица для указания значений видов благоустройств здания. Таблица состоит из следующих колонок:

  • «Вид благоустройства» – указывается вид благоустройства (горячее/холодное водоснабжение, канализация и т.д.) из справочника «Виды благоустройства объектов»;
  • «Значение» – указывается значение вида благоустройства.

Закладка «Прилегающая территория»

На данной закладке расположена таблица для указания свойств прилегающей территории здания. Таблица состоит из следующих колонок:

  • «Тип» – указывается свойство прилегающей территории (площадь газона, площадь грунта и т.д) из справочника «Прилегающая территория»;
  • «Значение» – указывается значение свойства прилегающей территории.

Закладка «Ответственные лица»

На данной закладке расположена таблица для указания ответственных лиц по обслуживанию здания. Таблица состоит из следующих колонок:

  • «Период» – указывается период назначения ответственного лица;
  • «Вид мастера» – указывается вид мастера из справочника «Виды мастеров»;
  • «Мастер» – указывается мастер по обслуживанию, элемент справочника «Физические лица»;

Закладка «Нормы потребления услуг»

На данной закладке расположена таблица для указания нормативов потребления коммунальных услуг на здание. Таблица состоит из следующих колонок:

  • «Период» - указывается дата начала действия нормы;
  • «Услуга» - указывается услуга из справочника «Услуги», по которой устанавливается норма потребления»;
  • «Норма потребления» - указывается норма потребления из справочника «Нормы потребления услуг». На здание, сооружение можно установить только те нормы потребления, в настройках которых указано, что норма устанавливается на здание, сооружение.
  • «Размер» - выводится размер нормы, указанной в поле «Норма потребления.

Закладка «Обслуживающие организации»

На данной закладке расположена таблица для указания обслуживающей организации здания и расчетных счетов, используемых при оплате начислений.

Закладка «Объекты учета и оборудование»

На данной закладке расположена таблица объектов учета и оборудования, установленных на текущее здание.

Для редактирования установленного на здание оборудования доступны команды «Установить» и «Отключить», предназначенные для установки и отключения оборудования соответственно.

 

На верхней командной панели расположены кнопки:

  • «Записать и закрыть» – по нажатию на кнопку данные справочника сохраняются и форма закрывается;
  • «Записать» – по нажатию на кнопку данные справочника сохраняются, форма не закрывается.

Справочник можно открыть из «Объекты учета / Здания, сооружения».
