Справочник «Типы родственных связей»

Данный справочник предназначен для хранения списка типов родственных связей с владельцем жилья.

Ввод нового элемента справочника и редактирование уже существующего элемента справочника производится в форме списка. При вводе нового элемента необходимо указать:

  • «Наименование» - наименование типа родственной связи;
  • «Близкое родство» - флажок необходимо устанавливать в том случае, если жильцы с данным типом родственной связи являются близкими родственниками по отношения к владельцу жилья;
  • «Код» - устанавливается автоматически.

Справочник можно открыть из «Учет в ЖКХ / Справочники / Прочие / Типы родственных связей».
