Справочник «Настройки обмена данными с сайтом ЖКХ»

Данный справочник предназначен для указания настроек обмена данными с сайтом ЖКХ, по которыми следует обмениваться с сайтом управляющей организации.

На форме справочника располагаются следующие элементы:

  • «Наименование» — наименование настройки;
  • «Настройка сбора данных» — настройка сбора данных, выбранная из справочника «Настройки сбора данных для квитанций и обменов»;
  • Вкладка «Загрузка показаний ПУ»:
    • Флаг «Загружать по зданиям» — если установлен, то при загрузке показаний приборов учета будет осуществляться загрузка с сайта сведений о показаниях индивидуальных проборов учета в разрезе зданий в отдельный документ "Ввод показаний счетчика".
    • Флаг «Загружать по услугам» — если установлен, то при загрузке показаний приборов учета будет осуществляться загрузка с сайта сведений о показаниях индивидуальных проборов учета в разрезе услуг в отдельный документ "Ввод показаний счетчика".
    • Если флаги «Загружать по зданиям» и «Загружать по услугам» не установлены, то будет осуществляться загрузка с сайта сведений о показаниях индивидуальных проборов учета в один документ "Ввод показаний счетчика".
  • Вкладка «Выгрузка данных по ЛС»:
    • Флаг «Выгружать одновременно одномерный штрихкод и QR-код» — если установлен, то при выгрузке сведений по лицевым счетам на сайт для каждой квитанции будут выгружаться как одномерный штрихкод, так и QR-код.
    • Таблица «Настройки выгрузки» — таблица, предназначенная для указания настроек выгрузки данных о начислениях и показаниях счетчиков в разрезе услуг. Данная таблица содержит следующие колонки:
      • «Услуга» — предназначена для выбора услуг, по которым необходимо установить индивидуальные настройки выгрузки сведений о начислениях и показаниях счетчиков. Данная колонка всегда содержит значение «<По умолчанию>», предназначенное для указания настроек выгрузки для услуг, не указанных в таблице.
      • «Выгружать начисления» — при выборе данного значения будут выгружаться размеры начислений по выбранной услуге.
      • «Выгружать показания» — при выборе данного значения будут выгружаться показания по приборам учета выбранной услуги.
