Справочник «Рассчитываемые тарифы»

Данный справочник предназначен для хранения рассчитываемых видов тарифов.

В справочнике необходимо указать:

  • "Арифметическая операция" - операция (сложить или умножить), которая будет выполняться между указанными параметрами расчета;
  • список вспомогательных параметров для расчета:
    • "Число" - произвольное число;
    • "Реквизит здания" - один из числовых реквизитов здания, к которому относится текущий лицевой счет;
    • "Свойство здания" - дополнительное свойство типа "Число" здания, к которому относится текущий лицевой счет;
    • "Реквизит квартиры" - один из числовых реквизитов квартиры, к которой относится текущий лицевой счет;
    • "Свойство квартиры" - дополнительное свойство типа "Число" квартиры, к которой относится текущий лицевой счет.
