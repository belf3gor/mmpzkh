Справочник «Нормы потребления услуг»

Данный справочник предназначен для хранения списка норм потребления услуг.

Справочник подчинен справочнику «Услуги», т.е. при вводе необходимо указать к какому виду услуг относится данная норма потребления.

При вводе нового элемента справочника необходимо указать:

  • «Наименование» – указывается наименование нормы потребления услуг;
  • «Услуга» – из справочника «Услуги» выбирается вид услуги, к которой относится данная норма потребления;
  • «Единица измерения» - единица измерения нормы потребления из справочника «Классификатор единиц измерения»;
  • «Норма на» - настройка, определяющая тип объекта, для которого устанавливается норма потребления услуги. Данная настройка может принимать следующие значения:
    • «Лицевой счет» – норма устанавливается для «Лицевых счетов»;
    • «Здание, сооружение» – норма устанавливается для «Зданий, сооружений»;
  • «Норма для владельцев (нанимателей) единственного жилья» - признак, определяющий, что получение размера нормы потребления нужно выполнять в соответствии с Постановлением Правительства Москвы №848-ПП от 13.12.2016 при расчете начислений услуг по содержанию жилого помещения. Данная настройка доступна, если одновременно выполняются следующие условия:
    • в «Настройках программы» на закладке "Функциональность программы" установлена настройка «Вести учет владельцев единственного жилья»;
    • выбрана услуга со способом расчета «С ограничением по нормам»;
    • в качестве объекта нормы указан лицевой счет.
  • «Размер» – указывается размер нормы потребления.
  • «Основание» – из списка выбирается значение, на которое будет умножаться размер нормы потребления.

Способ задания размеров норм зависит от реквизита «Вариант настройки».

  • Простой  - в качестве нормы в поле «Размер» указывается фиксированная норма (например, 0,55).
  • Расширенный - для задания нормы становятся доступными дополнительные поля: Таблица «Характеристики для задания размеров норм» и таблица «Размеры норм». 
    • В таблице «Характеристики для задания размеров норм» указываются различные параметры, от которых зависит норма (например, количество человек, количество комнат, характеристики зданий, помещений), а также вариант, по которому возможно будет задать значения параметров: "Значение" либо "Диапазон".  
    • В таблице «Размеры норм» задаются размеры норм в зависимости от заданных значений характеристик (значение поля «Другие» устанавливается, когда необходимо указать размер нормы для всех остальных вариантов характеристик).
  • Формула - для задания нормы становится доступен режим редактирования формул.

 

Норма на лицевые счета

При способе задания норм по расширенному варианту настройки доступны для выбора следующие характеристики:

    • Свойство здания,
    • Реквизит здания,
    • Площадь здания,
    • Свойство помещения,
    • Реквизит помещения,
    • Площадь помещения,
    • Количество человек - "Зарегистрированных", "Проживающих", "Льготников", "Владельцев единственного жилья", "Близких родственников" (данный вид характеристики можно использовать только для тех ограничений, которые используются при расчете льгот).
    • Количество комнат - "Жилых", "Всего".

В качестве основания можно указать:

    • «Количество дней»;
    • «Площадь». При этом в дополнительном поле необходимо выбрать вид площади помещения из справочника «Виды площадей»;
    • «Услуга»;
    • «Без основания»;
    • «Один человек» - при этом в дополнительном поле необходимо указать вид жильцов:
      • зарегистрированный,
      • проживающий,
      • близкий родственник (данный вид основания можно использовать только для тех ограничений, которые используются при расчете льгот),
      • льготник или
      • владелец единственного жилья (последнее значение можно выбрать только для норм, устанавливаемых для составных услуг со способом расчета «С ограничением по нормам»).

При выборе в качестве основания значения «Один человек» становится доступной дополнительная настройка «Вариант расчета основания нормы при отсутствии жильцов указанного вида» со значениями:

  • Не определять;
  • На одного человека;
  • По количеству собственников.

В случае, когда на лицевом счете отсутствуют жильцы указанного вида, основание для расчета нормы будет определяться с учетом данной настройки.

Внимание: Данная настройка влияет только на основание нормы! Т.е. если в таблице «Характеристики для задания размеров норм» указан, например, вид характеристик «Проживающих», в таблице «Размеры норм» указаны значения для некоторых значений количества проживающих (1 - 33 единицы, 2 - 21 единица, 3 - 18 единиц), в качестве основания выбрано значение «Один человек», вид жильцов «Проживающий» и установлен данный флаг, то значение размера нормы при отсутствии проживающих на лицевом счете будет взято "0 единиц", а не "33 единицы". Для корректного расчета нормы в описанной ситуации необходимо в таблицу «Размеры норм» добавить отдельное значение для количества человек равных 0. При этом если бы в таблице «Размеры норм» была отдельная строка для значения «Другие» - 15 единиц, то было бы взято именно это значение в "15 единиц" при отсутствии отдельной строки для значения количества человек равных 0.

Например:

Если в качестве основания нормы для лицевого счета указать «Общая площадь», то при определении нормы потребления размер общей площади помещения будет умножен на размер нормы.

 

Пример расширенного варианта настройки нормы на лицевой счет:

Пусть необходимо задать норму для лицевого счета следующим образом: если в семье проживает 1 человек и квартира однокомнатная, то норма равна 0,5, если проживает 2 человека и квартира не однокомнатная - 0,75, в остальных случаях - 1,0.

Для этого устанавливаем реквизит «Вариант настройки» в значение «Расширенный».

В таблице «Характеристики для задания размеров норм» указываем:

Вариант значения

Реквизит помещения Количество комнат Значение

В поле «Основание» указываем «Вся семья».

 

При способе задания нормы по формуле нобходимо перейти в режим редактирования формул, где имеется возможность выбрать следующие показатели расчета:

    • Здание - Свойства здания, Реквизиты здания, Площади здания;
    • Помещение - Свойства помещения, Реквизиты помещения, Площади помещения;
    • Количество человек - "Зарегистрированных", "Проживающих", "Льготников", "Владельцев единственного жилья", "Близких родственников";
    • Количество комнат - "Жилых", "Всего";
    • Показатели расчета - "Количество дней в месяце", "Месяц расчета";

       

      Примеры расчета нормы на лицевой счет при указанной настройке «Норма для владельцев (нанимателей) единственного жилья»:

      Пример 1.

      Пусть необходимо задать норму для лицевого счета следующим образом: на лицевом счете проживает 3 человека, из которых зарегистрировано в помещении 2 человека, площадь квартиры 100 кв.м., двое из жильцов являются собственниками с долями по 0.5 от общей площади, один из собственников является владельцем единственного жилья. Согласно Постановлению Правительства Москвы №848-ПП от 13.12.2016 при наличии на лицевом счете владельца единственного жилья для каждого зарегистрированного в помещении жильца используется пониженное начисление в пределах нормы. Для семьи из 2х человек (а в данном случае учитываются только зарегистрированные жильцы, которых двое) норма составляет 21+7 кв.м на каждого учитываемого жильца*.

      Устанавливаем реквизит «Вариант настройки» в значение «Расширенный».

      В таблице «Характеристики для задания размеров норм» указываем:

      Характеристика

      Вид характеристики

      Вариант значения

      Количество человек Зарегистрированных Значение

       

      В таблице «Размеры норм» указываем размеры норм в зависимости от количества зарегистрированных человек:

       

      Количество человек (Зарегистрированных)

      Размер

      1 40
      2 28
      Другие* 25

      Для семьи из 2х человек норма на лицевой счет составит 28*2 = 56 кв.м. Однако один из собственников, владеющих половиной площади помещения не является владельцем единственного жилья, поэтому значение нормы будет определяться как минимум из значений 56 и 50, т.е. будет равняться 50 кв.м.

       

      Пример 2.

      На лицевом счете проживает 3 зарегистрированных человека, площадь квартиры 100 кв.м., собственнику с признаком владелец единственного жилья принадлежит 0.7 от общей площади помещения. Настройки нормы аналогичные предыдущему примеру.

      Для семьи из 3х человек норма на лицевой счет составит 25*3 = 75 кв. м. Однако владельцу единственного жилья принадлежит только 70 кв.м. общей площади, поэтому значение нормы будет определяться как минимум из значений 75 и 70, т.е. будет равняться 70 кв.м.

       

      Пример 3.

      На лицевом счете проживает 2 зарегистрированных человека, площадь квартиры 100 кв.м., квартира муниципальная, наниматель квартиры является нанимателем единственного жилья.

      Для семьи из 2х человек норма на лицевой счет составит 28*2 = 56 кв.м.

       

      *Согласно Постановлению значения норм следующие: 33+7 кв.м. на одиноко проживающего гражданина; 21+7 кв.м. на каждого зарегистрированного жильца для семьи из 2х человек; 18+7 кв.м. на каждого зарегистрированного жильца для семьи из 3х человек и более.

       

      Примеры настройки нормы на лицевой счет для варианта «Формула»:

      1.Пусть необходимо задать норму для лицевого счета следующим образом: если в квартире проживает не более 2 человек, то норма задается как [Количество проживающих * 21], если более , то [Количество проживающих * 18].

      Для этого устанавливаем реквизит «Вариант настройки» в значение «Формула».

      В режиме редактирования формул необходимо ввести следующую формулу:

      ?([КоличествоЧеловек.Проживающих] < 3, [КоличествоЧеловек.Проживающих] * 21, [КоличествоЧеловек.Проживающих] * 18) - 

      где выражение ?( , , ) представляет собой конструкцию, в которой, если условие до первой запятой [КоличествоЧеловек.Проживающих] < 3

      выполняется, то выражение принимает вид [КоличествоЧеловек.Проживающих] * 21, если не выполняется, то [КоличествоЧеловек.Проживающих] * 18.

      В поле «Основание» указываем «Без основания».

       

      2. Также имеется возможность расчета нормы потребления в зависимости от даты.

      Для этого используется операнд [ПоказательРасчета.МесяцРасчета] (определяется как дата начала месяца, в котором производятся начисления), а также функция Дата(гггг,мм,дд).

      Пример использования:

      ?([ПоказательРасчета.МесяцРасчета] < Дата(2014,02,28), 5, 10).

      Данная формула означает, что, если начисления производятся ранее марта 2014 года, то размер норматива будет равен 5, если позже, то 10.

      Норма на здание / сооружение

       

      При способе задания нормы по расширенному варианту настройки доступны для выбора следующие характеристики:

        • Свойство здания,
        • Реквизит здания,
        • Площадь здания,
        • Количество проживающих / Площадь,
        • Количество зарегистрированных / Площадь,
        • Площадь / Количество проживающих,
        • Площадь / Количество зарегистрированных,
        • Количество человек - "Зарегистрированных", "Проживающих", "Льготников", "Владельцев единственного жилья".

      При этом параметры количества человек и площади рассчитываются для всего здания в целом.

       

      В качестве основания можно указать:

        • «Один человек» - при этом в дополнительном поле необходимо указать вид жильцов:
          • зарегистрированный или
          • проживающий.

      Количество зарегистрированных или проживающих жильцов в здании будет определяться на момент расчета как суммарное количество жильцов, зарегистрированных или проживающих по лицевым счетам здания, открытым на начало или в период расчета;

        • «Площадь». При этом в дополнительном поле необходимо выбрать вид площади здания:
          • Общая площадь всех помещений;
          • Общая площадь всех жилых и нежилых помещений;
          • Общая площадь всех помещений, входящих в состав общего имущества.

      Размер соответствующего вида площади будет браться из справочника «Здания, сооружения».

        • «Без основания».

      Например:

      Если в качестве основания нормы для зданий указать «Общая площадь всех помещений», то при определении нормы потребления размер общей площади всех помещений здания будет умножен на размер нормы, указанный в поле «Размер».

      При способе задания нормы на здание по формуле в режиме редактирования формул имеется возможность выбрать следующие показатели расчета:

        • Здание - Свойства здания, Реквизиты здания, Площади здания;
        • Количество человек - "Зарегистрированных", "Проживающих", "Льготников", "Владельцев единственного жилья";
        • Показатели расчета - "Количество дней в месяце", "Месяц расчета".

      Пример настройки нормы потребления на здание для варианта «Формула»:

      Пусть размер норматива потребления, устанавливаемого на здание, зависит от количества жителей, проживающих в многоквартирных домах (К), а также общей площади помещений, входящих в состав общего имущества в многоквартирном доме (Sои) и рассчитывается по формуле: 

      N = 0.09 * К / Sои.

      Для задания норматива устанавливаем реквизит «Вариант настройки» в значение «Формула».

      В режиме редактирования формул необходимо ввести следующую формулу:

      0.09 * [КоличествоЧеловек.Проживающих] / [Здание.РеквизитЗдания.ПлощадьЗданияМестОбщегоПользования]

      В поле «Основание» указываем «Без основания».

      Справочник можно открыть через меню «Начисление услуг / Нормы потребления услуг».
