Справочник «Проекты помещений»

Данный справочник предназначен для хранения списка и характеристик проектов помещений. Ввод нового и редактирование уже существующего элемента осуществляется в форме элемента справочника.

При вводе нового элемента справочника необходимо заполнить реквизиты:

  • «Код» – код элемента справочника. Устанавливается автоматически;
  • «Наименование» – указывается наименование проекта помещения;
  • «Здание» – указывается элемент справочнка Здания, к которому относиться проект помещения; 
  • «Индивидуальное отопление» – указывается признак использования в помещении индивидуального отопления. Используется для расчета услуг отопления по постановлению №1708 от 28.12.2018; 
  • «Описание» – указывается описание для проекта помещения, если требуется;
  • «Список стояков проекта»- табличная часть, предназначенная для хранения стояков (элемент справочника Стояки), относящихся к текущему проекту помещения.

Вверху формы расположены кнопки:

  • «Записать и закрыть» – записать элемент и закрыть форму элемента справочника;
  • «Записать» – записать элемент справочника.

 

Справочник можно открыть из «Объекты учета/ См. также/ Проекты помещений».
