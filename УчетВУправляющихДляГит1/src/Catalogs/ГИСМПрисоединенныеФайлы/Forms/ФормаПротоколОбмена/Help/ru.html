Форма протокол обмена

Протокол обмена предназначен для просмотра информации о выполненных и предстоящих операциях обмена с системой маркировки (ГИСМ).

Из меню Действия можно выполнить новые операции обмена (либо по двойному клику на соответствующем пункте протокола).

Команда Показать сообщения XML предназначена для просмотра технической информации для переданного сообщения обмена.
