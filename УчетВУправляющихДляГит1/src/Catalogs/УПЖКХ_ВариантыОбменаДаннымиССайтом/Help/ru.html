Справочник «Варианты обмена данными с сайтом»

Данный справочник предназначен для выбора видов данных, которыми следует обмениваться с сайтом управляющей организации.

На форме справочника располагаются следующие элементы:

  • «Код» — код элемента справочника устанавливается автоматически;
  • «Наименование» — наименование варианта обмена;
  • «Настройка доступа» — настройка доступа, выбранная из справочника «Настройки доступа к сайту»;
  • «Настройка обмена» — настройка обмена данными с сайтом, выбранная из справочника «Настройки обмена данными с сайтом ЖКХ»; 
  • «Выгрузка данных по л/с» — флаг, при установке которого будет осуществляться выгрузка на сайт учетных сведений по лицевым счетам.
    А именно:
    • Основные сведения по лицевым счетам (сведения о собственнике, адресе проживания, параметрах доступа на сайт и др.);
    • Сведения о начислениях за текущий месяц, с детализацией по услугам;
    • Сведения об индивидуальных и общедомовых приборах учета.
  • «Загрузка сведений об аккаунтах» — флаг, при установке которого будет осуществляться загрузка с сайта сведений об аккаунтах по лицевым счетам.
    А именно:
    • Факт наличия аккаунта по лицевому счету;
    • Контактный email лицевого счета;

    Если используется актуальная версия модуля «Сайт ЖКХ», данный флаг будет отсутствовать. 
  • «Выгрузка из программы на сайт электронных паспортов» — флаг, при установке которого будет осуществляться выгрузка на сайт электронных паспортов зданий, созданных документами «Электронные паспорта (пост. №1468)».
  • «Загрузка показаний по ПУ» — флаг, при установке которого будет осуществляться загрузка с сайта сведений о показаниях индивидуальных проборов учета;
  • «Загрузка реестра платежей (оператор «Монета»)» — флаг, при установке которого будет осуществляться загрузка с сайта сведений об оплатах, произведенных с помощью оператора «Монета»;
  • «Использовать регламентное задание» — флаг, при установке которого обмен данными будет осуществляться автоматически посредством регламентного задания.
  • «Описание» — описание текущего варианта обмена.
