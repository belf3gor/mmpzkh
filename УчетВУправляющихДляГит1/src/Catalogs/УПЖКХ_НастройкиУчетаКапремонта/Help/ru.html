Справочник «Настройки учета по услугам капитального ремонта»

Данный справочник предназначена для хранения настроек ведения учета по услугам капитального ремонта в организациях и подчинен справочнику «Организации».

Форма списка

В командной панели формы списка содержится кнопка «Включить действие настройки» / «Отменить действие настройки», предназначенная для изменения действия настроек учета по услугам капитального ремонта в разрезе организаций.

Для одной организации может быть создана только единственная настройка ведения учета по услугам капитального ремонта. При добавлении новой настройки предоставляется выбор из тех организаций, для которых настройка еще не определена. Если в информационной базе настройки учета по капитальному ремонту заданы для всех организаций, то при попытке создания новой настройки будет выведено соответствующее сообщение.

Форма элемента

На данной форме расположены следующие реквизиты и табличные части:

  • Организация - организация, для которой создана настройка (ссылка на элемент справочника «Организации»).
  • Вести раздельный учет по услугам капремонта - флажок, характеризующий признак ведения раздельного учета по услугам капитального ремонта в организации.
  • Настройка расчета пени - указывается настройка из справочника «Настройки расчета пени», в соответствии с которой будет выполняться расчет пени в случае несвоевременной и (или) неполной оплаты начисленных услуг капитального ремонта. При этом настройки расчета пени для услуг капитального ремонта могут отличаться от настроек прочих услуг.
  • Услуги капитального ремонта - табличная часть, предназначенная для хранения услуг, относящихся к капитальному ремонту. Табличная часть доступна для редактирования только при установленном флажке «Вести раздельный учет по услугам капремонта».

Особенности записи элементов справочника:

1. Если табличная часть «Услуги капитального ремонта» заполнена и флажок «Вести раздельный учет по услугам капремонта» не установлен, то при записи элемента произойдет очистка табличной части. Предварительно будет выведено соответствующее предупреждение.

2. Если флажок «Вести раздельный учет по услугам капремонта» установлен и табличная часть «Услуги капитального ремонта» не заполнена, то при записи элемента флажок будет снят. Предварительно будет выведено соответствующее предупреждение.
