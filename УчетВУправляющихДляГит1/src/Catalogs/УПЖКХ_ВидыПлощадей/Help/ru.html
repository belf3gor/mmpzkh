Справочник «Виды площадей»

Данный справочник предназначен для хранения списка видов площадей.

При добавлении нового элемента справочника, в поле «Наименование» указывается название вида площади. «Код» устанавливается автоматически.

Поле «Идентификатор для формул»содержит уникальный идентификатор для норм потребления, рассчитываемых по формуле. По умолчанию генерируется из значения поля «Наименование», при необходимости можно указать вручную.

Например, для элемента справочника с наименованием “Жилая площадь” будет сформирован идентификатор “ЖилаяПлощадь”, если предложенный вариант по каким-либо причинам не устраивает, можно указать свой, например, “Жилая”.
