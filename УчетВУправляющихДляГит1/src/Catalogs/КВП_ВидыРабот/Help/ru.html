Справочник «Виды работ»

Данный справочник используется для хранения списка работ и материалов, необходимых для выполнения заявок по ремонту жилого и нежилого фонда.

При вводе нового элемента справочника указываются:

  • «Группа»: отображается группа для текущего вида работ;
  • «Код»: код элемента справочника. Устанавливается автоматически. Может быть изменен вручную;
  • «Наименование»: указывается наименование вида работ;
  • «Номенклатура работ»: указывается номенклатура работ. Номенклатура работ используется для отражения операций списания материалов и оказания услуг в регламентированном учете по выполненным работам аварийно-диспетчерской службы;
  • «Норма труда рабочих (чел/час)»: указывается норма времени, необходимая для выполнения работником данного вида работ;
  • «Состав работ»:
    • «Номенклатура / Вид материала»: является ссылкой на элемент справочника «Номенклатура» или «Виды материалов» - номенклатура или вид материалов, используемые при выполнении работ;
    • «Расходуемое количество»: указывается количество (расход) материалов по норме, которое будет списано при выполнении данной работы;
    • «Единица измерения»: указывается единица измерения из справочника «Классификатор единиц измерения»; 
    • «Объем выполняемой работы»: указывается коэффициент (норма) в расчете на которую указывается расход материалов. 
  • «Комментарий»: поле для ввода комментария.
