Справочник "Шаблоны договоров"

Справочник Шаблоны договоров предназначен для хранения шаблонов, используемых при формировании печатных форм договоров с контрагентами.

Поставляемые в комплекте с конфигурацией шаблоны можно редактировать или создавать на их основе собственные.

При помощи выбранного шаблона, для каждого договора можно сформировать печатную форму.

При создании или редактировании шаблона договора можно использовать предопределенные параметры, которые будут заполнены в печатной форме данными информационной базы автоматически.
