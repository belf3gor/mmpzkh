Справочник "Параметры типовой операции"

Справочник предназначен для хранения списка параметров, которые используются для заполнения документа Операция. Данный справочник подчинен справочнику "Типовые операции". В форме элемента используется следующие поля:

  • Имя параметра содержит название параметра, как оно будет отображаться на форме документа Операция
  • Тип параметра содержит тип выбираемого значения
  • Связь по владельцу содержит параметр владелец. Например для параметра Договор, можно указать параметр Контрагент, который будет устанавливаться в качестве ограничения при выборе значения Договора
  • Подсказка используется для описания параметра. Отображается либо в подсказке ввода, либо в подсказке по кнопке в документе Операция для данного параметра
