Справочник «Настройки выгрузки для мобильного приложения»

Справочник «Настройки выгрузки для мобильного приложения» предназначен для настройки обмена данными с мобильным приложением «Монитор руководителя предприятия ЖКХ»;

Для того, чтобы начать пользоваться мобильным приложением, необходимо зарегистрироваться в приложении и настроить расписание обмена данными между 1С и мобильным приложением. Для этого:

  1. В поле «Организация» выбрать организацию, данные по которой необходимо выгружать в мобильное приложение.
  2. Нажать на кнопку «Зарегистрироваться». При этом:
    • Текущей организации будут присвоены идентификатор и ПИН-код, которые отобразаться в полях «ИД» и «ПИН» соответственно.
    • Будет выполнена автоматическая выгрузка данных из 1С на сервер мобильного приложения.
  3. Установить приложение на Ваше мобильное устройство. Ссылку на дистрибутив приложения можно получить, нажав на надпись «Google play».
  4. Ввести в мобильное приложение, идентификатор и ПИН-код, указанные в полях «ИД» и «ПИН» соответственно.
  5. Настроить расписание обмена, установив флаг «Выполнять автоматический обмен про расписанию» в группе «Настроить расписание обмена».

Обмен данными с мобильным приложением будет выполняться автоматически по расписанию, установленному в группе «Настроить расписание обмена».

Имеется возможность выполнить разовый обмен данными. Для этого следует нажать на кнопку «Выполнить сейчас», которая находится в группе «Выполнить обмен сейчас».


При необходимости можно сгенерировать новый ПИН-код (ключ, с помощью которого будут шифроваться данные при обмене с приложением). Для этого необходимо нажать на кнопку «Сгенерировать новый ПИН». После генерации нового ПИН-кода будет выполнена автоматическая выгрузка данных из 1С на сервер мобильного приложения.


В верхней части формы элемента справочника расположены кнопки:

  • «Записать и закрыть» – записать элемент и закрыть форму элемента справочника;
  • «Записать» – записать элемент справочника;
  • «Все действия» – открыть меню со всеми кнопками командной панели формы.
