Справочник "Виды регистров учета"

Справочник предназначен для хранения видов регистров учета, которые могут быть подписаны электронной подписью и хранятся в базе данных.
Данный справочник является служебным. Добавление новых элементов запрещено. Изменение существующих элементов доступно только администратору системы.
