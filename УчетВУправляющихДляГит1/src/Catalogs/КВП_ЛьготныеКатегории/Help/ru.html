Справочник «Льготные категории»

Данный справочник предназначен для хранения списка льготных категорий жильцов.

Список льготных категорий можно заполнить автоматически из предложенного списка по кнопке "Подбор". В открывшейся форме нужно выбрать льготную категорию и добавить ее в список двойным щелчком мыши.

При вводе нового элемента справочника указываются:

  • «Код» – код элемента справочника устанавливается автоматически;
  • «Наименование» – вводится наименование льготной категории.
  • «Полное наименование» – вводится полное наименование льготной категории.
