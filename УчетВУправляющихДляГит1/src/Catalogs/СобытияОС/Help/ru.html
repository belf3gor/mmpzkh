Справочник "События с основными средствами"

Справочник предназначен для хранения перечня событий с основными средствами организации, например, принятие к учету, перемещение, модернизация и т.п.

Для каждого события нужно выбрать из списка Вид события.

При регистрации документов по учету основных средств в поле Событие выбирается значение из справочника.
