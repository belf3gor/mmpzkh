Настройки универсальной выгрузки данных

Справочник хранит настройки отбора данных при выгрузке с помощь обработки "Универсальная выгрузка данных".

Доступны следующие отборы:

  • "Отбор по сумме задолженности" - выбирается один из параметров отбора:
    • "Все лицевые счета" - отбор данных по сумме задолженности не производится.
    • "Указать мин. сумму задолженности" - отбор данных по сумме задолженности производится по значению в поле "Минимальная сумма задолженности".
    • "С долгами" - отбор данных производится по сумме задолженности, которое больше нуля.
    • "С переплатами" - отбор данных производится по сумме задолженности, которое меньше нуля.
  • "Минимальная сумма задолженности" - сумма задолженности. которая устанавливается по настройке "Указать мин. сумму задолженности".
  • "Отбор по ответственному собственнику" - выбирается один из параметров отбора:
    • "Все лицевые счета" - отбор данных по ответственному собственнику не производится.
    • "Только физ.лица" - отбор данных производится по типу ответственного собственника лицевого счета, который является физическим лицом.
    • "Только юр.лица" - отбор данных производится по типу ответственного собственника лицевого счета, который является юридическим лицом.
  • "Отбор по наличию номера телефона" - выбирается один из параметров отбора:
    • "Все лицевые счета" - отбор данных по наличию номера телефона не производится.
    • "Только с номерами телефонов" - отбор данных производится по наличию номера телефона лицевого счета.
  • "Отбор по показаниям счетчиков" - выбирается один из параметров отбора:
    • "Только по счетчикам с показаниями" - отбор данных производится только по наличию показаний счетчиков.
    • "По всем счетчикам" - отбор данных по наличию показаний счетчиков не производится.
  • "Выгружать данные по" - выбирается один из параметров отбора на закладке "Выбор лицевых счетов":
    • "Всем лицевым счетам" - отбор по выбранным лицевым счетам не производится.
    • "Выбранным лицевым счетам" - отбор производится только по выбранным лицевым счетам.
    • "Всем лицевым счетам, кроме выбранных" - отбор производится только по лицевым счетам, не входящих в список выбранных.
  • "Выгружать данные по" - выбирается один из параметров отбора на закладке "Выбор услуг":
    • "Всем услугам" - отбор по выбранным лицевым счетам не производится.
    • "Выбранным услугам" - отбор производится только по выбранным услугам.
    • "Всем услугам, кроме выбранных" - отбор производится только по услугам, не входящих в список выбранных.


Справочник доступен в разделе "Интеграция со сторонними системами".
