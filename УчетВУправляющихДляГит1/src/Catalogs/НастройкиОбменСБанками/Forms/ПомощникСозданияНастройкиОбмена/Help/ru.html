Помощник создания настройки обмена

Воспользовавшись помощником, можно последовательно задать параметры организации, протестировать программу защиты информации, импортировать и настроить сертификат ЭП, изменить настройки, полученные из банка. Программа проверяет правильность выполненных настроек.
Результатом работы помощника будет открытая форма новой действующей настройки.

Набор параметров для заполнения на форме помощника отображается в зависимости от выбранного банка.
Возможно несколько вариантов создания настройки прямого обмена: Автоматическое получение настроек с сервера банка, Загрузка из файла, Ручное заполнение.

Автоматическое получение настроек с сервера банка
  • На форме помощника подключения выбрать вариант: Автоматическое получение настроек с сервера банка.

Программа предложит пройти аутентификацию на ресурсе банка, автоматически создаст настройки обмена по полученным от банка значениям и выполнит тестирование новой настройки. При этом возможны 2 варианта аутентификации на ресурсе банка:

  • По логину и паролю — электронные документы после отправки из программы импортируются в банковский модуль, где их нужно дополнительно подтвердить (акцептовать или подписать — в зависимости от регламента конкретного банка).
  • По сертификату подписи — обязательное использование электронной подписи для прямого обмена с банком. Необходимо сначала в операционной системе установить программу криптографии и добавить сертификат подписи в личное хранилище сертификатов операционной системы.

Получение настроек через загрузку из файла
  • На форме помощника подключения выбрать вариант: Загрузка из файла.
  • Указать путь к файлу настроек.

Файл для загрузки необходимо скачать из личного кабинета банка.

Ручное заполнение настроек
  • На форме помощника подключения выбрать вариант: Ручное заполнение.
  • Пройти аутентификацию на ресурсе банка (По логину и паролю/По сертификату).
  • Ввести идентификатор организации, который был присвоен в банке, и версию формата обмена.
  • Установить флажок, если перед отправкой в банк документы будут подписываться электронной подписью. Добавить сертификаты ключей электронной подписи.

Настройка прямого обмена через внешний модуль

Банк может представлять Внешний модуль собственной разработки для прямого обмена. В таком случае файл внешнего модуля загружается в информационную базу и используется при обмене. Для этого необходимо:

  • Выбрать Организацию и Банк.
  • Отметить вариант Ручное заполнение.
  • На следующем шаге установить флажок Внешний модуль и выбрать файл внешнего модуля.
