Справочник «Счетчики»

Данный справочник предназначен для хранения списка и характеристик используемых приборов учета. Ввод нового и редактирование уже существующего элемента осуществляется в форме элемента справочника.

При вводе нового элемента справочника необходимо заполнить реквизиты:

  • «Код» – код элемента справочника. Устанавливается автоматически;
  • «Наименование» – указывается наименование прибора учета;
  • «Идентификатор» – указывается номер или набор символов, по которому можно точно идентифицировать этот прибор учета. Индентификатор может быть использован, например, для загрузки данных по показаниям приборов учета из автоматизированных систем сбора показаний приборов учета;
  • «Модель» – указывается модель прибора учета;
  • «Заводской номер» – указывается заводской номер прибора учета;
  • «Услуга» – из справочника «Услуги» выбирается услуга, по которой осуществляется съём показаний данным прибором учета;
  • «Тарифность» – указывается тарифность счетчика (прибор учета может быть 1-но, 2-х или 3-х тарифный);
  • «Способ регистрации показаний» – указывается каким образом прибор учета регистрирует показания:
    • "Нарастающим итогом" (в этом случае при расчете начислений расход по прибору учета будет определяться как разность между текущими и предыдущими показаниями) или 
    • "Расход за период расчета" (в этом случае при расчете начислений в качестве расхода будет браться само показание);
  • «Разрядность» – указывается разрядность счетчика (разрядность не может быть более 15);
  • «Коэффициент трансформации» – указывается коэффициент трансформации показаний прибора учета (при расчете начислений расход по прибору умножается на коэффициент трансформации);
  • «Дата поверки» – справочно выводится дата поверки прибора учета. Дата поверки указывается при установке прибора учета в документе «Установка счетчика». С даты поверки прибор учета будет автоматически отключен;
  • «Использовать температурный коэффициент» – в зависимости от того, имеет ли прибор учета температурную компенсацию, устанавливается признак использования температурных коэффициентов. Если признак использования температурных коэффициентов установлен, то начисления по показаниям данного прибора учета производятся с учетом температурных коэффициентов, которые заданы в регистре сведений «Значения температурных коэффициентов»;
  • «Используется в качестве "выходного" прибора учета» – в зависимости от места установки прибора устанавливается признак является ли прибор учета "выходным". Начисления по показаниям прибора учета с установленным флажком не производятся.
    Расход "выходных" приборов учета может уменьшать расход "входных" приборов учета: для этого необходимо настроить зависимость приборов учета в регистре сведений «Зависимые приборы учета».

Вверху формы расположены кнопки:

  • «Записать и закрыть» – записать элемент и закрыть форму элемента справочника;
  • «Записать» – записать элемент справочника.

 

Справочник можно открыть из «Учет в ЖКХ / Справочники / Услуги / Счетчики».
