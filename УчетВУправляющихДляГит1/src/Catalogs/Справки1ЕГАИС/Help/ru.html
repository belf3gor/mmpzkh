Справки 1 ЕГАИС

Справочник предназначен для хранения данных по справкам 1, прилагаемых к ТТН. Справочник заполняется автоматически при загрузке из ЕГАИС Товарно-транспортных накладных.

Получить полные данные по одной справке можно, выполнив команду Запросить справку из формы списка. При этом, необходимо будет указать Регистрационный номер справки в системе ЕГАИС.
