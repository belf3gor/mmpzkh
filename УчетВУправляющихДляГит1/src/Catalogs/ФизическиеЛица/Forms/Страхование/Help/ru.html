Страхование

В форме указываются сведения, которые влияют на расчет страховых взносов по этому человеку.

Статус застрахованного лица

По умолчанию в программе с целью расчета страховых взносов человек считается гражданином Российской Федерации, даже если в его личных данных было указано гражданство другой страны. При этом период, с которого это статус действует, указывать не требуется. При расчете взносов за любой период будет считаться, что в этом периоде сотрудник является гражданином РФ.

Если на работу принимается иностранец

В случае если сотрудник является иностранцем, следует выбрать как соответствующий статус, так и месяц, с которого этот статус действует. При расчете взносов будет считаться, что за период, предшествующий этому месяцу, сотрудник является гражданином РФ. Начиная с указанного месяца при расчете будет учитываться уже другой, выбранный статус.

Статус выбирается из перечня, который разработан в соответствии с действующим законодательством. При выборе каждого статуса на форму выводится подсказка, поясняющая его смысл.

Для временно пребывающих иностранных граждан следует отдельно указать периоды, на которые с ними были заключены договоры длительностью более чем 6 месяцев. Сведения о менее продолжительных договорах указывать не следует.

Обращаем внимание, что для временно пребывающих высококвалифицированных иностранных специалистов следует выбрать статус Иностранец, временно пребывающий на территории РФ, при этом периоды работы не следует указывать независимо от длительности трудового договора.

Если страховой статус сотрудника изменился

Если статус сотрудника изменился, например, иностранец получил вид на жительство, то следует выбрать новый статус и указать месяц, с которого этот статус действует. Начиная с этого месяца при расчете взносов будет учитываться этот вновь установленный статус. До этого месяца будет учитываться ранее установленный статус.

Историю изменения сведений о страховом статусе можно просмотреть (а также отредактировать) по нажатию соответствующей ссылки.

Если в процессе редактирования статус был изменен, то в последующем, при записи данных сотрудника, будет предложено уточнить, с чем связаны произведенные изменения (исправлена ошибка в данных "задним числом" или же статус сотрудника в действительности изменился).

Помимо непосредственно влияния статуса на расчет взносов, он также используется при формировании отчетности по ним.

Сведения об инвалидности

Если лицо является инвалидом, то страховые взносы за него можно уплачивать по льготному тарифу (если этот льготный тариф в совокупности ниже чем основной тариф организации).
Сведения об инвалидности указываются в одноименном разделе карточки. Дата выдачи справки не влияет на расчет взносов, влияние оказывает месяц в поле Сведения действуют с – если флажок Есть справка об инвалидности устанавливается, то пониженный тариф будет применяться именно с этого месяца.

Прервать влияние сведений об инвалидности на расчет взносов можно и вручную, сняв флажок Есть справка об инвалидности и указав в поле Сведения действуют с месяц, начиная с которого взносы необходимо рассчитывать по общему тарифу.

Внимание! Если указан срок действия справки, то взносы автоматически будут рассчитываться по общему тарифу начиная с месяца, в который входит эта дата.
Например, если указан срок действия 31.12.2014, то уже в декабре взносы будут рассчитаны по общему тарифу.
