Справочник «Присоединенные файлы (Подсистема ЖКХ)»

Данный справочник предназначен для хранения присоединенных файлов.

 

P.S. На данный момент присоединенные файлы ко всем объектам конфигурации хранятся в одном месте, в данном справочнике.

Если мы для роли открываем возможность чтения и изменения данного справочника, то пользователю с этой ролью (теоретически) становятся доступны все присоединенные файлы.

Т.е. он может получить присоединенные файлы объектов, которые ему даже не доступны.

В типовой БП организовано разделение хранения присоединенных файлов по объектам конфигурации, что решает проблему доступа к присоединенным файлам недоступных пользователю объектов.
