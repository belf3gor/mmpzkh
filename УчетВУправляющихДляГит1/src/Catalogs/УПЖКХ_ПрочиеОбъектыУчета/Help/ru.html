Справочник «Прочие объекты учета, оборудование»

Данный справочник предназначен для учета коммунальной инфраструктуры (котельных, водозаборов, трубопроводов, электросетей), а также оборудования, устанавливаемого на объекты учета (здания, подъезды, помещения). Каждое оборудование принадлежит определенному виду оборудования, а также имеет собственный идентификатор оборудования.

В форме элемента можно указать следующие реквизиты:

  • Ответственный - ответственное физическое лицо или подразделение организации, отвечающее за прочий объект учета или оборудование;
  • Основное средство - выбирается из справочника «Основные средства». Основное средство для учета прочего объекта учета или оборудования в бухгалтерском учете.

Сведения о состоянии оборудования

На данной закладке указывается информация о состоянии оборудования на определенный период. Оборудование может быть введено в эксплуатацию либо снято с эксплуатации.

Виды выполняемых работ

На данной закладке есть возможность указать перечень видов работ из справочника «Виды работ», которые могут выполняться с текущим оборудованием. Если вид работ должен выполняться в обязательном порядке через определенные промежутки времени, то он определяется как "Плановый". Для планового вида работ указывается периодичность выполнения (День, Месяц, Год) и количество периодов, через которое требуется повторять выполнение работ.

Используемые материалы

На данной закладке есть возможность указать материалы (справочник «Номенклатура» ), совместимые с данным оборудованием, при этом каждый материал относится к определенному Виду материалов.

Ответственные мастера

Есть возможность указания ответственных мастеров по объекту. Можно указать вид мастера и самого мастера для каждого объекта.
