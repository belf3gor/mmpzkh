Справочник «Группы услуг»

Справочник предназначен для группировки услуг по произвольным логическим признакам. Для этого в справочнике предусмотрена возможность определения произвольных видов групп, а также групп услуг внутри каждого вида. Справочник имеет два уровня иерархии:

  • На первом уровне иерархии справочника задаются виды групп услуг. По умолчанию в справочнике уже заданы некоторые виды групп, которые используются, например, для установки параметров расчета льгот. При этом можно определить новые виды групп или изменить существующие. При вводе нового вида групп услуг необходимо ввести наименование и описание вида группировки.
  • На втором уровне иерархии справочника задаются группы услуг, при вводе которых необходимо ввести следующие сведения:
    • «Вид групп услуг» - вид групп услуг, к которому относится текущая группа услуг;
    • «Наименование» - наименование группы услуг;
    • «Список услуг группы» – список услуг, входящих в данную группу. Также отношение каждой отдельной услуги к той или иной группе услуг определенного вида можно задать в форме услуги.
Примечание
Любая услуга может быть включена в состав только одной группы услуг определенного вида групп.

Перечень механизмов учета в которых можно задействовать группировки услуг:

  • Установка некоторых параметров расчета льгот, доступных в Настройках учетной политики ЖКХ, в разрезе групп услуг.
Примечание
В дальнейшем список механизмов учета, в которых могут быть задействованы группировки услуг, может быть расширен.
