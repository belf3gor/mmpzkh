Справочник «Регистрации в налоговом органе»

Справочник предназначен для хранения списка налоговых органов, в которых зарегистрирована организация. В справочник заносятся все территориальные налоговые органы, где организация состоит на учете.

Информация справочника используется при заполнении регламентированных отчетов.

Как правило, справочник вызывается для ввода и редактирования информации из формы элемента справочника Организации.

Организация - в поле указывается организация, состоящая на учете в данной инспекции. При вызове справочника из формы элемента справочника Организации или формы элемента справочника Подразделения это поле недоступно для изменения.

Подразделение - в поле указывается обособленное подразделение организации, по месту нахождения которого организация зарегистрирована в данной инспекции. Это поле отображается только при вводе регистрации в налоговом органе из формы элемента справочника Подразделения.

КПП - код причины постановки на учет организации. Значение реквизит используется для заполнения поля КПП в регламентированных отчетах. В случае, если данные вносятся предпринимателем, данный реквизит на форме не отображается и не вводится.

Код налогового  органа - код налогового органа, обычно четыре цифры, например 7722. Реквизит будет использоваться на титульных листах регламентированных отчетов.

Краткое наименование налогового органа - краткое наименование налогового органа, в котором зарегистрирована организация. Краткое наименование используется для представления налогового органа в программе.

Полное наименование налогового органа - официальное наименование налогового органа, в котором зарегистрирована организация. Полное наименование используется при автозаполнении титульных листов регламентированных отчетов.

ОКТМО налогового органа - код налогового органа по ОКТМО.

ОКАТО налогового органа - код налогового органа по ОКАТО.

Наименование подразделения - в поле указывается полное наименование обособленного подразделения организации, по месту нахождения которого организация зарегистрирована в данной инспекции. Это поле отображается только при вводе регистрации в налоговом органе из формы элемента справочника Подразделения.

Отчетность подписывает - переключатель предназначен для выбора лица, подписывающего отчетность. Это может быть руководитель организации или иное лицо, действующее на основании предоставленных ему полномочий. В этом случае требуется заполнить сведения о представителе.  

Представитель

Представитель - в случае если интересы налогоплательщика в территориальном налоговом органе представляет законный представитель налогоплательщика (действующий на основании соответствующих документов), необходимо с помощью переключателя указать тип представителя (юридическое или физическое лицо) и выбрать его из соответствующего справочника.

ФИО уполномоченного лица - в поле указываются фамилия, имя и отчество лица, имеющего полномочия действовать от имени представителя-юридического лица. Если представитель - физическое лицо, это поле недоступно для изменения.

Номер, дата и вид документа - в поле указываются номер, дата и вид документа, подтверждающего полномочия представителя, например, доверенности.

Доверенность ЭДО - поле предназначено для выбора доверенности на осуществление электронного документооборота с налоговым органом. Это поле заполняется только в том случае, если электронный документооборот осуществляется через представителя.
