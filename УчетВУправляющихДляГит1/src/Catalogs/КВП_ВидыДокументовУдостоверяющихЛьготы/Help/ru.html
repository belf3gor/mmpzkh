Справочник «Виды документов, удостоверяющих льготы»

Данный справочник предназначен для хранения списка документов, удостоверяющих льготы граждан.

Список видом документов, удостоверяющих льготы можно заполнить автоматически из предложенного списка по кнопке "Подбор". В открывшейся форме нужно выбрать вид документа и добавить его в список по кнопке ОК. 

При вводе нового элемента справочника указываются:

  • «Код» – код элемента справочника устанавливается автоматически;
  • «Наименование» – вводится наименование документа, удостоверяющего льготы.

Справочник можно открыть через меню «Учет в ЖКХ / Льготы / Виды документов, удостоверяющих льготы».
