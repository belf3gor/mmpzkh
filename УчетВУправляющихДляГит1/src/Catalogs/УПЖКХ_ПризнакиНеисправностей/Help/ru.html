Справочник "Признаки неисправностей"

Данный справочник предназначен для хранения списка неисправностей аварийно-диспетчерской службы. Для каждого вида неисправности определяется перечень видов работ, которые необходимо выполнить для устранения данной неисправности.

При вводе нового элемента справочника можно заполнить реквизиты:

  • Группа - группа неисправностей, к которой относится текущий признак неисправности.
  • Наименование - наименование признака неисправности.

В табличной части Виды работ указывается перечень видов работ из справочника "Виды работ", которые необходимо выполнить для устранения данной неисправности.
