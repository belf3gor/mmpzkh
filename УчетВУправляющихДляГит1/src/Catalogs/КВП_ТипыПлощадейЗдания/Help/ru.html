Справочник «Типы площадей здания»

Данный справочник предназначен для хранения списка типов площадей здания.

При добавлении нового элемента справочника, в поле «Наименование» указывается название типа площади. «Код» устанавливается автоматически.

Поле «Идентификатор для формул» содержит уникальный идентификатор для норм потребления, рассчитываемых по формуле. По умолчанию генерируется из значения поля «Наименование», при необходимости можно указать вручную.

Например, для элемента справочника с наименованием “Общая площадь кровли” будет сформирован идентификатор “ОбщаяПлощадьКровли”, если предложенный вариант по каким-либо причинам не устраивает, можно указать свой, например, “ПлощадьКровли”.
