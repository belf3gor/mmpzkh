Справочник "Шаблоны сообщений"

Справочник содержит настройки шаблонов сообщений для заполнения документа СМС-сообщение.

На форме элемента справочника указываются следующие данные:

  • «Наименование»: поле, в котором указывается наименование шаблона;
  • «Код»: поле, в котором указывается код элемента справочника;
  • «Правило отбора получателей»: правило, по которому отбирается список получателей для данного шаблона;

Группа «Настройка шаблона»

Здесь заполняются сведения о тексте СМС-сообщения, такие, как:

  • «Содержание»: поле, в которое вводится текст СМС-сообщения, который будет использоваться при выборе данного шаблона в документе СМС-сообщение;
  • «Предварительный просмотр»: отображение текста СМС-сообщения, который будет видеть получатель;
  • «Добавить параметр»: кнопка, которая добавит в текст СМС-сообщения автоматическую вставку например имя пользователя или название организации;

Группа «Авторассылка»

Здесь расположены настройки авторассылки по данному шаблону:

  • «Использовать в авторассылке»: флаг, определяющий использование данного шаблона в автоматической рассылке СМС-сообщений;
  • «Настройки авторассылки»: ссылка для перехода на форму настроек рассылки СМС-сообщений.
