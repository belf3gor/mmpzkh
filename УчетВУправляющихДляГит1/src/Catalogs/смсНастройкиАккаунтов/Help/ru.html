Справочник "Учетные записи СМС-рассылки"

Справочник содержит настройки учётных записей для рассылки СМС-сообщений.

На форме элемента справочника расположены следующие реквизиты:

  • «Наименование»: указывается наименование настроек учётной записи;
  • «Код»: код элемента справочника;
  • «Организация»: организация, которой принадлежит аккаунт;
  • «Пользователь»: имя пользователя, необходимое для авторизации на сервере СМС-рассылки;
  • «Пароль»: пароль, необходимый для авторизации на сервере СМС-рассылки;
  • «Имя или номер отправителя»: номер телефона или имя, которое указывается в качестве отправителя при рассылке СМС;
  • «Начало периода запрета»: дата начала периода времени, в которое отправка сообщений будет запрещена;
  • «Конец периода запрета»: дата окончания периода времени, в которое отправка сообщений будет запрещена;
  • «Транслитерация кириллицы в латиницу»: признак использования переведа из кириллицы в латиницу при отправке сообщений, транслитерация происходит по правилам ГОСТ 7.79-2000 "Система стандартов по информации, библиотечному и издательскому делу. Правила транслитерации кирилловского письма латинским алфавитом.";
  • «Предупреждать, когда длина сообщения превышает N символов»: N - количество символов в сообщении при котором необходимо предупреждать;
  • «Использовать прокси-сервер»: признак использования прокси-сервера для подключения к серверу СМС-рассылки.
