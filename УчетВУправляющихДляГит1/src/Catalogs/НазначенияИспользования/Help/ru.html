Справочник "Назначения использования"

Справочник предназначен для хранения сведений о порядке погашения стоимости переданной в эксплуатацию специальной одежды и специальной оснастки.

Назначение использования указывается при передаче в эксплуатацию спецодежды и спецоснастки.

Просмотреть назначение использования спецодежды, спецоснастки можно в справочнике Номенклатура по ссылке Назначения использования.

При вводе назначения использования нужно заполнить реквизиты:

  • Наименование - наименование назначения использования.
  • Количество по нормативу выдачи - количество по нормативу передаваемой спецодежды, спецоснастки. Значение используется для подстановки по умолчанию в документы.
  • Способ погашения стоимости - способ погашения стоимости спецодежды, спецоснастки:
    • Линейный - погашение стоимости производится пропорционально в течение срока полезного использования.
    • Погашать стоимость при передаче в эксплуатацию - погашение стоимости производится сразу при передаче в эксплуатацию.
    • Пропорционально объему продукции (работ, услуг) - погашение стоимости производится пропорционально объему продукции (работ, услуг). Для регистрации объема продукции (работ, услуг) предназначен документ Выработка материалов.
  • Способ отражения расходов - способ отражения в затратах погашенной стоимости спецодежды, спецоснастки. Погашение стоимости спецодежды, спецоснастки производится при выполнении регламентной операции закрытия периода.
