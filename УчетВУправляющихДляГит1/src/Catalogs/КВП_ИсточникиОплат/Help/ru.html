Справочник «Источники оплат»

Справочник «Источники оплат» предназначен для хранения источников принятых оплат за услуги. Используется в документе «Регистрация оплаты».

Ввод новых элементов производится через форму списка элементов справочника.

Справочник содержит реквизиты:

  • «Код» – код элемента справочника, устанавливается автоматически;
  • «Наименование» – наименование элемента справочника.
