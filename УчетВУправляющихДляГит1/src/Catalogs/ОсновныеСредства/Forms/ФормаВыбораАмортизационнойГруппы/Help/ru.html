С помощью этой формы можно выбрать код основного средства по общероссийскому классификатору основных фондов (ОКОФ) и амортизационную группу, в которую данный код ОКОФ входит. В левой части формы находится классификатор ОКОФ в виде дерева. При выборе элемента этого дерева в правой части окна выводится список амортизационных групп, в которые выбранный элемент входит. В случае, если выбранный элемент не входит ни в одну из амортизационных групп по классификатору, будет предоставлена возможность выбрать любую из возможных амортизационных групп. По двойному щелчку мыши на амортизационной группе в данные элемента справочника "Основные Средства" будет перенесен код ОКОФ и амортизационная группа. При двойном нажатии на элемент дерева ОКОФ будет перенесено только значение кода ОКОФ. Также перенести выбранные данные можно по горрячей клавише (Ctrl+Enter). Позже,  при желании, в окне для редактирования элемента основных средств, вы сможете вручную поменять значение амортизационной группы и кода ОКОФ.
