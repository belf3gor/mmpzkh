Регистр «НДС предъявленный, реализация 0%»

Предназначен для хранения информации о суммах НДС, предъявленных поставщиками приобретенных ценностей, связанных с операциями реализации по нулевой ставке НДС.

Информация хранится в следующем разрезе:

  • Счет-фактура – первичный документ поступления;
  • Состояние – текущее положение по отношению к подтверждению нулевой ставки НДС, может принимать одно из следующих значений:
    • Ожидается подтверждение 0% - когда 180-дневный срок еще не истек;
    • Не подтверждена реализация 0% - когда начислен НДС на реализацию после истечения 180-дневного срока;
    • Подтверждена реализация 0% - когда собран необходимый пакет документов для подтверждения нулевой ставки НДС
  • Документ отгрузки – документ реализации, где была использована приобретенная ценность;
  • Вид ценности;
  • Ставка НДС;
  • Счет учета НДС – указывается субсчет счета 19 «НДС по приобретенным ценностям»;
  • Сумма без НДС;
  • НДС;
  • Дата события;
