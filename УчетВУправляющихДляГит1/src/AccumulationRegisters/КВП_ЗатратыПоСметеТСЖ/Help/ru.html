Регистр накопления «Затраты по смете»

Данный регистр предназначен для учета информации о планируемых затратах по смете ТСЖ.

В регистре используются измерения:

  • «Организация» - является ссылкой на элемент справочника «Организации»;
  • «Источник» - является ссылкой на элемент справочника «Назначение целевых средств»;
  • «Статья затрат» - является ссылкой на элемент справочника «Статьи затрат»;
  • «Периодичность» - определяет периодичность составления сметы;
  • «Начало периода» - начало периода;
  • «Конец периода» - окончание периода.

В регистре используются ресурсы:

  • «Сумма доход» - указана сумма доходов по смете;
  • «Сумма расход» - указана сумма расходов по смете.

Движения по данному регистру производятся с помощью документов «Смета ТСЖ».
