Регистр накопления «Взаиморасчеты с должниками»

Данный регистр предназначен для учета информации о взаиморасчетах с должниками.

В регистре используются измерения:

  • «Организация» - является ссылкой на элемент справочника «Организации»;
  • «Лицевой счет» - является ссылкой на элемент справочника «Лицевые счета» или «Контрагенты»;
  • «Договор» - является ссылкой на элемент справочника «Договоры контрагентов»;
  • «Дата суда» - значение типа "Дата";
  • «Дата передачи приставам» - значение типа "Дата";
  • «Решение суда» - значение типа "Срока";
  • «Окончание» - значение типа "Строка".

В регистре используются ресурсы:

  • «Сумма задолженности» - указана сумма задолженности должника;
  • «Сумма пени» - указана сумма пени должника;
  • «Госпошлина» - указана сумма госпошлины;
  • «Погашено до суда» - указана сумма погашения задолженности до суда;
  • «Сумма иска по решению суда» - указана сумма иска по решению суда;
  • «Взысканная сумма» - указана взысканная с должника сумма.

Движения по данному регистру производятся документом «Иск».
