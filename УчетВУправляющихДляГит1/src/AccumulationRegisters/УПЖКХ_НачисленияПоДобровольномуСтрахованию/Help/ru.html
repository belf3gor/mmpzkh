Регистр накопления «Начисления по добровольному страхованию»

Данный регистр предназначен для учета сведений о начислениях по добровольному страхованию имущества собственников помещений.

В регистре используются измерения:

  • «Организация» - является ссылкой на элемент справочника «Организации»;
  • «Лицевой счет» - является ссылкой на элемент справочника «Лицевые счета»;
  • «Договор» - является ссылкой на элемент справочника «Договоры контрагентов»;
  • «Услуга» - является ссылкой на элемент справочника «Услуги»;
  • «Месяц начисления» - значение типа «Дата»;
  • «Тариф» - значение типа «Число».

В регистре используются ресурсы:

  • «Количество» - значение типа «Число»;
  • «Сумма начисления» - указана сумма начисления.

Движения по данному регистру производятся документами: «Ввод фактических затрат на объект», «Начисление услуг», «Разовое начисление услуг», «Регистрация оплаты», «Начисление услуг в сторонней программе».
