Регистр накопления «Начисления по рассрочке»

Данный регистр предназначен для учета информации по предоставленной рассрочке.


В регистре используются следующие измерения:
  • «Организация» - ссылка на элемент справочника «Организации»;
  • «Заявление на рассрочку» - ссылка на документ «Заявление на рассрочку»;  
  • «Лицевой счет» - ссылка на элемент справочника «Лицевые счета»;
  • «Услуга» - ссылка на элемент справочника «Услуги»;
  • «Месяц рассрочки» - дата, соответствующая последнему дню месяца, за который предоставлена рассрочка;
  • «Вид операции» - ссылка на значение перечисления «Рассрочка»;

В регистре используются следующие ресурсы:
  • «Сумма начисления» - сумма изменения остатка по рассрочке;

Движения по данному регистру производятся документами «Начисление услуг», «Регистрация оплаты», «Заявление на рассрочку», «Закрытие периода по начислениям ЖКХ».
