Регистр накопления «Данные для отражения начислений в регл. учете»

Данный регистр предназначен для учета данных к отражению начислений в регл. учете в разрезе месяцев отражения.

В регистре используются измерения:

  • «Организация» - организация, по которой производится отражение начислений в регл. учете;
  • «Месяц отражения» - месяц, в рамках которого производится отражение начислений в регл. учете;
  • «Услуга» - услуга, начисление которой необходимо отразить в регл. учете;
  • «Лицевой счет» - лицевой счет, начисление которого отражается в регл. учете;
  • «Тип начисления» - тип отражаемого начисления, который может принимать следующие значения: «Начисление» - используется для отражения основных начислений, «Льгота» - для отражения предоставленных льгот, «Пени» - для отражения начисленных пени, «Корректировка» - для отражения корректировок начислений;
  • «Вид корректировки» - вид отражаемой корректировки, который может принимать следующие значения: «Перерасчет» – корректировка по перерасчету или по согласованию сторон, «Исправление ошибки» – корректировка по ошибке расчета или техническая корректировка. Значение вида корректировки требуется для определения вида операции документа «Корректировка реализации», с помощью которого необходимо производить отражение корректировки в регл. учете;
  • «Период корректировки» - месяц, за который произведена корректировка начисления в текущем месяце отражения. Значение данного измерения используется для определения месяца отражения, в котором было отражено скорректированное начисление, и поиска в нем документа отражения скорректированного начисления для отражения текущей корректировки путем формирования документа «Корректировка реализации» на исходный документ отражения начислений.

В регистре используются ресурсы:

  • «Сумма» - сумма начисления, пени, льготы или корректировки, подлежащая к отражению в регл. учете.

Техническая информация:

Приход начислений для отражения в регл. учете в данном регистре производится основными документами начисления услуг.

Списание отраженных в регл. учете начислений в данном регистре производится документом «Отражение начислений в регламентированном учете».
