Регистр накопления «Расчет приборов учета»

Данный регистр предназначен для хранения расхода по приборам учета.

В регистре используются измерения:

  • «Объект» - объект, на который установлен счетчик;
  • «Счетчик» - ссылка на элемент справочника «Счетчики»;
  • «Услуга» - ссылка на элемент справочника «Услуги».

В регистре используются ресурсы:

  • «Дневное показание» - дневной расход по счетчику;
  • «Ночное показание» - ночной расход по счетчику;
  • «Пиковое показание» - пиковый расход по счетчику.

Движения по данному регистру производятся документами «Ввод показаний счетчика», «Начисление услуг».
