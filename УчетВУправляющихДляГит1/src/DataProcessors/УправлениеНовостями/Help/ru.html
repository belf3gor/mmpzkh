Управление настройками и обновлением для новостей

В этой форме осуществляются все основные операции управления подсистемой новостей.

Закладка "Получение новостей"

На этой закладке собраны все настройки, касающиеся подключения к серверам новостей фирмы 1С

  • Разрешить обновления через интернет - если выключить, то подсистема новостей не будет делать попыток получения классификаторов и новостей с серверов фирмы 1С
  • Настройки прокси-сервера - по этой кнопке можно настроить параметры прокси-сервера
  • Обновление классификаторов, таймаут соединения - время (секунд), после которого попытка соединения с сервером классификаторов будет считаться неуспешной
  • Обновление классификаторов, количество попыток соединения - количество попыток соединения с сервером классификаторов
  • Обновление новостей, таймаут соединения - время (секунд), после которого попытка соединения с сервером новостей 1С будет считаться неуспешной
  • Обновление новостей, количество попыток соединения - количество попыток соединения с сервером новостей фирмы 1С
  • Обновление новостей, количество попыток получения файла новостей - количество попыток получения файла новостей
  • Интервал обновления списка новостей у пользователей - интервал (в минутах) автоматического обновления списка новостей на пользовательских компьютерах

Закладка "Обновление классификаторов"

На этой закладке можно посмотреть список всех классификаторов, которые задействованы в этой конфигурации. Также по кнопке "Обновить классификаторы" можно вручную обновить список классификаторов с серверов фирмы 1С.

Закладка "Ленты новостей"

На этой закладке можно посмотреть список всех лент новостей, которые задействованы в этой конфигурации. Также по кнопке "Получить новости" можно вручную обновить список новостей с серверов фирмы 1С.

Закладка "Автоудаление новостей"

На этой закладке можно настроить правила, по которым устаревшие новости будут помечаться на удаление для дальнейшего удаления из информационной базы.

Закладка "Настройки пользователей"

На этой закладке администратор системы может посмотреть или изменить настройки пользователей, связанные с новостями.

Закладка "Сервис"

На этой закладке можно настроить детальность ведения лога, связанного с подсистемой новостей. Также можно выгрузить лог (в случае, если он был включен) для дальнейшего анализа техническими специалистами фирмы 1С
