Обработка «Помощник установки приборов учета»

Данная обработка позволяет упростить установку приборов учета на здания, подъезды, помещения.

Установка приборов учета осуществляется в несколько этапов:

  1. Настройка сценария установки приборов учета.
  2. Параметры и установка приборов учета.
  3. Результат установки приборов учета.

Переход между этапами производится с помощью навигационных кнопок «Назад», «Далее», «Готово» на командной панели формы мастера.

 

Настройка сценария установки приборов учета

На данном этапе выполняется установка первоначальных параметров для установки приборов учета, таких как:

  • «Дата» - дата установки, на которую выполняется установка приборов учета;
  • «Организация» - организация, к которой относятся устанавливаемые приборы учета;
  • «Услуга» - услуга, приборы учета которой необходимо установить;

Например. Установка счетчиков производится по услуге «Холодное водоснабжение».

  • «Способ ввода параметров устанавливаемых приборов учета» - настройка определяет, каким образом необходимо вводить параметры счетчиков - «Сводно» или «Индивидуально».
    • «Сводно» - при данном способе параметры приборов учета указываются один раз и будут применены ко всем вновь создаваемым приборам учета. К таким параметрам ПУ относятся:
      • «Тарифность» - тарифность приборов учета;
      • «Разрядность» - разрядность приборов учета;
      • «Способ регистрации показаний» - способ регистрации показаний счетчиком;
      • «Коэффициент трансформации» - коэффициент трансформации приборов учета;
      • «Использовать температурный коэффициент» - признак наличия температурной компенсации.

Например. Если в настройках указан метод ввода параметров «Сводно», установлены следующие параметры счетчиков: тарифность - «2-тарифный», разрядность - «5», способ регистрации показаний - «Нарастающим итогом», то в результате работы мастера будут созданы двухтарифные счетчики с разрядностью 5 и способом регистрации показаний нарастающим итогом (т.е. расход будет считаться как разность текущих и предыдущих показаний).

    • «Индивидуально» - при данном варианте ввода параметров необходимо будет индивидуально для каждого счетчика указывать тарифность, разрядность, коэффициент трансформации, способ регистрации показаний и признак наличия температурной компенсации на этапе ввода данных для установки приборов учета.
  • «Имя услуги для наименования» - строковое представление услуги, которое автоматически включается в наименование устанавливаемых приборов учета. По умолчанию принимает значение краткого наименования (если не указано – полного наименования) вида услуги, к которому относится услуга;

Например. Если в качестве имени услуги указать «ХВС», а счетчик устанавливается на лицевой счет «л/с №32», и введен номер счетчика «90300021», то имя нового счетчика будет сформировано как «ХВС, л/с №32, 90300021».

  • «Ручной ввод наименований приборов учета с автодобавлением номера прибора учета» - настройка определяет, автоматически или вручную будут формироваться наименования приборов учета.
    Если флаг настройки не установлен, то формирование наименований счетчиков будет производиться автоматически.

Например. Если в качестве имени услуги указано «ХВС», а счетчик устанавливается на лицевой счет «л/с №102»,то наименование счетчика будет сформировано автоматически и примет следующее значение – «ХВС, л/с №102».

Например. Если в качестве имени услуги указано «ХВС», и устанавливаются два счетчика на лицевой счет «л/с №167»,то наименования счетчиков будут сформированы автоматически и примут следующие значения – «ХВС, л/с №167 (1)» и «ХВС, л/с №167 (2)».

Если же флаг настройки установлен, то наименования приборов учета будут задаваться пользователем вручную. При этом к заданному пользователем наименованию добавляется номер прибора учета в левую или правую часть наименования, в зависимости от выбранного значения переключателя:

    • «в левую часть наименования»;

Например. Например, в качестве имени услуги указано «ХВС», установлено ручное формирование имен счетчиков, номер счетчика располагается слева, в качестве номера счетчика указано «20005642» и счетчик устанавливается на лицевой счет «л/с №360». В данном случае наименование счетчика сформируется по умолчанию и будет иметь следующее значение – «20005642, ХВС, л/с №360». Наименование счетчика может быть отредактировано вручную.

    • «в правую часть наименования»;

Например. Например, в качестве имени услуги указано «ХВС», установлено ручное формирование имен счетчиков, номер счетчика располагается справа, в качестве номера счетчика указано «10007755» и счетчик устанавливается на лицевой счет «л/с №531». В данном случае  наименование счетчика сформируется по умолчанию и будет иметь следующее значение – «ХВС, л/с №531, 10007755». Наименование счетчика может быть отредактировано вручную.

  • «Ввести начальные показания приборов учета» - настройка определяет, будет ли выполнен ввод начальных показаний счетчиков после их установки. Если настройка установлена, то на этапе  ввода данных для установки приборов учета необходимо будет вводить начальные показания приборов учета в зависимости от тарифности;
  • «Назначить услугу на лицевые счета устанавливаемых приборов учета» - настройка определяет, будет ли выполнено назначение услуги, указанной в поле «Услуга» на лицевые счета помещений, в которые устанавливаются приборы учета.

 

Параметры и установка приборов учета

На данном этапе осуществляется ввод сведений об устанавливаемых приборах учета. Если на первом этапе был выбран «Сводный» способ ввода параметров устанавливаемых приборов учета, такие параметры как «Тарифность», «Способ регистрации показаний», «Разрядность», «Коэффициент трансформации», «Использовать температурный коэффициент» заполняются один раз для всех устанавливаемых приборов учета. Если же выбран «Индивидуальный» способ ввода параметров устанавливаемых приборов учета, то данные параметры необходимо заполнять для каждого прибора учета в табличной части «Устанавливаемые приборы учета».

  • «Объект» - объект, на который необходимо выполнить установку прибора учета. Список объектов можно заполнить автоматически с помощью специальной обработки через кнопку «Заполнить», расположенную в командной панели табличной части;
  • «№ прибора учета» - в данном поле можно ввести номер нового прибора учета вручную.
    Если номер счетчика будет указан вручную, то с указанным номером будет создан новый прибор учета и выполнена его установка на объект.
  • «Способ регистрации показаний» - способ регистрации показаний текущего прибора учета. Поле доступно, только если в настройках указан способ ввода параметров счетчиков «Индивидуально»;
    Способ регистрации показаний влияет на то, каким образом будет определяться расход по прибору учета – при выбранном значении «Нарастающим итогом» расход рассчитываться как разность между текущими и предыдущими показаниями; при выбранном значении «Расход за период расчета» в качестве расхода берутся текущие показания.
  • «Тарифность» - тарифность прибора учета;
  • «Разрядность» - разрядность прибора учета (не более 15);
  • «Коэффициент трансформации» - коэффициент трансформации прибора учета. При начислении расход по прибору учета умножается на коэффициент трансформации;
  • «Использовать температурный коэффициент» - признак наличия температурной компенсации. Температурный коэффициент используется в случае, когда прибор учета установлен вне помещения, не поддерживает температурную компенсацию, и ее необходимо учитывать;
  • «Дата включения» - дата включения текущего счетчика;
  • «Дата поверки» - дата поверки текущего счетчика;
  • «Дневное показание» - начальное ДНЕВНОЕ показание прибора учета, которое будет отражено в документе «Ввод показаний счетчика», сформированном после установки приборов учета. Поле доступно, только если установлена настройка «Ввести начальные показания приборов учета»;
  • «Ночное показание» - начальное НОЧНОЕ показание прибора учета, которое будет отражено в документе «Ввод показаний счетчика», сформированном после установки приборов учета. Поле доступно, только если установлена настройка «Ввести начальные показания приборов учета»;
  • «Пиковое показание» - начальное ПИКОВОЕ показание прибора учета, которое будет отражено в документе «Ввод показаний счетчика», сформированном после установки приборов учета. Поле доступно, только если установлена настройка «Ввести начальные показания приборов учета»;
  • «Наименование прибора учета» - автоматически формируемое наименование текущего прибора учета. Поле доступно для редактирования, только если была установлена настройка «Ручной ввод наименований приборов учета с автодобавлением номера прибора учета».
  • «Группа приборов учета» - группа справочника «Счетчики», в которой будут размещены создаваемые приборы учета.


Окончание работы мастера

По окончании работы мастера будут созданы элементы справочника «Счетчики», документы установки приборов учета «Установка счетчика» по видам объектов (здания, подъезды, помещения, лицевые счета), документ «Ввод показаний счетчика», если выполнялся ввод начальных показаний устанавливаемых приборов учета, документ «Управление услугами», если выполнялось назначение услуг на объекты.
