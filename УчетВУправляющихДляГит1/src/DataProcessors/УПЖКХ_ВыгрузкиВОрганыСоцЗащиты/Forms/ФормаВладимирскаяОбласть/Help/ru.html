Владимирская область

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Владимирской области за указанный период.
 
Перед началом выгрузки данных в обработке необходимо заполнить следующие параметры:
  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;
  • «Период» - месяц, за который необходимо предоставить данные;
  • «Файл загрузки» - путь к файлу формата *.DBF, из которого будет загружена необходимая информация для заполнения табличной части обработки;
  • «Каталог выгрузки» - каталог выгружаемого файла в формате *.DBF.
 
Перед началом выполнения обработки необходимо произвести её предварительную настройку. Для вызова диалогового окна настройки необходимо нажать на кнопку «Настройка» на форме обработки.
В открывшемся окне на закладке "Настройка соответствия услуг" необходимо произвести настройку соответствия услуг в ОСЗН и элементов справочника «Услуги» в базе. Чтобы загрузить услуги из дополнительного справочника услуг (из комплекта файлов для обмена) необходимо в поле "Файл загрузки услуг" указать файл справочника услуг в формате *.DBF и нажать на кнопку "Загрузить из файла".
Для корректного установления соответствия между лицевыми счетами из файла запроса и в БД необходимо на закладках «Населенные пункты» и «Улицы» загрузить соответствующую информацию из дополнительных справочников населенных пунктов и улиц (из комплекта файлов для обмена), для этого: необходимо в полях "Файл загрузки населенных пунктов" и "Файл загрузки улиц" указать соответствующие файлы справочников в формате *.DBF и нажать на кнопку "Загрузить из файла".
Для вывода необходимого реквизита справочника «Лицевые счета» в колонку «Лицевой счет» таблицы выгрузки, необходимо на закладке «Дополнительные настройки» выбрать соответствующий реквизит.

Для загрузки данных из файла и заполнение данными из базы необходимо нажать на кнопку "Заполнить". Если структура файла не будет соответствовать структуре, описанной в таблице, то загрузка данных будет невозможна.
Поиск данных из файла и БД происходит по следующим полям:
  • NSP
  • ULC
  • DOM
  • KOR
  • KVR
Для выгрузки данных из табличной части обработки в файл, каталог размещения которого указан в поле «Каталог выгрузки», необходимо нажать на кнопку «Выгрузить».
Формат файла выгрузки соответствует формату файла загрузки. Имя выгружаемого файла «FYYMMXXX.dbf», где
- F - фиксированное значение;
- YY - номер года периода выгрузки без века с лидирующим нулем;
- MM - номер месяца периода выгрузки с лидирующим нулем;
- XXX - код организации из файла загрузки.
