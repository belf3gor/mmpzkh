Приморский край

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Приморского края за указанный период.

Перед началом выгрузки данных в обработке необходимо заполнить следующие параметры:

  • «Месяц выгрузки» - месяц (далее - Месяц выгрузки), за который необходимо предоставить данные;
  • «Организация» - ссылка на элемент справочника «Организации» (далее - Основная организация), от имени которой будут выгружены необходимые данные за указанный период;
  • «Файл обмена» - путь к файлу в формате *.DBF, которые ОСЗН ежемесячно предоставляет в Основную организацию.

Перед началом выгрузки, для корректного заполнения таблицы обработки, необходимо произвести предварительную настройку. 

Для вызова диалогового окна настройки необходимо нажать на кнопку «2. Настройка».

В открывшейся форме на странице «Соответствие услуг» необходимо установить соответствие услуг из файла обмена с услугами из справочника "Услуги" в информационной базе. При этом:

  • Для того чтобы заполнить таблицу "Услуги в ОСЗН" необходимо нажать кнопку "Загрузить из файла обмена". В итоге из файла обмена будут загружены различные услуги, которые содержатся в колонке "GKU".
  • Редактирование строк в таблице "Услуги в ОСЗН" запрещено.
  • Добавление строк в таблице "Услуги в базе" доступно только в том случае, если заполнена таблица "Услуги в ОСЗН".

После того, как пользователь установит все необходимые настройки, можно приступить к этапу считывания данных из файла обмена. Для этого необходимо нажать на кнопку "3. Загрузить данные из файла обмена". В результате сведения из файла обмена будут загружены в таблицу обработки. При этом для лицевых счетов из файла обмена, которые были идентифицированы в справочнике "Лицевые счета" информационной базы, будет заполнена колонка "Лицевой счет в базе".
Идентификация данных из файла обмена происходит по следующим полям:

  • «NP» - наименование населенного пункта;
  • «UL» - наименование улицы;
  • «DOM» - номер дома;
  • «KOR» - номер корпуса;
  • «KV» - номер квартиры;
  • «LS» - номер лицевого счета.

Если в результате считывания данных из файла обмена в таблице обработки содержатся строки с идентифицированными лицевыми счетами, то можно приступить к этапу заполнения сведений о выполненных начислениях за Месяц выгрузки. Для этого необходимо нажать на кнопку "4. Заполнить сведения о начислениях". В итоге, при наличии в информационной базе необходимых данных, в таблице обработки будут заполнены колонки "Площадь"("PL"), "Количество проживающих"("KOLP"), "Норматив"("NORM"), "Сумма к оплате"("FAKTP"), "Фактическое потребление с учетом перерасчета"("FAKTPER"), "Установленный тариф"("TARF"), "Признак корректировки"("PRIZN"), "Количество месяцев долга"("MESD"), "Наличие соглашения о реструктуризации долга"("RESTRDOLG").

Если в результате получения сведений о начислениях за Месяц выгрузки заполнены колонки, описанные в предыдущем пункте, то можно приступить к этапу выгрузки данных из таблицы обработки в файл обмена. Для этого необходимо нажать на кнопку "5. Выгрузить в файл обмена". При этом:

  • Предварительно будет сохранено исходное состояние файла обмена. Для этого будет создана копия файла, к наименованию которого будет добавлено "_Исходный".
  • Сведения о полученных данных о начислениях из таблицы обработки будут записаны в файл обмена.
  • Файл обмена с полученными данными будет сохранён в каталог, указанный в поле «Файл обмена».
