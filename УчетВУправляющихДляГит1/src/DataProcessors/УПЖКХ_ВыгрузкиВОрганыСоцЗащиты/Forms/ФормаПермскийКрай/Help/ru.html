Пермский край

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Пермского края за указанный период. 
 
Перед началом загрузки данных в обработке необходимо заполнить следующие параметры:
  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;
  • «Период» - месяц, за который необходимо предоставить данные;
  • «Путь к файлам» - каталог, с которого будет загружены и выгружены данные (файлы формата *.DBF);
 
В обработке есть возможность настроить соответствие услуг из файла «SPR_UL.DBF» и БД, а так же выбрать услуги кап. ремонта. Для того, чтобы вызвать диалоговое окно настройки необходимо нажать на кнопку «Настройка».
Примечание: настройка даступна только после загрузки данных из файлов (нажатие кнопки кнопку «Загрузить из файлов»).
 
Для загрузки данных в обработке необходимо указать каталог, в котором должны присутствовать файлы формата *.DBF со следующей структурой и нажать кнопку «Загрузить из файлов»:

Если структуры файлов не будут соответствовать структурам описанной выше или один из файлов будет отсутствовать, то загрузка и выгрузка данных будет невозможна.
 
Поиск данных из файла и БД происходит по следующим полям:
  • FAM
  • IM

  • OT

  • UL

  • NDOM
  • NKORP
  • NKV
  • KVLIT
 
Для заполнения табличных частей обработки данными из файлов и данными БД необходимо нажать на кнопку «Заполнить». Для выгрузки данных из табличных частей обработки в каталог, указанный в поле «Путь к файлам», необходимо нажать на кнопку «Выгрузить». Формат файла выгрузки в формате *.DBF представлен ниже:
