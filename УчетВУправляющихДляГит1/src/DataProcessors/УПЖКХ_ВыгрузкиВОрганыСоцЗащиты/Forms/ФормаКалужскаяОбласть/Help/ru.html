Калужская область

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Калужской области за указанный период.

Перед началом выгрузки данных в обработке необходимо заполнить следующие параметры:
  • «Период» - месяц, за который необходимо предоставить данные в ОСЗН (далее - Месяц выгрузки);
  • «Организация» - организация, от имени которой будет предоставлены данные за Месяц выгрузки в ОСЗН (далее - Основаная организация);  
  • «Файл ОСЗН» - указывается путь к dbf-файлу, который получен из ОСЗН (далее - Файл обмена);


При первоначальном запуске обработки необходимо придерживаться следующей последовательности выполняемых операций:
1. Выбрать Файл обмена и при помощи нажатия на кнопку "1. Загрузить из файла" выполнить загрузку данных в таблицу обработки.
2. Установить соответствия услуг в ОСЗН с услугами из справочника "Услуги" в информационной базе. Для этого необходимо нажать на кнопку "2. Настройка".
    Таблица "Услуги в ОСЗН" будет заполнена кодами услуг В ОСЗН с соответствующими наименованиями, которые содержатся в колонках Файла обмена.

При последующем использовании обработки шаг с настройкой соответствия услуг можно пропускать. В случае, если в Файле обмена измениться состав услуг, то необходимо будет выполнить донастройки соответствия услуг.

Если все вышеперечисленные операции выполнены, то можно приступить к этапу получения сведений о начислениях и задолженности за Месяц выгрузки по Основной организации для лицевых счетов из Файла обмена, которые были идентифицированы в справочнике "Лицевые счета" информационной базы.
Для этого необходимо нажать на кнопку "3. Заполнить данными из базы".


Если таблица обработки заполнена сведениями о начислениях и задолженностях, то можно приступить к этапу выгрузки данных в Файл обмена. Для этого необходимо нажать на кнопку "4. Выгрузить в файл".
В момент записи будет создан исходный вариант Файл обмена. Для того, чтобы пользователи смогли определить данный файл, к его наименованию будет добавлен постфикс "_Исходный".
