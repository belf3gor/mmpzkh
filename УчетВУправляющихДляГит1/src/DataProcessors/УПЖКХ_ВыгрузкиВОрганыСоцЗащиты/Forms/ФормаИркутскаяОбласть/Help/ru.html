Иркутская область

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Иркутской области за указанный период.
Перед началом выгрузки данных в обработке необходимо заполнить следующие параметры:
  • «Месяц выгрузки» - месяц (далее - Месяц выгрузки), за который необходимо предоставить данные;
  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;
  • «Реестр л/с по ЖКУ» - путь к файлу, который предоставили из ОСЗН, в котором содержится реестр лицевых счетов получателей мер социальной поддержки по оплате жилого помещения и коммунальных услуг. Структура формат файла представлена в таблице «Структура файла с реестром л/с по ЖКУ».
  • «Реестр л/с по капремонту» - путь к файлу, который предоставили из ОСЗН, в котором содержится реестр лицевых счетов получателей мер социальной поддержки по оплате взносов на капитальный ремонт общего имущества в многоквартирных домах. Структура формат файла представлена в таблице «Структура файла с реестром л/с по капремонту».
  • «Реестр л/с по ТКО» - путь к файлу, который предоставили из ОСЗН, в котором содержится реестр лицевых счетов получателей мер социальной поддержки по оплате за обращение с твердыми коммунальными отходами. Структура формат файла представлена в таблице «Структура файла с реестром л/с по ТКО».
 

Перед началом выполнения обработки необходимо произвести ее настройку. Для вызова диалогового окна настройки необходимо нажать кнопку "1. Настройка", расположенную в верхней части формы обработки. В открывшемся окне необходимо установить следующие настройки:

  • На странице «Общие настройки» необходимо установить следующие настройки:

    • В разделе «Настройки идентификации лицевых счетов в базе» необходимо установить настройки идентификации лицевых счетов из файлов с реестрами с лицевыми счетами из справочника «Лицевые счета». При этом:

      • «Способ идентификации лицевых счетов в базе» - необходимо указать способ идентификации лицевых счетов из файлов с реестрами в информационной базе. Доступны 2 способа: «По адресу помещения» и «По наименованию/идентификатору лицевого счета».

      • «Реквизит лицевого счета из базы для идентификации» - необходимо указать реквизит справочника «Лицевые счета», по которому будет производиться идентификация лицевых счетов из файлов с реестрами. Доступно только для способа идентификации лицевых счетов в базе «По наименованию/идентификатору лицевого счета».

    • В разделе «Настройки форматов файлов с реестрами и их кодировок» необходимо установить форматы и кодировку для файлов с реестрами. При этом:
      • В подразделе «по ЖКУ» необходимо произвести следующие настройки:
        • «Формат файла с реестром л/с по ЖКУ» - необходимо установить формат файла с реестром л/с по ЖКУ. Доступны 2 варианта: «DBF» и «XLS».
        • «Кодировка файла (доступно только для формата *.DBF)» - необходимо установить кодировку DBF-файла с реестром л/с по ЖКУ. Доступны 2 варианта: «DOS» и «WIN».
      • В подразделе «по капремонту» необходимо произвести настройки, аналогичные тем, что описаны выше для подраздела «по ЖКУ».
      • В подразделе «по обращению с твердыми коммунальными отходами» необходимо произвести следующую настройку: 
        • «Кодировка файла» - необходимо установить кодировку DBF-файла с реестром л/с по ТКО. Доступны 2 варианта: «DOS» и «WIN».

  • На странице «Соответствие услуг» необходимо произвести настройку соответствия услуг в ОСЗН и элементов справочника «Услуги». При этом:

    • В левой таблице всегда содержится список из 11 услуг в ОСЗН, которые не доступны для редактирования.

    • В правой таблице содержится список услуг из справочника «Услуги» (далее - Услуга из базы), которые установлены в соответствие текущей услуге из ОСЗН. При этом:

      • В поле «Тип услуги» необходимо указать тип услуги. Доступны 3 варианта: «Индивидуальная», «Общедомовая» и «Индивидуальная и общедомовая».
        Предпосылкой появления данной настройки стало то, что в рамках программы пользователи создают Услуги в базе в наименовании которых предполагается что услуга является общедомовой, но при этом используется методика расчета как индивидуальной услуги. Н-р, для таких услуг как "ХВС на ОДН", "ГВС на ОДН". При этом расчет по данным услугам производится по нормативам потребления.
        Соответственно, чтобы оплаты по данным услугам попадали в колонки "Размер оплаты по услуге "Холодное водоснабжение" (ОДН)" и "Размер оплаты по услуге "Горячее водоснабжение" (ОДН)" необходимо для данных услуг установить тип услуги "Общедомовая".
        В случае, если для данных услуг тип установлен "Индивидуальная" или "Индивидуальная и общедомовая", то сведения об оплатах будут отражены в колонках "Размер оплаты по услуге "Холодное водоснабжение"" и "Размер оплаты по услуге "Горячее водоснабжение"" соответственно.
        Если выбран тип услуги "Индивидуальная и общедомовая", то колонки с оплатами на индивидуальную и общедомовую составляющие текущей услуги в ОСЗН будут заполняться в соответствии с расчетом, который настроен для соответствующих Услуг в базе.

      • В поле «Учитывать при расчете норматива» необходимо установить признак учета норматива текущей услуги из Базы при расчете норматива по соответствующей услуге в ОСЗН.

После того, как выполнены все предварительные настройки, необходимо нажать кнопку «2. Загрузить из файла(ов)», в результате которой будут заполнены соответствующие таблицы «Реестр ЖКУ», «Реестр капремонт» и «Реестр ТКО» данными из файлов, которые выбраны в полях «Реестр л/с по ЖКУ», «Реестр л/с по капремонту» и «Реестр л/с по ТКО».

После того, как произведено считывание данные из выбранных файлов с реестрами, необходимо нажать кнопку «3. Заполнить данными из базы», в результате которой будут заполнены соответствующие таблицы «Реестр ЖКУ», «Реестр капремонт» и «Реестр ТКО». При этом:

  • В таблице «Реестр ЖКУ» будут заполнены сведения об оплатах по ЖКУ, которые поступили от идентифицированных лицевых счетов в Месяце выгрузки, и задолженностях по ним;
  • В таблице «Реестр капремонт» будут заполнены сведения об оплатах на капремонт, которые поступили от идентифицированных лицевых счетов в Месяце выгрузки, и задолженностях по ним;
  • В таблице «Реестр ТКО» будут заполнены сведения об оплатах за обращение с ТКО, которые поступили от идентифицированных лицевых счетов в Месяце выгрузки, и задолженностях по ним;


Для выгрузки данных из таблиц «Реестр ЖКУ», «Реестр капремонт» и «Реестр л/с по ТКО» в соответствующие файлы для ОСЗН необходимо нажать на кнопку «4. Выгрузить в файл(ы)».
