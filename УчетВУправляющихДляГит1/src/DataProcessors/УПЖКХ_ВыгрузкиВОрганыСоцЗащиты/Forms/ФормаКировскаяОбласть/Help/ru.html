Кировская область

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Кировской области за указанный период.
Перед началом выгрузки данных в обработке необходимо заполнить следующие параметры:
  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;
  • «Период» - месяц, за который необходимо предоставить данные;
  • «Кодировка файла» - кодировка формируемого файла выгрузки.
  • «Каталог выгрузки» - путь к каталогу, в котором будет сохранен выгружаемый файл в формате *.DBF.
  • «Вид бюджета» - вид бюджета, по которому будет выполняться заполнение табличной части и выгрузка данных. Указать к какому виду бюджета относится каждая из льготных категорий можно в настройках выгрузки на закладке "Соответствие льготных категорий".
  • «Услуги» - услуги, по которым нужно заполнять табличную часть и выгружать данные: все услуги; услуги, кроме капитального ремонта; услуги капитального ремонта. Данный параметр можно заполнить только если ведется раздельный учет капитального ремонта.

Перед началом выполнения обработки необходимо произвести ее настройку. Для вызова диалогового окна настройки необходимо нажать кнопку "Настройка", расположенную в верхней части формы обработки.

На закладке «Соответствие льготных категорий» необходимо произвести настройку соответствия кодов льготных категорий, заданных в ОСЗН и элементов справочника «Льготные категории» из Базы, а также указать к какому виду бюджета (федеральный или областной) относится каждая льготная категория.

На закладке «Соответствие услуг» необходимо произвести настройку соответствия кодов услуг, заданных в ОСЗН и элементов справочника «Услуги» из Базы.
