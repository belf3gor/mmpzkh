Тверская область

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Тверской области за указанный период. Данная выгрузка разработана в соответствии с Законом Тверской области от 07.04.2016 № 21-ЗО «О компенсации расходов на уплату взноса на капитальный ремонт отдельным категориям граждан».

Перед началом выгрузки данных в обработке необходимо заполнить следующие параметры:
  • «Месяц выгрузки» - месяц, за который необходимо предоставить данные в ОСЗН (далее - Месяц выгрузки);
  • «Организация» - организация, от имени которой будет предоставлены данные за Месяц выгрузки в ОСЗН (далее - Основная организация);  
  • «Файл обмена» - указывается путь к dbf-файлу, который получен из ОСЗН (далее - Файл обмена) со следующей структурой:

где N - количество услуг, предоставляемых гражданину на льготных условиях.
В ходе выполнения загрузки, в таблицу "Начисленные льготы" будут загружены следующие поля: с 1-ого по 20-е и 26-е поле.

При запуске обработки необходимо выполнить следующую последовательность операций:

1. Необходимо произвести предварительную настройку. Для вызова диалогового окна настройки необходимо нажать на кнопку "2. Настройка". 
    В открывшейся форме настройки произвести следующие действия:
    a) На закладке "Таблица соответствия льготных категорий" необходимо установить соответствия между льготными категориями в ОСЗН и льготными категориями в базе. 
    b) На закладке "Таблица соответствия услуг" необходимо установить соответствия между наименованиями услуг в ОСЗН и услугами в базе, начисленные льготы которых должны попадать в соответствующее наименование льготы.

При последующем использовании обработки шаг с настройкой соответствия услуг и льготных категорий можно пропускать. Если в Файле обмена изменится состав услуг, то необходимо будет выполнить донастройки соответствия услуг.
Важно! Для того чтобы установленные настройки вступили в силу необходимо нажать кнопку «Сохранить и закрыть».

2. Если все вышеперечисленные операции выполнены, то можно приступить к этапу загрузки сведений из Файла обмена. Для этого необходимо нажать на кнопку "3. Загрузить сведения из файла обмена".
 
При загрузке одновременно срабатывают механизмы идентификации:

  • льготников (жильцов) в базе по фамилии, имени и отчеству (FAMIL, IMJA, OTCH);
  • льготников (жильцов) в базе по адресу (NNASP, NYLIC, NDOM, NKORP, NKW);
  • льготной категории в базе по наименованию льготной категории в ОСЗН (NISU_OSN1..N).

Если по каким–либо причинам идентификация лицевого счета не прошла, то будет выведено соответствующее сообщение с указанием Фамилии, Имени, Отчества и номера карточки учета, а поле "Лицевой счет" будет подчеркнуто красным пунктиром. Для данной строки можно указать лицевой счет в базе вручную, либо оставить эту строку не идентифицированной.


3. Для получения сведений о начислениях и задолженностях за Месяц выгрузки по Основной организации для идентифицированных лицевых счетов из Файла обмена, необходимо нажать на кнопку "4. Заполнить данными из информационной базы". 

4. Если таблица обработки заполнена сведениями о начислениях и задолженностях, то можно приступить к этапу выгрузки данных в Файл(ы) обмена. Для этого необходимо нажать на кнопку "5. Выгрузить в файл(ы)". 
    Все выгружаемые файлы будут сохранены в каталог, указанный в поле «Файл обмена».

Формат файла РPГММПКО.DBF выгрузки с данными о начисленных льготах аналогичен формату Файла обмена. При выгрузке данных в файл выгрузки предусмотрено заполнение полей с 21-ого по 25-е.

  • РР - буквенно-цифровой код Организации. Определяется Учреждением. Должен быть уникальным в данном муниципальном образовании. Коды из диапазона "00" - "10" выделяются Организациям, предоставляющим услуги в масштабе области"
  • ГММ - дата формирования файла (последняя цифра года, месяц);
  • П - признак направления передачи информации. Может принимать следующие значения:
    • 0 - данные передаются из Учреждения в Организацию;
    • 1 - данные передаются из Организации в Учреждение (при выгрузке используется по умолчанию);
  • КО - код Учреждения. Устанавливается в соответствии с Приложением.

 

Формат файла РPГММZКО.DBF выгрузки с данными о заявителях, не оплативших в установленные сроки взнос на капитальный ремонт общего имущества в многоквартирном доме за два месяца подряд на конец Месяца выгрузки. При выгрузке данных в файл выгрузки предусмотрено заполнение поля DAT_ZAD. 

  • Z - обозначает, что файл содержит сведения о гражданах, имеющих задолженности по оплате жилищно-коммунальных услуг.
