Саратовская область

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Саратовской области за указанный период.

Перед началом выгрузки данных в обработке необходимо заполнить следующие параметры:
  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;  
  • «Входящий ZIP» - указывается загружаемый файл в формате ZGGGGGММГГ.zip, где GGGGG - код организации, ММГГ - месяц и год отчетного периода. Поля «Период» и «Код организации» при выборе файла загрузки переопределяются в соответствии с данным форматом;
  • «Каталог выгрузки» - указывается каталог для выгружаемого файла;


Для заполнения данных о начислениях и оплатах необходимо произвести предварительную настройку. Для вызова диалогового окна настройки необходимо нажать на кнопку «Настройка». На форме настройки на закладке "Настройки соответствия услуг" необходимо установить соответствие услуг в ОСЗН и услуг в базе. Для начального заполнения (либо перезаполнения) предусмотрена кнопка "По умолчанию". Для того чтобы включить какую-либо услугу в выгрузку, необходимо установить флажок "Использовать" слева от неё в таблице, после чего строка выбранной услуги будет подсвечена (строки с неиспользуемыми услугами затемнены). Для каждой услуги в ОСЗН необходимо определить способ начисления:

  • «ОДН» - по данной услуге производятся общедомовые начисления по коллективным приборам учета;  
  • «Индивидуально» - по данной услуге производятся начисления на лицевой счет по индивидуальным приборам учета;
  • «Прочее» - прочие варианты начисления по услуге (кроме оговоренных выше).

Для всех используемых услуг в обязательном порядке должно быть установлено соответствие с одной или несколькими услугами в базе, в противном случае необходимо отказаться от её использования.
Внимание! Если в процессе использования выгрузки было произведено заполнение таблиц начислений и оплат, а после этого изменены настройки, то таблицы начислений и оплат не будут обновлены и необходимо будет заново заполнить их!

На закладке "Дополнительные настройки" необходимо указать поле справочника "Лицевые счета", по которому необходимо идентифицировать лицевые счета (код, наименование или идентификатор лицевого счета). Значение данного поля будет сравнивать с полем "ЛИЦ_СЧ" файлов загрузки.

Заполнение табличных частей производится следующим образом. По нажатии на кнопку "Загрузить все из файла" будет произведена загрузка данных из входящего ZIP-файла. Для заполнения таблиц начислений и оплат необходимо нажать кнопку "Заполнить все". Идентификация лицевых счетов происходит по настройке, описанной выше.

Для осуществления выгрузки данных из табличных частей необходимо нажать на кнопку «Выгрузить все». Выгрузка данных в обработке происходит в архив в формате ОGGGGGММГГ.zip, состоящего из фалов в формате ОGGGGGММГГКОД.CSV, где КОД - код услуги, по которой предоставлены данные о начислениях или оплатах в данном файле, со следующей структурой:
