Нижегородская область

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Нижегородской области за указанный период. 
 
Перед началом загрузки данных в обработке необходимо заполнить следующие параметры:
  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;
  • «Период» - месяц, за который необходимо предоставить данные;
  • «Путь к файлам» - каталог с файлами переданные из социальной защиты (формат файлов *.DBF);
  • «Каталог выгрузки» - каталог выгрузки данных из БД в файлы в формата *.DBF (отличен от Пути к файлам);

 

Для заполнения табличных частей обработки необходимо указать каталог в поле «Путь к файлам», где должны присутствовать файлы формата *.DBF со следующей структурой:

Также перед началом загрузки для корректного заполнения табличных частей обработки необходимо произвести предварительную настройку. Для вызова диалогового окна настройки необходимо нажать на кнопку «Настройка». В появившемся окне необходимо произвести настройку соответствия данных из файлов и данных БД. Для автоматического заполнения данными табличных частей настройки из файлов нужно нажать на кнопку «Заполнить таблицы соответствий». Если структуры файлов не будут соответствовать структуре описанной выше, то загрузка и выгрузка данных будет невозможна.
