Тамбовская область

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Тамбовской области за указанный период и загрузки льгот в базу из файлов.
Перед началом выгрузки данных в обработке необходимо заполнить следующие параметры:
  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;
  • «Период» - месяц, за который необходимо предоставить данные;
  • «Код организации» - указывается код Организации вместе с регистрационным номером в ПФР;
  • «Вид отапливаемой площади» - указывается вид площади из базы соответствующий виду отапливаемой площади;
 
Перед началом загрузки и выгрузки данных для корректного заполнения табличных частей обработки необходимо произвести предварительную настройку. Для вызова диалогового окна настройки необходимо нажать на кнопку «Настройка». На закладке "Настройка соответствия услуг" необходимо загрузить услуги из специального справочника услуг ОСЗН (имя файла - "SVERUSL.DBF"), а затем установить соответствие услуг из базы. На закладке "Настройка соответствия льготных категорий" необходимо загрузить льготные категории из специального справочника льготных категорий ОСЗН (имя файла - "VOCKATEG.DBF"), а затем установить соответствие кодов законов и реестров льготным категориям. На закладке "Дополнительные настройки" необходимо в поле "Код в ОСЗН для услуги "Отопление"" указать код услуги в ОСЗН, который присвоен для услуги отопления. Если код указан, то в выгружаемых данных для строк с такими услугами в поле "Объем услуги" будет выводиться размер отапливаемой площади, а не реальный объем начисленной услуги. В правой таблице кодов услуг в ОСЗН необходимо указать те коды услуг, для которых необходимо размер норматива потребления считать по следующей формуле: 1 зарегистрированный человек - размер нормы 33, 2 зарегистрированных человека - размер нормы 21, 3 и более зарегистрированных человека - размер нормы 18. Для остальных услуг размер нормы будет получаться стандартным способом.

 

Перед началом загрузки и выгрузки данных в обработке на закладке "Таблица выгрузки начислений" необходимо заполнить следующие параметры:

  • «Каталог загрузки начислений» - указывается каталог файлов, в которых содержатся файлы с данными о льготниках, по которым необходимо выгрузить данные;
  • «Каталог выгрузки начислений» - указывается каталог выгружаемых файлов.

Для заполнения табличной части начислений данными из БД необходимо нажать на кнопку «Заполнить». Идентификация лицевых счетов происходит по полю "ACCOUNT" из файла загрузки и идентификатору лицевого счета в базе. Для выгрузки данных из табличной части начислений в каталог указанный в поле «Каталог выгрузки начислений» необходимо нажать на кнопку «Выгрузить». По каждой льготной категории из поля "KODK" и по каждой услуге из ОСЗН будет сформирован отдельный файл выгрузки.
 
 

Перед началом загрузки льгот в обработке на закладке "Таблица загрузки льгот" необходимо заполнить следующие параметры:

  • «Каталог загрузки льгот» - указывается каталог файлов, в которых содержатся файлы с данными о суммах льгот, которые необходимо загрузить в базу;

Имя файла загрузки льгот должен содержать следующую подстроку "VIDU-Код услуги", где Код услуги - код услуги в ОСЗН, для которой установлено соответствие на форме настроек.

Для заполнения табличной части льгот данными из БД необходимо нажать на кнопку «Заполнить». Идентификация лицевых счетов происходит по полю "ACCOUNT" из файла загрузки и идентификатору лицевого счета в базе. Для создания документа «Начисление услуг в сторонней программе» необходимо нажать на кнопку «Создать документ».
