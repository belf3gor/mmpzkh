Забайкальский край

Обработка предназначена для выгрузки сведений об объемах фактического потребления коммунальных услуг и суммах оплаты за истекший месяц в отдел социальной защиты населения Забайкальского края за указанный период.
 
Перед началом выгрузки данных в обработке необходимо заполнить следующие параметры:
  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;
  • «Период» - месяц, за который необходимо предоставить данные;
  • «Файл загрузки» - путь к файлу формата *.DBF, из которого будет загружена необходимая информация для заполнения табличной части обработки;
  • «Кодировка файла» - кодировка файла загрузки и выгрузки в формате *.DBF, которое может принимать значения DOS и Windows;
  • «Каталог выгрузки» - каталог выгружаемого файла в формате *.DBF.
 
Перед началом выполнения обработки обязательно необходимо произвести её настройку. Для вызова диалогового окна настройки необходимо нажать на кнопку «Настройка» на форме обработки.
В открывшемся окне необходимо произвести настройку соответствия услуг в ОСЗН и элементов справочника «Услуги» в базе и установить номера полей файла загрузки/выгрузки.
 
Для загрузки данных в обработке необходимо указать файл загрузки формата *.DBF со следующей структурой:

Наименование заданное ОСЗН Улица

Наименование заданное ОСЗН Сумма оплаты

Для загрузки данных из файла и заполнение данными из базы необходимо нажать на кнопку "Заполнить". Если структура файла не будет соответствовать структуре, описанной в таблице, то загрузка данных возможно выполнена не будет.
Поиск данных из файла и БД происходит по следующим полям:
  • Населенный пункт
  • Улица

  • Номер дома

  • Номер квартиры

Для выгрузки данных из табличной части обработки в файл, каталог размещения которого указан в поле «Каталог выгрузки», необходимо нажать на кнопку «Выгрузить».
Формат файла выгрузки полностью соответствует формату файла загрузки. Имя выгружаемого файла «out_file.dbf».
