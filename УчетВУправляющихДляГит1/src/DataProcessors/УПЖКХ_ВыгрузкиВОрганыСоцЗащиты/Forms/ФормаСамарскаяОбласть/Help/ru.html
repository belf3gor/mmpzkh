Самарская область

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Самарской области за указанный период.

Перед началом выгрузки данных в обработке необходимо заполнить следующие параметры:
  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;
  • «Период» - месяц, за который необходимо предоставить данные;
  • «Идентификатор организации» - уникальный идентификатор организации в ГИС НМСПН;
  • «Каталог загрузки» - указывается каталог, в котором расположены файлы формата dbf, которые присылает НМСПН;
  • «Каталог выгрузки» - указывается каталог для выгружаемых файлов;


Для заполнения данных о начислениях и оплатах необходимо произвести предварительную настройку. Для вызова диалогового окна настройки необходимо нажать на кнопку «Настройка». На форме настройки на закладке "Настройки соответствия услуг" необходимо установить соответствие услуг в ГИС НМСПН и услуг в Базе.

Для каждой услуги в ГИС НМСПН необходимо определить способ свертки тарифа:

  • «Максимум» - по данной услуге будет браться максимально возможное значение тарифа;
  • «Минимум» - по данной услуге будет браться минимально возможное значение тарифа;
  • «Усреднять» - по данной услуге будет браться среднее значение тарифа;
  • «Суммировать» - по данной услуге будет браться суммарное значение тарифа.

 

Для каждой услуги в Базе необходимо определить способ определения начислений:

  • «Общедомовая» - по данной услуге производятся общедомовые начисления по коллективным приборам учета;  
  • «Индивидуальная» - по данной услуге производятся начисления на лицевой счет по индивидуальным приборам учета;
  • «Индивидуальная и общедомовая» - прочие варианты начисления по услуге.

 

Для всех услуг в ГИС НМСПН в обязательном порядке должно быть установлено соответствие с одной или несколькими услугами в базе, в противном случае данные по услуге из ГИС НМСПН выгружены не будут.

На закладке "Дополнительные настройки" необходимо указать способ идентификации в базе льготополучателей из файла ("по ФИО и адресу" - полная идентификация по ФИО жильца и адресу помещения, в котором открыт лицевой счет; "по Адресу" - идентификация только по адресу помещения, в котором открыт лицевой счет; "по коду ФИАС" - полная идентификация по ФИО жильца, коду ФИАС здания(GUID дома в ФИАС) и по номеру квартиры), указать кодировку загружаемых и выгружаемых файлов формата DBF. Так же доступны 2 команды по очистке данных во всех заполненных на основной форме таблицах и очистке настроек соответствия услуг.

Заполнение табличных частей производится следующим образом. По нажатии на кнопку "Загрузить из каталога" будет произведена загрузка данных из входящих файлов формата DBF. Для заполнения таблиц необходимо нажать кнопку "Заполнить".

Для осуществления выгрузки данных из табличных частей необходимо нажать на кнопку «Выгрузить в каталог». Выгрузка данных в обработке происходит в фалы в формате ИдентификаторОрганизации_КодУслуги_ГГГГ_ММ.DBF, где ИдентификаторОрганизации - уникальный идентификатор организации в ГИС НМСПН, КодУслуги - код услуги, ГГГГ - год отчетного периода, ММ - месяц отчетного периода.
