Калининградская область

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Калининградской области.

Перед работы с обработкой необходимо заполнить следующие параметры:

  • «Период» - отчетный период;

  • «Файл загрузки» - пусть к заполняемому файлу;

  • «Наименование организации» - наименование организации, от имени которой формируется файл выгрузки;

  • «Код организации» - наименование организации, от имени которой формируется файл выгрузки (Код организации в системе расчетов Предприятия).

Также необходимо произвести настройку выгрузки. Для этого необходимо перейти на форму настройки нажав кнопку «Инструкция».

Выгрузка данных в ОСЗН

Для заполнения файла выгрузки данными из базы необходимо загрузить его в обработку нажатием кнопки "Загрузить". При загрузке автоматически устанавливается соответствие данных файла данным в базе и его заполнение. Результат загрузки отображается в таблице выгрузки на форме данной обработки.

Заполненную таблицу выгрузки можно выгружать в файл выгрузки нажатием кнопки «Выгрузить».

Формат файла выгрузки:

Наименование поля

Значение поля

SCH

Номер лицевого счета в организации

KOD_ORG

Код организации в системе расчетов Предприятия

N_ORG

Наименование организации

KLADR

Код адреса в КЛАДРе (улицы)

KLADR_NAME

Наименование населенного пункта

STREET_NAM

Наименование улицы

NDOM

Номер дома

NKORP

Корпус

NKW

Номер квартиры

KOD_USL

Код услуги

N_USL

Наименование услуги

TARIF

Тариф (руб)

RAS

Расход по прибору учета/норматив потребления

NAC

Сумма начислений по лицевому счету за месяц DAT

Первое число, месяц и год начисления

Сальдо на 1 число месяца

Код результата обработки запроса
