Свердловская область (версия 6)

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Свердловской области за указанный период.
Перед началом загрузки данных в обработке необходимо заполнить следующие параметры:
  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;
  • «Период» - месяц, за который необходимо предоставить данные;
  • «Имя файла загрузки» - путь к файлу формата *.DBF, с которого будет загружена необходимая информация для заполнения табличной части обработки;
  • «Код организации» - код организации, необходимый для заполнения в таблицу выгрузки;
  • «Дата определения задолженности» - дата, на которую необходимо получать данные о задолженностях лицевых счетов (данная дата должна принадлежать периоду выгрузки);
Перед началом загрузки и выгрузки данных, для корректного заполнения табличной части обработки, необходимо произвести предварительную настройку. Для вызова диалогового окна настройки необходимо нажать на кнопку «Настройка».
 
В открывшейся форме настройки на закладке "Настройка соответствия услуг" необходимо установить соответствия между услугами в ОСЗН и услугами в базе, для этого необходимо на данной закладке в поле "Имя файла загрузки" указать файл услуг, который предоставляет ОСЗН и нажать на кнопку "Загрузить из файла", а затем уже установить соответствие. Если данные необходимо выгружать в разрезе видов тарифов услуг, то необходимо установить флаг "Учитвать виды тарифов услуг". При установке данного флага в таблице услуг базы станет доступна колонка "Вид тарифа", в которой необходимо указать вид тарифа.
 
На закладке "Услуги без ОДН" при необходимости можно указать услуги, для которых необходимо выгружать начисления за ОДН в составе индивидуальных начислений.
 
На закладке "Настройка соответствия видов благоустройств" необходимо установить соответствия между видами благоустройств в ОСЗН и видами благоустройств в базе. Если некоторые виды благоустройств остутствуют в базе, то необходимо создать их и настроить благоустройства для зданий или помещений. Для остальных видов благоустройств из ОСЗН уже существуют виды благоустройства в базе, их достаточно просто настроить для зданий или помещений.
 
На закладке "Настройка соответствия льготных категорий" необходимо указать льготные категории, которые относятся к льготным категориям детей-сирот.

Поиск данных из файла и БД происходит по следующим полям:

  • FIRSTNAME

  • MIDDLENAME

  • CITY

  • STREET
  • HOUSE
  • BUILDING
  • FLAT
