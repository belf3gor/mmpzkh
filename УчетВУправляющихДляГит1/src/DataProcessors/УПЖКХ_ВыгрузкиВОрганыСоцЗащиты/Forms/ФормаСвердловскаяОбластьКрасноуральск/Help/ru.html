Свердловская область (Красноуральск)

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Свердловской области по городскому округу Красноуральска за указанный период. 
 
Перед началом выгрузки данных в обработке необходимо заполнить следующие параметры:
  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;
  • «Период» - месяц, за который необходимо предоставить данные;
  • «Каталог выгрузки» - путь к каталогу на диске, в котором будет сохранен файл выгружаемых данных формата *.dbf;
  • «Идентификатор УК» - идентификатор организации, необходимый для заполнения в таблицу выгрузки;
  • «Только льготники» - флаг, при установке которого в таблицу выгружаемых данных будут попадать только льготники;
 
Перед началом выгрузки данных, для корректного заполнения табличной части обработки, необходимо произвести предварительную настройку. Для вызова диалогового окна настройки необходимо нажать на кнопку «Настройка».
В открывшейся форме настройки необходимо установить соответствия между кодами услуг в ОСЗН и услугами в базе.
 
Для заполнения табличной части обработки данными из БД необходимо нажать на кнопку «Заполнить». Для выгрузки данных из табличной части обработки в файл необходимо нажать на кнопку «Выгрузить». Формат файла выгрузки в формате *.DBF представлен ниже:
