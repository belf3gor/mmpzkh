Красноярский край (вариант 1)

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Красноярского края за указанный период. 
 
Перед началом выгрузки данных в обработке необходимо заполнить следующие параметры:
  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;
  • «Период» - месяц, за который необходимо предоставить данные;
  • «Лицевые счета» - группа лицевых счетов справочника «Лицевые счета»;
  • «Выберете файл dbf» - для каждой вкладки обработки указывается имя выгружаемого файла в формат *.DBF;
 
В обработке есть возможность настройки услуг. Для открытия диалогового окна настройки необходимо нажать на кнопку «Услуги». В диалоговом окне необходимо добавить те услуги, по которым необходимо настроить соответствие услуг и его представление в обработке выгрузки.
 
Для заполнения табличных частей обработки данными необходимо нажать на кнопку «Заполнить все». Для заполнения табличной части на определенной вкладки обработки нужно нажать на кнопку «Заполнить».
 
Для выгрузки данных из табличной части обработки в файл в формата *.DBF  необходимо указать имя выгружаемого файла в поле «Выберете файл dbf». Форматы файлов выгрузки в формате *.DBF представлены ниже:
