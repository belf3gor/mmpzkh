Челябинская область

Обработка предназначена для формирования обменного файла за указанный период и последующей передачи его в отдел социальной защиты населения Челябинской области.
Формат файла обмена соответствует формату утвержденному в Соглашении, заключенном на период опытного внедрения программного комплекса "ЕИС в сфере социальной защиты населения" в целях подготовительной работы для организации обмена информацией, необходимой для расчета компенсации расходов по оплату жилого помещения и коммунальных услуг в соответствии с Законом Челябинской области от 18.12.2014 г. №89-ЗО «О возмещении расходов на оплату жилого помещения, отопления и освещения педагогическим работникам областных государственных и муниципальных образовательных организаций, проживающим и работающим в сельской населенных пунктах, рабочих поселках (поселков городского типа) Челябинской области» (далее - Закон) и постановлением Правительства Челябинской области от 20.07.2011 г. №230-П «О Положении о порядке назначения отдельным категориям граждан компенсации расходов на оплату жилых помещений и коммунальных услуг, рассчитанной на основании фактических начислений на оплату жилых помещений и коммунальных услуг, в Челябинской области».

Перед началом выгрузки данных в обработке необходимо заполнить следующие параметры:

  • «Месяц выгрузки» - месяц (далее - Месяц выгрузки), за который необходимо предоставить данные;
  • «Организация» - ссылка на элемент справочника «Организации» (далее - Основная организация), от имени которой будут выгружены необходимые данные за указанный период;
  • «Каталог сохранения файла для ОСЗН» - каталог, в который будет выгружен файл в формате *.DBF (далее - файл для ОСЗН). При этом наименование файла для ОСЗН будет "OUT_FILE.dbf";
  • «Код района» - код района, который будет записан в поле "K_RAION" файла для ОСЗН;
  • «Идентификатор кода города по ФИАС» - идентификатор кода города по ФИАС, который будет записан в поле "FIAS_CITY" файла для ОСЗН.

Перед началом выгрузки, для корректного заполнения таблицы обработки, необходимо произвести предварительную настройку. 

Для вызова диалогового окна настройки необходимо нажать на кнопку «1. Настройка».

В открывшейся форме на странице «Общие настройки» необходимо установить следующие настройки:

  • «Тип выгружаемого наименования организации» - определяет признак записи в поле "POSTAV" файл для ОСЗН сокращенного или полного наименования Основной организации;
  • «Вид отапливаемой площади помещений» - ссылка на элемент справочника «Виды площадей». С учетом выбранного вида площади будет определяться соответствующий размер площади выгружаемого помещения, на основе которого будет заполнено поле «PL_OTAP» файла для ОСЗН;
  • «Учитывать пени при выгрузке» - определяет признак включения всех сумм по операциям по пени (начисления, перерасчеты и оплаты) за Месяц выгрузки по Основной организации; 

    • После того, как выполнены все предварительные настройки, необходимо нажать кнопку «2. Заполнить данные для выгрузки», в результате которой будет заполнена таблица обработки сведениями о лицевых счетах с произведенными на них начислениями в Месяце выгрузки.

  • «Учитывать перерасчеты за текущий месяц начисления» - определяет признак включения в сумму, выводимую в поле "Перерасчеты за прошлые периоды, руб.", перерасчетов, рассчитанных за текущий месяц начисления.

    Для выгрузки данных из таблицы выгрузки в файл для ОСЗН необходимо нажать на кнопку «3. Выгрузить данные в файл для ОСЗН».
