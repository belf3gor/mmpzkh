Архангельская область

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Архангельской области за указанный период. 
 
Перед началом выгрузки данных в обработке необходимо заполнить следующие параметры:
  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;
  • «Период» - месяц, за который необходимо предоставить данные;
  • «Файл загрузки» - путь к файлу формата *.DBF, с которого будет загружена необходимая информация для заполнения табличной части обработки;
 
Перед началом загрузки и выгрузки данных, для корректного заполнения табличной части обработки, необходимо произвести предварительную настройку. Для вызова диалогового окна настройки необходимо нажать на кнопку «Настройка».
В открывшейся форме настройки на закладке "Соотвтетсвие льготных категорий" необходимо установить соответствия между льготными категориями в ОСЗН и льготными категориями в базе, на закладке "Соотвтетсвие наименований льгот и услуг" необходимо установить соответствия между наименованиями льгот в ОСЗН и услугами, начисленные льготы которых должны попадать в соответствующее наименование льготы.

Если структура файла не будет соответствовать структуре описанной в таблице, то выгрузка данных будет невозможна.
 
Поиск данных из файла и БД происходит по следующим полям:
  • FAMIL
  • IMJA

  • OTCH

  • NNASP

  • NYLIC
  • NDOM
  • NKORP
  • NKW
  • NKOD_PU
Для заполнения табличной части обработки данными из файла и данными БД необходимо нажать на кнопку «Заполнить». Для выгрузки данных из табличной части обработки в файл указанный в поле «Файл загрузки» необходимо нажать на кнопку «Выгрузить». Формат файла выгрузки в формате *.DBF соответствует формату файла загрузки.
