Ярославская область

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Ярославской области за указанный период.

Перед началом выгрузки данных в обработке необходимо заполнить следующие параметры:
  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;
  • «Месяц выгрузки» - месяц, за который необходимо предоставить данные;  
  • «Файл загрузки *.dbf» - указывается имя загружаемого файла в формат *.DBF;
  • «Каталог выгрузки» - указывается каталог для выгружаемого файла в формате *.DBF;

Если структура файла загрузки/выгрузки не будет соответствовать структуре описанной в таблице, то загрузка/выгрузка данных будет невозможна.
 
Для загрузки данных в обработке необходимо указать на закладке «Загрузка» файл формата *.DBF в поле «Файл загрузки *.dbf» со структурой описанной выше.

Для заполнения табличной части на закладке «Загрузка» данными из файла и БД необходимо нажать на кнопку «Заполнить». При этом цвет фона строк табличной части и доступность использования поля «Использовать» зависит от различных условий заполнения строк табличной части. Строка окрашивается в голубой цвет и поле «Использовать» становится недоступным в следующих ситуациях:
  1. Если не заполнены значение данных «Проживающий», «Категория льгот», «Срок начала»;
  2. Если льготная категория уже установлена на проживающего и производится попытка установить эту же льготную категорию (тип операции "N");
  3. Если льготная категория не действует на проживающего и производится попытка прекращения этой же льготной категории (тип операции "D");
 
В обработке есть возможность настроить соответствие услуг. Для того, чтобы вызвать диалоговое окно настройки необходимо нажать на кнопку «Настройка». Данная настройка доступна на закладке «Выгрузка».
 
Для заполнения табличной части на закладке "Выгрузка" данными из БД необходимо нажать кнопку «Заполнить». При этом цвет фона строк табличной части зависит от следующих условий:
  1. Для строк, в которых не заполнено значение ячейки "Код услуги", фон закрашивается в голубой цвет.
  2. Для строк, в которых заполнено значение ячейки "Код услуги", фон закрашивается в "Белый" цвет.
 
Для выгрузки данных в обработке необходимо указать во вкладе «Выгрузка» каталог выгрузки в поле «Каталог выгрузки». Формат файла выгрузки в формате *.DBF представлен выше.
