Московская область (г. Жуковский)

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Московской области г.Жуковский за указанный период.

Перед началом работы с выгрузкой необходимо заполнить следующие параметры:

  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;
  • «Период» - месяц и год, за который необходимо предоставить данные;
  • «Каталог выгрузки» - каталог выгрузки файлов в формате *.DBF;
  • «Код организации» - код организации, который назначен в ОСЗН;
  • «Отбор по зданию/группе зданий» - ссылка на элемент/группу справочника «Здания, сооружения», по которму будут браться необходимые данные за указанный период;
  • «Вид отапливаемой площади» - указывается вид площади из базы соответствующий виду отапливаемой площади.

Настройка

Перед началом заполнения таблиц выгрузки необходимо произвести её настройку. Для вызова диалогового окна настройки необходимо нажать на кнопку «Настройка» на форме обработки.

На странице «Соответствие видов услуг» необходимо произвести настройку соответствия услуг в ОСЗН и элементов справочника «Услуги». При этом:

  • В левой таблице всегда содержится список из фиксированных согласно формату выгрузки услуг в ОСЗН, которые не доступны для редактирования.

  • В правой таблице содержится список услуг из справочника «Услуги» (далее - Услуга из базы), которые установлены в соответствие текущей услуге из ОСЗН. При этом:

    • В поле «Тип услуги» необходимо указать тип услуги. Доступны 3 варианта: «Индивидуальная», «Общедомовая» и «Индивидуальная и общедомовая».
      Предпосылкой появления данной настройки стало то, что в рамках программы пользователи создают Услуги в базе в наименовании которых предполагается что услуга является общедомовой, но при этом используется методика расчета как индивидуальной услуги. Н-р, для таких услуг как "ХВС на ОДН", "ГВС на ОДН". При этом расчет по данным услугам производится по нормативам потребления.
      Соответственно, чтобы оплаты по данным услугам попадали в колонки "Начислено к оплате ОДН по услуге "Холодное водоснабжение"", "Начислено к оплате ОДН по услуге "Горячее водоснабжение"" и "Начислено к оплате ОДН по услуге "Электрическая энергия"" необходимо для данных услуг установить тип услуги "Общедомовая".
      В случае, если для данных услуг тип установлен "Индивидуальная" или "Индивидуальная и общедомовая", то сведения об оплатах будут отражены в колонках "Начислено к оплате по услуге "Холодное водоснабжение"", "Начислено к оплате по услуге "Горячее водоснабжение"" и "Начислено к оплате по услуге "Электрическая энергия"" соответственно.
      Если выбран тип услуги "Индивидуальная и общедомовая", то колонки с оплатами на индивидуальную и общедомовую составляющие текущей услуги в ОСЗН будут заполняться в соответствии с расчетом, который настроен для соответствующих Услуг в базе.

Выгрузка данных в ОСЗН

Для заполнения табличных частей обработки данными необходимо нажать на кнопку «Заполнить». Для выгрузки данных из табличных частей обработки в каталог, указанный в поле «Путь к файлам», необходимо нажать на кнопку «Выгрузить». Для удобства заполнения и выгрузки всех табличных частей выгрузки добавлены кнопки «Заполнить все» и «Выгрузить все» соответственно.
