Мурманская область

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Мурманской области. 

 

Перед началом выгрузки данных в обработке необходимо заполнить следующие параметры:

  • «Организация» - ссылка на элемент справочника «Организации» от имени которой будут выгружены данные;  
  • «Каталог выгрузки» - каталог, в котором будут размещены выгружаемые файлы;
  • Настройки, которые необходимо установить на вкладке «Перечень физических лиц»:
    • «Дата выгрузки» - дата, по состоянию на которую необходимо предоставить перечень физических лиц;
  • Настройки, которые необходимо установить на вкладке «Паспортные данные»:
    • «Дата выгрузки» - дата, по состоянию на которую необходимо предоставить сведения о паспортных данных жильцов;
  • Настройки, которые необходимо установить на вкладке «Информация о выбывших жильцах»:
    • «Период» - месяц, за который необходимо предоставить данные о выбывших жильцах;
  • Настройки, которые необходимо установить на вкладке «Информация о начислениях»:
    • «Период» - месяц, за который необходимо предоставить информацию о произведенных начислениях;
  • Настройки, которые необходимо установить на вкладке «Информация о тарифах»:
    • «Дата выгрузки» - дата, по состоянию на которую необходимо предоставить информацию об установленных тарифах;
  • Настройки, которые необходимо установить на вкладке «Параметры домов и жилых помещений»:
    • «Дата выгрузки» - дата, по состоянию на которую необходимо предоставить данные о параметрах домов и жилых помещений;
    • «Подключение к газу» - элемент справочника «Виды благоустройства объектов», согласно которому необходимо определять признак подключения помещения к газовому снабжению.
  • «Только льготники» - признак выгрузки сведений только по тем лицевым счетам, для жильцов которых установлена действующая льготная категория по состоянию на дату выгрузки или действовавшая в период выгрузки (с учетом выбранной вкладки выгрузки);
  • На панели настроек данной выгрузки доступны следующие вкладки:
    • «Здания».
      На данной странице можно указать список зданий с учетом которых необходимо выполнить выгрузку.
    • «Соответствие услуг».
      На данной странице необходимо установить соответствия между услугами в ОСЗН и услугами из справочника «Услуги».
      В таблице «Услуги в ОСЗН» содержится фиксированный набор услуг из ОСЗН. Не доступен для редактирования! 
      В таблице «Услуги в базе» необходимо из справочника «Услуги» выбрать перечень услуг, которые соответствуют текущей выбранной услуге из ОСЗН.

Для того, чтобы начать выгрузку данных, необходимо перейти на соответствующую вкладку и нажать на кнопку «Выгрузить».
