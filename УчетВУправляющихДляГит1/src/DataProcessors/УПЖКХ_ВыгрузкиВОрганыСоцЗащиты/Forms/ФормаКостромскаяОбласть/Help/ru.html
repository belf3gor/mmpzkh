Костромская область

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Костромской области.

Перед работы с обработкой необходимо заполнить следующие параметры:

  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;
  • «Период» - месяц, за который необходимо предоставить данные.
  • «Код организации» - код организации, заполняется для соответствия данных в файле обмена ОСЗН. Если в файле обмена в текущей строке значение поля "KPRED" отличается, то строка при чтении файла игнорируется.
  • «Вид площади отопления» - ссылка на элемент справочника "Виды площадей". Выбирается вид площади помещения, по которой определяется размер площади помещения для рассчета размера начисления по услуге "Отопление".
  • «Каталог загрузки» - каталог загрузки файлов формата *.DBF.
  • «Каталог выгрузки» - каталог выгрузки файлов формата *.DBF.
  • Соответствие кодов услуг в файле обмена и услуг в информационной базе. Соответствие устанавливается в окне настроек обработки (открывается по нажатию кнопки "Настройка" на командной панели формы).

 

Для заполнения табличной части обработки данными необходимо нажать на кнопку «Заполнить». Для выгрузки данных из табличных частей обработки в файлы в формате *.DBF  в каталог указанный в поле «Каталог выгрузки» необходимо нажать на кнопку «Выгрузить».
