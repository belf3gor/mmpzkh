Алтайский край

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Алтайского края за указанный период. 
 
Перед началом выгрузки данных в обработке необходимо заполнить следующие параметры:
  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;
  • «Период» - месяц, за который необходимо предоставить данные.
 
Для заполнения табличной части обработки данными необходимо нажать на кнопку «Заполнить». Для выгрузки данных из табличной части обработки в файл необходимо указать каталого выгрузки в поле «Каталог выгрузки», указать его имя в поле «Имя файла» и нажать на кнопку «Выгрузить».
