Курская область

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Курской области. Данная выгрузка разработана в соответствии с постановлением Правительства Курской области от 24.04.2009 года №58 "О предоставлении отдельным категориям граждан компенсационных выплат в связи с расходами по оплате жилых помещений и коммунальных услуг".
 
Перед началом загрузки данных в обработке необходимо заполнить следующие параметры: 
  • «Период» - месяц, за который необходимо предоставить данные;
  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;
  • «Файл загрузки» - путь к файлу (далее - Файл загрузки), в который будут загружены сведения о начислениях из информационной базы (файлы формата *.DBF);


Перед началом использования данной выгрузки, необходимо выполнить её настройку. Для этого необходимо нажать кнопку "Настройка", расположенную в верхней части основной формы.
Далее откроется форма настройки состоящая из следующих страниц:
1. «Настройка соответствия услуг».
    На данной странице устанавливаются соответствия услуг из Файла загрузки с услугами из базы. В левой части страницы содержится таблица (далее - Таблица услуг в ОСЗН), в которой необходимо указать все услуги из Файла загрузки с их кодами.
    Для удобства предусмотрено автоматическое заполнение Таблицы услуг в ОСЗН на основе данных из Файла загрузки. Для этого необходимо нажать кнопку «Загрузить из файла».
2. «Прочие настройки».
    На данной странице в поле «Кодировка файла загрузки DBF» необходимо указать кодировку Файла загрузки. Доступны два значения: «Windows» и «DOS».

Функционально обработка разделена на три части:

  • Загрузка из Файла загрузки данных о льготниках, производится при помощи нажатия кнопки «Загрузить из файла». Предварительно в поле «Файл загрузки» необходимо указать DBF-файл, формат которой аналогичен с форматом файла «IN_FILE.DBF», структура которого представлена ниже. 
  • Заполение данных о начислениях и оплатах за указанный период, производится при помощи нажатия кнопки «Заполнить». Если ни один льготник не идентифицирован, то будет выведено соответствующее информационное сообщение.
  • Выгрузка данных в файл о начислениях и оплатах за указанный период, производится при помощи нажатия кнопки «Выгрузить в файл». При этом выгружаемые данные будут записаны в Файл загрузки, а исходный вариант Файла загрузки будет записан в новый файл, наименование которого формируется следующим образом: [Наименование Файла загрузки] + "_Исходный". При выгрузке данных в Файле загрузки предусмотрено заполнение только тех полей, которые в структуре формата файла «IN_FILE.DBF» выделены зеленым цветом.
