Московская область (г.Подольск)

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Московской области за указанный период на основании Постановления Правительства Московской области от 30.12.2008 №1235/52 "Об утверждении Порядка предоставления компенсации расходов по оплате жилого помещения и коммунальных услуг отдельным категориям граждан, имеющим место жительства в Московской области". Перед началом работы с выгрузкой необходимо заполнить следующие параметры:
  • «Период» - месяц и год, за который необходимо предоставить данные;
  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;
  • «Путь к каталогу» - каталог выгрузки файла в формате *.XLS;

Настройка

Перед началом заполнения таблицы выгрузки на основной форме обработки необходимо произвести её настройку. Для вызова диалогового окна настройки необходимо нажать на кнопку «Настройки» на форме обработки. На форме «Настройки» необходимо указать следующие данные:
  • Соответствие услуг ОСЗН и услуг в базе. Для этого в таблице на закладке «Таблица соответствия» необходимо указать услуги или список услуг из справочника "Услуги" в соответствующей колонке таблицы для каждой услуги ОСЗН:
    • Содержание и тех.обслуживание жилья;
    • Горячее водоснабжение;
    • Холодное водоснабжение;
    • Отопление;
    • Электроэнергия;
    • ОДН ГВС;
    • ОДН Э/Э.
  • Порядок выгружаемых данных по зданиям. Для этого в таблице на закладке «Порядок зданий» необходимо задать порядок вывода данных в файле выгрузки по зданиям. Данная таблица автоматически заполняется всеми зданиями справочника "Здания".

Выгрузка данных в ОСЗН

Для заполнения таблицы на основной форме обработки с выгружаемыми данными необходимо нажать на кнопку «Заполнить». Для выгрузки данных в файл выгрузки, путь к которому указан в поле «Путь к каталогу», необходимо нажать на кнопку «Выгрузить». Имя файла выгрузки формируется следующим образом:

<название месяца выгрузки> <год выгрузки>.xls
