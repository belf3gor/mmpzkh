Красноярский край (вариант 2)

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Красноярского края за указанный период.

Перед началом выгрузки данных в обработке необходимо заполнить следующие параметры:

  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;

  • «Период» - месяц, за который необходимо предоставить данные;

  • «Лицевые счета» - группа лицевых счетов справочника «Лицевые счета».
     Выгрузка будет осуществлена только по тем лицевым счетам, которые содержатся в выбранной группе.
     Если группа не выбрана, то выгрузка производится по всем лицевым счетам, помещения которых обслуживает выбранная организация;

  • «Каталог выгрузки» - каталог для выгружаемых файлов в формате *.DBF;

Перед заполнением табличных частей обработки необходимо установить предварительные настройки. Для открытия формы настройки необходимо нажать кнопку «Настройка», расположенную в верхней части обработки.

Если все необходимые настройки установлены, то можно начать заполнение таблиц обработки. Для этого в каждой вкладке имеется кнопка «Заполнить».


При этом для корректного заполнения всех таблиц выгрузки необходимо произвести следующую последовательность действий:

1. Во-первых, необходимо заполнить таблицу «Квартиросъемщики».

2. Далее можно заполнить либо таблицу «Сумм оплат по услугам», либо таблицу «Сведения о членах семьи».

Для выгрузки данных, содержащихся в соответствующей таблице обработки, необходимо нажать кнопку «Выгрузить».


На форме обработки содержатся кнопки «Заполнить все» и «Выгрузить все», предназначенные для одновременного заполнения или выгрузки данных во всех таблицах.
