Вологодская область

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Вологодской области за указанный период.

Перед началом выгрузки данных в обработке необходимо заполнить следующие параметры:
  • «Период» - месяц, за который необходимо предоставить данные;
  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;
  • «Файл загрузки» - путь к файлу формата *.DBF, с которого будет загружена необходимая информация для заполнения табличной части обработки.

Перед началом выгрузки для корректного заполнения табличных частей обработки необходимо произвести предварительную настройку. Для вызова диалогового окна настройки необходимо нажать на кнопку «Настройка». В появившемся окне необходимо произвести настройку соответствия данных из файлов и данных БД. На закладке "Виды собственности" произвести настройку соответствия кодов видов собственности в ОСЗН видам собственности из БД. На закладке "Услуги" произвести настройку соответствия кодов услуг в ОСЗН услугам из БД. На закладке "Коды организаций" произвести настройку соответствия кодов организаций в ОСЗН организациям из БД. На закладке "Коды льготных категорий" произвести настройку соответствия кодов льготных категорий в ОСЗН льготным категориям из БД. На закладке "Формы управления МКД" произвести указание форм управления многоквратирными зданиями. Для заполнения соответствующего табличной части необходимо нажать на кнопку «Заполнить» командной панели соответствующей таблицы.

Если структура файла не будет соответствовать структуре описанной в таблице, то загрузка и выгрузка данных будет невозможна.
 
Идентификация льготников в базе происходит по следующим полям загружаемого файла:
FAM, IM, OT, PUNKT_NM, STREET_NM, HOUSE, KORP, FLAT.
Данные поля должны быть заполнены в файле загрузки, иначе идентификация льготников произойдет некорректно.
 
Для осуществления выгрузки данных из табличной части необходимо нажать на кнопку «Выгрузить». Структура файла выгрузки соответствует структуре загружаемого файла.
