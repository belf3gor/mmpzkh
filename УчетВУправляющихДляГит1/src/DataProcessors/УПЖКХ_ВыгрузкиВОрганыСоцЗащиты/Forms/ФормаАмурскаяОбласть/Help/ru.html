Амурская область

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Амурской области за указанный период.

Перед началом выгрузки данных в обработке необходимо заполнить следующие параметры:
  • «Период» - месяц, за который необходимо предоставить данные в ОСЗН (далее - Месяц выгрузки);
  • «Организация» - ссылка на элемент справочника «Организации», по которой будут получены и выгружены необходимые данные за Месяц выгрузки;  
  • «Каталог загрузки» - указывается каталог, в котором расположены файлы(*.dbf или *.xls), которые присылает ОСЗН;
  • «Каталог выгрузки» - указывается каталог для выгружаемых файлов;


Для заполнения данных о задолженностях, начислениях и оплатах необходимо произвести предварительную настройку. Для вызова диалогового окна настройки необходимо нажать на кнопку «1. Настройка». На форме настройки на закладке "Настройки соответствия услуг" необходимо установить соответствие услуг в ОСЗН и услуг из справочника "Услуги" (далее - Услуги в базе).

Для каждой услуги в ОСЗН необходимо определить способ свертки тарифа:

  • «Максимум» - для данной услуги ОСЗН будет браться максимально возможное значение тарифа среди установленных в соответствие Услуг из базы (*);
  • «Минимум» - для данной услуги ОСЗН будет браться минимально возможное значение тарифа среди установленных в соответствие Услуг из базы (*);
  • «Усреднять» - для данной услуги будет браться среднее значение тарифа среди установленных в соответствие Услуг из базы (*);
  • «Суммировать» - для данной услуги будет браться суммарное значение тарифа среди установленных в соответствие Услуг из базы (*).

(*) - все перечисленные операции производятся только по тем услугам, которые назначены на идентифицированные лицевые счета по состоянию на конец Месяца выгрузки.

Для всех услуг в ОСЗН должно быть установлено соответствие с одной или несколькими Услугами в базе. Если для услуг в ОСЗН не установлено соответствие ни с одной Услугой в базе, то сведения по ним не будут выгружены в файлы.

На закладке "Дополнительные настройки" необходимо указать способ идентификации в базе льготополучателей из файла ОСЗН ("по адресу регистрации" - полная идентификация по адресу помещения, в котором открыт лицевой счет; "по номеру лицевого счета" - идентификация по номеру лицевого счета), указать кодировку загружаемых и выгружаемых файлов. Также необходимо указать признак включения в конечное сальдо сведений о начислениях, произведенных в Месяце выгрузки.

Заполнение табличных частей производится следующим образом. По нажатии на кнопку "2. Загрузить файл из каталога загрузки" будет произведена загрузка данных из входящих файлов. Для заполнения таблиц необходимо нажать кнопку "3. Заполнить данными из информационной базы".

Для осуществления выгрузки данных из табличных частей необходимо нажать на кнопку «4. Выгрузить файл в каталог выгрузки». Выгрузка данных в обработке происходит в файлы в формате "ММ-ГГГГZ(P).dbf" (xlsx), где ММ – отчетный месяц (01-12), ГГГГ – отчетный год (2015 и т.д.), Z- константа (признак файла от поставщика услуг - задолженность). P - константа (признак файла от поставщика услуг)
