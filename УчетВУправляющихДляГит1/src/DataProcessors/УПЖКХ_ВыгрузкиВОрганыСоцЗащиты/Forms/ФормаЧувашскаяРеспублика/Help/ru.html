Чувашская республика

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Чувашской республики за указанный период.

Перед началом выгрузки данных, в обработке необходимо заполнить следующие параметры:
  • «Период» - месяц, за который необходимо предоставить данные;
  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;
  • «Файл загрузки» - указывается имя загружаемого файла в формат *.CSV;
  • «Кодировка файла» - указывается кодировка файлов загрузки/выгрузки.


Перед началом загрузки/выгрузки данных, для корректного заполнения табличных частей обработки, необходимо произвести предварительную настройку. Для вызова диалогового окна настройки необходимо нажать на кнопку «Настройка».
1. В табличной части вкладки «Настройка соответствия услуг» необходимо установить соответствия между услугами в ОСЗН и услугами в организации.
2. В табличной части вкладки «Настройка соответствия степеней родства» необходимо установить соответствия между степенями родства в ОСЗН и степенями родства в базе.
3. В табличной части вкладки «Настройка соответствия норм зданий» по необходимости можно установить соответствия между зданиями в Базе и нормами потребления услуг в Базе, которые необходимо выгружать по всем лицевым счетам, принадлежащим текущему зданию в поле "Норматив потребления" таблицы "Сведения об услугах ЖКХ". Если норма для здания не указана, то по умолчанию будут выгружены те нормы потребления по услугам, которые актуальны для выгружаемых лицевых счетов по состоянию на конец отчетного периода.
4. На вкладке «Тариф услуги подогрева» по необходимости можно указать услугу, соответствующую услуге "Подогрев" в ОСЗН и указать для нее тариф, который необходимо выгружать.

Для загрузки данных в обработке обязательно необходимо указать файл в поле «Файл загрузки» и указать его кодировку в поле «Кодировка файла».

Типы записей в файлах загрузки/выгрузки данных.
A – идентификация источника сведений;
О – основная запись об учетных данных льготника;
Ж – сведения о гражданах и характеристиках жилья;
С – сведения о составе семьи льготника;
У – сведения об услугах ЖКХ.

Запись типа «A» должна быть всегда первой записью каждого файла.
Запись типа «О» является уникальной и может быть представлена на одно лицо, имеющее право на льготы по оплате жилья и коммунальных услуг.
Запись типа «Ж» должна идти после записи типа «O», обязательно должна присутствовать в файле выгрузке.

В файле загрузки должны обязательно присутствовать запись типа «A» и записи типа «О».
В файле выгрузки будут присутствовать запись типа «A» и записи типа «О». Далее если адрес идентифицирован на каждую запись типа «О» обязательно будет присутствовать запись типа «Ж» и допускаются запись(и) типа «С», запись(и) типа «У».
В записях типа «С» данные из записи типа «О» не указываются, т.е. в составе семьи сам льготник не указывается.

Для заполнения таблиц обработки данными необходимо нажать на кнопку «Считать данные из файла загрузки и заполнить таблицы» на верхней командной панели формы.
Для выгрузки заполненных в таблицах обработки данных в файл формата *.CSV необходимо нажать на кнопку «Выгрузить данные в файл» на верхней командной панели формы.

Выгружаемый файл будет сохранен в тот же каталог, где находится файл загрузки.
