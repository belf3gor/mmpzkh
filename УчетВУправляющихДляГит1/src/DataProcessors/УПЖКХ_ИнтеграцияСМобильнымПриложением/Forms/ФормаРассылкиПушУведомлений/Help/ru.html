Форма «Рассылка push-уведомлений»

Данная форма предназначена для рассылки push-уведомлений пользователям мобильного приложения «Ломбард:Личный кабинет».

 

На форме имеются следующие поля ввода:

    • Заголовок сообщения - текст заголовка уведомления. Данный заголовок первым делом отобразиться на мобильных устройствах пользователей;
    • Текст сообщения - текст самого уведомления. Данный текст пользователи мобильных устройств увидят после открытия уведомления;
    • Дата начала рассылки - дата, начиная с которой будет происходить рассылка уведомлений;
    • Дата окончания рассылки - дата, когда рассылка текущего уведомления будет прекращена.

 

Для того чтобы начать рассылку push-уведомлений, необходимо нажать на кнопку "Отправить". После успешной отправки уведомлений появится соответствующее сообщение.
