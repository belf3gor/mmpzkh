Обработка «Работа с должниками»

Данная обработка предназначена для работы с лицевыми счетами, имеющими задолженность.

Список лицевых счетов, имеющих задолженность, заполняется автоматически согласно установленному отбору:

  • «Сведения на» - период, на который имеется задолженность по лицевым счетам;
  • «по организации» - организация, по которой имеется задолженность - ссылка на справочник "Организации";
  • «Объект» - здание или помещение, которому соответствуют лицевые счета - ссылка на справочник "Здания", или на справочник "Помещения";
  • «Услуга» - услуга или группа услуг, по которым имеется задолженность - ссылка на справочник "Услуги";
  • «Минимальная сумма» - минимальная сумма задолженности;
  • «Минимальный срок» - минимальный срок задолженности.

Для выбранного лицевого счета отображается дополнительная информация на вкладке «Журнал документов» в разрезе следующих документов:

  • «Предупреждение» - перечень отправленных уведомлений собственнику/нанимателю - документы "Предупреждение";
  • «Исковое заявление» - перечень составленных исковых заявлений по собственнику/нанимателю - документы "Исковое заявление";
  • «Судебное соглашение» - перечень заключенных судебных соглашений по собственнику/нанимателю - документы "Судебное соглашение".

Таблицы на вкладках "Взаиморасчеты" и "Карточка должника", "Отчеты" формируются отчеты по задолженности по выбранному лицевому счету/выбранным счетам.
