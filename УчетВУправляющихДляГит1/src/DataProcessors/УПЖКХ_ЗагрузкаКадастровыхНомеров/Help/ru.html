Обработка «Загрузка кадастровых номеров зданий и помещений»

Обработка предназначена для загрузки с сервиса Росреестра кадастровых номеров зданий и помещений, используемых для выгрузки в ГИС ЖКХ.

Для начала работы с обработкой необходимо указать токен (кодовое слово), полученный на сервисе Росреестра в соответствии с инструкцией.

Обработка содержит следующие таблицы:

  • Таблица объектов учета, для которых необходимо заполнить кадастровые номера;
  • Таблица загруженных кадастровых номеров, в которой выводятся все загруженные кадастровые номера объекта, выбранного в таблице объектов учета. Выбор кадастрового номера из предложенных в таблице выполняется двойным кликом по его строке.

Заполнение таблицы объектов учета можно выполнить одним из следующих способов:

  • Вручную, с помощью кнопки «Добавить»;
  • Всеми объектами учета с незаполненным кадастровым номером с помощью кнопки «Заполнить объектами без кадастровых номеров» меню «Заполнить»;
  • Всеми объектами учета независимо от заполненности кадастрового номера с помощью кнопки «Заполнить всеми объектами» меню «Заполнить».

Загрузить кадастровые номера с сервиса Росреестра можно одним из следующих способов:

  • С помощью кнопки «Загрузить кадастровые номера», по нажатию на которую будет выполнена загрузка кадастровых номеров для всех объектов в таблице объектов учета, у которых не заполнена колонка «Новый кадастровый номер»;
  • Вручную для объекта, выбранного в таблице объектов учета, путем ввода адреса объекта учета в поле «Адрес» и нажатия кнопки «Обновить кадастровые номера объекта».

Если в результате загрузки для объекта учета найден единственный кадастровый номер, то он автоматически будет добавлен в колонку «Новый кадастровый номер» соответствующей строки таблицы объектов учета.

Если в результате загрузки для объекта учета не найден кадастровый номер или кадастровых номеров найдено несколько, то будет необходимо указать кадастровый номер вручную из таблицы загруженных кадастровых номеров или изменить представление адреса объекта в поле «Адрес» и повторить загрузку кадастрового номера.

Отобрать только те объекты учета, для которых не удалось определить новый кадастровый номер, можно с помощью кнопки «Показать объекты учета с незаполненными кадастровыми номерами».

Сохранить новые кадастровые номера можно с помощью кнопки «Сохранить кадастровые номера», по нажатию на которую существующие кадастровые номера объектов учета будут заменены на кадастровые номера, указанные в колонке «Новый кадастровый номер» таблицы объектов учета. При этом, замена кадастровых номеров выполняется только для тех объектов учета, у которых новый кадастровый номер заполнен.

В случае изменения вашего токена на сервисе Росреестра необходимо ввести новый токен с помощью «Формы изменения токена». Для этого нажмите на кнопку «Изменить токен» и в открывшейся форме укажите новый токен.
