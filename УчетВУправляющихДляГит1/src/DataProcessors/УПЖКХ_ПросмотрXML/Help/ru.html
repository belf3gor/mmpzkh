Обработка «Просмотр XML»

Данная обработка предназначена для чтения файлов XML, а так же ZIP-архивов, содержащих файлы XML, и позволяет расшифровывать содержимое файлов XML для представления в удобном для пользователя виде.

Расшифровка файлов осуществляется с помощью библиотек расшифровки тегов и атрибутов, представляющих собой макеты «Библиотека расшифровки тегов XML» и «Библиотека расшифровки атрибутов XML», в которые можно вносить изменения с помощью пользовательских макетов печати.


Макет «Библиотека расшифровки тегов XML» содержит следующие колонки:

  • «Наименование тега» - наименование тега, для которого необходимо выполнять расшифровку.
  • «Наименование для замены» - текст, который необходимо использовать в качестве наименования тега.
  • «Значение дочернего атрибута для вывода в наименовании тега» - наименование атрибута в тега, значение которого необходимо использовать в качестве наименования тега.
  • «Выводить текстовое содержимое тега» - любое значение колонки, отличное от пустой строки, указывает, что для тега необходимо выводить его текстовое содержимое. Вывод текстового содержимого осуществляется в отдельном поле на форме обработки.
  • «Родительский тег» - наименование тега, в котором должен располагаться тег, указанный в колонке «Наименование тега».
    • Допускается наличие нескольких строк с одинаковым значением колонки «Наименование тега» для различных способов расшифровки в зависимости от расположения тега в файле и от его содержимого.

    Макет «Библиотека расшифровки атрибутов XML» содержит следующие колонки:
  • «Наименование атрибута» - наименование атрибута, для которого необходимо выполнять расшифровку.
  • «Наименование для замены» - текст, который необходимо использовать в качестве наименования атрибута.
  • «Родительский тег» - наименование тега, в котором должен располагаться атрибут.
  • «Тип атрибута» - тип атрибута. Может принимать значение «Число» или «Дата». Если не заполнен, то считается, что содержимое атрибута является строкой.
    • Допускается наличие нескольких строк с одинаковым значением колонки «Наименование атрибута» для различных способов расшифровки в зависимости от расположения атрибута в файле.
