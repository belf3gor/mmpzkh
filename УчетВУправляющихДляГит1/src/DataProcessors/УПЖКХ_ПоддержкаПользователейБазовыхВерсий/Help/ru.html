Обработка «Поддержка пользователей базовых версий»

Данная обработка предназначена для проверки наличия действующей подписки ИТС Отраслевой, которая требуется пользователям БАЗОВОЙ версии программы, без которой некоторый функционал программы недоступен. Для проверки наличия подписки необходимо указать регистрационный номер продукта, с которым вы сейчас работаете. Регистрационный номер можно найти на коробке, компакт-диске или книгах документации к программному продукту.
