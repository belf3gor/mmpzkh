Автоматическая активация (при наличии доступа к Интернету)

Если ваш компьютер подключен к Интернету и у вас есть учетная запись на Портале 1С:ИТС, то такой способ предпочтителен.

В диалоге «Активация программы» укажите логин и пароль к вашей учетной записи на Портале 1С:ИТС (http://portal.1c.ru/) и нажмите «Активировать через Интернет».

Программа подключится к порталу и получит список поставок программы, которые зарегистрированы в личном кабинете 1С:ИТС. Если среди них найдется регистрация организации, для которой ведется учет в информационной базе, то будет выполнена активиция программы.

Если нужной регистрации нет, то будет выдано сообщение об ошибке. В этом случае нужно указать логин и пароль к той учетной записи Портала 1С:ИТС, на которую зарегистрирована программа, либо выполнить активацию с помощью файла регистрации.

Активация с помощью файла регистрации (при отсутствии доступа к Интернету)

Если доступа к Интернету нет, то можно активировать программу с помощью файла активации. Файл активации можно скачать из раздела «Программные продукты» Портала 1С:ИТС.

Для выполнения активации нужно нажать кнопку «Выбрать файл регистрации» и указать файл, который вы скачали с Портала 1С:ИТС. Файл регистрации представляет собой zip-архив с регистрационными данными. Если файл регистрации содержит сведения организации,для которой ведется учет в информационной базе, то будет выполнена активиция программы. Если файл регистрации содержит сведения о другой организации, то будет выдано сообщение об ошибке.
