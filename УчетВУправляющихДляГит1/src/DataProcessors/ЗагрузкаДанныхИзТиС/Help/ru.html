Обработка "Загрузка данных из конфигурации "Торговля и Склад"

Продажи ®  Обмен данными с ТиС ® Загрузка из Торговля и Склад 77

Обработка предназначена для загрузки данных оперативного учета, предварительно выгруженных из конфигурации "Торговля и склад".

В поле Файл данных укажите имя файла, из которого будут загружены данные.

Данные загружаются в текущую информационную базу по кнопке Загрузить данные.
