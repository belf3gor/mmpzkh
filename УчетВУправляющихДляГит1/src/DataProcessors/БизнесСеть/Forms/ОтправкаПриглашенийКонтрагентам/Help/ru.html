Отправка приглашений контрагентам

В форме производится приглашение клиентов (поставщиков и покупателей) в сервис 1С:Бизнес-сеть.

Приглашения отправляется на адреса электронной почты контрагентов. Отправка производится от имени организации, зарегистрированной в сервисе.

Список контрагентов может быть сформирован вручную командой Добавить, Подобрать и автоматически по совершенным сделкам (по поставкам, по закупкам, по всем сделкам).

  • Для всех контрагентов - общий шаблон письма приглашения клиентов в сервис.
  • Для поставщиков - шаблон приглашения поставщиков.
  • Для покупателей - шаблон приглашения покупателей.

Если контрагент уже зарегистрирован в сервисе, в поле Электронная почта будет указано "Зарегистрирован, приглашение не требуется". После отправки приглашений незарегистрированным контрагентам в поле e-mail появится надпись Приглашение отправлено".
