Обработка "Загрузка заявок аварийно-диспетчерской службы"

Обработка предназначена для загрузки заявок аварийно-диспетчерской службы из файла формата xml.

Перед выполнением загрузки указываются:

  • «Организация» – организация, по которой загружаются данные;  
  • «Имя файла загрузки» – выбирается загружаемый файл формата xml;

На панели инструментов табличной части расположены кнопки:

  • «Загрузить заявки» – при нажатии данной кнопки табличная часть обработки заполняется данными из загружаемого файла.
  • «Сформировать документы» – при нажатии данной кнопки на основании данных из табличной части формируются документы «Заявка на выполнение работ».

Отчет открывается через меню «Учет в ЖКХ / Сервис / Загрузка заявок аварийно-диспетчерской службы».
