Обработка "Выгрузка в Систему Город"

Обработка предназначена для формирования файла экспорта данных в Систему Город в формате TXT, VEK или DBF, включающего информацию о плательщиках, задолженностях, показаниях счетчиков или начислениях за услуги ЖКХ.

Перед выполнением выгрузки необходимо указать основные настройки:

  • «Организация» – организация, по которой выгружаются данные;
  • «Период» («Дата выгрузки») – период выгружаемых данных (дата, на которую необходимо выгрузить данные);
  • «Формат выгрузки» – вариант выгрузки данных;
  • «Каталог выгрузки» – выбирается путь для сохранения файла выгрузки;

После выбора определенного формата выгрузки данных на закладке «Дополнительные настройки» будут доступны одни из следующих настроек (для каждого формата доступны свои настройки):

  • Настройки формирования строк файла выгрузки:
    • «Выводить только инициалы ФИО» - признак вывода в строках файла выгрузки инициалов фамилии, имени и отчества ответственного собственника/нанимателя лицевого счета. Доступен только для тех ответственных собственников/нанимателей, которые являются жильцами, и которые являются контрагентами с видом "Физическое лицо";
    • «Кодировка» – кодировка символов файла выгрузки;
    • «В поле лицевой счет выводить» – значение вывода в поле лицевого счета файла выгрузки (код, наименование или идентификатор лицевого счета);
    • «В поле счетчик выводить» – значение вывода в поле счетчика файла выгрузки (код, наименование, идентификатор или заводской номер счетчика);
    • «Константа» – значение константы для вывода в строках файла выгрузки;
    • «Префикс» – значение префикса лицевого счета для вывода в строках файла выгрузки;
    • «Номер дополнения» – номер дополнения для вывода в строках файла выгрузки;
    • «Номер экранной формы» – номер экранной формы для вывода в строках файла выгрузки;
    • «Символ резделителя суммы» – символ, используемый для разделения целой и дробной части выгружаемой суммы долга;
    • «Символ разделителя корпуса здания» – символ разделителя корпуса здания в поле адреса лицевого счета (символ "/" или символ "К");
    • «Не выводить город в поле адреса» – признак выгрузки наименования города в поле адреса лицевого счета файла выгрузки;
    • «Услуга ИПУ» – услуга ИПУ, по которой выгружаются показания счетчиков;
    • «Не выгружать данные по счетчикам» – признак того, что в файл выгрузки не будут выгружаться сведения по приборам учета;
    • «Выгружать лицевые счета с переплатами» – признак включения в файл выгрузки информации по лицевым счетам с отрицательной задолженностью (переплатой);
    • «Выводить конечную дату» – признак вывода конца периода выгрузки в строках файла выгрузки;
    • «Символ разделителя полей выгрузки» – символ разделителя полей строк файла выгрузки;
    • «Определять задолженность на» – признак получения задолженности по выгружаемым лицевым счетам на начало или конец отчетного периода;
    • «Холодная вода», «Горячая вода», «Электроэнергия» – услуги соответствия, по которым будут выгружаться показания индивидуальных приборов учета;
  • Настройки формирования заголовка файла выгрузки:
    • «Номер услуги банка» – номер услуги банка для вывода в заголовке файла выгрузки;
    • «Примечание» – примечание, выводимое в заголовке файла выгрузки;
  • Настройки формирования имени файла выгрузки:
    • «Порядковый номер файла» – порядковый номер файла в реестре;
    • «Постоянный символ» – постоянный символ;
    • «Расчетный счет» – номер расчетного банковского счета в Отделении Сбербанка;
    • «Код ОСБ» – код Отделения Сбербанка, с которым заключен договор;
    • «Номер договора» – номер заключенного с Отделением Сбербанка договора;

На закладке «Отбор лицевых счетов» можно произвести отбор по выгружаемым лицевым счетам.

«Выгружать» - настройка отбора лицевых счетов. При выборе пункта «все лицевые счета» отбор по лицевым счетам выполнен не будет. При выборе пункта «указанные лицевые счета» становится доступной табличная часть, в которой необходимо указать лицевые счета, по которым необходимо выгружать данные.

На закладке «Отбор услуг» можно произвести отбор по выгружаемым услугам. «Выгружать» – вариант получения данных о задолженности, показаниях счетчиков или начислений за услуги ЖКХ с разделением по учету услуг капремонта ("по всем услугам" и "по указанным услугам". Если по отчетной организации ведется раздельный учет по услугам кап. ремонта, то становятся доступными еще 2 варианта отбора услуг: "по услугам отличным от кап. ремонта" и "по услугам кап. ремонта"). При выборе пункта «Указанные услуги» становится доступной табличная часть для указания услуг, по которым необходимо выгрузить данные.

При нажатии на кнопку «Описание» можно увидеть описание формата выгрузки данных.

В верхней части диалогового окна формы расположена кнопка «Выгрузить сведения», при нажатии которой будет сформирован файл экспорта в формате TXT, VEK или DBF и сохранен в каталоге выгрузки.

Отчет открывается через меню «Интеграция / Сервис / Выгрузка в Систему Город».
