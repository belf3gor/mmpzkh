Выгрузка в систему приема платежей «Платон»

Обработка предназначена для выгрузки данных в систему приема платежей "Платон" за указанный период.

Перед началом выгрузки данных в обработке необходимо заполнить следующие параметры:

  • «Период» - месяц, за который необходимо выгрузить данные;
  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;
  • «Каталог выгрузки» - указывается каталог для выгружаемых файлов в формате *.CSV.


Обработка предназначена для получения данных, предназначенных для выгрузки в следующие файлы:

1. «Файл счетов». Данный файл предназначен для использования в платежной системе «Платон» с целью идентификации лиц, осуществляющих платеж, и привязки платежей к конкретным лицевым счетам.
    Структура текущего файла должна удовлетворять следующему формату:

Номер поля

Наименование поля

1 №
2 Номер лицевого счета плательщика
3 Тип населенного пункта
4 Название населенного пункта
5 Тип улицы
6 Название улицы
7 Номер дома
8 Номер корпуса
9 Номер квартиры
10 Фамилия имя отчество плательщика
11 Имя плательщика
12 Отчество плательщика
13 Общая площадь квартиры/помещения или ее части приходящаяся на данный лицевой счет (кв.м)
14 Жилая площадь квартиры/помещения или ее части приходящаяся на данный лицевой счет (кв.м)
15 Число постоянно проживающих (чел.)
16 Число временно проживающих (чел.)
17 Число временно отсутствующих (чел.)

2. «Файл начислений».  Данный файл предназначен для использования в платежной системе «Платон» с целью загрузки данных о начислениях за указанный период.
    Структура текущего файла должна удовлетворять следующему формату:

Номер поля

Наименование поля

1 №
2 Расчетный месяц
3 Номер лицевого счета плательщика
4 ФИО плательщика
5 Код услуги
6 Наименование услуги
7 Сумма, начисленная  по услуге, с учетом всех льгот и перерасчетов (руб.)
8 Входящее сальдо по услуге или по счету в целом (руб.)
9 Сумма платежей по услуге или по счету в целом (руб.)
10 Сумма начислений по счету в целом (руб.)
11 Исходящее сальдо по услуге или по счету в целом (руб.)
12 Входящее сальдо (задолженность на начало расчетного периода)  по пени  по счету в целом (руб. )
13 Начисленная сумма пени по счету в целом с учетом возможных перерасчетов (руб.)
14 Исходящее сальдо по пени по счету в целом (руб.)
15 Сумма льгот по услуге (руб.)
16 Сумма перерасчетов по услуге (руб.)
17 Сумма субсидий по услуге (руб.)
18 Объем потребленного ресурса по услуге
19 Способ определения расхода (по счетчику, по норме расхода)
20 Единица измерения объема потребленного ресурса по услуге
21 Размер текущего тарифа по услуге (руб.)
22 Начисленная сумма по услуге (руб.)
23 Сумма перерасчета входящего сальдо по услуге или по счету в целом (руб.)
24 Сумма льгот по счету в целом (руб.)
25 Сумма перерасчетов по счету в целом (руб.)
26 Сумма субсидий по счету в целом (руб.)
27 Сумма перерасчета пени (руб.)

3. «Файл счетчиков». Данный файл предназначен для использования в платежной системе «Платон» с целью загрузки данных о индивидуальных приборах учета действовавших в течение указанного месяца.
    Структура текущего файла должна удовлетворять следующему формату:

Номер поля

Наименование поля

1 №
2 Номер лицевого счета плательщика
3 Код услуги
4 Идентификатор (код) счетчика
