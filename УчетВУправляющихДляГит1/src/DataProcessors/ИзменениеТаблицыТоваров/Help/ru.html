Изменение таблицы товаров

Обработка предназначена для выполнения следующих действий:

  • объединение в одном документе несколько табличных частей документов других видов, например, можно оформить документ реализации, объединив в нем табличные части нескольких документов поступления;
  • изменение цены номенклатуры на определенный процент;
  • установка цен по позициям номенклатуры в соответствии с выбранным типом цен;
  • распределение количества или суммы на несколько частей;
  • уменьшение или увеличение точности цен номенклатуры;
  • изменение ставки НДС по всем позициям номенклатуры и т.д.

Обработка вызывается в документах, имеющих табличную часть, по кнопке Изменить. Например, в документе Установка цен номенклатуры, Поступление (акт, накладная).

Объединение в одном документе несколько табличных частей документов различных видов

Выполняется с помощью кнопки Добавить из документа - после нажатия которой происходит копирование содержимого табличной части документов выбранного вида. Возможно объединение в одной табличной части данных документов нескольких различных видов. Например, можно оформить документ реализации, объединив в нем табличные части нескольких документов поступления.

После нажатия кнопки откроется окно, в котором необходимо выбрать вид документа, из которого будут добавлены номенклатурные позиции. После чего открывается журнал документов выбранного вида, в котором возможно указать один или несколько конкретных документов.

Копирование табличной части производится по следующему алгоритму: для каждой номенклатурной позиции в табличной части выбранного документа проверяется - имеется ли такая же номенклатурная позиция с такой же ценой в табличной части исходного документа. Если нет, то она добавляется, если да, то увеличивается количество исходной позиции. Цены для новой позиции устанавливаются в соответствии с тем типом цен, который задан в документе.

Список действий по изменению табличной части

Список действий с выбранными позициями табличной части, выполняемых обработкой:

  • Изменить цены на процент - изменение цены позиции номенклатуры на определенный процент. Величина процента задается в текстовом поле.

    • При изменении цены позиций номенклатуры в процессе переоценки происходит автоматический пересчет суммы по позиции номенклатуры и суммы НДС.

    • Если процент задан с плюсом, то происходит увеличение цены, а если с минусом, то ее уменьшение.
  • Установить цены по типу - цены по отмеченным позициям номенклатуры будут установлены в соответствии с выбранным типом цен. Используется при необходимости для разных позиций в табличной части документа устанавливать разный тип цен. Например, часть позиций в одном документе продается по оптовым ценам, а часть позиций – по мелкооптовым ценам.
  • Распределить сумму по количеству - разделение заданной суммы на несколько частей, чтобы величина каждой части соотносилась с величиной остальных следующим образом: суммы по позициям изменяются пропорционально количеству позиции номенклатуры. При этом цены позиции номенклатуры изменяются на одинаковую величину для единицы измерения каждой позиции, независимо от того, что это за единица.
    Произвольная сумма в валюте документа вводится в текстовом поле.
  • Распределить сумму по суммам - разделение заданной суммы на несколько частей, чтобы величина каждой части соотносилась с величиной остальных следующим образом: суммы по позициям изменяются пропорционально самим суммам. При этом цены по позициям изменяются пропорционально самим ценам.
    Произвольная сумма в валюте документа вводится в текстовом поле.
  • Округлить цены до - уменьшение или увеличение точности цен на позиции номенклатуры. Желаемую точность от 0.01 до 100 можно выбрать в списке по кнопке "…".
  • Установить ставку НДС - изменение ставки НДС. Ставка НДС выбирается из списка. При изменении ставки НДС автоматически пересчитываются суммы НДС и итоговые суммы.

Настройка и выполнение действий

Параметры изменения табличной части задаются выбором действия из списка и вводом соответствующего значения в поле, расположенном рядом с выбранным действием.

Изменение табличной части документа по заданным параметрам без перехода в документ осуществляется по кнопке Выполнить. Таким образом, можно произвести несколько действий по изменению табличной части, не выходя в форму документа. Действия выполняются только по отмеченным флажком позициям.

В верхней части списка расположены иконки позволяющие установить или снять пометку сразу для всех строк табличной части.

Сохранение произведенных изменений выполняется по кнопке Перенести в документ.
