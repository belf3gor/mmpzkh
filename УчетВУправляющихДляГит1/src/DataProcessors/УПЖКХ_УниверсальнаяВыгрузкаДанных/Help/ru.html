Обработка «Универсальная выгрузка данных»

Данная обработка позволяет выгрузить данные по лицевым счетам во внешний файл. Состав выгружаемых данных настраивается в справочнике "Настройки загрузки/выгрузки".

Перед выгрузкой данных необходимо заполнить поля:

  • «Период» - указывается период (месяц и год), за который выгружаются данные.
  • «Организация» - указывается организация из справочника "Организации", по которой выгружаются данные.
  • «Формат выгрузки» - указывается настройка выгрузки из справочника "Настройки загрузки/выгрузки".
  • «Настройки выгрузки» - указывается настройка выгрузки из справочника "Настройки универсальной выгрузки данных".
  • «Имя файла реестра» - указывается имя файла с выгружаемыми данными. Если в имени файла присутствую значения: [ИНН] и/или [Расчетный счет], - то данные сокращения будут заменены на ИНН организации и основной расчетный счет организации.

Обработка доступна в разделе "Интеграция со сторонними системами".
