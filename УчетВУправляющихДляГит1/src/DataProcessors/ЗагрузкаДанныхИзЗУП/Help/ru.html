Обработка предназначена для загрузки данных о расчете заработной платы, предварительно выгруженных из конфигурации "Зарплата и управление персоналом".

В поле Файл данных укажите имя файла, из которого будут загружены данные.

Данные загружаются в текущую информационную базу по кнопке Выполнить.
