Обработка «Интеграция с АСКУЭ «Ресурс» от компании «Bolid»

Данная обработка предназначена для настройки интеграции программного продукта «1С:Учет в управляющих компаниях ЖКХ, ТСЖ и ЖСК» и АСКУЭ «Ресурс» от компании «Bolid».

Обработка предполагает поэтапное ознакомление с механизмом интеграции и включает в себя следующие страницы:

1. Начальная страница - предназначена для ознакомления с механизмами интеграции с АСКУЭ «Ресурс» от компании «Bolid».

2. Страница регистрации - предназначена для создания учетной записи пользователя, необходимой для обмена.

3. Страница проверки сведений в программе - предназначена для проверки сведений в Вашей программе на их готовность к интеграции с сервисом.
Для того, чтобы перейти к организациям, у которых не заполнен ИНН, или к счетчикам, у которых не указан заводской номер, необходимо воспользоваться соответствующими гиперссылками.

4. Страница настроек обмена данными с сервисом - предназначена указания настроек, в соответствии с которыми будет выполняться автоматический обмен с сервисом. Возможны следующие варианты:

  • «Не использовать» - обмен в автоматическом режиме выполняться не будет; 
  • «По умолчанию» - автоматический обмен будет выполняться 25-го числа каждого месяца в 15.00;
  • «Ручная настройка» - расписание автоматического обмена можно задать вручную. 

5. Страница завершения мастера настроек обмена данными с сервисом - предназначена для уведомления о завершении первоначальной настройки интеграции с АСКУЭ «Ресурс» от компании «Bolid». 

6. Страница управления обменами - предназначена для мониторинга выполненных обменов, изменения настроек обмена с сервисом, а так же для выполнения обмена с сервисом вручную. Данная страница содержит следующие разделы:

  • «Об обмене» - содержит краткое описание обмена данными с АСКУЭ «Ресурс» от компании «Bolid». 
  • «Сведения о подписке на пользование сервисом» - содержит информацию о текущем статусе подписки на пользование сервисом.
  • «Настройка обмена» - предназначен для изменения настроек обмена данными с сервисом и содержит гиперссылку «Изменить расписание обмена», по нажатию на которую можно изменить расписания обмена с сервисом.
  • «Управление обменом» - предназначен для мониторинга истории выполнения обмена и обмена данными с АСКУЭ «Ресурс» от компании «Bolid» вручную. Содержит следующие пункты:
    • «Выполнить разовый обмен данными» - по нажатию на кнопку будет выполнен разовый обмен данными с сервисом.
    • «Журнал обмена» - предназначен для просмотра информации об истории обмена.
  • «Справочная информация» - содержит гиперссылки на страницу сайта разработчиков с полезной информацией об обмене.
