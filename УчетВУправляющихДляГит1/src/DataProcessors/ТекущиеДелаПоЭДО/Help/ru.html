Текущие дела ЭДО

Рабочее место позволяет просмотреть незавершенные дела по работе с электронными документами и выполнить требующиеся действия над ними.

Работа с электронными документами разделена по разным группам и зависит от характера незавершенного события. Слева расположены группы возможных действий, которые выполняются с электронными документами, справа — список электронных документов для которых требуется совершить то или иное действие. Для списка электронных документов в форме расположены быстрые отборы.

Режимы работы

Режимы работы переключаются по кнопке Настройка просмотра. Рабочее место может работать в трех различных режимах:

  • К исполнению - отображаются только те документы, с которыми текущий пользователь может совершить соответствующее действие. Скрываются сервисные разделы (Отправить и Распаковать) и разделы, предназначенные для контроля состояния документов.
  • Мои документы - отображаются документы, за которые отвечает (является ответственным) текущий пользователь.
  • Весь документооборот - отображаются все документы по всем разделам без ограничений.

Группы возможных действий

Входящие — документы, поступающие от других организаций (контрагентов). Для группы доступна кнопка Утвердить, подписать и отправить. При нажатии будут сформированы документы, которые необходимо утвердить, подписать и отправить.

  • Отразить в учете — электронные документы, для которых в настройке Способ отражения в учете выбран вариант Вручную в форме электронного документа;
  • Утвердить — электронные документы, которые необходимо утвердить. Для выделенных в списке элементов доступна команда Утвердить;
  • Подписать — электронные документы, которые необходимо подписать. Для выделенных в списке элементов доступна команда Подписать и отправить;
  • Исправить — документы, отправленные организацией, но отклоненные контрагентом по определенным причинам и требуют исправления;
  • Аннулировать — документы, которые по определенным причинам (например, допущена ошибка) требуют аннулирования;
  • На контроле — отправленные электронные документы, для которых ожидаются действия со стороны других участников обмена, например, получателя или оператора ЭДО.

Исходящие — документы, предназначенные для отправки в другие организации (контрагентам).

  • Создать — перечень документов информационной базы, для которых необходимо сформировать электронные документы. Для выделенных в списке элементов доступны команды Создать, подписать и отправить, Изменить документ и Создать электронный документ;
  • Подписать — электронные документы, которые необходимо подписать. В списке элементов доступна команда Подписать и отправить;
  • Исправить — отправленные электронные документы, требующие корректировки, которые отклонил получатель или по какой-либо причине их не удалось отправить; также в этом разделе отображаются документы, которые были отклонены в процессе подписания;
  • Аннулировать — документы, которые по определенным причинам (например, допущена ошибка) требуют аннулирования;
  • На контроле — отправленные электронные документы, для которых ожидаются действия со стороны других участников обмена, например, получателя или оператора ЭДО;

Отправить — подготовленные документы, которые ожидают отправки. Для выделенных в списке элементов доступны команды Отправить и Отменить.

Распаковать — пакеты, которые были получены, но не распознаны программой по неизвестной причине. Для выделенных в списке элементов доступны команды Распаковать и Отменить.

Приглашения —необходимы для настройки обмена между контрагентами.

  • Требуется пригласить — настройка была создана, записана, но не отправлено приглашение контрагенту;
  • Ждем согласия — приглашения, которые были отправлены контрагентам и требуют ответа;
  • Требуется согласие — приглашения, которые поступили от сторонних организаций на подключение к ЭДО.

Ознакомиться — электронные документы, документооборот по которым завершился при выполнении обмена. Например, при получении служебного сообщения. Для выделенных в списке элементов доступна кнопка Отправить в архив, которая окончательно исключает документ из текущих дел. Отображение раздела регулируется в настройках обмена электронными документами.
