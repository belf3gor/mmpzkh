Обработка «Загрузка из конфигурации «Отчетность предпринимателя» ред. 2.0»

Обработка предназначена для загрузки данных учета, предварительно выгруженных из конфигурации "Отчетность предпринимателя", ред 2.0.

В поле "Файл данных" укажите имя файла, из которого будут загружены данные.

Данные загружаются в текущую информационную базу по кнопке "Загрузить данные".
