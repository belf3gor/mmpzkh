Обработка «Мастер обеспечения заявок АДС»

Данная обработка позволяет автоматизировать обеспечение мастеров АДС материалами, требуемыми для выполнения заявок на выполнение работ.

Мастер обеспечения заявок АДС доступен из раздела «Аварийно-диспетчерская служба - Сервис», а также непосредственно из формы наряд-заказа заявки на выполнение работ.

Работа с Мастером обеспечения заявок АДС разбита на 2 этапа:

Этап 1. Подбор заявок:

На данном этапе указываются:

  • «Период обеспечения» - указываются дата начала и дата окончания периода обеспечения. Заявка попадает в период обеспечения, если ее время, удобное заказчику для выполнения работ, входит в интервал обеспечения;
  • «Организация» - организация, для которой выполняются заявки на выполнение работ АДС;
  • «Мастер» - мастер, ответственный за выполнение заявки на выполнение работ;
  • «Склад мастера» - склад, закрепленный за мастером АДС. Предполагается, что за одним мастером закреплен только один склад. Поле недоступно для редактирования и выводится информативно;
  • «Список заявок мастера, которые необходимо обеспечить материалами» - таблица, содержащая список заявок на выполнение работ, удовлетворяющих критериям отбора по периоду обеспечения, организации и мастеру. Таблица заполняется автоматически по нажатию на кнопку «Заполнить список заявок». Из списка отобранных заявок пользователь может указать те, которые необходимо обеспечить материалами, и перейти к следующему этапу.

Для перехода к следующему этапу необходимо нажать на кнопку «Далее».

Этап 2. Обеспечение заявок материалами:

На закладке «Обеспечение заявок материалами» отображаются 4 таблицы:

1. «Материалы, требующиеся мастеру для выполнения его заявок» - в этой таблице отображается список всех материалов, требующихся мастеру для выполнения его заявок. Материалы отсортированы на три группы, выделенные разными цветами:

  • Красным цветом выделены материалы, которых недостаточно на складе мастера и которые невозможно получить с прочих складов, т.к. суммарно на всех складах их недостаточно;

  • Синим цветом выделены материалы, которых недостаточно на складе мастера, но которые возможно восполнить с прочих складов;

  • Зеленым цветом выделены материалы, которых достаточно на складе мастера.

2. «Заявки мастера, для выполнения которых требуется текущий материал» - в этой таблице отображается список заявок мастера, для выполнения которых требуется материал, выделенный в таблице «Материалы, требующиеся мастеру для выполнения его заявок», а также количество материалов «В наличии» на «Складе мастера», которое определяется как остаток количества материалов по дебету счетов 10 «Материалы».

3. «Остатки текущего материала на прочих складах» - в этой таблице отображаются остатки материала на прочих складов для выделенного материала в таблице «Материалы, требующиеся мастеру для выполнения его заявок». Остатки материала «В наличии» на прочих складах определяются как остаток количества материалов по дебету счетов 10 «Материалы» на складах, для которых не назначено материально ответственное лицо.

4. «Заявки мастеров прочих складов, входящих в период обеспечения, на выполнение которых им требуется текущий материал» - в этой таблице отображается список заявок мастера прочего склада, выделенного в таблице «Остатки текущего материала на прочих складах», для выполнения которых ему требуется текущий материал.

Для обеспечения заявок материалами необходимо выполнить перемещение материалов с прочих складов на склад мастера. Это можно сделать вручную в таблице «Остатки текущего материала на прочих складах», заполнив колонку «Получить», но в пределах имеющегося на прочем складе количества материала. Также возможно автоматическое заполнение количества перемещаемых материалов при помощи кнопки «Заполнить недостающие материалы», которая имеет два режима:

  • «Имеющимися свободными остатками» - в данном случае выполняется заполнение колонки «Получить» свободными материалами с прочих складов. При этом материал считается свободным, если он не требуется мастеру прочего склада для выполнения его заявок, входящих в период обеспечения. Для общих складов (у которых не указан ответственный) все материалы считаются свободными. В первую очередь выполняется перемещение материалов с общих складов, а потом уже со складов прочих мастеров.

  • «Только по заявкам, которые можно полностью обеспечить» - в данном случае механизм заполняет колонки «Получить» в рамках заявок мастера по следующему принципу: если хотя бы по одной позиции наряд-заказа заявки невозможно обеспечение материала, то и для всех позиций материалов данной заявки обеспечение выполнено не будет. В случае использования данного метода автоматического заполнения те строки таблицы «Заявки мастера, для выполнения которых требуется текущий материал», заявки которых полностью обеспечить невозможно, будут выделены красным цветом. При этом в случае дальнейшего ручного изменения перемещаемого количества цветовое выделение строк будет снято.

    По нажатию на кнопку «Обновить» выполняется обновление таблицы «Материалы, требующиеся мастеру для выполнения его заявок» и все введенные данные по перемещаемым количествам материалов будут очищены.

    По нажатию на кнопку «Сформировать документы» выполняется формирование документов «Перемещение товаров» в соответствии с введенными количествами перемещаемых материалов. Сформированные документы отображаются на закладке «Документы перемещения».
