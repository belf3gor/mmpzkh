Выгрузки на Интернет-портал ЖКХ

Обработка предназначена для выгрузки данных по начислениям ЖКХ за указанный период на Интернет-порталы ЖКХ различных регионов.

Обработку можно открыть через меню "Учет в ЖКХ / Сервис / Выгрузки на Интернет-портал ЖКХ".

Для каждого региона из списка существует свой формат выгрузки:

  • Чувашская республика

Чувашская республика
На Интернет-портал ЖКХ Чувашской республики выгружаются данные по домам, начислениям, расчетам по домовым приборам учета, платежам, задолженностям. Обработка выгружает данные за выбранный период в формате *.XLS.
