Обработка "Формирование аналитических графиков по данным учета"

Обработка используется для формирования и вывода в табличный документ графиков по лицевым счетам, зданиям и приборам учета.

Данные, показываемые в графиках, собираются за период, указанный в поле "Период" формы обработки.

Общие настройки графиков доступны по нажатию гиперссылки "Настройки графиков".

Формирование графиков

Перед формированием графика необходимо выбрать лицевой счет, здание или прибор учета в поле "Объект". После выбора объекта необходимо в поле "Вид графика" выбрать вид формируемого графика.

По лицевым счетам, выбранным из справочника "Лицевые счета", доступны для формирования следующие виды графиков:

  • «Взаиморасчеты» - выводятся графики, сведения которых содержат следующие данные: начисления и оплаты, процент собираемости оплат, начисления и оплаты пени, процент собираемости оплат по пени, задолженность по месяцам. Отбор сведений по организации в общей информации и графиках осуществляется по выбранной организации в поле «Организация».
  • «Расход по приборам учета» - выводятся графики, сведения которых содержат данные о расходах по приборам учета текущего лицевого счета.
  • «Начисления по льготам» - выводятся графики, сведения которых содержат данные о начислениях и начисленных льготах по текущему лицевому счету либо группе лицевых счетов. Отбор сведений по организации в общей информации и графиках осуществляется по выбранной организации в поле «Организация».

По зданиям, выбранным из справочника "Здания, сооружения", доступны для формирования следующие виды графиков:

  • «Взаиморасчеты» - выводятся графики, сведения которых содержат следующие данные: начисления и оплаты, процент собираемости оплат, начисления и оплаты пени, процент собираемости оплат по пени, задолженность по месяцам. Отбор сведений по организации в общей информации и графиках осуществляется по выбранной организации в поле «Организация».
  • «Расход по приборам учета» - выводятся графики, сведения которых содержат данные о расходе по приборам учета, установленным на текущее здание.

По приборам учета, выбранным из справочника "Счетчики",доступны для формирования следующие виды графиков:

  • «Расход по приборам учета» - выводятся графики, сведения которых содержат данные о расходе по выбранным приборам учета.
