Форма «Настройки графиков»

Данная форма предназначена для настройки графиков в справочниках «Здания, сооружения», «Лицевые счета», «Счетчики».

На форме расположены следующие элементы:

  • «Выводить графики» - настройка, определяющая необходимость формирования графиков.
  • «Виды графиков» - настройки, определяющие виды формирования графиков, доступных пользователю.

Примечание
Данная группа настроек видна при открытии формы из вкладки «Взаиморасчеты» справочника «Здания, сооружения» и из вкладок «Взаиморасчеты» и «Карточка расчетов» справочника «Лицевые счета», а также при открытии из формы обработки «Формирование аналитических графиков по данным учета», когда выбран вид графика «Взаиморасчеты».

  •  График «Расход по показаниям счетчиков» - настройка, определяющая вид формирования данного графика.

Примечание
Данная настройка видна при открытии формы из справочника «Счетчики» и закладки «Счетчики» справочников «Здания, сооружения» и «Лицевые счета», а также при открытии из формы обработки «Формирование аналитических графиков по данным учета», когда выбран вид графика «Расход по приборам учета».

  • «Параметры графика» -  группа настроек графика «Расход по показаниям счетчиков» , состоящая из:   
    • «Услуга для формирования» - настройка определяет, по каким услугам, рассчитываемым по показаниям приборов учета, необходимо формировать график "Расход по показаниям счетчиков". Если в данной настройке услуга задана, то график формируется только по услуге, указанной в настройке. Если настройка не заполнена, то график будет сформирован для всех услуг, имеющих расход по приборам учета.
    • «Способ группировки данных по услугам» - настройка предназначена для установки способа группировки данных по услугам. Доступна, когда настройка «Услуга для формирования» не заполнена. Данная настройка может принимать следующие значения:
      • «По каждой услуге формировать отдельный график» - при выборе настройки независимо от единицы измерения услуг для каждой услуги будет сформирован отдельный график.
      • «Услуги одной единицы измерения в одном графике» - при выборе данной настройки данные по услугам с одной единицей измерения будут включены в один общий график.

Примечание
Данная группа настроек видна при открытии формы из справочника «Счетчики» и из вкладки «Счетчики» справочников «Здания, сооружения» и «Лицевые счета», а также при открытии из формы обработки «Формирование аналитических графиков по данным учета», когда в качество объекта выбран элемент справочников «Здания, сооружения» и «Лицевые счета» и выбран вид графика «Расход по приборам учета». 

  • График «Начисления по льготам» - настройка, определяющая вид формирования данного графика.

Примечание
Данная настройка видна при открытии формы из вкладки «Льготники» справочника «Лицевые счета», а также при открытии из формы обработки «Формирование аналитических графиков по данным учета», когда выбран вид графика «Начисления по льготам».

Внизу формы расположена кнопка «Сохранить и закрыть», при нажатии на которую настройки графиков сохраняются, а форма закрывается.
