Обработка "Выгрузка данных лицевых счетов для сайта ЖКХ"

Обработка предназначена для формирования файла экспорта данных в формате XML, включающего информацию по начислению и оплате за услуги ЖКХ.

Перед выполнением выгрузки указываются:

  • «Организация» – организация, по которой выгружаются данные;
  • «Период» – период выгружаемых данных;
  • «Файл данных» – выбирается файл для выгрузки данных в формате XML и путь для его сохранения;

На панели инструментов табличной части расположены кнопки:

  • «Заполнить» – при нажатии данной кнопки заполняется табличная часть формы списком лицевых счетов для выгрузки в зависимости от выбранной организации и периода.
  • «Очистить» – при нажатии данной кнопки табличная часть формы очищается.

В нижней части диалогового окна формы расположены кнопки:

  • «Выгрузить» – при нажатии данной кнопки будет сформирован файл экспорта в формате XML.

Отчет открывается через меню «Учет в ЖКХ / Сервис / Выгрузка в Web-приложение».
