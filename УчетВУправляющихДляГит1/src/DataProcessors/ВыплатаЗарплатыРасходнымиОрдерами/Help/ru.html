Выплата зарплаты расходными ордерами

Обработка предназначена для автоматического формирования кассовых документов для выдачи зарплаты работникам.

В поле Платежная ведомость следует указать документ, на основании которого выплачивается зарплата. Табличная часть заполнится автоматически.

Сформировать список кассовых документов по отмеченным работникам можно по кнопке Создать документы.

Провести созданные документы Выдача наличных можно по кнопке Провести документы.

По одной ведомости создавать и проводить кассовые документы можно в разное время. В этом случае уже созданные по документу кассовые документы будут автоматически найдены и заполнены в списке.
