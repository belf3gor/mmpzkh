Подбор основных средств

Обработка предназначена для подбора позиций основных средств в табличную часть документа.

Обработка вызывается в документах, имеющих табличную часть, по кнопке Подбор. Подбор осуществляется из справочника ОсновныеСредства, открывающегося в режиме подбора.

 

   В верхней части окна располагается строчка поиска основных средств. Она используется для фильтрации по наименованию. В случае когда содержимое строчки не пустое в окно списка основных средств (правая часть формы) попадут только те основные средства, в наименовании которых содержится эта строчка. В левой части формы находится структура справочника основные средства в виде дерева. Выбирая элемент дерева можно фильтровать какие элементы основных средств попадут в список основных средств. В правой части окна находится список основных средств. Их можно переносить в список подбора с помощью двойного нажатия на левую кнопку мыши, а также с помощью клавиши Enter. В нижней части окна располагается список основных средств, которые должны быть перенесены в документ. Удалить элемент из списка можно вызвав его контекстное меню по правой кнопке мыши и выбрав в нем элемент "Удалить". Перенести выбранные позиции основных средств в табличную часть документа можно нажав на кнопку "Перенести в документ" в левой верхней части формы.
