Обработка «Обмен данными с ГИС ЖКХ»

Данная обработка предназначена для формирования и чтения файлов в формате «xlsx», которые используются для обмена данными с ГИС ЖКХ.


Вид обмена: «Сведения о зданиях»

Выгрузка сведений о зданиях


Перед выгрузкой сведений о зданиях необходимо указать следующие настройки:

  • «Выгружать общие сведения о зданиях» - флаг, символизирующий необходимость выгрузки сведений о зданиях;
  • «Выгружать сведения о договорах управления» - флаг, символизирующий необходимость выгрузки договоров управления зданиями;
  • «Выгружать способы формирования фонда кап. ремонта» - флаг, символизирующий необходимость выгрузки сведений о выбранном способе формирования фонда кап. ремонта;
  • «Здания» - список зданий из справочника «Здания», сведения по которым необходимо выгрузить.


При выгрузке сведений о зданиях сведения группируются по типам зданий: МКД или жилые дома.
Для МКД формируется файл с наименованием «Сведения о МКД от [дата формирования файла].xlsx».
Для жилых домов формируется файл с наименованием «Сведения о ЖД от [дата формирования файла].xlsx».
Так как в ГИС ЖКХ имеется ограничение на размер файлов загрузки, файлы большого размера разбиваются на несколько более маленьких файлов.

При выгрузке сведений о договорах управления зданиями формируется один файл с наименованием «Сведения о договорах управления от [дата формирования файла].xlsx».

Загрузка идентификаторов зданий и помещений


Имеется возможность выполнять загрузку уникальных номеров жилых домов/помещений/комнат. Для этого для вида обмена «Сведения о зданиях» на закладке «Загрузка идентификаторов зданий и помещений» в поле «Каталог поиска» необходимо указать каталог, в котором содержится файл (файлы) экспорта зданий и помещений из ГИС ЖКХ. Для корректного сопоставления данных в информационной базе должны быть заполнены коды по ФИАС для соответствующих зданий.

По нажатию на кнопку «Поиск идентификаторов» производится поиск и чтение файлов экспорта, а по данным файла заполняется таблица идентификаторов. По нажатию на кнопку «Записать идентификаторы» в информационную базу заносятся данные по идентификаторам объектов на основании сведений таблицы.

Вид обмена: «Сведения о лицевых счетах»

Выгрузка сведений о лицевых счетов

Перед выгрузкой сведений необходимо заполнить таблицу с лицевыми счетами, которые необходимо выгрузить. Для этого следует воспользоваться подменю «Заполнить л/с»:

  • «Всеми невыгруженными л/с» - таблица заполняется всеми лицевыми счетами, которые ранее не были выгружены. Сведения о выгруженных лицевых счетах сохраняются в регистре сведений «Объекты, выгруженные в ГИС ЖКХ»;
  • «Вручную» - будет открыта форма выбора лицевых счетов из справочника «Лицевые счета»;
  • «Автоматически с отбором» - будет открыта специальная форма подбора лицевых счетов по заданным критериям.


При выгрузке сведений о лицевых счетах формируется один файл с наименованием «Сведения о лицевых счетах от [дата формирования файла].xlsx»

ВАЖНО:
В случае, когда организация ведет раздельный учет по услугам капитального ремонта, при выгрузке лицевых счетов необходимо указать Тип лицевого счета, установив переключатель в одно из значений «Выгружать лицевые счета капитального ремонта» или «Выгружать лицевые счета управляющей организации»:
- Если лицевые счета созданы только для начисления услуг капитального ремонта (КР), то выгрузку данных по таким лицевым счетам необходимо выполнять с установленной настройкой "Выгружать лицевые счета капитального ремонта";
- Если на лицевые счета начисления по КР не производятся или если начисления по КР производятся, но при этом печатается единая квитанция на основные услуги и на услуги кап. ремонта, то выгрузку данных по таким лицевым счетам необходимо выполнять с установленной настройкой "Выгружать лицевые счета управляющей организации;
- Если для начисления услуг капитального ремонта отдельные лицевые счета не создаются, но при этом печатаются отдельные платежные документы по КР, то выгрузку таких лицевых счетов необходимо выполнить дважды - с установленной настройкой "Выгружать лицевые счета управляющей организации", а затем повторно с установленной настройкой "Выгружать лицевые счета капитального ремонта". При этом в ГИС ЖКХ будут созданы 2 лицевых счета с одинаковым наименованием, но разных типов и с разными идентификаторами ЕЛС и ЖКУ.

Загрузка идентификаторов лицевых счетов

Имеется возможность выполнять загрузку уникальных номеров ЕЛС и идентификаторов ЖКУ лицевых счетов. Для этого для вида выгрузки «Сведения о лицевых счетах» на закладке «Загрузка идентификаторов» в поле «Каталог поиска» необходимо указать каталог, в котором содержится файл (файлы) экспорта ЕЛС из ГИС ЖКХ. Для корректного сопоставления данных в информационной базе должны быть заполнены идентификаторы в ГИС ЖКХ для соответствующих помещений и комнат, а для жилых домов - уникальные номера в ГИС ЖКХ.

По нажатию на кнопку «Поиск идентификаторов» производится поиск и чтение файлов экспорта, а по данным файла заполняется таблица идентификаторов. По нажатию на кнопку «Записать идентификаторы» в информационную базу заносятся данные по идентификаторам объектов на основании сведений таблицы.

ВАЖНО:
В случае, когда организация ведет раздельный учет по услугам капитального ремонта, при загрузке лицевых счетов также необходимо указать Тип лицевого счета, установив переключатель в одно из значений «Загружать идентификаторы лицевых счетов капитального ремонта» или «Загружать идентификаторы лицевых счетов управляющей организации»:
- Если лицевые счета созданы только для начисления услуг капитального ремонта (КР), то загрузку идентификаторов по таким лицевым счетам необходимо выполнять с установленной настройкой "Загружать идентификаторы лицевых счетов капитального ремонта";
- Если на лицевые счета начисления по КР не производятся или если начисления по КР производятся, но при этом печатается единая квитанция на основные услуги и на услуги кап. ремонта, то загрузку идентификаторов по таким лицевым счетам необходимо выполнять с установленной настройкой "Загружать идентификаторы лицевых счетов управляющей организации";
- Если для начисления услуг капитального ремонта отдельные лицевые счета не создаются, но при этом печатаются отдельные платежные документы по КР, то загрузку идентификаторов таких лицевых счетов необходимо выполнить дважды. С установленной настройкой "Загружать идентификаторы лицевых счетов управляющей организации", выгрузив идентификаторы из раздела "Оплата за жилое помещение и коммунальные услуги" на портале ГИС ЖКХ, а затем с установленной настройкой "Загружать идентификаторы лицевых счетов капитального ремонта", выгрузив идентификаторы из раздела "Оплата взноса на капитальный ремонт" на портале ГИС ЖКХ.

Вид обмена: «Платежные документы»

Выгрузка сведений о платежных документах

Перед выгрузкой необходимо выбрать файл шаблона, предварительно выгруженный из ГИС ЖКХ. В данном файле обязательно должен быть заполнен лист «Услуги исполнителя».

Также перед выгрузкой сведений необходимо заполнить таблицу лицевыми счетами, сведения о начислениях которых необходимо выгрузить. Для этого следует воспользоваться меню «Заполнить л/с»:

  • «Всеми невыгруженными л/с» - таблица заполняется всеми лицевыми счетами, которые ранее не были выгружены. Сведения о выгруженных лицевых счетах сохраняются в регистре сведений «Платежные документы, выгруженные в ГИС ЖКХ»;
  • «Вручную» - будет открыта форма выбора лицевых счетов из справочника «Лицевые счета»;
  • «Автоматически с отбором» - будет открыта специальная форма подбора лицевых счетов по заданным критериям.


При выгрузке сведений о лицевых счетах формируется один файл с наименованием «Сведения о платежных документах от [дата формирования файла].xlsx»

Загрузка идентификаторов платежных документов

Перед загрузкой идентификаторов все файлы проектов либо размещенных платежных документов, выгруженные и принятые в ГИС ЖКХ, должны быть загружены из ГИС ЖКХ и помещены в отдельный каталог. Данный каталог выбирается в поле «Каталог поиска».

После этого по команде «Выполнить поиск» в указанном каталоге производится поиск идентификаторов принятых платежных документов, а по команде «Сохранить идентификаторы» найденные идентификаторы записываются в регистр сведений «Платежные документы, выгруженные в ГИС ЖКХ» для соответствующего лицевого счета и периода выгрузки.

Отзыв и удаление проектов

По команде «Выбрать платежные документы» открывается форма выбора регистра сведений «Платежные документы, выгруженные в ГИС ЖКХ», в которой можно выбрать платежные документы за определенный месяц с заполненными (загруженными) идентификаторами.

После выбора платежных документов по команде «Сформировать файл отзыва» формируется файл отзыва платежных документов с наименованием «Сведения об отзыве платежных документов от [дата формирования файла].xlsx» для последующей передачи в ГИС ЖКХ.

Вид обмена: «Приборы учета»

Выгрузка / загрузка сведений о приборах учета

Перед выгрузкой необходимо выбрать счетчики, сведения по которым необходимо выгрузить. Для этого следует воспользоваться подменю «Добавить приборы учета»:

  • «из справочника «Счетчики» - подбор счетчиков будет осуществляться путем их выбора из соответствующего справочника;
  • «по объектам установки» - подбор счетчиков будет осуществляться путем выбора объектов, на которые установлены счетчики;
  • «все приборы учета, не выгруженные в ГИС ЖКХ» - таблица заполняется всеми счетчиками, которые ранее не выгружали в ГИС ЖКХ. Сведения о выгруженных счетчиках сохраняются в регистре сведений «Объекты, выгруженные в ГИС ЖКХ».

ВАЖНО:
В соответствии с приказом Министерства связи и массовых коммуникаций РФ и Министерства строительства и жилищно-коммунального хозяйства РФ от 29 февраля 2016 г. № 74/114/пр "Об утверждении состава, сроков и периодичности размещения информации поставщиками информации в государственной информационной системе жилищно-коммунального хозяйства" информация о датчиках температуры для индивидуальных приборов учета не заполняется. И соответственно, она не указывается ни в шаблоне, ни в личном кабинете.
В случае, даже если на индивидуальный прибор учета установлен температурный датчик, либо датчик давления, в выгрузку данная информация попадать не будет и графы "Наличие датчиков температуры" и "Наличие датчиков давления" на листе "Сведения о ПУ" будут заполнены значением "Нет".

Загрузка номеров ПУ из ГИС ЖКХ

Для выгрузки показаний приборов учета в ГИС ЖКХ у счетчиков должен быть заполнен номер, присвоенный на стороне ГИС ЖКХ при выгрузке сведений о приборах учета. Номера ПУ в ГИС ЖКХ могут быть загружены автоматически на закладке «Загрузка номеров ПУ в ГИС ЖКХ».

Для этого необходимо:

  1. Выгрузить из ГИС ЖКХ файл экспорта сведений о приборах учета, сохранить в каталог на диске.
  2. Выбрать данный каталог в поле «Каталог загрузки номеров ПУ».
  3. По нажатию кнопки «Найти номера приборов учета» в табличную часть будут заполнены все найденные приборы учета из справочника «Счетчики» с их номерами в ГИС ЖКХ.
  4. По нажатию кнопки «Записать номера приборов учета» данные номера записываются в соответствующие элементы справочника «Счетчики».

Выгрузка показаний приборов учета

Могут быть выгружены показания индивидуальных приборов учета (вид обмена ПУ - «Выгрузка показаний ИПУ») либо показания общедомовых приборов учета (вид обмена ПУ - «Выгрузка показаний ОДПУ»). Перед выгрузкой необходимо выбрать счетчики, показания которых нужно выгрузить. Для этого следует воспользоваться меню «Добавить приборы учета»:

  • «из справочника «Счетчики» - подбор счетчиков будет осуществляться путем их выбора из соответствующего справочника;
  • «по объектам установки» - подбор счетчиков будет осуществляться путем выбора объектов, на которые установлены счетчики;
  • «все приборы учета, имеющие показания» - таблица заполняется всеми счетчиками, по которым на дату выгрузки имеются введенные показания не выгружали в ГИС ЖКХ.


Вид обмена: «Состояние расчетов с РСО»


Перед выгрузкой необходимо выбрать договоры с РСО - ссылки на документы «Управление договорами с поставщиками услуг».

Суммы оплаты в РСО формируются по данным регистра бухгалтерии «Хозрасчетный»: выбираются дебетовые обороты с отбором по контрагенту (РСО) и его договору, указанным в документе «Управление договорами с поставщиками услуг».

Вид обмена: «Договоры ДПОИ»


Перед выгрузкой необходимо выбрать договоры ДПОИ - ссылки на элементы справочника «Договоры ДПОИ».

Вид обмена: «Выгрузка работ и услуг организации»

Перечни работ и услуг, Планы работ и услуг

Перед выгрузкой необходимо указать:

  • «Путь к файлу шаблона» - шаблон файла выгрузки (скачивается из личного кабинета ГИС ЖКХ).
  • «Перечень работ и услуг» - элемент справочника «Перечни работ и услуг».

Планы работ и услуг формируются по данным документов «Учет планов работ и услуг».

Факты работ и услуг

Перед выгрузкой необходимо указать:

  • «Путь к файлу шаблона» - шаблон файла выгрузки (скачивается из личного кабинета ГИС ЖКХ).
  • «Перечень работ и услуг» - элемент справочника «Перечни работ и услуг».
  • «Месяц выгрузки фактов работ и услуг» - месяц, за который необходимо выгрузить сведения о фактах.

Факты работ и услуг формируются по данным документов «Учет фактов работ и услуг».

Вид обмена: «Выгрузка сведений товарищества, кооператива»

Сведения об отчетах ТСЖ

Перед выгрузкой необходимо выбрать отчеты товарищества, кооператива - ссылки на документы «Смета ТСЖ». Это можно сделать либо вручную c помощью кнопки «Добавить», либо при помощи кнопки «Заполнить», указав предварительно организацию и период отчета ТСЖ.

Сведения о членах ТСЖ

Выгружаются все члены ТСЖ, действующие на конец указанного месяца. Члены ТСЖ фиксируются документом «Вступление в ТСЖ». Роли членов ТСЖ - документом «Состав правления ТСЖ и ревизионной комиссии». 

Вид обмена: «Выгрузка оплат»


Перед выгрузкой необходимо указать месяц, за который следует выгрузить оплаты.


Обработка доступна в разделе командного интерфейса «Интеграция с ГИС ЖКХ».

ВНИМАНИЕ: Для выгрузки данных на рабочем месте должен быть установлен MS Office (2007 SP3, 2010, 2013) с полной версией MS Excel.
