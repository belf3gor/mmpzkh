Настройки обмена данными с ГИС ЖКХ

Данная форма предназначена для настройки обмена данными с ГИС ЖКХ.

Общие настройки


«Идентифицировать лицевые счета по» - способ идентификации лицевых счетов, используемый при некоторых видах обмена с ГИС ЖКХ. Возможно указать следующие способы идентификации лицевых счетов: «по наименованию», «по наименованию без префикса», «по идентификатору».

При формировании файла импорта сведений о лицевых счетах в колонку «Номер ЛС (иной идентификатор потребителя)» будет выгружено, соответственно, либо полное наименование лицевого счета, либо наименование лицевого счета без префикса «л/с №», либо идентификатор лицевого счета.
При загрузке идентификаторов ЕЛС и ЖКУ поиск лицевых счетов в базе будет осуществляться в соответствие с данной настройкой.
При формировании файла импорта сведений о приборах учета в колонку «Номер лицевого счета/Единый лицевой счет/Идентификатор ЖКУ» будет выгружаться:
- Идентификатор ЖКУ, если он заполнен;
- ЕЛС, если не заполнен Идентификатор ЖКУ и заполнен ЕЛС;
- если Идентификатор ЖКУ и ЕЛС не заполнены, то идентификация лицевых счетов будет осуществляться в соответствие с данной настройкой.


«Доступные виды обмена» - определяет перечень видов обмена, доступный для текущей организации, в соответствии с её типом, указанным в справочнике «Организации». При выборе значения «Соответствующие типу организации» будут доступны только те виды обмена, которые соответствует текущей организации. При выборе значения «Все возможные» - все имеющиеся виды обмена.

«Каталог выгрузки файлов» - предназначена для выбора каталога, в который будут сохранены выгруженные файлы.

Сведения о зданиях


«Не выгружать сведения о комнатах для жилых зданий» - определяет признак того, что не нужно заполнять закладку "Комнаты" файла выгрузки жилых домов. Это может потребоваться в тех случаях, когда учет помещений для жилыъ домов в программе не ведется, но создаются фиктивные помещения для того, чтобы открыть на них лицевые счета.

Платежные документы


«Настройка сбора данных для платежных документов» - предназначено для выбора настройки сбора данных, в соответствии с которой будет произведена выгрузка данных для вида обмена «Платежные документы».
