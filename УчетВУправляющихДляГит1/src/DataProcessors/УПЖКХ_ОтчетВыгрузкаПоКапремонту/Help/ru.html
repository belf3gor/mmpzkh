Отчет-выгрузка по услуге капитального ремонта

Отчет предназначен для выгрузки  данных по начислениям и оплатам по услугам капитального ремонта за отчетный период в файл формата "CSV".

Перед формированием отчета-выгрузки необходимо заполнить следующие поля:

  • «Вариант отчета» – предназначен для выбора варианта формирования отчета. Доступные значения:
    • "Отчет по начислению взносов" - при выборе которого в отчет-выгрузку попадают данные о начислениях по услугам капитального ремонта;
    • "Отчет по оплате взносов" - при выборе которого в отчет-выгрузку попадают данные об оплатах по услугам капитального ремонта;
    • "Отчет по начислению и оплате взносов (Вариант1)" - при выборе которого в отчет-выгрузку попадают данные о начислениях и оплатах по услугам капитального ремонта по Варианту 1.
    • "Отчет по начислению и оплате взносов (Вариант2)" - при выборе которого в отчет-выгрузку попадают данные о начислениях и оплатах по услугам капитального ремонта по Варианту 2.
  • «Организация» – организация, оказывающая услуги;
  • «Период» – предназначен для выбора периода выгрузки;
  • «Ответственный» – предназначен для выбора ответственного из справочника "Физические лица". Поле доступно для варианта отчета "Отчет по начислению и оплате взносов (Вариант2)".
  • «Путь к файлу» – полный путь к файлу выгрузки.


В верхней части формы расположена кнопка «Настройки», по нажатию на которую открывается форма настроек отчета-выгрузки, на которой расположены следующие поля: 

  • «Характеристика» – выбирается элемент плана видов характеристик "Дополнительные реквизиты", в котором хранятся коды зданий;
  • «В качестве идентификатора л/с выводить» – список доступных значений: "Наименование", "Идентификатор". Если выбрано значение "Наименование", то в поле "Номер лицевого счета" таблицы отчета будет выведено наименование лицевого счета, если "Идентификатор" - будет выведен идентификатор лицевого счета.
  • «Услуги кап. ремонта» – по нажатию на гиперссылку открывается таблица "Услуги капитального ремонта", предназначенная для выбора услуг, относящихся к капитальному ремонту.


На форме расположена таблица «Таблица отчета», которая содержит данные по начислениям и оплатам по услугам капитального ремонта по состоянию на конец отчетного периода. Таблица заполняется нажатием на кнопку «Заполнить», расположенную на командной панели таблицы "Таблица отчета".

В нижней части формы расположены кнопки:

  • «Сформировать отчет», по нажатию на которую формируется печатная форма с данными по услугам капитального ремонта на основе данных, имеющихся в таблице "Таблицы отчета".
  • «Выгрузить данные», по нажатию на которую данные по услугам капитального ремонта выгружаются в файл формата ".CSV".
  • «Закрыть», по нажатию на которую форма закрывается.


Файл выгрузки представляет собой текстовый файл формата "CSV", в котором данные разделены символом ";" (точка с запятой), и данные по каждому лицевому счету выводятся с новой строки.

Отчет-выгрузка доступен на панели разделов "Раскрытие информации", команда "Сервис".
