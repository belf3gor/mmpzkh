Дополнительная информация, ЖКХ

Форма предназначена для показа справочной, нормативной и другой информации, поставляемой в составе конфигурации.  

Для открытия формы используйте команду "Справка - Дополнительная информация ЖКХ" главного меню. При открытии сразу показывается одна из страниц дополнительной информации. Выбор страницы к показу при открытии формы выполняется программой произвольным образом.

В командной панели формы размещены навигационные кнопки, с помощью которых осуществляется переход по уже просмотренным страницам, а также пункты меню, соответствующие названиям разделов дополнительной информации. При выборе в командной панели раздела открывается список страниц дополнительной информации, относящихся к данному разделу. Страница показывается в форме при выборе пункта меню с названием страницы.

Текст страницы может содержать ссылки на другие страницы дополнительной информации (внутренние ссылки), а также ссылки на разделы сайтов, размещенных в сети Интернет (внешние ссылки). Для перехода по внешним ссылкам необходимо, чтобы компьютер имел доступ в сеть Интернет.

Показ отдельных страниц дополнительной информации может быть настроен поставщиком конфигурации таким образом, чтобы эти страницы показывались в режиме рабочего стола при начале работы с программой. Если таких страниц несколько, то выбор страницы к показу выполняется программой произвольным образом. Если форм для показа в режиме рабочего стола нет, то открывается обычная форма, содержащая командную панель с перечнем разделов и страниц, и показывается одна из имеющихся страниц.

Для пользователей конфигураций ПРОФ-версий доступна настройка показа дополнительной информации при начале работы программы. Если дополнительная информация показывается в режиме рабочего стола, то в командной панели размещается флажок "Показывать при начале работы программы". Если снять его установку, то дополнительная информация не будет показываться при начале работы. Для включения показа при начале работы откройте данную форму и установите режим показа с помощью пункта "Настройка - Показывать при начале работы". Данный пункт отсутствует, если в текущем сеансе уже осуществляется показ дополнительной информации в режиме рабочего окна. В этом случае нужно выполнить настройку с помощью флажка "Показывать при начале работы программы", расположенном в командной панели рабочего стола.

Для пользователей базовых версий конфигураций показ дополнительной информации при начале работы программы выполняется всегда.
