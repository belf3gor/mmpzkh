Оплата поставщикам

Помощник предназначен для формирования списка платежных поручений на оплату поставщикам.

В шапке обработки указывается:

  • Организация - организация, для которой будет заполнен список планируемых платежей. 
  • Банковский счет - счет организации, с которого производится оплата.

В список попадают задолженности по документам расчетов: "Поступление (акты, накладные)", "Поступление доп. расходов", "Поступление НМА".
При этом анализируются остатки по счетам расчетов, которые можно настроить по гиперссылке "Настройка счетов учета расчетов с поставщиками".

По каждой задолженности (для каждого документа) указан срок планируемой оплаты, список представлен в хронологическом порядке.

Оплата по счету

Также в обработку включена планируемая оплата поставщикам по документам "Счет поставщика".
Счета имеют статус "Не оплачен" или "Оплачен частично".

Если есть документы расчетов связанные со счетом поставщика, тогда они отображаются вместе со счетом поставщика, одной строкой.
Оплата в таком случае производится по счету, сумма документов расчетов недоступна для редактирования и отображается серым цветом.

Формирование оплаты

Строки, по которым нужно сформировать платежные поручения, нужно отметить флажком.

Платежные документы формируются автоматически по кнопке "Создать платежные поручения".

После выполнения команды открывается список, содержащий сформированные документы.
