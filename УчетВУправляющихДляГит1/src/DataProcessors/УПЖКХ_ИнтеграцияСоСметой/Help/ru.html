Обработка «Интеграция с 1С:Смета»

Данная обработка предназначена для  интеграции с системой "1С: Смета": выгрузка данных о зданиях и загрузка данных по расчету тарифов.

Обмен происходит через файлы формата XML.

Для выгрузки данных о зданиях необходимо заполнить поля:

  • «Дата выгрузки» - дата, на которую выгружаются данные;
  • «Путь к файлу данных» - путь к файлу, в который будут выгружены данные;

Для загрузки данных по расчету тарифов необходимо заполнить поля:

  • «Путь к файлу данных» - путь к файлу, из которого будут загружены данные;
  • «Организация» - организация, производящая расчет коммунальных услуг;
  • «Услуга» - услуга, по которой загружаются данные;
  • «Тип площади» - тип площади здания.
