Обработка «Интеграция с голосовыми сервисами»

Данная обработка предназначена для интеграции между программным продуктом «1С:Учет в управляющих компаниях ЖКХ, ТСЖ и ЖСК» и сервисами автообзвона должников и автоматического приема показаний.

Обработка предполагает поэтапное ознакомление с механизмом интеграции и его настройку и включает в себя следующие страницы:

  1. Начальная страница - предназначена для ознакомления с механизмами интеграции с сервисами автообзвона должников и автоматического приема показаний.
  2. Страница настроек подключения к сервису - предназначена для указания настроек подключения к сервису, полученных при приобретении сервиса.
  3. Страница настроек обмена данными с сервисом - предназначена указания настроек, в соответствии с которыми будет выполняться обмен с сервисом. Данная страница содержит следующие разделы:
    • «Настройте расписание выгрузок данных о лицевых счетах на сервис» - предназначен для настройки расписания выгрузок данных о лицевых счетах на сервис. Доступны следующие варианты настройки:
      • «Не использовать» - при данной настройке, выгрузка по расписанию производиться не будет.
      • «По умолчанию» - при данной настройке, выгрузка будет производиться с 16.00 один раз в день.
      • «Ручная настройка» - при данной настройке, расписание выгрузки можно настроить вручную. Вы можете настроить расписание выгрузки по дням, неделям, месяцам или указать конкретную дату выгрузки, указав количество дней для повторения.
    • «Настройте расписание загрузок данных из сервиса» - предназначен для настройки расписания загрузок данных о наиболее актуальных показаниях счетчиков с сервиса. Доступны следующие варианты настройки:
      • «Не использовать» - при данной настройке, загрузка по расписанию производиться не будет.
      • «По умолчанию» - при данной настройке, загрузка будет производиться с 15.00 один раз в день.
      • «Ручная настройка» - при данной настройке, расписание загрузки можно настроить вручную. Вы можете настроить расписание загрузки по дням, неделям, месяцам или указать конкретную дату загрузки, указав количество дней для повторения.
    • «Настройте параметры автообзвона должников» - предназначен для настройки критериев, по которым будет выполняться отбор должников для автообзвона. Доступны следующие критерии:
      • «Сумма задолженности» - сумма задолженности, при достижении которой, будет производиться автообзвон должника.
      • «Количество месяцев задолженности» - минимальное количество месяцев задолженности, при достижении которого, будет производиться автообзвон должника.

    • ВАЖНО: Наличие тех или иных разделов настроек определяется оплаченным видом подписки на сервисы.

  4. Страница завершения мастера настроек обмена данными с сервисом - предназначена для уведомления о завершении первоначальной настройки интеграции с сервисами автообзвона должников и автоматического приема показаний. Возврат на предыдущие страницы более не будет возможен, изменить настройки можно будет на Странице управления обменами.
  5. Страница управления обменами - предназначена для мониторинга статусов подписки и выполненных обменов, изменения настроек обмена с сервисами автообзвона должников и автоматического приема показаний, а так же для выполнения обменов с сервисами вручную. Данная страница содержит следующие разделы:
    • «О сервисе» - содержит подробное описание работы сервиса автообзвона и приема показаний.
    • «Статус активности сервисов» - предназначен для отображения информации об активности сервисов. Каждый сервис может иметь один из следующих статусов:
      • «Сервис активен» - обозначает, что сервис активен и доступен для работы с ним.
      • «Сервис не активен» - обозначает, что сервис не активен и не доступен для работы с ним. По ссылке «Как активировать?» Вы можете посмотреть подробное описание сервиса и его стоимости и оправить запрос на активацию сервиса.
      • «Сервис не оплачен» - обозначает, что сервис не оплачен и для его активации необходимо произвести оплату.
    • «Настройка обменов» - предназначен для изменения настроек обмена данными с сервисами и содержит следующие пункты:
      • «Изменить настройки подключения» - позволяет изменить параметры подключения, предоставленные при приобретении сервиса: логин, пароль и сайт.
      • «Изменить расписания обменов» - позволяет изменить расписания обменов с сервисами автообзвона должников и автоматического приема показаний по аналогии с разделами «Настройте расписание выгрузок данных о лицевых счетах на сервис» и «Настройте расписание загрузок данных из сервиса» Страницы настроек обмена данными с сервисом.
      • «Настройка параметров автообзвона должников» - позволяет изменить настройки критериев, по которым будет выполняться отбор должников по аналогии с разделом «Настройте параметры автообзвона должников» Страницы настроек обмена данными с сервисом.
    • «Управление обменами» - предназначен для мониторинга истории выполнения обменов и обмена данными с сервисами автообзвона должников и автоматического приема показаний вручную. Содержит следующие пункты:
      • «Разовые обмены данными» - предназначен для разовой выгрузки или загрузки данных. При этом в рамках выгрузки данных на сервис возможен выбор лицевых счетов, по которым необходимо выполнить выгрузку данных.
      • «Журнал обмена» - предназначен для просмотра информации об истории обменов. Здесь указывается информация о виде обмена, дате обмена, статусе выполнения обмена, варианте запуска обмена (по расписанию или вручную), комментарий о результате выполнения обмена, а так же имеется возможность открытия файла обмена.
    • «Информирование жильцов о возможности передачи показаний по телефону» - предназначен для информирования жильцов о возможности передачи показаний счетчиков по телефону и содержит следующий пункт:
      • «Печать листовки» - предназначен для печати информационной листовки, уведомляющей о наличии возможности передачи показаний по телефону. Листовки предназначены для размещения в подъездах или для индивидуальной рассылки жильцам. При этом можно указать номер телефона, заполнить «подвал» листовки и сформировать ее. В нижней части отображается сформированная листовка. По клавише «Печать», будет осуществлена печать листовки.
    • «Справочная информация» - содержит гиперссылки на страницы с подробным описанием сервиса приема показаний и сервиса автообзвона должников.
