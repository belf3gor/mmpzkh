Обработка «Мастер начислений»

Данная обработка позволяет автоматизировать операции начисления услуг, начисления пени и закрытия периода по начислениям ЖКХ.

Мастер доступен из раздела "Начисление услуг" -> "Сервис".

 

Работа с Мастером начислений разбита на 3 этапа:

Этап 1. Общие настройки:

Раздел «Общие настройки» - в данном разделе указываются:

 

  • «Месяц начисления» - указывается месяц, за который необходимо выполнить операции начисления;
  • «Организация» - организация, для которой выполняются операции начисления;
  • «Ответственный» - пользователь, ответственный за выполнение операций начисления. 

    • Раздел «Настройки начислений» - в данном разделе указываются:

  • «Начислять пени» - если данный флаг установлен, то после завершения операции начисления услуг будет выполнено начисление пени;
  • «Закрыть период по начислениям ЖКХ» - если данный флаг установлен, то после завершения операции начисления услуг (в случае если установлен флаг «Начислять пени», то после завершения операции начисления пени) будет выполнено закрытие периода по начислениям ЖКХ. 

    • Раздел «Настройки многопоточности» - данный раздел доступен только в клиент-серверном режиме работы информационной базы. В разделе указываются:

  • «Использовать многопоточность» - если данный флаг установлен, то в поле «Количество потоков»* появляется возможность указать количество параллельно используемых потоков для выполнения операций начисления.

    * Чрезмерное увеличение количества потоков может привести к существенному снижению производительности системы вплоть до ее подвисания.

    Рекомендации по использованию количества потоков, в зависимости от мощности сервера. 

    • Этап 2. Выбор объектов начисления:

    В верхней части формы необходимо заполнить таблицу объектов учета. В качестве объекта указывается элемент справочника «Здания, сооружения».

     

    Раздел «Настройки начислений услуг» - в данном разделе указываются:

  • «Типы операций начисления» - выбирается одно из 3-х доступных значений переключателя:
    • «Начисления на лицевые счета» - будет выполнено начисление услуг только с видом операции "Начисления на лицевые счета";
    • «Начисления по ПУ» - будет выполнено начисление услуг только с видом операции "Начисления по приборам учета";
    • «Все» - будет выполнено начисление услуг с обоими видами операций.

    Если выбрано значение «Начисления на лицевые счета» или «Все», то необходимо заполнить таблицу услуг для вида операций "Начисления на лицевые счета". Кроме того, есть возможность установить флаг «Начислять на все услуги», при этом будут формироваться документы начисления услуг для всех услуг, подходящих по типу операции, содержащихся в информационной базе.

    Если выбрано значение «Начисления по ПУ» или «Все», то необходимо заполнить таблицу услуг для вида операций "Начисления по приборам учета". Кроме того, есть возможность установить флаг «Начислять на все услуги», при этом будут формироваться документы начисления услуг для всех услуг, подходящих по типу операции, содержащихся в информационной базе.

  • «Организация» - организация, для которой выполняются операции начисления;
  • «Ответственный» - пользователь, ответственный за выполнение операций начисления.

    • Этап 3. Формирование документов:

    На данном этапе пользователь запускает формирование документов в фоновом режиме по нажатию на кнопку «Сформировать документы».

    После завершения формирования документов можно посмотреть список документов по каждому типу документа. Документов "Начисление услуг" кроме того, разделяются по видам операции начисления.

    Примечание:
    На данном этапе мастер выполняет поиск ранее сформированных документов по месяцу начисления и организации, указанным на первом этапе, для зданий и услуг, установленных на втором этапе, и если такие документы найдены, то перезаполняет их, а не создаёт новые. Найденные документы отображаются пользователю в таблице мастера до старта формирования начислений.

    Пример: Для здания «ул. Вавилова д. 45» сформирован документ «Начисление услуг УК00-000001 от 12.12.2018 23:59:59» по услуге «Антенна». Если в мастере начислений на первом этапе установить настройки:

      - месяц начисления - «Декабрь 2018»

      - настройки начислений - «Начислять услуги»

    На втором этапе выбрать параметры:

      - объект - «ул. Вавилова д. 45»

      - услуга - «Антенна»

    То на третьем этапе сформированный документ отобразится в таблице начислений услуг, и при запуске процесса формирования документов мастер предложит перезаполнить и перепровести его.

    Продолжении незавершенных задач "Мастера начислений":

    В случае, если по каким-либо причинам задача мастера не была завершена (технический сбой или ручная остановка работы "Мастера начислений"), то при последующем открытии обработки пользователю будет доступен список незавершенных задач. Незавершенную задачу можно продолжить или удалить.
