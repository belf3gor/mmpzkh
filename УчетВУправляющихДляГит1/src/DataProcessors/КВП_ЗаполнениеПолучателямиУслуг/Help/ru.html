Обработка «Заполнение получателями услуг»

Данная обработка предназначена для формирования списка зданий, квартир или подъездов, отобранных по определенным критериям. Обработка не используется самостоятельно, а вызывается из форм ввода документов для заполнения табличной части.

Внешний вид формы обработки зависит от значения поля «Тип объекта». В зависимости от типа объекта будут отображаться разные поля отбора. 

Если в качестве типа объекта определены «Помещения», можно сформировать список занесенных в базу помещений по следующим критериям:

  • «Здание» – если выбрано здание, то будут отобраны только помещения, принадлежащие к данному зданию;
  • «Комнат» – если установлено, то будут отобраны только те помещения, где количество комнат равно установленному значению;
  • «Тип помещения» – если установлено, то будут отобраны только помещения, относящиеся к указанному типу;
  • «Этаж» – если установлено, то будут отобраны только помещения, расположенные на указанном этаже;
  • «Приватизация» – значение устанавливается из списка.

Если типом объекта будут являться «Здания», то будут предложены следующие поля отбора:

На вкладке «Общее» устанавливается отбор по следующим полям:

  • На вкладке «Общее» устанавливается отбор по следующим полям:

    • «Группа зданий» – если указана, то в список будут занесены только здания, относящиеся к указанной группе;
    • «Вид мастера» – если указан, то в список будут занесены только здания, находящиеся на обслуживании у определенного типа мастера;
    • «Мастер» – если указан, то в список будут занесены только здания, находящиеся на обслуживании у определенного типа мастера; 
  • На вкладке «Адрес» устанавливается отбор по адресным данным:
    • «Регион» – если указан, то в список будут занесены только здания, находящиеся в указанном регионе;
    • «Город» – если указан, то в список будут занесены только здания, находящиеся в указанном городе;
    • «Населенный пункт» – если указан, то в список будут занесены только здания, находящиеся в указанном населенном пункте;
    • «Улица» – если указана, то в список будут занесены только здания, находящиеся на указанной улице.
  • На вкладке «Характеристики» устанавливается отбор по выбранным дополнительным свойствам справочника «Здания».

Если типом объекта будут являться «Подъезды», то будут предложены следующие поля отбора:

  • «Здание» – если указано здание, то в список будут занесены только подъезды, относящиеся к выбранному зданию;
  • «Квартиры с» – если указан номер первой квартиры, то будут отобраны только подъезды, где номер первой квартиры равен указанному;
  • «по» – если указан номер последней квартиры, то будут отобраны только те подъезды, где номер последней квартиры равен указанному.

Если типом объекта будут являться «Лицевые счета», то будут предложены следующие поля отбора:

  • «Период» – если указана начальная дата, то будут отобраны лицевые счета, действующие в указанном периоде;
  • «Объект» – если указан, то будут отобраны только лицевые счета, относящиеся к указанному объекту.
  • «Здание» – если указано здание, то в список будут занесены только подъезды, относящиеся к выбранному зданию;
  • «Список контрагентов» – если заполнен список контрагентов, то будут отобраны только лицевые счета, владельцем которых является контрагент из списка;
  • «Услуга» – если указана, то будут отобраны только лицевые счета, для которых назначена или не назначена указанная услуга;
  • «Отображать л/с, где услуга» – если выбрано значение "Действует", тогда выбираются счета, для которых услуга назначена, если - "Не действует", тогда выбираются счета, для которых услуга не назначена.

Если типом объекта будут являться «Паркинги», то будут предложены следующие поля отбора:

  • «Группа паркингов» – если указано, то в список будут занесены паркинги, относящиеся к выбранной группе.

Если типом объекта будут являться «Парковочные места», то будут предложены следующие поля отбора:

  • «Владелец» – если указано, то в список будут занесены только парковочные места, принадлежащие выбранному владельцу;
  • «Список контрагентов» – если заполнен список контрагентов, то будут отобраны только парковочные места, владельцем которых является контрагент из списка.

 

При нажатии кнопки «Заполнить» таблица будет заполнена значениями в соответствии с указанным отбором.

Таблица состоит из трех колонок:

  • «N» – порядковый номер строки в таблице;
  • «Использовать» – если флажок снят, то объект не будет передан в документ, вызвавший обработку;
  • «Объект» – здание, подъезд или квартира, параметры которых соответствуют критериям отбора.

На панели, расположенной над таблицей присутствуют кнопки:

  • «Установить флажки» – при нажатии в колонке «Использовать» будут установлены флажки для всех строк;
  • «Снять флажки» – при нажатии в колонке «Использовать» будут сняты флажки для всех строк.

Внизу формы обработки расположены кнопки:

  • «Добавить» – добавить значения из списка в документ. Добавлены будут только те значения, у которых установлен флажок «Использовать»;
  • «Закрыть» – закрыть форму обработки.
