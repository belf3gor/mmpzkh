Обработка «Установка долей собственности л/с»

Данная обработка позволяет установить доли собственности, соответствующие лицевым счетам.

Вызывается обработка из форм справочников «Лицевые счета».

Для выполнения обработки необходимо заполнить поля:

  • «Дата» - дата, на которую будет внесены изменения долей лицевого счета;
  • «Здание» - ссылка на элемент справочника «Здания», в котором открыт лицевой счет;
  • «Помещение» - ссылка на элемент справочника «Помещения», в котором открыт лицевой счет;
  • «Делитель» - делитель для расчета долей лицевых счетов в помещении; (в качестве делителя как правило устанавливается общая площадь помещения; при нажатии на кнопку "площадь помещения" общая площадь в поле делитель установится автоматически).

На форме расположена табличная часть "Доли", в которой заполняются доли по каждому лицевому счету. Табличная часть состоит из колонок:

  • «Лицевой счет» – ссылка на элемент справочника «Лицевые счета»;
  • «Владелец» – владелец лицевого счета;
  • «Доля собственности л/с» - площадь лицевого счета в помещении; (на одно помещение может быть зарегистрировано несколько лицевых счетов, при этом доля собственности позволяет определить площадь для каждого из лицевых счетов);

При выполнении обработки будут созданы записи об изменении долей лицевых счетов.
