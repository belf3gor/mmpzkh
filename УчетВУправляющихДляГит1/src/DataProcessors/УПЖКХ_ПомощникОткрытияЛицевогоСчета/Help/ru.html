Обработка «Помощник открытия лицевого счета»

Данная обработка позволяет упростить открытие нового лицевого счета.

Помощник открывается автоматически при создании нового элемента в справочнике «Лицевые счета» или при создании документа «Открытие лицевого счета».

Открытие лицевого счета осуществляется в несколько этапов, которые зависят от текущего варианта учета собственников помещений (настройка находится на закладке «Учет собственников» формы «Настройка программы (ЖКХ)»:

Подробный вариант учета собственников помещений:

  • «Выбор помещения лицевого счета» - страница «Данные о помещении» - сбор информации о помещении, где будет открыт новый лицевой счет. На данном этапе необходимо указать дату открытия лицевого счета, организацию, здание и помещение;
  • «Регистрация прав собственности на помещение» - страница «Регистрация собственности» - данный этап будет иметь место, когда данные о собственности на выбранное помещение еще не введены. Если ранее уже были введены данные о собственности с помощью документа «Установка собственников помещений», то ввод данных о собственности пропускается. При вводе данных о собственности необходимо указать Вид собственности из списка:
    • Государственная
    • Муниципальная
    • Частная
    Если выбирается вид собственности «Частный», то появляется флажок «Несколько собственников». Если флажок остается не отмеченным, то подвидом частной собственности будет являться "Индивидуальный". При установке данного флажка поля собственника, основание, реквизиты документа собственности исчезают и появляется таблица с такими же колонками, подвидом же будет являться "Совместный". Таким образом можно указать несколько собственников на данное помещение. Также для вида собственности «Частная» доступен флажок «Определить доли собственности» и при этом подвид частной собственности будет "Долевой". Он устанавливается, если необходимо указать доли собственников. При установке данного флажка в таблице появляется колонка «Доля». Доли собственников указываются в виде простой дроби. Сумма долей должна быть равна 1;
  • «Выбор собственников лицевого счета» - страница «Данные о собственниках» - если ранее на помещение не было открыто лицевых счетов, то можно выбрать будет открыт один или несколько лицевых счетов. Далее возможны следующие варианты:
    • У помещения один собственник - тогда он автоматически становится ответственным владельцем лицевого счета;
    • У помещения несколько собственников -
      • если собственность совместная, то тогда необходимо выбрать ответственного владельца лицевого счета. Для этого в списке "Собственники лицевого счета" установите ответственного в начало списка.
      • если на помещение установлена долевая собственность, то тогда сначала необходимо переместить собственников из списка "Собственники помещения" в список "Собственники лицевого счета";
  • «Настройка параметров лицевого счета» - страница «Данные по лицевому счету» - на данном этапе производится настройка параметров лицевого счета, таких как: "Группа размещения л/с", "Имя лицевого счета", "Идентификатор л/с", "Контрагент" и "Договор". Если договор на обслуживание отсутствует, то его можно создать автоматически нажатием кнопки "Создать". Доля собственности лицевого счета рассчитывается автоматически при совместной собственности как сумма долей собственников, на кого открывается лицевой счет. При необходимости доля может быть рассчитана от площади комнат лицевого счета (имеет смысл для коммунальных квартир). При этом сумма размеров долей всех лицевых счетов на помещении не должна превышать 1;
  • «Назначение услуг для лицевого счета» - страница «Назначение услуг» - на данном этапе можно выбрать шаблон лицевого счета, из которого будут скопированы назначенные услуги и список норм потребления услуг, или можно установить услуги вручную, путем редактирования таблицы услуг.
  • «Настройка списка жильцов» - страница «Данные о жильцах» - физические лица из числа собственников лицевого счета, указанные на этапе "Выбор собственников лицевого счета", попадают в список жильцов автоматически. Так же можно добавить новых жильцов. Для этого используется «Помощник создания проживающего». Если в системе уже содержится физическое лицо с введенным ФИО, то будет предложено выбрать из списка имеющихся, в противном случае, физическое лицо будет создано по завершению работы. На закладке можно отметить флажками зарегистрированных и проживающих жильцов. Данные о собственности на данной закладке не изменяются, а выводятся информативно;
  • «Окончание работы мастера» - страница «Завершение работы» - по окончании работы помощника будут созданы элемент справочника «Лицевые счета», документ «Открытие лицевого счета», документ «Установка собственников помещения», если выполнялся ввод данных о регистрации прав собственности, документ «Назначение услуг», если выполнялся ввод данных об услугах. Их можно открыть, нажав на соответствующую ссылку.

Так же на данном этапе необходимо указать ответственного собственника;
