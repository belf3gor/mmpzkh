Обработка «Загрузка кодов зданий по ФИАС»

Обработка предназначена для загрузки с сервиса «Dadata.ru» кодов зданий по ФИАС, используемых для выгрузки в ГИС ЖКХ.

Для начала работы с обработкой необходимо указать API-ключ (кодовое слово), полученный на сервисе «Dadata.ru» в соответствии с инструкцией.

Обработка содержит следующие таблицы:

  • Таблица зданий, для которых необходимо заполнить коды по ФИАС;
  • Таблица загруженных кодов по ФИАС, в которой выводятся все загруженные коды по ФИАС здания, выбранного в таблице зданий. Выбор кода по ФИАС из предложенных в таблице выполняется двойным кликом по его строке.

Заполнение таблицы зданий можно выполнить одним из следующих способов:

  • Вручную, с помощью кнопки «Добавить»;
  • Всеми зданиями с незаполненным кодами по ФИАС с помощью кнопки «Заполнить зданиями без кодов по ФИАС» меню «Заполнить»;
  • Всеми зданиями независимо от заполненности кодов по ФИАС с помощью кнопки «Заполнить всеми зданиями» меню «Заполнить».

Загрузить коды по ФИАС с сервиса «Dadata.ru» можно одним из следующих способов:

  • С помощью кнопки «Загрузить коды по ФИАС», по нажатию на которую будет выполнена загрузка кодов по ФИАС для всех зданий в таблице зданий, у которых не заполнена колонка «Новый код дома по ФИАС»;
  • Вручную для здания, выбранного в таблице зданий, путем ввода адреса здания в поле «Адрес» и нажатия кнопки «Обновить коды здания по ФИАС».

Если в результате загрузки для здания найден единственный код по ФИАС и представление адреса совпадает с имеющимся в базе, то он автоматически будет добавлен в колонку «Новый код дома по ФИАС» соответствующей строки таблицы зданий.

Если в результате загрузки для здания не найден код по ФИАС или кодов по ФИАС найдено несколько, то будет необходимо указать код по ФИАС вручную из таблицы загруженных кодов по ФИАС или изменить представление адреса здания в поле «Адрес» и повторить загрузку кода по ФИАС.

Отобрать только те здания, для которых не удалось определить новый код по ФИАС, можно с помощью кнопки «Показать здания с незаполненными кодами по ФИАС».

Сохранить новые коды по ФИАС можно с помощью кнопки «Сохранить коды по ФИАС», по нажатию на которую существующие коды по ФИАС зданий будут заменены на коды по ФИАС, указанные в колонке «Новый код дома по ФИАС» таблицы зданий. При этом, замена кодов по ФИАС выполняется только для тех зданий, у которых новый код по ФИАС заполнен.

В случае изменения вашего API-ключа на сервисе «Dadata.ru» необходимо ввести новый API-ключ с помощью «Формы изменения API-ключа». Для этого нажмите на кнопку «Изменить API-ключ» и в открывшейся форме укажите новый API-ключ.
