Формирование платежных поручений ресурсоснабжающим организациям

Обработка предназначена для формирования списка платежных документов на перечисление собранных за услуги ЖКХ средств ресурсоснабжающим организациям (РСО).

В шапке обработки необходимо указать:

  • период, за который будут анализироваться оплаты населения за услуги ЖКХ; 
  • организацию и банковский счет, с которого переводятся денежные средства.

Список строк заполняется автоматически данными об оплатах:

    При заполнении анализируются оплаты, зарегистрированные с помощью документов «Регистрация оплаты», по выбранной организации за заданный период.

    Оплаты группируются по объектам (зданиям, подъездам) и видам услуг.

    Далее оплаты дополняются данными РСО, которыми осуществляется поставка видов услуг по данным объектам. Данные по РСО определяются в соответствии с документами «Управление договорами с поставщиками услуг» по состоянию на конец заданного периода.

    ВАЖНО: Учитываются только РСО по тем объектам, по которым отмечено, что оплата осуществляется через управляющую организацию.

    Суммы оплат по видам услуг, для которых нет данных по РСО или по которым оплата осуществляется не через управляющую организацию, из данных об оплатах исключаются.

Выполняется расчет суммы вознаграждения с учетом процента вознаграждения от общей суммы оплат.

Формирование платежных документов


Строки, по которым необходимо сформировать платежные поручения, должны быть отмечены флажком Оплатить.

Платежные документы формируются автоматически по кнопке Создать платежные документы.

После выполнения команды открывается список, содержащий сформированные документы. Если создается только один платежный документ, то открывается сразу форма этого документа.

В результате по каждому договору с РСО и виду услуг за период по организации будет сформирован один документ «Платежное поручение».

Настройка видов услуг


Предусмотрена возможность группировки оплат по видам услуг, если собранные с населения средства по этим видам услуг необходимо перечислить РСО одним платежным документом.

Для этого в окне настроек группировки видов услуг (открывается по кнопке Настройка) можно ввести наименование группировочной услуги и выбрать виды услуг, из которых она состоит.

В результате оплаты по видам услуг, входящих в одну группу видов услуг, будут определяться как оплата за один группировочный вид услуг.

Примечание
Если для одной из группировочных услуг указать, что она является услугой содержания общего имущества, то при формировании платежных документов для РСО, осуществляющей поставку услуги содержания, сумма платежа заполняется за минусом рассчитанной суммы вознаграждения. Если таких РСО за период несколько, то сумма платежа уменьшается на сумму вознаграждения пропорционально собранным по ним средствам.

Пример:

За период по организации с помощью документов «Регистрация оплаты» были отражены следующие оплаты от населения:

  • По зданию №1 за Отопление - 10000, за Содержание общего имущества - 40000.
  • По зданию №2 за Отопление - 6000, за Горячее водоснабжение - 3000, за Содержание общего имущества - 10000.
  • По зданию №3 за Отопление - 13000, за Горячее водоснабжение - 8000, за Содержание общего имущества - 30000.
  • По зданию №4 за Отопление - 7000.

Итого было собрано средств за Отопление - 36000, за Горячее водоснабжение - 11000, за Содержание общего имущества - 80000.

По каждому виду услуги в документах «Управление договорами с поставщиками услуг» установлены следующие РСО (для простоты предположим, что услуги оказываются по одному общему договору по всем домам):

  • ТЭЦ 1 осуществляет поставку услуг Отопление и Горячее водоснабжение в здания №1 и №2.
  • ТЭЦ 2 осуществляет поставку услуг Отопление и Горячее водоснабжение в здания №3 и №4.
  • УК 1 выполняет услуги по Содержанию общего имущества в зданиях №1 и №2.
  • УК 2 выполняет услуги по Содержанию общего имущества в зданиях №3 и №4.

Процент вознаграждения укажем равным 5,00.

Сумма вознаграждения рассчитывается от общей суммы оплат (36000+11000+80000) *5% = 127000*5% = 6350.

В результате создаются следующие документы "Платежное поручение":

  • 1) Оплата поставщику ТЭЦ 1 за отопление на сумму 16000.
  • 2) Оплата поставщику ТЭЦ 1 за горячее водоснабжение на сумму 3000.
  • 3) Оплата поставщику ТЭЦ 2 за отопление на сумму 20000.
  • 4) Оплата поставщику ТЭЦ 2 за горячее водоснабжение на сумму 8000.
  • 5) Оплата поставщику УК 1 за содержание общ. имущества на сумму 50000-6350*(50000/80000) = 46031,25.
  • 6) Оплата поставщику УК 2 за содержание общ. имущества на сумму 30000-6350*(30000/80000) = 27618,75.
