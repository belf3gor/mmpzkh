Обработка «Загрузка реестра показаний счетчиков»

Данная обработка предназначена для загрузки реестра показаний счетчиков из систем автоматического сбора показаний приборов учета.

Поддерживаются следующие форматы загружаемых файлов: TXT, XLS, CSV, DBF, XML.

Для выполнения обработки необходимо заполнить поля:

  • «Организация» - организация, производящая расчет коммунальных услуг;
  • «Формат файла загрузки» - ссылка на элемент справочника «Настройки загрузки из систем приема платежей или систем сбора показаний приборов учета»; 
  • «Файл загрузки» - путь и имя загружаемого файла;

    По кнопке «Настройка» открывается форма для редактирования настроек загрузки:

  •  «Организация» - организация, производящая расчет коммунальных услуг;

  • «Формат файла загрузки» - ссылка на элемент справочника «Настройки загрузки из систем приема платежей или систем сбора показаний приборов учета»; 
  • «Файл загрузки» - путь и имя загружаемого файла;  
  • «Виды создаваемых документов» - список документов для автоматического формирования;

    При нажатии на кнопку «Загрузить реестр» заполняется табличная часть обработки данными из загружаемого файла. Состав колонок зависит от настроек выбранного формата загрузки (реквизит «Формат файла загрузки»).

    При нажатии на кнопку «Сформировать документы» происходит формирование документов, выбранных в форме настройки.
