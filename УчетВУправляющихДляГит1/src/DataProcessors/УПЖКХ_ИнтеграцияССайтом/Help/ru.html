Обработка «Интеграция с сайтом»

Данная обработка предназначена для интеграции с программным продуктом «1С: Сайт управляющей компании ЖКХ, ТСЖ и ЖСК».

Она состоит из следующих форм:

  1. Форма «Новости сайта» - предназначена для редактирования новостей сайта непосредственно из учетной программы.
  2. Форма «Управление параметрами доступа» - предназначена для подготовки параметров доступа в личные кабинеты собственников лицевых счетов на сайте.
  3. Форма «Разовая выгрузка данных» - предназначена для разовой выгрузки данных по лицевым счетам на сайт за определенный период.
  4. Форма «Мониторинг обмена данными» - предназначена для выполнения и мониторинга обменов данными с сайтом.
