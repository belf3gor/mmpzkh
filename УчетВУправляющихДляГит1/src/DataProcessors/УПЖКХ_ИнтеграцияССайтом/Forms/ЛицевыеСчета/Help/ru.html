Форма «Управление параметрами доступа»

Данная форма предназначена для подготовки параметров доступа в личные кабинеты собственников лицевых счетов на сайте.

При открытии формы необходимо указать настройку доступа, выбрав ее в соответствующем поле из справочника «Настройки доступа к сайту».

На форме располагаются следующие элементы:

  • Таблица «Генерация параметров доступа на сайт». При этом:
    • В командной панели таблицы располагаются следующие кнопки:
      • «Заполнить» - кнопка-меню, с помощью которой осуществляется подбор лицевых счетов.
      • «Сгенерировать» - по нажатию на которую осуществляется генерация параметров доступа в соответствии с настройками, указанными в «Настройка программы».
      • «Записать логины» - по нажатию на которую происходит запись сгенерированных логинов в справочник «Лицевые счета» (пароли в базе данных не сохраняются). Если используется актуальная версия модуля «Сайта ЖКХ», данная кнопка может отсутствовать.
      • «Передать на сайт» - по нажатию на которую сгенерированные логины и пароли передаются на сайт в соответствии с выбранной настройкой доступа.
      • «Оповестить жильцов» - кнопка-меню, состоящая из следующих кнопок:
        • «Отправить по электронной почте» - по нажатию на которую осуществляется отправка параметров доступа по электронной почте для всех лицевых счетов, у которых зафиксирован электронный ящик в справочнике «Лицевые счета».
        • «Распечатать памятки» - по нажатию на которую осуществляется печать памяток с параметрами доступа.
    • Таблица содержит следующие колонки:
      • «Лицевой счет» - элемент справочника «Лицевые счета», для которого формируются параметры доступа.
      • «Владелец» - ответственный собственник/наниматель лицевого счета.
      • «Адрес» - адрес помещения лицевого счета.
      • «Логин» - логин для входа в личный кабинет на «Сайте ЖКХ». Данная колонка отсутствует в случае, когда на сайте используется регистрация с помощью пинкода.
      • «Пароль/Пинкод» - пароль/пинкод для входа в личный кабинет на «Сайте ЖКХ».
      • «Карточка распечатана» - указывает, сформирована ли карточка с параметрами доступа в личный кабинет на «Сайте ЖКХ» для лицевого счета.
      • «Есть параметры доступа на сайте» - указывает, имеются ли для лицевого счета параметры доступа в личный кабинет на «Сайте ЖКХ».
  • Флаг «Обновлять параметры доступа» - при установке которого для каждого лицевого счета, имеющегося в таблице будут сгенерированы новые параметры доступа (старые параметры будут утеряны).
