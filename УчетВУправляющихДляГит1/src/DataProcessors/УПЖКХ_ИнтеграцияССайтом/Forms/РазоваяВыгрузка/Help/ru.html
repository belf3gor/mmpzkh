Форма «Разовый обмен данными»

Данная форма предназначена для разового обмена данными с сайтом как за актуальный, так и за произвольный период.

На форме располагаются следующие элементы:

  • Тумблер «Обмен за актуальный период/Обмен за произвольный период» - определяет, необходимо выполнить обмен за актуальный или произвольный период. При обмене за произвольный период доступна только выгрузка на сайт данных по лицевым счетам. Загрузка с сайта сведений о показаниях приборов учета и оплатах доступна только для обмена за актуальный период.
  • Поле ввода «Настройка доступа» - необходимо указать настройку доступа, выбрав ее в соответствующем поле из справочника «Настройки доступа к сайту».
  • Поле ввода «Настройка обмена» - необходимо указать настройку обмена данными с сайтом, выбрав ее в соответствующем поле из справочника «Настройки обмена данными с сайтом ЖКХ».
  • Поле ввода «Месяц выгрузки» - предназначено для выбора месяца, за который необходимо выполнить обмен. Поле доступно только при обмене за произвольный период.
  • Флаг «Выгрузка данных по ЛС» - если флаг установлен, будет выполнена выгрузка на сайт сведений по лицевым счетам. Флаг доступен только при обмене за актуальный период. При обмене за произвольный период по умолчанию считается, что данный флаг установлен.
  • Флаг «Указать лицевые счета» - если флаг установлен, выгрузка данных по лицевым счетам будет выполнена по лицевым счетам, выбранным Пользователем. Если снят - по всем имеющимся лицевым счетам. Данный флаг доступен, только если флаг «Выгрузка данных по ЛС» установлен.
  • Таблица «Лицевые счета» - предназначена для выбора лицевых счетов, по которым будет выполнена выгрузка данных на сайт, из справочника «Лицевые счета». Таблица доступна только при установленном флаге «Указать лицевые счета».
  • Флаг «Загрузка сведений об оплатах» - если флаг установлен, будет выполнена загрузка с сайта сведений об оплатах. Флаг доступен только при обмене за актуальный период.
  • Флаг «Загрузка сведений о показаниях» - если флаг установлен, будет выполнена загрузка с сайта сведений о последних показаниях приборов учета. Флаг доступен только при обмене за актуальный период.
