Форма «Мониторинг обмена данными»

Данная форма предназначена для выполнения и мониторинга обменов данными с сайтом.

При открытии формы необходимо указать настройку доступа, выбрав ее в соответствующем поле из справочника «Настройки доступа к сайту».

На форме располагаются следующие элементы:

  • Таблица «Журнал обмена» - отображает журнал совершенных обменов данными в разрезе выбранной настройки доступа. 
    В командной панели таблицы располагаются следующие кнопки:
    • «Найти» - предназначена для поиска записей в журнале в соответствии с заданными критериями поиска.
    • «Обновить» - предназначена для обновления данных в журнале обмена.
  • Таблица «Виды обмена» - отображает перечень доступных видов обмена для выбранной настройки доступа.
    В командной панели таблицы располагаются следующие кнопки:
    • «Изменить» - предназначена для изменения текущего варианта обмена.
    • «Выполнить обмен сейчас» - предназначена для выполнения текущего варианта обмена.
    • «Выполнить разовую выгрузку» - предназначена для открытия формы «Разовая выгрузка данных».
