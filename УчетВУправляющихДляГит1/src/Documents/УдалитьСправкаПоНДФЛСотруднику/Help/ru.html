Документ "Справка о доходах (2-НДФЛ) для сотрудников"

Зарплата ® Учет НДФЛ и налогов (взносов) с ФОТ ® Справка о доходах (2-НДФЛ) для сотрудников

Документ предназначен для выдачи справок о доходах сотрудникам организации (и другим физическим лицам, получавшим доходы) за налоговые периоды с 2010 года.

Заполнение полей документа

В соответствующих полях документа указываются:

  • Налоговый период
    • , за который передаются сведения;
  • Организация — по умолчанию в качестве значения реквизита указывается организация, установленное в настройках пользователя. Это может быть как головная организация, так и ее обособленное подразделение с выделенным балансом;
  • Сотрудник — выбирается из списка сотрудников организации;
  • Телефон и данные подписывающего справку лица (по умолчанию используются телефон и данные главного бухгалтера организации);

На закладках документа отображаются данные подготавливаемой справки.

Автоматическое заполнение документа

При выборе сотрудника или изменении порядка формирования справки данные автоматически перечитываются.

При необходимости данные справок могут быть дополнены или уточнены. Для перезаполнения справки по данным ИБ предназначены кнопки Обновить.

Печать справки

Кнопка Печать позволяет получить формы 2-НДФЛ для вывода на принтер. При ее формировании автоматически проводятся проверки выводимых данных и в случае их недостаточности либо ошибочности выдаются диагностические сообщения.
