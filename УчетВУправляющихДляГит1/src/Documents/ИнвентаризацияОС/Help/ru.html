Документ "Инвентаризация ОС"

Документ предназначен для отражения результатов инвентаризации основных средств.

На закладке Основные средства указывается список основных средств и результаты проведения инвентаризации. Для автоматического заполнения табличной части документа предназначена кнопка Заполнить. Она заполняет список основных средств по остаткам на дату проведения инвентаризации, с отбором по:

  • подразделению организации, указанному в поле Местонахождение ОС (если поле пустое, выводится список ОС по всем подразделениям),
  • материально-ответственному лицу, указанному в поле МОЛ (если поле пустое, выводится список ОС по всем ответственным лицам).

При этом для всех ОС в списке автоматически заполняется данными учета поле Стоимость и проставляются обе галочки: и Наличие по данным учета, и Фактическое наличие. Для основных средств, отсутствующих по итогам инвентаризации, необходимо снять галочку Фактическое наличие вручную.

На закладке Дополнительно указываются сведения, необходимые для заполнения приказа о проведении инвентаризации, а также сведения о составе инвентаризационной комиссии.

  • В поле Документ указывается наименование документа-основания для проведения инвентаризации. Его можно выбрать из списка или указать вручную.

  • Председатель инвентаризационной комиссии выбирается путем установки флажка в колонке Председатель табличного поля Инвентаризационная комиссия. 

Печатные формы

Для документа Инвентаризация ОС предусмотрены следующие печатные формы:

  • Инвентаризационная опись ИНВ-1
  • Сличительная ведомость ИНВ-18
  • Приказ о проведении инвентаризации ИНВ-22

Ввод на основании

На основании документа Инвентаризация ОС можно ввести следующие документы:

  • Принятие к учету ОС
  • Перемещение ОС
  • Списание ОС
