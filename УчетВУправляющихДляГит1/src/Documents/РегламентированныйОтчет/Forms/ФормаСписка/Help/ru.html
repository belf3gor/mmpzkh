Список регламентированных отчетов

Список «Регламентированная отчетность» содержит список всех введенных форм отчетности. Список открывается по кнопке «Журнал» из формы списка справочника «Регламентированные отчеты».
При открытии ранее введенных отчетов сразу открывается форма отчета, без вызова «стартовой» формы.
