Документ «СМС сообщение»

Документ предназначен для рассылки смс сообщений.

При вводе нового документа необходимо указать:

  • «Дата» – дата документа;
  • «Номер» – номер документа;
  • «Аккаунт» – аккаунт, по которому производится рассылка смс сообщений;
  • «Шаблон» – из справочника «Шаблоны» выбирается шаблон смс сообщения;
  • «Дата и время отправки» – дата и время, в которое необходимо отправить смс сообщения.

Средняя часть формы представляет собой табличную часть «Критерии отбора получателей», которая состоит из колонок:

  • «Использование» – признак использования отбора;
  • «Поле отбора получателей» - указавает по каким параметрам будет осуществлятся формирование списка получателей, 
  • «Вид сравнения» - указывает какое условие будет накладываться на параметр,
  • «Значение» –  содержит конкретное значение с которым будет осуществлятся сравнение.

Имеется возможность применять группировку критериев отбора с использованием групп с условиями И, ИЛИ, НЕ.

Нижняя часть формы представляет собой табличную часть «Получатели», которая состоит из колонок:

  • «№» – порядковый номер строки в документе;
  • «Получатель» – получатель смс сообщения; 
  • «Номер телефона» – номер телефона получателя;
  • «СМС» – количество смс сообщений;
  • «Передано» – количество переданных сообщений;
  • «Состояние сообщения» – состояние смс сообщения.
