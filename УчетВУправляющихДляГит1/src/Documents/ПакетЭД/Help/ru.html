Пакет электронных документов

Документ формируется автоматически и предназначен для хранения реквизитов отправителя и получателя электронных документов, а также связи электронных документов и документов учетной системы.

Для исходящих документов данные по Электронному документу и Объекту владельцу заполняются в момент формирования пакета, для входящих документов — в момент распаковки, выполняемой либо автоматически программой, либо по команде Распаковать пакет (статус пакета должен быть К распаковке).

В некоторых случаях документ может быть использован для разбора проблемных ситуаций службой технической поддержки. Для этого необходимо выгрузить содержимое пакета по команде Сохранить пакет на диск.
