Документ «Отражение начислений в регламентированном учете»

Документ предназначен для отражения оперативных начислений по жилищно-коммунальным услугам в регламентированном учете на счетах бухгалтерского и налогового учета (для правильного расчета налога на прибыль, налога по упрощенной системе налогообложения и НДС).

Перед вводом информации в новый документ необходимо выбрать один из следующих видов операций:

  • «Сводное отражение (упрощенное)» – данный вид операции предназначен для отражения начислений сводными проводками;
  • «Отражение типовыми документами реализации услуг» – данный вид операции предназначен для отражения начислений при помощи типовых документов.

Имеются дополнительные настройки в «Учетной политике ЖКХ», влияющие на работу документа. Настройки располагаются на закладке «Регламентированный учет».

При вводе нового документа необходимо указать:

  • «Дата» – дата документа;
  • «Номер» – номер документа;
  • «Организация» – организация, по которой необходимо отразить начисления в регламентированном учете;
  • «Зачет авансов» – можно выбрать один из вариантов производить автоматический зачет авансов или не производить;
  • в группе «Параметры для отбора начислений» в соответствующих полях: контрагента, номенклатуру, номенклатурную группу. Так же имеется возможность установить отбор по виду взаиморасчетов (настройка доступна, если в настройках «Учетной политики ЖКХ» выбран способ отражения льгот "Отдельно от начислений", либо установлен флажок "Отражать пени отдельно от начислений"):
    • Все - при автоматическом заполнении в табличную часть попадут и произведенные начисления и рассчитанные льготы;
    • Только начисления - в табличную часть попадут только начисления;
    • Только льготы - в табличную часть попадут только льготы. Данный вид доступен, если в настройках «Учетной политики ЖКХ» установлен способ отражения льгот отдельно от начислений.
    • Только пени - в табличную часть попадут только пени. Данный вид доступен, если в настройках «Учетной политики ЖКХ» установлена настройка "Отражать пени отдельно от начислений" или "Отражать пени за капитальный ремонт отдельно от начислений" при ведении раздельного учета по услугам капитального ремонта.

Вид операции «Сводное отражение (упрощенное)»

При выборе данного вида операции отражение начислений производится сводными проводками.

Для заполнения табличной части "Контрагенты" необходимо воспользоваться кнопкой «Заполнить». При этом, счета учета расчетов по начислениям заполняются автоматически согласно настройкам «Учетной политики ЖКХ».

По каждой строке табличной части можно просмотреть расшифровку начислений.

При проведении документа будут сформированы проводки лишь по тем строкам, у которых отмечен флажок. Если выбран автоматический «Способ зачета авансов», то будут также сформированы проводки по зачету авансов.

Вид операции «Отражение типовыми документами реализации услуг»

При выборе данного вида операции отражения начислений производятся при помощи типовых документов.

При этом на форме располагаются следующие закладки:

  • «Параметры документов», на которой необходимо установить следующие настройки:
    • «Тип документа» – выбрать либо «Акт об оказании производственных услуг», либо «Реализация товаров, услуг»;
    • «Подразделение затрат» - подразделение затрат. Данное поле доступно только если в поле "Тип документа" указан «Акт об оказании производственных услуг» и ведется учет затрат по подразделениям;
    • при необходимости установить флажки «Учитывать НДС» и «Сумма включает НДС»;
    • в группе «Параметры отражения льгот» определить контрагента и договор с контрагентом для формирования документа «Корректировка долга» для возмещения затрат по предоставленным льготам;
  • «Начисления», на которой заполнение таблицы происходит по кнопке «Заполнить».

    При этом:

    • черным цветом будут выделены суммы, которые еще не отражены в регламентированном отчете;
    • серым цветом – суммы, отраженные в оперативном учете, которые соответствуют суммам, отраженным в регламентированном отчете;
    • красным цветом – суммы, которые не совпадают в оперативном и регламентированном учете.

    Для вида операции «Отражение типовыми документами реализации услуг» при автоматическом заполнении документа в табличную часть попадают только начисления по договорам с видом "С покупателем" (подробнее см. «Настройки программы ЖКХ», закладка «Учет договоров»). Если необходимо с помощью документов реализации отражать отрицательные начисления (например, могут возникнуть в случае проведения корректировок начислений), то для этого необходимо установить флаг «Формировать типовые документы с отрицательными суммами» в «Учетной политике ЖКХ» на закладке «Регламентированный учет».

    ВАЖНО: При отражении типовыми документами в регл. учете отрицательных начислений или начислений по агентской схеме независимо от выбранного «Типа документа» будут сформированы документы «Реализация товаров, услуг».

    На панели табличной части расположена группа команд «Документы» со следующими командами:

    • «Сформировать документы» - при нажатии кнопки (доступна только при виде операции «Отражение типовыми документами реализации услуг») будут автоматически сформированы документы для тех строк табличной части, у которых установлен флажок.
    • «Удалить документы в текущей строке» - при нажатии кнопки будет помечен на удаление документ, сформированный для текущей строки табличной части.
    • «Удалить все документы» - при нажатии кнопки будут помечены на удаление все сформированные и отраженные в табличной части документы.


    По каждой строке табличной части можно просмотреть расшифровку начислений.

Документ можно открыть с помощью пункта меню «Регламентные операции / Отражение начислений в регламентированном учете».
