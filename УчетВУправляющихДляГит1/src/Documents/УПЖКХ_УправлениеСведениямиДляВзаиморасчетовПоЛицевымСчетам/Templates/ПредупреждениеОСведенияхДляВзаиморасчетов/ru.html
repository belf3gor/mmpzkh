По [СписокОрганизаций] имеются действующие договоры контрагентов лицевых счетов, отсутствующие в сведениях для взаиморасчетов по лицевым счетам. В связи с этим некоторые операции (например, обмен с сайтом) могут выполняться некорректно.

По указанным организациям рекомендуется сформировать документы "Управление сведениями для взаиморасчетов по лицевым счетам", заполнив их договорами контрагентов лицевых счетов, которые отсутствуют в сведениях для взаиморасчетов по лицевым счетам.

Для формирования документов в автоматическом режиме нажмите кнопку "Сформировать документы". Сформированные документы будут записаны и открыты, после чего их можно будет скорректировать.
