Документ «Управление сведениями для взаиморасчетов по лицевым счетам»

Документ предназначен для управления сведениями для взаиморасчетов по лицевым счетам в разрезе организаций.

При вводе нового документа указываются следующие сведения:

  • «Дата» – дата документа;
  • «Номер» – номер документа;
  • «Организация» – организация, к которой относятся указываемые в таблице взаиморасчетов сведения для взаиморасчетов;
  • «Объект» – выбирается из справочника «Здания, сооружения», «Подъезды» или «Помещения». Используется для заполнения табличной части сведений для взаиморасчетов. Не обязателен для заполнения.
  • «Таблица сведений для взаиморасчетов» - в данной таблице указываются сведения для взаиморасчетов по лицевым счетам.
    • Лицевой счет - указывается лицевой счет из справочника «Лицевые счета».
    • Контрагент - указывается контрагент из справочника «Контрагенты».
    • Договор контрагента - указывается договор контрагента по организации, указанной в шапке документа, с видом договора «С покупателем» либо «Прочее».

Предусмотрена функция автоматического заполнения таблицы сведений для взаиморасчетов, вызываемая по команде «Заполнить». По нажатию на нее выполняется заполнение таблицы сведений для взаиморасчетов всеми открытыми на дату документа лицевыми счетами, которые относятся к объекту, выбранному в поле «Объект» шапки документа.

В поле «Ответственный» указывается пользователь, ответственный за создание документа.

В поле «Комментарий» можно указать произвольный комментарий к документу.

 

Документ автоматически формируется при открытии лицевого счета с помощью обработки «Помощник открытия лицевого счета», а также может быть введен на основании документа «Открытие лицевого счета».

Документ формирует движения по регистру сведений «Сведения для взаиморасчетов по лицевым счетам».

 

Документ доступен в разделе «Управление лицевыми счетами» - «Управление сведениями для взаиморасчетов по лицевым счетам».
