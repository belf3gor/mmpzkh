Документ «Подготовка данных для раскрытия»

Документ предназначен для подготовки данных, которые требуется раскрывать в соответствии с Постановлениями правительства РФ №731 и №1468.

 

В документе предусмотрены 2 вида операций:

  • «Сведения о деятельности УО» – операция предназначена для подготовки данных для раскрытия сведений о деятельности управляющей организации;
  • «Сведения о здании» – операция предназначена для подготовки данных для раскрытия сведений об одном здании и соответствующей ему прилегающей территории;

 

При вводе документа необходимо указать:

  • «Номер» – номер документа устанавливается автоматически;
  • «Дата» – дата документа устанавливается автоматически;
  • «Организация» – выбирается организация из справочника «Организации»;
  • «Период» – выбирается месяц, на конец которого подготавливаются данные;

 

Сведения о деятельности УО

Подготавливаются следующие данные о деятельности управляющей организации:

  • Общая информация об организации
    • Основные сведения об организации
    • Сведения о работе диспетчерской службы
    • Участие в саморегулируемых организациях
  • Сведения об основных показателях финансово-хозяйственной деятельности
    • Информация о ТСЖ
    • Информация о задолженности УО перед РСО
  • Перечень многоквартирных домов

 

Сведения о здании

Для данного вида операции необходимо указать:

«Здание» – выбирается здание из справочника «Здания»;

  • Общие сведения о здании
  • Общая характеристика

    • Общие сведения о земельном участке
    • Парковки в границах земельного участка
    • Контейнерные площадки
    • Элементы благоустройства
  • Конструктивные элементы
    • Фундамент
    • Фасады
    • Крыши
    • Несущие конструкции
    • Мусоропроводы
    • Лифты
    • Общедомовые приборы учета
    • Система электроснабжения
    • Система теплоснабжения
    • Система горячего водоснабжения
    • Система холодного водоснабжения
    • Система водоотведения
    • Система газоснабжения
    • Система вентиляции
    • Система пожаротушения
    • Система водостоков
    • Дополнительное оборудование
    • Сведения о состоянии МКД
    • Энергоэффективность МКД
  • Иные сведения
    • Сведения об услугах по содержанию и ремонту общего имущества
    • Сведения об иных услугах
    • Сведения о предоставляемых коммунальных услугах
    • Сведения об использовании общего имущества
    • Сведения о капитальном ремонте имущества
    • Сведения о планируемых общих собраниях
    • Сведения о проведенных общих собраниях
  • Отчет об исполнении договора управления
    • Общая информация об оказании услуг
    • Прочие поступления денежных средств
    • Капитальный ремонт
    • Общая информация по коммунальным ресурсам
    • Информация об объемах коммунальных услуг
    • Информация о расчетах за коммунальные услуги

  • Отчет об исполнении договора оказания услуг
    • Общая информация об оказании услуг организацией
    • Прочие поступления денежных средств
    • Сведения о нарушениях качества предоставления услуг

 

В поле «Ответственный» – указывается ответственный за работу с документом пользователь.

При необходимости можно заполнить поле «Комментарий», которое будет отображаться в списке документов.

Документ не формирует движения.

Документ позволяет формировать печатную форму «Раскрытие постановлений». В данной печатной форме в виде таблицы выводятся все показатели раскрытия информации.

Документ можно открыть через меню «Учет в ЖКХ / Раскрытие информации / Подготовка данных для раскрытия».
