Документ "Передача товаров"

Документ предназначен для отражения операций по передаче собственных материалов стороннему переработчику для выпуска продукции, передачи заказчику продукции, выпущенной из давальческого сырья, для передачи комитенту товаров, закупленных по договору комиссии, а также для безвозмездной передачи, т.е. когда товары предоставляются другой стороне, без получения от нее платы или иного встречного предоставления.

Передача сырья в переработку

Для отражения передачи сырья в переработку необходимо выбрать в поле Вид операции операцию Передача сырья в переработку. Материалы, переданные в переработку, учитываются на счете 10.07 "Материалы, переданные в переработку на сторону".

Если материалы передаются на производство определенного количества продукции, их состав и количество можно рассчитать с помощью спецификаций по кнопке Заполнить. В открывшейся форме подбора номенклатуры указывается продукция, которую планируется произвести, и ее количество, далее указываются спецификации.

Примечание
Для организаций, уплачивающих налог на прибыль, суммовая оценка прямых расходов, связанных со стоимостью материалов, переданных в переработку, определяется также, как в бухгалтерском учете, с учетом переработчиков (контрагентов).

Передача продукции заказчику

Для документального оформления передачи продукции заказчику (формирования печатной формы) необходимо выбрать в поле Вид операции операцию Передача продукции заказчику. Операция применяется только в том случае, когда передача продукции заказчику и оказание услуг заказчику по ее производству приходятся на разные моменты  времени. Если передача продукции заказчику и оказание услуг заказчику отражаются одновременно, то нужно регистрировать документ Реализация услуг по переработке.

Передача товаров комитенту

Для передачи товаров, закупленных в рамках договора комиссии (агентирования) комитенту (принципалу), в интересах которого был закуплен товар, используется вид документа Передача товаров комитенту. В поле Контрагент указывается комитент (принципал), в поле договор – договор комиссии (агентский договор) по закупкам. Табличная часть может быть заполнена автоматически по остаткам.

Безвозмездная передача

Для оформления безвозмездной передачи необходимо выбрать в поле Вид операции операцию Безвозмездная передача.  Поле Получатель заполняется при передаче ценностей определенному лицу.  
Табличную часть можно заполнить по данным документа: Поступление (акт, накладная), нажав на кнопку Заполнить. В поле Цена необходимо указать рыночную стоимость товара, с которой будет исчисляться НДС.
По умолчанию сумма расходов и сумма НДС учитываются на счете 91.02.

На основании документа Передача товаров можно зарегистрировать документ Счет-фактура выданный по гиперссылке Ввести счет-фактуру. В счете-фактуре автоматически указывается номер и дата счета-фактуры.
Для организаций, использующих Раздельный учет НДС, сумма "без НДС" включается в 7 Раздел Декларации по НДС.  

Печатные формы

Для документа Передача товаров с видами операций: Передача сырья в переработку, Передача продукции заказчику, Передача товаров комитенту предусмотрены следующие печатные формы:

  • М-15 (Накладная на отпуск материалов на сторону)
  • ТОРГ-12 (Товарная накладная)
  • 1-Т (Товарно-транспортная накладная)
  • Транспортная накладная
  • Универсальный передаточный документ (УПД)
         

с видом операции Безвозмездная передача предусмотрены следующие печатные формы:

  • М-11 (Требование-накладная)
  • ТОРГ-12 (Товарная накладная)
  • Счет-фактура выданный
  • Универсальный передаточный документ (УПД)
