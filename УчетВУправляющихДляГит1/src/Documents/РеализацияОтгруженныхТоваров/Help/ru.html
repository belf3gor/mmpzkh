Документ "Реализация отгруженных товаров" 

Документ предназначен для оформления перехода права собственности по ранее состоявшейся отгрузке. Документ можно ввести на основании документа Реализация (акт, накладная) с видом операции Отгрузка без перехода права собственности.

При вводе документа в шапке обязательно нужно указать Документ отгрузки - документ Реализация (акт, накладная) с видом операции Отгрузка без перехода права собственности, по которому оформляется переход права собственности.

На основании документа Реализация отгруженных товаров можно ввести следующие документы:

  • Счет-фактура выданный
  • Отражение начисления НДС
