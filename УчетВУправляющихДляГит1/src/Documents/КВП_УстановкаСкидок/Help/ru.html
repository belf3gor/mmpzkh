Документ «Установка скидок»

В документе по льготной категории задаются размеры скидок на услуги и указываются лица, пользующиеся льготой.

Поля документа заполняются в следующем порядке:

  • «Номер» – номер документа устанавливается автоматически;
  • «Дата» – дата документа устанавливается автоматически;
  • «Льготная категория» – указывается льготная категория, по которой будет произведена установка размеров скидок.

На нижней половине формы расположена таблица, состоящая из колонок:

  • «N» – порядковый номер строки;
  • «Услуга» – из справочника «Услуги» выбирается услуга, по которой будет предоставлена скидка;
  • «Размер скидки» – устанавливается размер скидки в процентах на указанную услугу;
  • «Льгота предоставляется» – из списка выбирается круг лиц пользующихся льготой (вся семья, близкие родственники или один человек);
  • «Ограничение по объему услуги» – из справочника «Нормы потребления услуг» выбирается норма, ограничивающая объем услуги, на который предоставляется скидка. Льгота будет предоставлена на объем потребления, не превышающий размер данного ограничения;
  • «Площадь для расчета ограничения» – из справочника «Виды площадей» выбирается вид площади для расчета объема ограничения.
    Колонка доступна, когда норма ограничения задана в единицах измерения площади, а объем начисленной услуги измеряется в единицах, отличных от единицы измерения площади;

    Примечание
    Единицы измерения нормы ограничения и вида рассчитываемой услуги должны задаваться в соответствии с общероссийским классификатором единиц измерения (ОКЕИ), а именно выбираться из справочника «Классификатор единиц измерения», заполненного по кнопке «Подбор из ОКЕИ».

    Пример:
    Общая площадь квартиры 60 м2. В квартире проживает 1 человек, льготник.
    Расчет услуги отопление выполняется в Гкал по тарифу 1200 руб. Объем потребления за месяц составил 2,178 Гкал.
    Льгота по отоплению предоставляется 50% на самого льготьника в пределах соц.нормы площади 33 м2.
    Площадь для расчета ограничения указываем "Общая площадь".
    Расчет льготы будет выполнен следующим образом:
    Норма ограничения составит 2,178 * 33 / 60 = 1,1979 Гкал.
    Итого размер льготы минимум(2,178, 1,1979) * 1200 * 50% = 1,1979 * 1200 * 50% = 718,74 руб.

  • «Льготная категория для сверхобъема» – из справочника «Льготные категории» выбирается льготная категория для расчета льготы на объем услуги, превышающий установленный объем ограничения.
    Колонка доступна, если по услуге установлено ограничение по объему;
    Следующие пары льгот применяются вместе на одном лицевом счете:
    • Участники ВОВ и Семьи участников ВОВ
    • Инвалиды ВОВ и Семьи инвалидов ВОВ
    Участникам ВОВ и инвалидам ВОВ предоставляется 100% скидка на оплату жилищных услуг и отопления в пределах 33 м2 (соцнорма на одного человека). 
    Семьям участников и инвалидов ВОВ предоставляется 50% скидка на оплату жилищных услуг и отопления на всю оставшуюся площадь квартиры (S - 33 м2).

    Примечание
    В качестве носителя льготы во всех вышеперечисленных 4-х льготах должен быть указан сам участник / инвалид ВОВ, а не его близкие родственники, как это могло бы показаться на первый взгляд для семейной пары льгот.

    Пример:
    Общая площадь квартиры 65 м2. В квартире проживает 2 человека: отец и сын, отец - инвалид ВОВ.
    Расчет услуги отопление выполняется в Гкал по тарифу 2100 руб. Объем потребления за месяц составил 1,900 Гкал.
    Скидки по отоплению установлены следующим образом:
     - Инвалиду ВОВ - 100% на одного человека в пределах соц.нормы площади 33 м2. Площадь для расчета ограничения "Общая площадь", льготная категория для сверхобъема "Семьи инвалидов ВОВ". 
     - Семье инвалида ВОВ  - 50% на семью без ограничения.
    Расчет льготы будет выполнен следующим образом:
     - по инвалиду ВОВ льгота составит минимум(1,900, 1,900 * 33 / 65) *  2100 * 100% = минимум(1,900, 0,965) *  2100 * 100%  = 0,965 *  2100 * 100%  = 2026,5 руб.
     - по семье инвалида ВОВ льгота составит (1,900 - 0,965) *  2100 * 50% = 0,935 *  2100 * 50%  = 981,75 руб.
    Итого размер льготы 3008,25 руб.

  • «Ограничение по сумме льготы» – устанавливается размер ограничения на итоговую сумму льготы. Льгота будет рассчитана, но не более размера данного ограничения.

    Пример:
    В квартире проживает 1 льготник, является собственником.
    Расчет услуги капремонт выполняется по общей площади квартиры 75 м2, по тарифу 5,20 руб.
    Скидка составляет 50% на самого льготьника, но не более 150 руб. 
    Расчет льготы будет выполнен следующим образом:
    минимум (75 * 5,20 * 50%, 150) = минимум (195, 150) = 150 руб.

В нижней части формы документа расположены поля:

  • «Ответственный» – указывается ответственный из справочника «Пользователи»;
  • «Комментарий» – вводится комментарий к документу;

Документ открывается через раздел «Льготы и скидки» -> «Установка скидок».
