Документ "Счет-фактура полученный"

Документ предназначен для регистрации счетов-фактур, полученных от поставщиков:

Счет-фактура по поступлению
Счет-фактура на выданный аванс
Корректировочный счет-фактура

Общими для всех видов документа являются реквизиты:

  • Контрагент которым выставлен счет-фактура.
  • Договор с контрагентом в рамках которого совершена операция, к которой выставлен счет-фактура.
  • Вх. номер и дата - номер и дата, указанные в документе поставщика.
  • Исправление номер и дата позволяют зарегистрировать счет-фактуру на исправление.
  • Код вида операции - указывается код, необходимый для регистрации в журнале учета счетов-фактур, книге покупок и книге продаж.
  • Выбирается Способ получения: на бумажном носителе или в электронном виде.
  • Сумма всего и сумма НДС (в т.ч.) по счету-фактуре.

Флажок Отразить вычет НДС в книге покупок позволяет автоматически включить запись в книгу покупок и делает бухгалтерскую проводку по вычету НДС.

Счет-фактура по поступлению

Счет-фактуру можно ввести на основании проведенного документа поступления: Поступление (акт, накладная), Поступление доп. расходов и пр. Для ввода счета-фактуры на основании в документах предусмотрена кнопка Зарегистрировать.

При вводе счета-фактуры по поступлению вручную необходимо в дополнение к общим реквизитам заполнить документ Основание (возможно выбрать несколько документов).

 

Счет-фактура на выданный аванс

Счет-фактуру от поставщика на выданный аванс рекомендуется вводить на основании проведенного документа на списание денежных средств: Списание с расчетного счета, Выдача наличных.

При вводе счета-фактуры на выданный аванс вручную необходимо в дополнение к общим реквизитам заполнить документ Основание. А в табличной части "Реквизиты счета-фактуры на аванс" указать величину аванса по ставкам НДС.

Корректировочный счет-фактура

Основанием корректировочного счета-фактуры может являться документ Корректировка поступления с видом операции "Корректировка по согласованию сторон" или документ Возврат товаров поставщику.

Исправление собственной ошибки

В документе предусмотрено исправление ошибок ввода данных, допущенных пользователем при регистрации полученного счета-фактуры с корректными реквизитами. Исправленная версия счета-фактуры создается автоматически при записи документа Корректировка поступления с видом операции «Исправление собственной ошибки».

 

Печатные формы

Для документа Счет-фактура полученный предусмотрены следующие печатные формы:

  • Счет-фактура

Ввод на основании

На основании документа Счет-фактура полученный можно ввести документ:

  • Отражение начисления НДС 
    Для ручного отражения восстановления НДС с выданных авансов при упрощенном учете НДС.

На основании счета-фактуры полученного на аванс можно создать исправленный счет-фактуру на аванс.
