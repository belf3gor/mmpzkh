Документ «Реестр участников голосования»

Документ предназначен для составления реестра участников голосования.

При вводе нового документа указываются:

  • «Номер» – номер документа. Устанавливается автоматически;
  • «Дата» – дата документа;
  • «Организация» – из справочника «Организации» выбирается организация;
  • «Объект проведения» – выбирается место проведения голосования из справочников "Здания", "Лицевые счета" или "Подъезды".

Ниже на форме располагается табличная часть «Участники». Данная табличная часть состоит из колонок:

  • «Номер строки» - порядковый номер строки;
  • «Лицевой счет» - из справочника «Лицевые счета» заполняется по указанному объекту проведения автоматически при нажатии на кнопку «Заполнить» ;
  • «Владелец» - из справочника «Проживающие» устанавливается автоматически, если заполнена колонка «Лицевой счет»;
  • «Квартира» - из справочника «Помещения» устанавливается автоматически, если заполнена колонка «Лицевой счет»;
  • «Площадь» - общая площадь квартиры, заполняется автоматически при нажатии на кнопку «Заполнить».

В поле «Описание реестра» – указывается дополнительная информация к документу.

В поле «Комментарий» – указывается комментарий к документу.

Документ не формирует движения.

Внизу формы документа расположены кнопки:

  • «ОК» – записать, провести документ и закрыть форму документа;
  • «Записать» – записать документ;
  • «Закрыть» – закрыть форму документа.

Документ позволяет формировать печатную форму «Реестр участников голосования».

Документ можно открыть через меню «Учет в ЖКХ / Документы / Голосование / Реестр участников голосования».
