Документ «Назначение услуг»

Документ используется для назначения услуги для указанных лицевых счетов.

При вводе документа необходимо указать:

  • «Номер» – номер документа устанавливается автоматически;
  • «Дата» – дата документа устанавливается автоматически;
  • «Организация» – ТСЖ выбирается из предложенного списка;
  • «Услуга» – из справочника «Услуги» выбирается назначаемая услуга.

Табличная часть документа состоит из следующих колонок:

  • «N» – порядковый номер строки;
  • «Объект» – из справочника «Лицевые счета» выбирается лицевой счет, для которого назначается услуга;
  • «Квартира» – указывается квартира, соответствующая данному лицевому счету;
  • «Контрагент» – указывается квартиросъемщик по данному лицевому счету;
  • «Установить с» – указывается дата начала предоставления услуги;
  • «Установить по» – указывается дата окончания предоставления услуги.

При нажатии кнопки «Заполнить» будет вызвана обработка «Заполнение получателями услуг», используемая для быстрого заполнения табличной части.

При проведении документ для каждой строки в табличной части делает две записи в регистр сведений «Назначенные начисления». В поле «Дата изменения» первой записи указывается дата начала предоставления услуги (в регистре сведений ресурс «Действует» будет установлен в значение «Истина»), во второй записи в поле «Дата изменения» указывается дата окончания предоставления услуги (в регистре сведений ресурс «Действует» будет установлен в значение «Ложь»).


Внизу формы расположены кнопки:

  • «ОК» – провести документ и закрыть форму;
  • «Записать» – записать документ;
  • «Закрыть» – закрыть форму документа.

Документ можно открыть через меню «Учет в ЖКХ / Документы / Начисление / Оплата / Назначение услуг».

Техническая информация

  • Подробнее о правилах формирования периодов действия услуг можно ознакомиться в справке регистра «Назначенные начисления».
