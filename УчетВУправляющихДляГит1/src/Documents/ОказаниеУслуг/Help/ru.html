Документ "Оказание услуг"

Продажа ® Оказание услуг

Документ предназначен для отражения в учете операции по оказанию услуг нескольким контрагентам.

При вводе документа в шапке нужно указать следующие реквизиты:

  • Вид расчетов - вид оказываемых услуг. Выбирается из справочника Виды расчетов. Вид расчетов предназначен для ведения раздельного учета расчетов с контрагентами по различным видам услуг.
  • Услуга - оказываемая услуга. Выбирается из справочника Номенклатура. По данной номенклатурной позиции отражается выручка от оказания услуг.
  • Учитывать НДС, Сумма включает НДС, Ставка НДС - параметры налогообложения НДС.

Для отражения операции нужно заполнить закладки:

  • Контрагенты
  • Счета-фактуры
  • Счета учета

Контрагенты

В табличной части Контрагенты указываются:

  • Контрагент - контрагент, которому были оказаны услуги.
    • Список контрагентов можно заполнить автоматически по виду расчетов, указанному в шапке документа. По кнопке Заполнить - По виду расчетов в табличную часть включаются контрагенты, у которых есть договоры с указанным видом расчетов и сроком действия, истекающим не ранее месяца, в котором производится начисление.
  • Договор - договор, по которому выполняется начисление за оказанные услуги. Договор выбирается из справочника Договоры с отбором по виду расчетов, установленному в шапке документа.
  • Цена, Сумма - заполняются автоматически, если в договоре установлен тип цен. Цена заполняется на основании регистра Цены номенклатуры.
  • Счет расчетов, Счет авансов - заполняются автоматически при указании контрагента и договора на основании регистра Счета учета расчетов с контрагентами.

Счета-фактуры

На закладке Счета-фактуры для каждой строки документа выписывается отдельный счет-фактура. Для этого должен быть установлен флажок в графе Выписан счет-фактура. Счета-фактуры создаются автоматически при проведении документа и фиксируются в графе Счет-фактура.

Счета учета

На закладке Счета учета указываются счета и аналитика для формирования бухгалтерских проводок.

  • Счет доходов, Счет расходов и Счет учета НДС по реализации - заполняются автоматически при указании номенклатуры на основании регистра Счета учета номенклатуры.
    • Счет доходов и Счет расходов определяют, относится ли данная строка реализации к деятельности с общим порядком налогообложения или к отдельным видам деятельности с особым порядком налогообложения. Недопустимо указание для доходов и расходов счетов, относящихся к разным видам деятельности.
  • В поле Субконто подставляется значение номенклатурной группы выбранной услуги.

Печатные формы

Для документа Оказание услуг предусмотрены следующие печатные формы:

  • Акты об оказании услуг
  • Счета-фактуры
