Документ «Заявка на выполнение работ»

Документ используется для регистрации заявок и жалоб от жильцов, для планирования и отражения факта выполнения работ по ремонту жилого фонда.

При вводе документа необходимо указать:

  • «Номер» – номер документа устанавливается автоматически;
  • «Дата» – дата документа устанавливается автоматически;
  • «Организация» – из справочника «Организации» выбирается организация;
  • «Вид операции» – вид операции документа. Могут быть выбраны значения "Выполнение работ по лицевому счету", "Выполнение работ по дому";
  • «Приоритет» – приоритет выполнения работ (высокий, средний, низкий);
  • «Категория работ» – категория выполняемых работ. Могут быть выбраны значения "Плановый ремонт", "Текущий ремонт", "Капитальный ремонт", "Аварийный ремонт";

    Нижняя часть формы состоит из трех закладок «Основные сведения», «Наряд заказ» и «Выполнение».

    • На закладке «Основные сведения» расположены поля:
      • «Здание» – из справочника «Здания, сооружения» выбирается здание;
      • «Подъезд» – из справочника «Подъезды» выбирается подъезд. При заполненном здании подъезд можно выбрать только из числа подъездов этого здания;  
      • «Помещение» – из справочника «Помещения» выбирается помещение (доступно для вида операции "Выполнение работ по лицевому счету"). При заполненном здании помещение выбирается из числа помещений этого здания, при заполненном подъезде - из числа помещений подъезда;
      • «Лицевой счет» – из справочника «Лицевые счета» выбирается лицевой счет (доступно для вида операции "Выполнение работ по лицевому счету"). Заполняется автоматически, если заполнено помещение, а в помещении открыт только один лицевой счет;
      • «Код домофона» – код домофона подъезда, заполняется автоматически при выборе подъезда;
      • «Адрес» – адрес лицевого счета, заполняется автоматически при выборе лицевого счета;
      • «Контактное лицо» – заполняется автоматически при выборе лицевого счета, определяется по ответственному собственнику / нанимателю лицевого счета.
        • Если ответственный собственник / наниматель лицевого счета является жильцом, то заполняется данным жильцом.
        • Если ответственный собственник / наниматель лицевого счета является контрагентом, заполняется основным контактным лицом данного контрагента. Если основное контактное лицо контрагента не определено, то заполняется первым из списка контактных лиц данного контрагента.
      • «Телефон» – телефон, заполняется автоматически при выборе лицевого счета. Может быть отредактирован вручную. Принцип заполнения определяется по реквизиту "Контактное лицо": 
        • Если "Контакное лицо" является жильцом, заполняется номером телефона связанного с жильцом физического лица. Если номеров телефона у физического лица несколько, становится доступен список выбора номера телефона.
        • Если "Контактное лицо" является контактным лицом контрагента, заполняется номером телефона контактного лица контрагента. Если номеров телефона у контактного лица несколько, становится доступен список выбора номера телефона. 
      • «Договор контрагента» - заполняется автоматически договором, участвующим во взаиморасчетах по выбранному лицевому счету.
      • «Дата и время, удобные для заказчика» – указывается дата и время, на которое запланировано выполнение заявки;
      • «Состав звена»:
        • «Ответственный»  – указывается мастер, ответственный за выполнение работ по текущей заявке;
        • В табличной части указывается список мастеров, выполняющих работы по текущей заявке;
      • «Текст заявки» – вводится текст жалобы от клиента или комментарий к выполнению работ.
      • «Это повторное обращение» - флаг устанавливается в случае, если текущая заявка является повторным обращением. При установке флага из списка введенных ранее документов «Заявка на выполнение работ» выбирается первичное обращение.
    • На закладке «Наряд заказ» расположено поле «Признак неисправности» и 2 табличные части:
      • «Признак неисправности» - указывается признак неисправности из справочника «Признаки неисправностей». При выборе признака неисправности осуществляется автоматическое заполнение табличных частей «Работы» и «Наряд-заказ».
      • Табличная часть «Работы» состоит из колонок:
        • «№» – порядковый номер строки;
        • «Вид работ» – указывается вид работ из справочника «Виды работ» ;
        • «Норма труда» – норма труда мастера, заполняется автоматически при выборе вида работ;
        • «Объем (план)» – указывается планируемое количество работ по заявке;
        • «Объем (факт)» – указывается количество выполненных работ по заявке (заполняется по факту);
        • «Время (план)» – отображается планируемое на выполнение работы время. Рассчитывается автоматически исходя из нормы труда и планируемого объема работ. Может быть изменено вручную;
        • «Время (факт)» – отображается фактическое время, затраченное на выполнение работы. Рассчитывается автоматически исходя из нормы труда и фактического объема работ. Может быть изменено вручную;

      • При нажатии кнопки «Заполнить факт по плану» автоматически во всех строках значение колонки «Объем (факт)» будет установлено равным значению колонки «Объем (план)».

      • Табличная часть «Наряд-заказ» состоит из колонок:

        • «№» – порядковый номер строки;
        • «Вид работ» – из справочника «Виды работ» выбирается вид работ, по которой будет списан материал;
        • «Номенклатура» – выбирается материал из справочника «Номенклатура»;
        • «Количество (план)» – указывается количество материала, которое планируется потратить для выполнения данного вида работ;
        • «Количество (факт)» – указывается количество материала, которое потрачено при выполнении данного вида работ;
        • «Ед. измерения» – указывается ед.измерения из справочника «Классификатор единиц измерения»;
        • «Цена» – указывается цена номенклатуры;
        • «Ставка НДС» – указывается ставка НДС;
        • «Сумма НДС» – отображается сумма НДС;
        • «Сумма (план)» – отображается планируемая сумма материалов, которая необходима для выполнения данного вида работ;
        • «Сумма (факт)» – отображается фактическая сумма материалов, которая была потрачена при выполнении данного вида работ.

          Кнопка «Заполнить» предназначена для автоматического заполнения табличной части «Наряд заказ» по выбранным видам работ из табличной части «Работы».

          При нажатой кнопке «Отбор по виду работ» в табличной части «Наряд-заказ» будут отображаться материалы для выделенного(активного) в табличной части «Работы» вида работ.

          При отжатой кнопке «Отбор по виду работ» в табличной части «Наряд-заказ» будут отображаться все материалы по заявке.

    • На закладке «Выполнение» поля заполняются в следующем порядке:
      • «Выполнено» – устанавливается признак выполнения заявки;
      • «Время фактического выполнения» – указывается время, в которое были выполнены работы по заявке;
      • «Оценка выполнения» – выбирается оценка выполнения мастером работ;
      • «Дата и время начала работ» - указывается дата и время фактического начала работ по заявке;
      • «Дата и время окончания работ» - указывается дата и время фактического окончания работ по заявке. В случае, если заполена дата и время начала работ, оцениваются фактические трудозатраты по заявке.
      • «Замечания по выполнению» – указываются замечания по выполнению заявки, если необходимо;
      • «Требование-накладная» – ссылка для автоматического ввода документа "требование-накладная" для списания по бух.учету материалов, фактически потраченных при выполнении работ.


Внизу формы документа расположены кнопки:

  • «ОК» – провести документ и закрыть форму;
  • «Записать» – записать документ;
  • «Закрыть» – закрыть форму документа.

Документ можно открыть через меню «Учет в ЖКХ /Аварийно-диспетчерская служба», а также отражается в журнале «Заявки на выполнение работ».
