Документ "Формирование записей раздела 7 декларации по НДС"

Документ предназначен для отражения операций в разделе 7 декларации по НДС.

Документ заполняется автоматически по необлагаемым НДС операциям, доступным к отражению в разделе 7 декларации по НДС на дату ввода документа. Это операции, отраженные в учете с указанным кодом операции для раздела 7 декларации по НДС.

Документ можно заполнить вручную, при этом дополнительно требуется распределить включенный в стоимость НДС по указанным в документе операциям, нажав на кнопку "Распределить НДС".
