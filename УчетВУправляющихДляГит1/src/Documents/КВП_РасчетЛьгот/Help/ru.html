Документ «Расчет льгот»

Документ предназначен для ежемесячного расчета льгот по лицевым счетам.

В зависимости от настройки «Использовать новый алгоритм расчета льгот», которая задается на форме «Настройки параметров учета (ЖКХ)», документ оформляется следующим образом:

 

1. Если установлен флаг «Используется новый алгоритм расчета льгот», то:

 

При вводе документа необходимо указать:

  • «Номер» – номер документа, устанавливается автоматически;
  • «Дата» – дата документа, устанавливается автоматически;
  • «Организация» – из справочника «Организации» выбирается организация, от имени которой необходимо произвести расчет льгот;
  • «Услуга» – из справочника «Услуги» выбирается услуга, по которой необходимо произвести расчет льгот. Допускается не заполнять данное поле;
  • «Объект» – указывается объект, по которому необходимо произвести расчет льгот. В качестве объекта можно указать:
    • Здание;
    • Подъезд;
    • Помещение;
    • Лицевой счет;
  • «Период» – месяц, за который необходимо произвести расчет льгот.

 

Для удобства заполнения таблицы расчета льгот предусмотрена кнопка «Заполнить и рассчитать». При нажатии на данную кнопку производится автоматическое получение сведений о начислениях и расчет на их основании льгот за выбранный месяц.

Табличная часть состоит из следующих колонок:

  • «Лицевой счет» – лицевой счет (далее – Текущий лицевой счет), по которому проживает льготник (далее – Текущий льготник), для которого произведен расчет льгот;
  • «Услуга» – услуга (далее – Текущая услуга), по которой произведен расчет льгот для Текущего льготника;
  • «Льготник» – Текущий льготник;
  • «Период» – период (далее – Текущий период), за который произведен расчет льгот;
  • «Льготная категория» – льготная категория, по которой произведен расчет льгот для Текущего льготника по Текущей услуге в Текущем периоде;
  • «Сумма услуги» – итоговая сумма начислений, произведенных в Текущем периоде на Текущий лицевой счет по Текущей услуге (с учетом настройки «Учитывать корректировки при расчете льгот», которая задается в «Настройках параметров учета ЖКХ»;
  • «Тариф» – размер тарифа, с учетом которого произведено начисление на Текущий лицевой счет по Текущей услуге в Текущем периоде;
  • «Вид тарифа» – вид тарифа, по которому произведено начисление на Текущий лицевой счет по Текущей услуге в Текущем периоде (отображается только при включенной настройке «Использовать функционал выгрузки в ГЦЖС», которая задается в «Функциональности программы ЖКХ»);
  • «Количество» – объем потребления, с учетом которого произведено начисление на Текущий лицевой счет по Текущей услуге в Текущем периоде;
  • «Льготный тариф» – размер льготного тарифа, с учетом которого произведен расчет льгот для Текущего льготника по Текущей услуге в Текущем периоде;
  • «Вид тарифа льготы» – вид тарифа, по которому произведен расчет льгот для Текущего льготника по Текущей услуге в Текущем периоде  (отображается только при включенной настройке «Использовать функционал выгрузки в ГЦЖС», которая задается в «Функциональности программы ЖКХ»);
  • «Льготное количество» – объем рассчитанной льготы для Текущего льготника по Текущей услуге в Текущем периоде;
  • «Льготная площадь» – размер площади, с учетом которого произведен расчет льгот по Текущей услуге для Текущего льготника в Текущем месяце;
  • «Начислено» – сумма льгот, рассчитанная по Текущей услуге для Текущего льготника в Текущем периоде.

 

Рекомендуется использовать данный вариант настройки «Использовать новый алгоритм расчета льгот», так как он предусматривает более гибкий и удобный механизм для расчета льгот!

Примечание
При расчете льготы с ограничением по нормативному объему по услугам, начисляемым по многотарифным приборам учета, ограничение по объему таких услуг выполняется в порядке уменьшения тарифа. Т.е. сначала в «льготном» объеме учитывается потребление по самому дорогому тарифу, если при этом норматив не был полностью использован, зачитывается потребление по менее дорогому тарифу и т.д.

Пример:
Трехтарифный прибор учета. Льгота 50% с ограничением по нормативу 70*2=140 кВт.ч;
За месяц израсходовано: по Дневному тарифу (4,92 руб.) 100 кВт.ч, по ночному (1,26 руб.) 50 кВт.ч, по пиковому (4,08 руб.) 70 кВт.ч.
Расчет льготы будет выполнен следующим образом:
По дневному тарифу минимум(100, 140) * 4,92 * 50% = 100 * 4,92 * 50% = 246 руб.
По пиковому тарифу минимум(70, 140-100) * 4,08 * 50% = 40 * 4,08 * 50% = 81,60 руб.
Итого размер льготы 327,60 руб.

2. Если не установлен флаг «Используется новый алгоритм расчета льгот», то:

 

При вводе документа необходимо указать:

  • «Номер» – номер документа, устанавливается автоматически;
  • «Дата» – дата документа, устанавливается автоматически;
  • «Организация» – из справочника «Организации» выбирается организация, от имени которой необходимо произвести расчет льгот;
  • «Вид операции» – вид операции расчета льгот;
  • «Услуга» – из справочника «Услуги» выбирается услуга, по которой необходимо произвести расчет льгот. Допускается не заполнять данное поле;
  • «Объект» – указывается объект, по которому необходимо произвести расчет льгот. В качестве объекта можно указать:
    • Здание;
    • Подъезд;
    • Помещение;
    • Лицевой счет;
  • «Начислять за» – количество дней, за которые необходимо произвести расчет льгот;

 

Для удобства заполнения таблицы расчета льгот предусмотрена кнопка «Заполнить». При нажатии на данную кнопку производится автоматическое получение сведений о начислениях за выбранный месяц. Для удобства расчета льгот предусмотрена кнопка «Рассчитать». При нажатии на данную кнопку производится расчет льгот за выбранный месяц.

Табличная часть состоит из следующих колонок:

  • «Лицевой счет» – лицевой счет (далее – Текущий лицевой счет), по которому произведен расчет льгот;
  • «Квартира» – помещение, в котором открыт Текущий лицевой счет;
  • «Владелец» – ответственный собственник (наниматель) Текущего лицевого счета;
  • «Услуга» – услуга (далее – Текущая услуга), по которой произведен расчет льгот для Текущего лицевого счета;
  • «Начислено услуг» – итоговая сумма начислений, произведенных на Текущий лицевой счет по Текущей услуге (с учетом настройки «Учитывать корректировки при расчете льгот», которая задается в «Настройках параметров учета ЖКХ»;
  • «Сумма льгот» – сумма льгот, рассчитанная по Текущей услуге для Текущего лицевого счета.

 

Документ открывается через меню «Льготы и скидки/ Расчет льгот».
