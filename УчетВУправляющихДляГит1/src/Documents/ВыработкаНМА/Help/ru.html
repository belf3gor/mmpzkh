Документ "Выработка НМА"

НМА ® Параметры амортизации ® Выработка НМА

Документ предназначен для регистрации объема выработанной продукции (выполненных работ) для начисления амортизации нематериальных активов.

В документе указывается объем выработанной продукции (выполненных работ) по каждому нематериальному активу.

  • Нематериальный актив - нематериальный актив.
  • Объем работ - объем выработанной продукции (выполненных работ) по нематериальному активу.
