Документ «Разовое начисление услуг»

Документ предназначен для начисления разовых услуг по лицевому счету.

При вводе нового документа необходимо указать:

  • «Дата» – дата документа;
  • «Номер» – номер документа;
  • «Организация» – из справочника «Организации» выбирается организация, производящая начисления по услугам;
  • «Вид операции» – выбирается вид операции из возможных значений:
    • "Прочие начисления" (для новых документов значение устанавливается по умолчанию), при выборе которого данные о количестве расхода по услуге вводятся вручную пользователем;
    • "Тех.обслуживание ПУ", при выборе которого количество расхода по услуге рассчитывается автоматически и равняется количеству счетчиков, установленных на момент начисления;
    • "Установка ПУ", при выборе которого количество расхода по услуге рассчитывается автоматически и равняется количеству счетчиков, которые были установлены в месяце начисления.
  • «Объект» – указывается объект, по соответствующим лицевым счетам которого производится начисление платежей за услуги. Данное поле доступно для вида операций "Тех.обслуживание ПУ" и "Установка ПУ". В качестве объекта можно указать:
    • Здание;
    • Подъезд;
    • Помещение;
    • Лицевой счет.
  • «Услуга для начисления» – выбирается услуга из справочника «Услуги». При этом выбирается услуга со способом расчета "Фиксированное начисление", по которой необходимо произвести начисления. Данное поле доступно для вида операций "Тех.обслуживание ПУ" и "Установка ПУ".
  • «Услуга для анализа приборов учета» – выбирается услуга из справочника «Услуги». При этом выбирается услуга со способами расчета: «По показаниям счетчика», «По показаниям счетчика и норме», по которым рассчитывается количество предоставленной услуги, выбранной в поле "Услуга для начисления".

Нижняя часть формы представляет собой табличную часть «Услуги»

  • «№» – порядковый номер строки в документе;
  • «Лицевой счет» – из справочника «Лицевой счет» выбирается лицевой счет;
  • «Договор» – из справочника «Договоры» выбирается договор лица (контрагента), ответственного за взаиморасчеты по лицевому счету.
  • «Услуга» – из справочника «Услуги» выбирается предоставленная услуга;
  • «Количество» – количество предоставленной услуги;
  • «Цена» – цена предоставленной услуги;
  • «Сумма» – сумма начисления по услуге.


Внизу формы документа расположены кнопки:

  • «ОК» – записать и провести документ;
  • «Записать» – записать документ;
  • «Закрыть» – закрыть форму документа.

Документ можно открыть через меню «Начисление / Разовое начисление услуг».
