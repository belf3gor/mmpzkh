Документ «Открытие лицевого счета»

Данный документ предназначен для открытия нового лицевого счета на помещение (при этом на одно помещение может быть открыто несколько лицевых счетов).

Примечание:
Документ формируется автоматически. При попытке создания нового документа вызывается обработка «Помощник открытия лицевого счета».
Обработка «Помощник открытия лицевого счета» также может быть вызвана нажатием кнопки «Открыть л/с» в справочниках «Здания», «Помещения» или при попытке создания лицевого счета в справочнике «Лицевые счета».

В документе указываются сведения о собственниках и жильцах помещения.

При вводе документа необходимо указать:

  • «Номер» – номер документа; устанавливается автоматически;
  • «Дата» – дата документа; по умолчанию заполняется рабочей датой;
  • «Помещение» – помещение (квартира, офис и др.), на которое открывается лицевой счет; если выбрано нежилое помещение, то регистрация проживающих в табличной части будет невозможна;
  • «Лицевой счет» – открываемый лицевой счет; ссылка на справочник «Лицевые счета»;
  • «Контрагент» – из справочника «Контрагенты» выбирается контрагент, который будет использован для отражения начислений в регламентированном учете;
  • «Договор контрагента» – из справочника «Договоры контрагентов» выбирается договор контрагента, который будет использован для отражения начислений в регламентированном учете.

Также необходимо указать, кто будет выступать в качестве ответственного собственника, нанимателя:

  • Контрагент,
  • Один из жильцов.

Для этого предназначен одноименный переключатель, расположенный в шапке документа. При выборе значения «Контрагент», в качестве ответственного собственника, нанимателя будет установлен контрагент, выбранный в поле «Контрагент». При выборе значения «Один из жильцов» в качестве ответственного собственника, нанимателя будет установлен один из жильцов, указанных в табличной части документа. Установить ответственным собственником, нанимателем одного из жильцов можно с помощью кнопки «Ответственный», расположенной на командной панели табличной части. При этом, у выбранного ответственного жильца изменится степень родства на «Ответственный собственник, наниматель» и он будет выделяться жирным шрифтом.

На форме данного документа расположена табличная часть, содержащая сведения о жильцах, относящихся к лицевому счету. Табличная часть состоит из следующих колонок:

  • «N» – порядковый номер строки;
  • «Проживающий» – из справочника «Жильцы» выбирается жилец-собственник, проживающий и/или зарегистрированный в помещении, на которое открыт лицевой счет;
  • «Степень родства» – из справочника «Типы родственных связей» выбирается тип родственной связи по отношению к ответственному собственнику, нанимателю жилого помещения;
  • «Проживает» – признак, показывающий, что человек проживает в помещении, на которое открывается лицевой счет;
  • «Дата заселения» – дата заселения жильца;
  • «Зарегистрирован» – признак, показывающий, что человек зарегистрирован в помещении, на которое открывается лицевой счет;
  • «Дата регистрации» – дата регистрации жильца;

Примечание:
Если ведется учет готовыми числами (в «настройках параметров учета (ЖКХ)» на закладке «Учет жильцов» установлен флажок «Учитывать количество проживающих и зарегистрированных готовыми числами»), то для ввода информации о количестве жильцов на лицевом счете в шапке документа отображаются поля ввода количества проживающих и зарегистрированных:

   - «Проживает» – количество проживающих на лицевом счете;
   - «Зарегистрировано» – количество зарегистрированных на лицевом счете.

Если ведется полноценный учет жильцов (в «настройках параметров учета (ЖКХ)» на закладке «Учет жильцов» снят флажок «Учитывать количество проживающих и зарегистрированных готовыми числами»), то количество проживающих и зарегистрированных жильцов на лицевом счете определяется по данным табличной части документа (колонки «Проживает» и «Зарегистрирован»).

  • «Собственник» – признак, показывающий, что человек является собственником помещения, на которое открывается лицевой счет;
  • «Доля собственности» – в случае долевой собственности указывается доля собственности проживающего на помещение;

Примечание:
В программе имеется возможность использовать 2 варианта установки собственников помещений:

  - Упрощенный вариант (в «настройках параметров учета (ЖКХ)» на закладке «Учет собственников» установлен флажок «Использовать упрощенный вариант учета собственников помещений»):

При данном варианте учета установка собственников помещения выполняется  документом «Открытие лицевого счета». В качестве собственников могут быть установлены жильцы, указанные в табличной части. Для этого необходимо устанавливать флажок в колонке «Собственник». При этом, установить собственником контрагента, выбранного для отражения начислений в регламентированном учете, нельзя.

  - Не упрощенный вариант (в «настройках параметров учета (ЖКХ)» на закладке «Учет собственников» снят флажок «Использовать упрощенный вариант учета собственников помещений»):

При данном варианте учета установка собственников помещений производится с помощью документа «Установка собственников помещений». При этом в документе «Открытие лицевого счета» данные о собственности отображаются лишь справочно и не могут быть изменены.

  • «Владелец (наниматель) единственного жилья» – флажок устанавливается, если человек является владельцем единственного жилья. Данный признак доступен, когда в «Настройках программы (ЖКХ)» на закладке "Функциональность программы" установлена настройка «Вести учет владельцев единственного жилья»;
  • «Дополнительная площадь» – указывается количество дополнительной площади;

    Следующие поля табличной части выводятся информативно:
  • «Вид документа» – вид документа, удостоверяющего личность жильца; 
  • «Серия документа» – серия документа;
  • «Номер документа» – номер документа;
  • «Дата выдачи документа» – дата выдачи документа;
  • «Кем выдан документ» – организация, выдавшая документ;
  • «Код подразделения» – код подразделения, выдавшего документ. 

При необходимости можно заполнить поле «Комментарий», которое будет отображаться в списке документов.

Внизу формы расположены кнопки:

  • «Печать» – позволяет вывести печатные формы документа;
  • «ОК» – провести документ и закрыть форму;
  • «Записать» – записать документ;
  • «Закрыть» – закрыть форму документа.


Документ можно открыть через пункт главного меню «Учет в ЖКХ / Лицевые счета / Открытие лицевого счета».
