Справку можно использовать как доказательство проявления должной осмотрительности при работе с контрагентом, а при необходимости – в суде. Формат справки адаптирован под потребности финансовых служб и включает в себя всю необходимую информацию. Справка подписана электронной подписью Интерфакс.

  • Контрагент - выбор контрагента для отбора справок в списке и для запроса новой справки;
  • Запросить справку - запросить новую справку в сервисе 1СПАРК Риски. Новая справка будет добавлена в список с состоянием "Готовится". Необходимо дождаться смены состояния справки на "Получена", после справка станет доступной для просмотра. Для просмотра справки необходимо выполнить двойной щелчок мышью по строке справки;
  • Гиперссылка "Все справки в личном кабинете на Портале 1С:ИТС" - перейти на Портал 1С:ИТС для просмотра всех справок, полученных с текущим логином пользователя Интернет-поддержки;

 

Список справок

В списке отображаются все справки по контрагентам, которые были запрошены из программы. Для просмотра справки необходимо выполнить двойной щелчок мышью по строке справки.

  • Дата - дата запроса справки;
  • Контрагент - контрагент, для которого была запрошена справка;
  • ИНН - ИНН, который был указан для контрагента на момент запроса справки;
  • Состояние - состояние получения справки:
    • Готовится - в настоящий момент выполняется подготовка справки в сервисе 1СПАРК Риски;
    • Получена - справка успешно получена, для просмотра справки дважды щелкните левой клавишей мыши по строке справки;
    • Неизвестный ИНН - не удалось подготовить справку, т.к. в сервисе 1СПАРК Риски нет сведений о контрагенте с указанным ИНН;
    • Не удалось подготовить справку - не удалось подготовить справку в связи с временной недоступностью сервиса 1СПАРК Риски;
    • <Не удалось проверить состояние справки> - состояние справки не удалось проверить из-за возникновения нештатной ситуации: возникла ошибка при подключении к сервису 1СПАРК Риски или функциональность 1СПАКР Риски была отключена в настройках программы.
      Необходимо проверить готовность справки самостоятельно, выполнив двойной щелчок по строке справки;
  • Пользователь - пользователь, который запросил справку;
