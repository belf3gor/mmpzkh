Документ "Реализация услуг по переработке"

Документ предназначен для отражения операций по переработке давальческого сырья. Документ можно ввести на основании документов Отчет производства за смену, Требование-накладная.

Оказание услуг по переработке

Для отражения оказания услуг по переработке материалов заказчика нужно заполнить закладку Продукция и услуги. На этой закладке указывается:

  • Продукция - наименование выпущенной из сырья заказчика продукции (оказание услуг по переработке такого сырья признается этим документом).
  • Сумма (плановая) - плановая себестоимость выпущенной из сырья заказчика продукции.
  • Спецификация - список нормативов расходов, необходимых для выпуска продукции. Значение поля Спецификация будет использоваться при заполнении закладки Материалы заказчика.

Данные закладки Зачет авансов нужно заполнить, если в шапке документа выбран способ зачета авансов По документу.

Отчет об использованных материалах заказчика

Для отражения отчета перед заказчиком об использованных в переработке материалах нужно заполнить закладку Материалы заказчика. На этой закладке указывается:

  • Счет учета - счет учета материалов заказчика, списанных в производство. В поле Счет передачи необходимо указать счет 003.02 "Материалы, переданные в производство" или его субсчет.

Примечание
Для налогового учета (по налогу на прибыль) операция списания материалов заказчика в производство не отражается, т.к. она не влияет на признание доходов и расходов.

Печатные формы

Для объекта Реализация услуг по переработке предусмотрены следующие печатные формы:

  • Акт об оказании услуг
  • Отчет о продукции
  • Отчет о материалах
  • Справка-расчет "Рублевая сумма документа в валюте"

Ввод на основании

На основании документа Реализация услуг по переработке можно ввести следующие объекты:

  • Отчет производства за смену
  • Требование-накладная
  • Счет-фактура выданный
  • Отражение начисления НДС
