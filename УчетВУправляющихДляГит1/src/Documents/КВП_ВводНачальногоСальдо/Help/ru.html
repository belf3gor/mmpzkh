Документ «Ввод начального сальдо»

Документ предназначен для ввода начальных данных о начислениях по выбранным лицевым счетам.

При вводе нового документа заполняются реквизиты:

  • «Номер» – номер документа устанавливается автоматически;
  • «Дата» – дата документа устанавливается автоматически;
  • «Организация» – из справочника «Организации» выбирается организация, по которой необходимо ввести начальное сальдо за услуги;
  • «Объект» – указывается объект, по которому производится ввод начального сальдо за услуги. В качестве объекта можно указать:
    • Здание;
    • Подъезд;
    • Помещение;
    • Лицевой счет.

Ниже расположена табличная часть, состоящая из колонок:

  • «N» – порядковый номер строки табличной части устанавливается автоматически;
  • «Объект» – из справочника «Лицевые счета» выбирается лицевой счет, на который будет начислена сумма к оплате за предоставленную услугу.
  • «Услуга» – из справочника «Услуги» выбирается предоставленная услуга;
  • «Раздел учета» – раздел учета, по которому будет введено начальное сальдо за услугу;
  • «Месяц начисления» – месяц начисления, на который будет введено начальное сальдо за услугу;
  • «Сумма» – указывается начисляемая сумма.


Вверху формы документа расположены кнопки:

  • «Провести и закрыть» – записать, провести документ и закрыть форму документа;
  • «Записать» – записать документ;
  • «Провести» – проводит документ.
