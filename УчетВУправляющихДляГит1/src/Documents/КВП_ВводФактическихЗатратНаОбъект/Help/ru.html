Документ «Ввод фактических затрат на объект»

Документ предназначен для регистрации сумм фактических затрат на оплату услуг организациям, предоставляющим услуги указанному объекту.

При вводе нового документа необходимо указать следующие параметры:

  • «Номер» – уникальный номер документа (устанавливается автоматически);
  • «Дата» – дата документа. Указывается дата, на которую вводятся данные сумм фактических затрат;
  • «Организация» – ссылка на справочник «Организации». Выбирается организация, по которой будут производиться начисления сумм фактических затрат;
  • «Вариант распределения» – вариант распределения затрат на лицевые счета, выбранные в таблице распределения. Принимает одно из следующих значений:
    • «По объему»,
    • «По сумме»,
    • «По объему и сумме».

Документ состоит из двух табличных частей:

  • Объекты учета – в таблице указываются объекты (здания), по которым необходимо распределить сумму фактических затрат (и количество) на лицевые счета.
  • Таблица распределения – в таблице отображаются распределенные на лицевые счета суммы (и количество).

Таблица объектов учета состоит из следующих колонок:

  • «Здание» – ссылка на справочник «Здания». Выбирается здание, по которому необходимо распределить количество и сумму предоставленной услуги.
  • «Услуга» – ссылка на справочник «Услуги». Из справочника выбирается предоставленная услуга, по которой необходимо распределить количество и сумму.
  • «Способ распределения» – способ распределения затрат на лицевые счета, выбранные в таблице распределения. Принимает одно из следующих стандартных значений:
    • «поровну»,
    • «пропорционально площади»,
    • «пропорционально количеству жильцов»,
    • «пропорционально объему потребления услуги»,
    • «пропорционально количеству комнат».

Также к перечисленным способам добавляется дополнительные числовые свойства справочников «Помещения» и «Лицевые счета».

  • «Доп. параметр распределения» - дополнительный параметр для распределения затрат на лицевые счета, выбранные в таблице распределения. Доступен для выбора только для способов распределения «пропорционально площади» и «пропорционально количеству жильцов».
  • «Сумма» – сумма фактических затрат на услугу по зданию. Доступна для ввода при выбранном варианте распределения «По сумме» и «По объему и сумме».
  • «Количество» – количество предоставленной услуги по зданию. Доступно для ввода при выбранном варианте распределения «По объему» и «По объему и сумме».

Для распределения фактических затрат в строке таблицы необходимо указать количество и/или сумму (в зависимости от выбранного варианта распределения). В противном случае для выделенной строки распределение производиться не будет.

Таблица распределения состоит из следующих колонок:

  • «Лицевой счет» – ссылка на справочник «Лицевые счета». Выбирается лицевой счет, на который будет распределяться сумма фактических затрат;
  • «Помещение» – отображается помещение, на которое открыт лицевой счет;
  • «Владелец» – отображается ответственный собственник, наниматель лицевого счета;
  • «Количество распределения» – количество предоставленной услуги по лицевому счету;
  • «Тариф» – тариф распределяемой услуги;
  • «Сумма» – сумма фактических затрат на лицевой счет.

Таблицу распределений документа можно заполнить вручную или автоматически. Для автоматического заполнения таблицы распределения документа необходимо воспользоваться кнопкой «Заполнить», которая расположена над таблицей распределения документа. Имеется два варианта автоматического заполнения таблицы распределения:

  • «Заполнить автоматически» - при выборе данного варианта заполнения будет произведено автоматическое заполнение таблицы распределения документа всеми лицевыми счетами выбранного здания, открытыми по состоянию на дату документа;
  • «Заполнить с отбором» - при выборе данного варианта заполнения будет выведено диалоговое окно обработки «Заполнение получателями услуг», где можно отобрать нужные лицевые счета, например, по подъезду или по выбранным помещениям.

Распределение затрат производится при нажатии кнопки «Рассчитать» следующим образом:

  • Если выбран вариант распределения «По объему и сумме», при этом в таблице объектов учета заполнено количество и сумма фактических затрат на здание, то при распределении по лицевым счетам:
    • количество распределения по лицевому счету рассчитывается по формуле K * D / P, где:
      • K - общее количество фактических затрат по зданию;
      • D - количество услуги по лицевому счету, рассчитанное с учетом выбранного способа распределения;
      • P - суммарное количество услуги по всем лицевым счетам, выбранным в таблице распределения.

        Например, если выбран способ распределения услуги «пропорционально площади» и дополнительный параметр распределения «общая площадь», то для лицевого счета количество услуги D будет равно размеру общей площади помещения, приходящейся на лицевой счет, а суммарное количество услуги по всему лицевым счетам P будет равно сумме размеров общих площадей всех лицевых счетов в таблице распределения.
    • сумма распределения по лицевому счету рассчитывается по формуле S * D / P, где:
      • S - общая сумма фактических затрат по зданию;
      • D - количество услуги по лицевому счету, рассчитанное с учетом выбранного способа распределения;
      • P - суммарное количество услуги по всем лицевым счетам, выбранным в таблице распределения.

        Например, если выбран способ распределения услуги «поровну», то для лицевые счета количество услуги D будет равно «1», а суммарное количество услуги по всем лицевым счетам P будет равно количеству различных лицевых счетов в таблице распределения.
    Тариф по лицевому счету рассчитается исходя из распределенных суммы и количества.
  • Если выбран вариант распределения «По сумме», при этом в таблице объектов учета заполнена сумма фактических затрат, то при распределении затрат по лицевым счетам будет рассчитываться только сумма распределения (формула описана выше). Количество потребления по лицевому счету устанавливается в соответствии с выбранным способом распределения. Тариф рассчитается исходя из распределенной суммы затрат по лицевому счету и количества.
  • Если выбран вариант распределения «По объему», при этом в таблице объектов учета заполнено количество фактических затрат, то при распределении затрат по лицевым счетам будет рассчитываться только объем распределения (формула описана выше). Сумма затрат по лицевому счету рассчитается в соответствии с распределенным количеством и действующим на дату документа по услуге тарифом.
  • Если суммарное количество услуги по всем лицевым счетам равно «0», то выводится сообщение об отсутствии данных для распределения.

Например:

1. На здание «г.Москва, ул.Перова, дом №1» по услуге «Отопление» нужно распределить фактические затраты в размере 1000 рублей и в количестве 10 ГКал пропорционально общей площади (способ распределения «По площади» и дополнительный параметр распределения «Общая площадь»). В данном здании открыто два лицевых счета: «л/с №1» с общей площадью 40 кв.м. и «л/с №2» с общей площадью 50 кв.м. Тариф по услуге установлен в размере 1006.04 руб. Расчет выполняется при установленной точности объема начислений до 2 знаков после запятой (задается в «Настройках параметров учета (ЖКХ)»).

Если выбрать вариант распределения «По объему и сумме» и указать количество и сумму фактических затрат, то будет распределяться и количество, и сумма затрат.

Внизу формы документа расположены кнопки:

  • «ОК» – записать, провести документ и закрыть форму документа;
  • «Записать» – записать документ;
  • «Закрыть» – закрыть форму документа.

Документ можно открыть через меню «Учет в ЖКХ / Документы / Начисление/Оплата / Ввод фактических затрат на объект».
