Документ «Судебное соглашение»

Документ предназначен для отражения факта заключения судебного соглашения на погашение задолженности по услугам с распределением оплаты по месяцам.

При вводе нового документа необходимо указать:

  • «Дата» – дата документа;
  • «Номер» – номер документа;
  • «Организация» – организация, к которой относится данный документ;
  • «Лицевой счет» – из справочника «Лицевые счета» выбирается лицевой счет, по которому заключено судебное соглашение;
  • «Дата начала платежей» – дата начала платежей по судебному соглашению;
  • «Сумма по соглашению» – распределяемая сумма задолженности;
  • «Срок по соглашению» – количество месяцев, на которые распределяется сумма задолженности;
  • табличная часть «График платежей» заполняется автоматически при нажатии кнопки «Заполнить график платежей»:
    • «N» - номер строки табличной части устанавливается автоматически;
    • «Месяц платежа» - месяц платежа;
    • «Сумма» - сумма платежа.

В поле «Комментарий» – указывается комментарий к документу.

Внизу формы документа расположены кнопки:

  • «ОК» – записать и провести документ;
  • «Записать» – записать документ;
  • «Закрыть» – закрыть форму документа.

Документ можно открыть с помощью пункта меню «Учет в ЖКХ / Документы / Судебное соглашение».
