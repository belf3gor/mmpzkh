Исходящий электронный документ

Документ предназначен для хранения и отображения исходящих электронных документов.
Создать электронный документ можно двумя способами:

  • Из списков прикладных документов конфигурации, для которых предусмотрена возможность обмена электронными документами.
  • Из рабочего места Текущие дела по ЭДО.
