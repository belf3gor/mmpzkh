Документ "Изменение понижающего коэффициента для расчета амортизации НМА (налоговый учет)"

НМА ® Параметры амортизации ® Изменение понижающего коэффициента для расчета амортизации НМА (налоговый учет)

Документ предназначен для изменения понижающего коэффициента для расчета амортизации НМА для целей налогового учета по налогу на прибыль.

В документе указывается список нематериальных активов, для которых изменяется понижающий коэффициент.

  • Нематериальный актив - нематериальный актив.
  • Понижающий коэффициент - понижающий коэффициент для расчета амортизации НМА для целей налогового учета по налогу на прибыль. Указанный коэффициент будет применяться для начисления амортизации, начиная со следующего месяца. Значение можно заполнить автоматически по кнопке Заполнить - Для списка НМА.
