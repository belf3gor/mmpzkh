Документ «Установка счетчика»

Документ предназначен для отражения фактов включения/отключения Приборов учета на Здания, Подъезды, Помещения и/или Лицевые счета.

При установке прибора учета на лицевой счет следует обратить внимание, что лицевой счет должен был открыт по состоянию на дату включения прибора учета.

Документ доступен в разделе «Приборы учета» интерфейса программы.

Установка приборов учета

Для установки приборов учета необходимо создать новый документ, в поле «Действие» выбрать значение «Включить счетчик», в табличной части указать:

  • «Объект» для которого устанавливается прибор учета. При этом еще справочно в отдельной колонке отображается Здание, где располагается объект. Список объектов может быть заполнен автоматически с помощью обработки «Заполнение получателями услуг», которая вызывается с помощью кнопки «Заполнить» командной панели табличной части.
  • «Стояк» на который закрепляется указанный счетчик (выбрается из справочника Стояки).

Данное поле является необязательным для заполнения и доступно, если в программе включен функционал использования учета по стоякам.

  • «Счетчик» который закрепляется за указанным объектом (выбрается из справочника Счетчики).
  • «Номер пломбы» при необходимости.
  • «Дату включения» прибора учета.
  • «Дату поверки» прибора учета. С даты поверки прибор учета считается отключенным.

    Даты включения и поверки можно установить/поменять автоматически сразу для всех приборов учета в табличной части, воспользовавшись соответствующей командой в подменю «Установить» командной панели табличной части.

При установке приборов учета может быть сформирована печатная форма «Акт допуска в эксплуатацию узла учета», которая доступна в подменю «Печать» командной панели формы документа.

При этом можно сформировать акты допуска:

  • «По всем строкам» - акты допуска будут сформированы для всех приборов учета из табличной части.
  • «По объекту» - акт допуска будет сформирован только для прибора учета в активной строке табличной части.

Отключение приборов учета

Для отключения приборов учета необходимо создать новый документ, в поле «Действие» указать значение «Отключить счетчик» и выбрать «Причину отключения». Это может быть поверка, выход из строя, утрата или прочее. Далее в табличной части указать:

  • «Объект» по которому установлен прибор учета.
  • Отключаемый «Счетчик».
  • «Дату отключения» прибора учета.

    ВАЖНО: Дата отключения считается последним днем действия прибора учета. Фактическое отключение прибора производится на следующий день после даты отключения.

    Дату отключения можно установить/поменять автоматически сразу для всех приборов учета в табличной части, воспользовавшись соответствующей командой в подменю «Установить» командной панели табличной части.

Ввод начальных и конечных показаний приборов учета

Для ввода показаний предназначен документ «Ввод показаний счетчика», который также доступен в разделе «Приборы учета» интерфейса программы.

Но если на момент установки прибора учета известны начальные показания, то осуществить их ввод можно непосредственно из документа «Установка счетчика».

Для этого достаточно:

  • Установить флаг «Ввести начальные показания» - в табличной части станут доступными колонки для ввода показаний.
  • Ввести показания приборов учета в соответствии с тарифностью.
  • Нажать кнопку «Сформировать» под табличной частью.

В результате будут автоматически сформированы документы ввода показаний.

После того как показания отражены в документах ввода показаний они больше не доступны для изменения в документе установки счетчика – отображаются только справочно.

Ссылки на сформированные документы ввода показаний отображаются под табличной частью.

Если обнаружится, что были введены неверные показания, то изменить их можно уже непосредственно в документе ввода показаний, перейдя по ссылке на соответствующий документ.

Аналогично при отключении приборов учета можно ввести последние показания, снятые на момент отключения, установив флаг «Ввести конечные показания».

Техническая информация

При проведении документ делает записи в регистр сведений «Закрепленные счетчики».

  • Подробнее о правилах формирования периодов действия приборов учета можно ознакомиться в справке регистра «Закрепленные счетчики».
