Документ "Начисление пеней"

Документ предназначен для начисления пеней по договорам с покупателями.

В шапке документа указываются следующие реквизиты:

  • Контрагент - покупатель, по договору с которым начисляются штрафные санкции;

  • Договор - договор, служащий основанием для начисления штрафных санкций;

  • Ставка пени - ставка процента за период расчета. В качестве периода расчета может быть принят День, Месяц и Год. При указании расчетного периода в Месяц или Год, пени рассчитываются из расчета 30 и 360 дней соответственно;

  • Счет расчетов - счет для отражения задолженности покупателя по расчетам по пени.

Заполнение табличной части Задолженность возможно выполнить как в ручном так и в автоматическом режиме с помощью меню Заполнить. При автоматическом заполнении в табличную часть попадают остатки по документам расчетов с покупателями с истекшим сроком оплаты. Срок оплаты может быть задан в карточке договора по флажку Установлен срок оплаты. Если указанный флажок не установлен, то используется срок оплаты по умолчанию, указанный в Сроках оплаты покупателями. В случае, если срок оплаты в карточке договора и в целом по информационной базе не установлен - задолженность считается просроченной с момента возникновения.

В табличной части указываются следующие реквизиты:

  • Документ расчетов - документ, по которому возникла задолженность;

  • Представление - представление документа для печатной формы;

  • Задолженность - сумма задолженности, на которую начисляются пени;

  • Просрочка - срок просрочки в днях;

  • Пени - сумма пени, рассчитанная исходя из задолженности, срока просрочки в днях и ставки пени, указанной в шапке документа.

    Документ имеет печатную форму "Начисление пеней".
