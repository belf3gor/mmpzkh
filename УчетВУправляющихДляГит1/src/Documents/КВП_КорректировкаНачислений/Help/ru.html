Документ «Корректировка начислений»

 

Документ предназначен для корректировки начислений за услуги по лицевым счетам.

Перед вводом информации в новый документ необходимо выбрать один из следующих видов операций:

  • «Корректировка» – операция предназначена для ручной корректировки начислений;
  • «Перерасчет по площади» – операция предназначена для корректировки начислений по услугам, которые зависят от площади помещения, в случаях, когда изменялась эта площадь;
  • «Перерасчет по проценту» – операция предназначена для корректировки начислений, где можно указать процент перерасчета;
  • «Недопоставка» – операция предназначена для перерасчета услуг, по которым возникла недопоставка в расчетном месяце;
  • «Перерасчет по изменению лицевых счетов» – операция предназначена для перерасчета начислений по услугам, которые зависят от количества жильцов, в случаях, когда в расчетном месяце состав жильцов менялся; 
  • «Перерасчет комиссии банка» – операция предназначена для перерасчета комиссий банка.

Для всех операций, кроме операции «Корректировка», предусмотрено автоматическое заполнение табличной части. Заполнение можно произвести из меню «Заполнить» командной панели табличной части документа. Возможные варианты автоматического заполнения табличной части:

  • Пункт меню «Заполнить» - автоматический анализ начислений, по которым необходимо произвести корректировку;
  • Пункт меню «Заполнить с отбором» - открывает форму подбора объектов учета;
  • Пункт меню «Заполнить по документу оплаты» - данные для корректировки буду считаны с выбранного документа «Регистрация оплаты».

Также имеется возможность автоматического расчета суммы корректировки при помощи кнопки «Рассчитать».

При вводе нового документа необходимо указать:

  • «Дата» – дата документа;
  • «Номер» – номер документа;
  • «Организация» – организация, производящая начисление пени;
  • «Корректировать» – режим корректировки услуг:
    • «Только основные услуги» – при установке данного режима можно откорректировать только выбранные услуги;
    • «Основные и зависимые услуги» – при установке данного режима можно откорректировать как основные, так и зависимые от них услуги. Зависимость услуг задается в регистре сведений «Зависимости услуг»;
    • «Только зависимые услуги» - при установке данного режима можно откорректировать только зависимые услуги. Зависимость услуг задается в регистре сведений «Зависимости услуг»;
На форме данного документа расположена табличная часть, состоящая из колонок:
  • «N» – порядковый номер строки;
  • «Лицевой счет» – из справочника «Лицевые счета» выбирается лицевой счет, по которому будет произведена корректировка начисления по указанной услуге;
  • «Месяц начисления» – месяц, в котором производилось начисление;
  • «Услуга» – из справочника «Услуги» выбирается услуга, по которой будет произведена корректировка начисления. Если услуга занесена в список услуг, не подлежащих перерасчету, в учетной политике (ЖКХ), то корректировка начислений по данной услуге запрещена;
  • «Услуга-основание» – из справочника «Услуги» выбирается услуга-основание, для зависимой услуги;
  • «Квартира» – отображается квартира, соответствующая данному лицевому счету;
  • «Владелец» – отображается ответственный собственник (наниматель) данного лицевого счета;
  • «Сумма начисления» – указывается сумма начисления по указанной услуге;
  • «Тип начисления» – указывается тип перерасчета по указанной услуге. Доступные значения:
    • «Индивидуальное» – сумма перерасчета будет относиться к индивидуальным начислениям.
    • «Общедомовое» – сумма перерасчета будет относиться к общедомовым начислениям
    • «Без разделения» – общая сумма перерасчета без разделения на типы. 

Выбор Типа начисления доступен для ненулевого значения Суммы начисления.

  • «Сумма льготы» – указывается сумма льготы по начислению по указанной услуге;
  • «Льготная категория» – из справочника «Льготные категории» выбирается льготная категория, на основании которой предоставляется льгота по данной услуге;
  • «Сумма пени» – указывается сумма пени по начислению по указанной услуге.


Корректировка

Данная операция используется, когда необходимо вручную указать сумму перерасчета.

Для возможности сторнирования нормативных начислений по показаниям приборов учета необходимо установить галочку «Корректировка нормативных начислений по приборам учета».

Перерасчет по площади

Перед началом работы с данным видом операции, необходимо указать:

    • «Период» - имеется возможность задать начальный и конечный месяц;
    • «Услуга» – из справочника «Услуги» выбирается услуга, по которой будет произведена корректировка начисления. Если услуга занесена в список услуг, не подлежащих перерасчету, в учетной политике (ЖКХ), то корректировка начислений по данной услуге запрещена;
    • «Вид площади» - вид площади помещения, от которого зависит расчет услуги.

Для данного вида операции возможен перерасчет услуг со способами расчета "По площади" и "По нормам потребления". Автоматический перерасчет услуг со способом расчета "По нормам потребления" выполняется, если в характеристиках или в основании нормы потребления указана "Площадь". Площадь, по которой выполняется перерасчет услуги, определяется по виду площади, указанному в шапке документа, на дату документа корректировки.

Перерасчет по проценту

Перед началом работы с данным видом операции, необходимо указать:

    • «Период» - имеется возможность задать начальный и конечный месяц;
    • «Услуга» – из справочника «Услуги» выбирается услуга, по которой будет произведена корректировка начисления. Если услуга занесена в список услуг, не подлежащих перерасчету, в учетной политике (ЖКХ), то корректировка начислений по данной услуге запрещена;
    • «Процент понижения» - указывается процент, который необходимо вычесть из суммы начисления;
    • «Вариант начисления» - расчет будет производиться по одному из следующих видов начислений:
      • «Только начисления по показаниям ПУ» - в расчет будут браться только начисления, произведенные по показаниям приборов учета;
      • «Только начисления без показаний ПУ» - в расчет будут браться только начисления, произведенные при отсутствии показаний или при отсутствии действующего прибора учета (по среднему значению и/или норме потребления услуг);
      • «Все начисления» - в расчет попадет сумма двух выше описанных видов начислений.

Недопоставка

Перед началом работы с данным видом операции, необходимо указать:

    • «Период» - имеется возможность задать начальный и конечный месяц;
    • «Произвольный период» - при включенном флажке появляется возможность задать расширенный диапазон периодов, если флажок выключен период можно установить только в диапазоне одного месяца;

      Примечание
      Если используется произвольный период, то расчет корректировок по недопоставке производится за все дни периода перерасчета.

    • «Услуга» – из справочника «Услуги» выбирается услуга, по которой будет произведена корректировка начисления. Если услуга занесена в список услуг, не подлежащих перерасчету, в учетной политике (ЖКХ), то корректировка начислений по данной услуге запрещена;
    • «Время недопоставки» - указывается время недопоставки в днях или часах;
    • «Частичный перерасчет» - производится частичный перерасчет, который указывается в процентах. Возможные варианты:
      • «С понижением тарифа» - расчет производится по формуле: S = P * K * (V/Vs), где S - Сумма перерасчета, P - Процент, K - Количество потребленной услуги, V - Время перерасчета, Vs - Всего времени перерасчета за период;
      • «По проценту понижения» - расчет производится по формуле: S = Q * (P/100) * (V/Vs), где S - Сумма перерасчета, Q - Сумма начисления, P - Процент, V - Время перерасчета, Vs - Всего времени перерасчета за период;
    • «Вариант начисления» - расчет будет производится по одному из следующих видов начислений:
      • «Только начисления по показаниям ПУ» - в расчет будут браться только начисления, произведенные по показаниям приборов учета;
      • «Только начисления без показаниям ПУ» - в расчет будут браться только начисления, произведенные не по показаниям приборов учета(по среднему значению и норме потребления услуг);
      • «Все начисления» - в расчет попадет сумма двух выше описанных видов начислений.

Перерасчет по изменению лицевых счетов

Перед началом работы с данным видом операции, необходимо указать:

    • «Период» - имеется возможность задать начальный и конечный месяц;
    • «Услуга» – из справочника «Услуги» выбирается услуга, по которой будет произведена корректировка начисления. Если услуга занесена в список услуг, не подлежащих перерасчету, в учетной политике (ЖКХ), то корректировка начислений по данной услуге запрещена.

Примечание
В случае, если производится корректировка начисления, произведенного за счет повышающего коэффициента (при отсутствии действующего ИПУ), то такая корректировка имеет следующие особенности:
1. При корректировке начисления за счет повышающего коэффициента зависимые услуги не учитываются, т.е. корректировка зависимых услуг на основании корректировки начисления за счет повышения не производится.
2. Перерасчет начисления, произведенного за счет повышающего коэффициента, производится по результатам перерасчета нормативного начисления, от которого произведено повышение. При этом исходная сумма начисления за счет повышения умножается на коэффициент перерасчета, который определяется по формуле: K = S / Q, где S - Сумма текущего расчета нормативного начисления, Q - Сумма исходного нормативного начисления.

Перерасчет комиссии банка

Перед началом работы с данным видом операции, необходимо указать:

    • «Период» - имеется возможность задать начальный и конечный месяц;
    • «Услуга» – из справочника «Услуги» выбирается услуга, по которой будет произведена корректировка начисления. Если услуга занесена в список услуг, не подлежащих перерасчету, в учетной политике (ЖКХ), то корректировка начислений по данной услуге запрещена;
    • «Процент комиссии банка» - указывается процент комиссии банка, по которому произведется перерасчет.

 

Способы отражения перерасчетов по типам начислений:

  1. Для всех видов операции, кроме «Корректировка», при заполнении таблицы по кнопке «Заполнить» или «Заполнить с отбором» сведения о начислениях по услугам автоматически компонуются в разрезе типов начислений, и перерасчеты при этом будут разделены на индивидуальные, общедомовые и те перерасчеты ("Без разделения"), для которых тип определить нельзя:
    Например:
    - По услугам: "Антенна" и "Антенна (спутник)", - в периоде перерасчета начисления относятся к индивидуальным. Поэтому "Тип начисления" автоматически устанавливается в соответствующее значение.
    - По услуге "Водоотведение" в периоде перерасчета начисления были произведены по индивидуальному, общедомовому расходу, а также был произведен перерасчет без разделения по типу начисления. Поэтому значение "Тип начисления" автоматически устанавливается в соответствии с начислениями.

  2. Для изменения значений в колонке «Тип начисления» для всех услуг в таблице добавлена кнопка «Установить тип начисления» в командной панели таблицы:
