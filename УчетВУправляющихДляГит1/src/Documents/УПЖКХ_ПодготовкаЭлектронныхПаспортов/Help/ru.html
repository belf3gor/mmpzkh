Документ «Электронные паспорта для выгрузки на сайт»

Документ предназначен для подготовки электронных паспортов зданий, которые требуется выгрузить на сайт Управляющей организации.

 

При вводе документа необходимо указать:

  • «Номер» – номер документа устанавливается автоматически;
  • «Дата» – дата документа устанавливается автоматически;
  • «Организация» – выбирается организация из справочника «Организации»;
  • «Вид здания» – выбирается вид здания: МКД или жилой дом;
  • «Здание» – выбирается здание из справочника «Здания»;
  • «Период» – выбирается месяц, на конец которого подготавливаются данные;

    После выбора вида здания документ заполняется полями, необходимыми для формирования нужного вида электронного паспорта.

     

    Для заполнения электронного паспорта необходимо воспользоваться кнопкой «Заполнить», находящейся на форме документа.

    После заполнения электронного паспорта необходимо подтвердить факт его заполнения. Для этого следует установить тумблер в положение «Паспорт заполнен полностью».

    Электронные паспорта, заполненные полностью, могут быть отправлены на сайт Управляющей организации.

     

    В поле «Ответственный» – указывается ответственный за работу с документом пользователь.

    При необходимости можно заполнить поле «Комментарий», которое будет отображаться в списке документов.

     

    Документ позволяет:

  • Сформировать печатную форму со всеми данными электронного паспорта. Для этого необходимо воспользоваться кнопкой «Печать электронного паспорта» в меню «Печать».
  • Сформировать непосредственно электронный паспорт, который представляет собой файл формата XML установленной структуры. Для формирования файла необходимо воспользоваться кнопкой «Только сформировать» в меню «Формирование и отправка паспорта».
  • При нажатии на кнопку «Сформировать и поместить в архив» в меню «Формирование и отправка паспорта» сформировать электронные паспорта (XML-файлы) и поместить их в архив.
  • При нажатии на кнопку «Отправить на свой сайт» в меню «Формирование и отправка паспорта» отправить электронные паспорта на сайт Управляющей организации.

      Для перехода к Настройкам электронных паспортов необходимо воспользоваться кнопкой «Настройки», которая находится в верхней части формы документа.

      Документ располагается в подсистеме «Интеграция с сайтом ЖКХ».
