Документ «Закрытие лицевого счета»

Документ предназначен для закрытия лицевого счета.

При вводе документа необходимо указать:

  • «Номер» – номер документа устанавливается автоматически;
  • «Дата» – дата документа. Закрытие лицевого счета так же производится на дату документа;
  • «Лицевой счет» – из справочника «Лицевые счета» выбирается лицевой счет;
  • «Комментарий» – указывается комментарий для закрытия, например, причина закрытия лицевого счета;
  • «Обнулить сведения о задолженностях (авансах) закрываемого лицевого счета» – при установленном флажке сведения о задолженностях (авансах) выбранного лицевого счета будут обнулены.
    Если на момент закрытия лицевого счета имеются неотраженные в регламентированном учете данные о начислениях, то они также будут обнулены. При этом обнуляются данные только по актуальным сведениям для взаиморасчетов закрываемого лицевого счета.
  • «Аннулировать сведения о собственности на помещение и всех её собственниках» – при установленном флажке сведения о виде собственности и собственниках помещения, соответствующего выбранному лицевому счету, будут аннулированы. Сведения о собственности аннулируются только в том случае, если они действуют на момент закрытия лицевого счета.
    Данный флажок доступен только при упрощенном учете собственников.

Примечание
День закрытия лицевого счета считается последним днем действия лицевого счета.

После закрытия:

  • Если лицевой счет закрыт в середине текущего периода, то на данный лицевой счет начисление будет произведено автоматически за фактическое количество дней действия лицевого счета. Начиная со следующего периода в документе "Начисление услуг" автоматическое начисление на данный лицевой счет производиться не будет;
  • По закрытому лицевому счету можно произвести начисление документом "Разовое начисление услуг", но при этом пользователь будет предупрежден о том, что лицевой счет уже закрыт;
  • При автоматическом заполнении документа "Ввод фактических затрат на объект", закрытые лицевые счета учитываться не будут. При ручном добавлении закрытого лицевого счета в табличную часть, пользователь будет предупрежден об этом;
  • Если лицевой счет закрыт, но по нему имеется задолженность, то можно регистрировать оплату с помощью документа "Регистрация оплаты", при этом пользователю будет выведено сообщение о закрытии лицевого счета;
  • По закрытому лицевому счету можно провести только ручную корректировку начислений с помощью документа "Корректировка начислений", вид операции "Корректировка";
  • По закрытому лицевому счету имеется возможность производить корректировку взаиморасчетов документом "Корректировка взаиморасчетов ЖКХ";
  • Отсутствует возможность назначения новых услуг на закрытые лицевые счета;
  • На закрытый лицевой счет можно установить счетчик, но при этом пользователь будет предупрежден о том, что лицевой счет уже закрыт;
  • По закрытым лицевым счетам имеется возможность осуществлять закрытие периода документом "Закрытие периода по начислениям ЖКХ".

Внизу формы расположены кнопки:

  • «ОК» – провести документ и закрыть форму;
  • «Записать» – записать документ;
  • «Закрыть» – закрыть форму документа.

Документ можно открыть через меню «Учет в ЖКХ / Документы / Лицевые счета / Закрытие лицевого счета».

Техническая информация

Примечание
При закрытии лицевого счета автоматически выполняется:

  • отмена назначенных на лицевой счет услуг,
  • отключение установленных на данный лицевой счет приборов учета,
  • снятие отметок о проживании и регистрации жильцов,
  • отмена действующих льгот (при наличии льготников).
Все изменения регистрируются следующим днем после даты закрытия (т.к. день закрытия лицевого счета считается последним днем действия лицевого счета).

Подробнее о правилах формирования периодов действия услуг и приборов учета можно ознакомиться в справке регистров «Назначенные начисления», «Закрепленные счетчики».
