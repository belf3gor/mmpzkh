Документ "Подтверждение нулевой ставки НДС"

Документ предназначен для регистрации фактов подтверждения или не подтверждения ставки НДС 0% по операциям реализации.

На закладке Документы реализации список документов реализации заполняется автоматически по кнопке Заполнить. При этом заполняются данные обо всех операциях по реализации на дату документа по ставке НДС 0%, по которым не была подтверждена правомерность применения ставки.

Далее необходимо внести в документ информацию о том, по каким из операций применение ставки НДС 0% подтверждено или не подтверждено. 

Записи, по которым правомерность применения ставки 0% еще не определена, необходимо удалить из табличной части.

При не подтверждении экспорта нужно указать статью прочих расходов.

При проведении документа по строкам, в которых указано событие "Не подтверждена ставка 0%" регистрируются счета-фактуры выданные. В таких счетах-фактурах указываются сведения на основании данных документа отгрузки, ставка НДС 0% заменяется на ставку НДС для реализованной номенклатуры.

Все счета-фактуры, выписанные при неподтверждении ставки 0%, можно распечатать из формы документа "Подтверждение нулевой ставки НДС" по кнопке "Счета-фактуры".
