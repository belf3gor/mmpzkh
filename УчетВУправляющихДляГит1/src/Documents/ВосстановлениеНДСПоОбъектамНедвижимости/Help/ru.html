Документ "Восстановление НДС по объектам недвижимости"

Документ предназначен для восстановления суммы НДС по введенным в эксплуатацию объектам недвижимости, которые начали использоваться для операций, не облагаемых НДС (в случаях, указанных в пункте 2 статьи 170 НК РФ).

При вводе документа нужно задать порядок отражения суммы восстановленного НДС при проведении документа: 

  • Восстановление НДС отражается в книге продаж, формируются проводки по бухгалтерскому и налоговому учету - установлен флажок Отражать восстановление в книге продаж.
  • Документ не формирует движений, но его данные используются для заполнения декларации - не установлен флажок Отражать восстановление в книге продаж.

Заполнение списка объектов недвижимости

На закладке Объекты недвижимости указываются все объекты недвижимости организации, в том числе объекты, по которым требуется произвести восстановление сумм НДС, ранее принятых к вычету.

Восстановление суммы НДС по объекту отражается в налоговой декларации по НДС, если установлен флажок в графе Используется для операций, не облагаемых НДС.

Предусмотрены следующие варианты автоматического заполнения табличной части Объекты недвижимости:

  • Заполнить - Подбор объектов недвижимости - табличная часть заполняется объектами недвижимости, которые должны быть отражены в налоговой декларации по НДС в соответствии с требованиями НК РФ:
    • объект недвижимости введен в эксплуатацию не ранее 2006 года (это условие может быть отключено соответствующим флажком в форме подбора);
    • у объекта основного средства установлена группа учета ОС - здание, сооружение, земельный участок (список групп можно менять в форме подбора, если требуется отобрать основные средства с незаполненной группой, в список нужно добавить пустой элемент);
    • объект недвижимости не полностью самортизирован.

Если табличная часть документа уже заполнена объектами недвижимости, то они будут отражены в форме подбора. В этом случае по кнопке Заполнить будут отобраны только те объекты, которые отсутствуют в списке.

Кроме списка объектов недвижимости заполняются их параметры - стоимость, дата ввода в эксплуатацию для целей бухгалтерского учета, дата начисления первой амортизации для целей налогового учета. После заполнения формы подбора, некоторые объекты могут быть исключены из списка. Для этого нужно снять флажок в соответствующей строке.

  • Заполнить - Заполнить параметры объектов недвижимости - заполняются параметры объектов недвижимости, указанных в табличной части.

Заполнение списка счетов-фактур

Строки табличной части Счета-фактуры соответствуют одному объекту недвижимости. Выбранный объект недвижимости отображается в заголовке табличной части.

Счета-фактуры по объекту недвижимости можно подобрать автоматически по кнопке Заполнить - Подбор счетов-фактур. Список заполняется документами, которыми была сформирована стоимость объекта недвижимости.

В колонке Код операции для декларации указывается операция по соответствующему коду для заполнения декларации по НДС:

  • объект недвижимости, завершенный капитальным строительством подрядными организациями (1011801);
  • объект недвижимости, завершенный капитальным строительством при выполнении строительно-монтажных работ для собственного потребления (1011802);
  • объект недвижимости приобретен по договору купли-продажи (1011803).

При заполнении списка документов код декларации определяется по виду ценности документа, сформировавшего стоимость объекта недвижимости. Если по документу определился вид ценности, не соответствующий ни одному коду декларации, то такой документ не попадет в табличную часть при заполнении. В этом случае флажок следует установить вручную и сопоставить счету-фактуре один из доступных кодов операций для декларации.

    Заполнение суммы НДС, предъявленного к вычету по объектам недвижимости

    Восстанавливаемая доля НДС, ранее принятого к вычету, рассчитывается исходя из выручки по операциям реализации, не облагаемым НДС, в общей выручке организации. НДС восстанавливается в течение десяти лет, начиная с года, в котором объект недвижимости начал использоваться для операций, не облагаемых НДС. Восстановление отражается в книге продаж, заполняется приложение к налоговой декларации по НДС (раздел 5). Приложение заполняется по всем объектам недвижимости организации, включая те, по которым нет восстанавливаемых сумм НДС.

    Долю выручки, не облагаемой НДС, можно рассчитать автоматически по кнопке Заполнить - Заполнить долю выручки табличной части Объекты недвижимости.

    Суммы НДС, предъявленные к вычету по объекту недвижимости, в табличной части Объекты недвижимости можно заполнить автоматически по данным табличной части Счета-фактуры. Автоматическое заполнение возможно, если не установлен флажок Используется для операций, не облагаемых НДС по строкам табличной части Объекты недвижимости, а также не установлен флажок Отражать восстановление в книге продаж. Заполнить данные можно по кнопке Заполнить - Заполнить суммы НДС по данным счетов-фактур или Заполнить - Заполнить суммы НДС по данным счетов-фактур в текущей строке табличной части Объекты недвижимости.

    На закладке Счета списания НДС можно выбрать счет списания НДС из плана счетов бухгалтерского учета, а также указать аналитику к счету списания.
