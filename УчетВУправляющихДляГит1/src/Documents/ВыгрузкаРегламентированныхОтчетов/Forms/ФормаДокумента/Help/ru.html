Документ «Выгрузка регламентированных отчетов»

Документ «Выгрузка регламентированных отчетов» предназначен для выгрузки данных форм отчетности на магнитный носитель (дискету) или жесткий диск компьютера в формате, утвержденном ФНС России.

Документ может быть создан двумя способами:

  • вводом вручную нового документа в журнале «Выгрузка регламентированных отчетов»
  • автоматически - при нажатии кнопка «Выгрузить» в форме регламентированного отчета.

Независимо от способа создания документа, в дальнейшем работа с документом выполняется одинаковым образом.

Номер и дата документа заполняются автоматически, и, в общем случае, изменять их нет необходимости.

Параметры отбора отчетов

Управляющие элементы этой группы позволяют указать: какие регламентированные отчеты будут включены в документ выгрузки.

В список выгружаемых отчетов буду включаться те отчеты, у которых в период отбора попадает дата окончания отчетного периода.

«Организация» - заполняется выбором из выпадающего списка, который, в свою очередь, формируется по справочнику «Организации». В документ «Выгрузка регламентированных отчетов» будут отбираться отчеты, у которых в диалоге отчета указана такая же организация.

«Заполнить» - нажатие кнопки заполняет список «Выгружаемые отчеты» отчетами, отобранными в соответствии с установками отбора.

При заполнении списка отчетов также составляется список кодов налоговых органов, в которые будут представляться отобранные формы отчетности. Список формируется по данных отобранных отчетов. Сформированный список можно увидеть, нажав кнопку справа от поля «Код налогового органа».

Так как вместе могут выгружаться только те отчеты, которые представляются в одну налоговую инспекцию, в списке следует выбрать код той ИФНС, в которую будут представляться выгруженные отчеты.

Выгружаемые отчеты

В список «Выгружаемые отчеты» включаются отчеты, которые удовлетворяют условиям отбора отчетов, указанным в группе элементов «Параметры отбора отчетов». Список заполняется автоматически при нажатии кнопки «Заполнить». Возможность добавлять отчеты в список вручную отсутствует.

Слева от наименования отчета можно поставить отметку-флажок, и таким образом, указать, что этот отчет будет выгружаться. Для выгрузки следует выбирать только те отчеты, которые представляются в одну налоговую инспекцию. При выборе кода ИФНС в соответствующем реквизите формы документа, отметки будут автоматически расставлены у тех отчетов, которые представляются в указанную инспекцию.

«Установить отметки»/«Снять отметки» - кнопки позволяют установить/снять отметки сразу для всех отчетов.

Порядок выгрузки отчетов

Выгрузка данных регламентированной отчетности выполняется в 2 этапа:

  • формируются тексты выгрузки;
  • тексты выгрузки записываются на магнитный носитель.

Для успешной выгрузки необходимо, чтобы формы отчетности были заполнены данными, а сами данные - сохранены на диске.

Проверить заполнение любого отчета можно, открыв его прямо из документа выгрузки, для этого неолбходимо дважды щелкнуть мышью по строке с наименованием отчета.

Формирование текстов выгрузки

Формирование текстов выгрузки выполняется при нажатии кнопки «Сформировать тексты выгрузки» в верхней командной панели документа «Выгрузка регламентированных отчетов».

Если в списке выгружаемых отчетов выбраны для выгрузки отчеты, представляемые в разные инспекции, будет выдано соответствующее предупреждение и выгрузка будет остановлена. В этом случае необходиом отметить только те отчеты, которые представляются в одну налоговую инспекцию, и повторить выгрузку.

В процессе выгрузки будут извлекаться сохраненные данные отчетов, отобранных для выгрузки. Если отчет, отобранный для выгрузки, в момент выполнения выгрузки открыт, и в него внесены какие-либо изменения, для выгрузки будут извлекаьбся данные, сохраненные в информационной базе, а не те данные, которые отредактировны в открытом отчете. По этой причине перед выгрузкой рекомендуется сохранить отредактированные отчеты.

Если тексты выгрузки сформированы успешно - будет выдано соответствующее сообщение в окно сообщений

В случае, если в процессе выгрузки обнаружены ошибки в данных отчета, которые не позволяют выполнить выгрузку корректно, на экран будет выдан запрос вида «Не удалось выгрузить <наименование отчета>. Прервать выгрузку?».

Если ответить «Да» - процесс выгрузки будет прерван и в окно сообщений будет выдана информация о том, какую форму отчетности не удалось выгрузить.

Если ответить «Нет» или вообще не отвечать на вопрос - в этом случае процесс выгрузки будет продолжен. По окончании процесса выгрузки в окно сообщений будет выдана информация о формах отчетности, которые не удалось выгрузить, и будет выдано дополнительное прикрепленное окно «Навигация по ошибкам», которое содержит список ошибок, обнаруженных в процессе выгрузки.

В процессе выгрузки документ «Выгрузка регламентированных отчетов» будет сохранен. а из списка «Выгружаемые отчеты» будет удалены те отчеты, которые не были отмечены для формирования текстов выгрузки.

Навигация по ошибкам

Окно «Навигация по ошибкам» содержит таблицу, состоящую из двух колонок: наименование отчета и описание ошибки.

При выделении какой-либо строки в таблице, в правой части окна выдаются «координаты» ошибочного поля отчета.

После того, как все обнаруженный ошибки будут исправлены, можно выполнить выгрузку повторно.

Внимание! После исправления ошибок отчет следует сохранить на диске!

Запись текстов выгрузки на магнитный носитель

После того, как тексты выгрузки успешно сформированы, их можно будет записать на магнитный носитель - дискету или жесткий диск.

Для записи предназначена кнопка «Сохранить» в верхней командной панели документа «Выгрузка регламентированных отчетов».

При ее нажатии будет выдан диалог выбора места сохранения файлов выгрузки: дискеты в дисководе или каталога на жестком дике компьютера. После указания места сохранения необходимо нажать кнопку «Сохранить».
