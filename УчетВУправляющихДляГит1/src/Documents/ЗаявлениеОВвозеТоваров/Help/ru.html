Документ "Заявление о ввозе товаров"

Документ предназначен для отражения в учете ввоза товаров из государств таможенного союза. Документ может быть составлен на неограниченное количество поставок от одного контрагента за отчетный период (месяц). Табличная часть документа заполняется автоматически по данным документов "Поступление (акт, накладная)".

Дополнительно в документе указывается:

- код по товарной номенклатуре внешне-экономической деятельности (ТН ВЭД);
- количество товара в единице измерения, которая установлена для данной группы товаров (кода ТН ВЭД) Единым таможенным тарифом;
- ставка НДС;
- реквизиты товаросопроводительных документов и счета-фактуры;
- страна происхождения и номер декларации на товары;
- вес товара;
- код вида транспорта;
- реквизиты спецификаций и участников сделки (при покупке товаров через комиссионера).

При проведении документа производится начисление налога на добавленную стоимость, подлежащего уплате.
По кнопке "Выгрузить" формируется файл выгрузки "Заявление о ввозе товаров и уплате косвенных налогов".
По кнопке "Сдать в ФТС" формируется "Статистическая форма учета и перемещения товаров".
