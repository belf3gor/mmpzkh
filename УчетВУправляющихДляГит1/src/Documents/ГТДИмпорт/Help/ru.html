Документ "ГТД по импорту"

Документ предназначен для отражения таможенных сборов и пошлин, зафиксированных при поступлении импортных товаров в грузовой таможенной декларации (ГТД). Документ можно ввести на основании документа Поступление товаров и услуг.

При вводе документа закладке Главное нужно указать следующие реквизиты:

  • Таможня - заполняется наименование таможни из справочника Контрагенты,
  • Номер ГТД - номер грузовой таможенной декларации, по которой необходимо ввести информацию о таможенных пошлинах и сборах,
  • Таможенные сборы и штрафы могут быть оформлены в любой валюте. Валюта документа изменяется по гиперссылке Цены и валюта,
  • Валютный депозит и Депозит на таможне (руб) - указываются договора по расчетам с таможней, по которым учитывается взимание соответственно валютных и рублевых таможенных платежей.


Здесь же необходимо заполнить сведения о таможенных сборах и платежах, а также счета учета расчетов с таможней для отражения санкций таможни и аналитику учета расходов по штрафам.

Флажок Отразить вычет в книге покупок позволяет автоматически принять НДС к вычету и отразить в книге покупок.

На закладке Разделы ГТД указывается сумма таможенной пошлины, сумма взимаемого НДС, а также список товаров по каждому разделу ГТД.

Грузовая таможенная декларация может иметь несколько разделов, в которые группируются товары с одинаковым порядком расчета таможенных платежей. Для ввода информации отдельно по каждому из разделов предназначены две связанные табличные части. В верхней таблице отображается сводная информация о таможенных пошлинах и сборах по всем разделам ГТД. При установке курсора на одну из строк этой таблицы в табличной части "Товары по разделу..." отображается информация о товарах по данному разделу ГТД.

  • Таможенная стоимость - вводится в валюте документа.
  • Ставка пошлины, Ставка НДС - указывается ставка пошлины и ставка НДС.
  • Сумма пошлины - сумма таможенной пошлины вычисляется, исходя из введенной таможенной стоимости товаров и ставки пошлины. Таможенная пошлина пересчитывается в рубли или в валюту валютного депозита с таможней, в зависимости от выбранного в табличной части варианта заполнения (графа Валюта пошлины).
  • Сумма НДС - сумма взимаемого НДС, может быть указана в рублях или в валюте (графа Валюта НДС).
  • Табличные части закладки Разделы ГТД заполняются информацией о товарах в валюте документа.
    • Список номенклатуры можно заполнить автоматически из документа Поступление (акт, накладная). По кнопке Заполнить выбирается документ поступления, из которого весь перечень товаров попадает в табличную часть соответствующего раздела ГТД. Товары, которые не относятся к данному разделу, можно удалить, после чего повторить процедуру для следующего раздела ГТД.
    • Список номенклатуры также можно заполнить автоматически по кнопке Подбор.

    • Введенная информация о таможенной пошлине и НДС автоматически распределяется пропорционально стоимости товаров, указанной в счете-фактуре, по кнопке Распределить. При необходимости информацию о таможенной пошлине и взимаемом НДС с каждого товара может скорректировать.

    • Добавлять и удалять разделы ГТД можно по кнопке Разделы ГТД.
  • Счет учета (БУ), Счет учета НДС (БУ), Счет учета (НУ) - счета бухгалтерского и налогового учета, в дебет которых будут отнесены таможенные платежи, за исключением санкций таможни.
