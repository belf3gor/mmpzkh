Документ «Выход из ТСЖ»

Документ предназначен для ввода физических лиц, выбывших из ТСЖ.

При вводе нового документа заполняются реквизиты:

  • «Номер» – номер документа устанавливается автоматически;
  • «Дата» – дата документа устанавливается автоматически;
  • «Организация» – ТСЖ выбирается из предложенного списка;
  • «Здание» – из справочника «Здания» выбирается здание;

Ниже расположена табличная часть, состоящая из колонок:

  • «N» – порядковый номер строки табличной части устанавливается автоматически;
  • «Член ТСЖ» – из справочника «Физические лица» физическое лицо, выбывшее из ТСЖ.


Внизу формы документа расположены кнопки:

  • «ОК» – записать, провести документ и закрыть форму документа;
  • «Записать» – записать документ;
  • «Закрыть» – закрыть форму документа.

Документ можно открыть через меню «Учет в ЖКХ / Смета ТСЖ / Вступление/Выход из ТСЖ / Вступление в ТСЖ».
