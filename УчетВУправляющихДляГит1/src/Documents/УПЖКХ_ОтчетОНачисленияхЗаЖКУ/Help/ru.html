Документ «Отчет о начислениях за ЖКУ»

Данный документ предназначен для подготовки сведений о платежах, начисленных за жилищно-коммунальные и сопутствующие услуги, для передачи в государственное казенное учреждение города Москвы "Городской центр жилищных субсидий" (ГКУ "ГЦЖС"). 

 

При вводе документа необходимо указать:

  • «Номер» - номер документа (устанавливается автоматически);
  • «Дата» - дата документа (устанавливается автоматически);
  • «Организация» - организация, предоставляющая сведения ГКУ "ГЦЖС";
  • «Объект» - объект (здание, подъезд или лицевой счет), по которому необходимо предоставить сведения по начислениям. Поле не обязательно для заполнения. Если поле «Объект» будет не заполнено, то при автоматическом заполнении данные будут собраны по всем объектам;
  • «Услуга» - услуга или группа услуг справочника «Услуги», по которой необходимо предоставить сведения по начислениям за ЖКУ. Поле не обязательно для заполнения. Если поле будет не заполнено, то при автоматическом заполнении данные будут собраны по всем услугам;
  • «Период» - месяц, за который предоставляются сведения по начислениям в ГКУ "ГЦЖС".

 

На форме данного документа расположены 3 табличные части:

  • «Сведения о плательщике жилищно-коммунальных и прочих услуг (файл I)» - содержит детальную информацию о плательщике;
  • «Сведения о начисленной плате за жилые помещения, коммунальные и прочие услуги (файл II)» - содержит детальную информацию о начислениях по лицевому счету плательщика, выделенного в табличной части «Сведения о плательщике жилищно-коммунальных и прочих услуг (файл I)»;
  • «Сведения о лицах, которые учитываются при расчете платы за жилищно-коммунальные и прочие услуги (файл III)» - содержит детальную информацию о зарегистрированных, фактически проживающих без регистрации лицах или собственниках (в табличную часть не попадают собственники контрагенты справочника «Контрагенты»), которые учитываются при расчете платы за жилищно-коммунальные и прочие услуги. При этом каждый житель учитывается один раз. Сведения о лицах выводятся для плательщика, выделенного в табличной части «Сведения о плательщике жилищно-коммунальных и прочих услуг (файл I)».

 

Для автоматического заполнения документа необходимо на командной панели нажать кнопку «Заполнить».

 

ВАЖНО: Перед началом работы с документом необходимо выполнить настройку выгрузки в регистре сведений «Настройка выгрузки в ГЦЖС» (можно открыть через меню «Учет в ЖКХ / Сервис / Выгрузки в ГЦЖС / Настройки выгрузки в ГЦЖС») или нажать кнопку «Настройки» на командной панели формы документа. Настройка выгрузки должна быть произведена по времени раньше, чем создан документ.

 

Колонки в табличных частях документа, которые необязательны для заполнения и предназначены для расшифровки кодов, заданных в настройках выгрузки, можно скрыть/отобразить с помощью кнопки «Расшифровка».

 

Проверить корректность заполнения данных о начислениях по лицевому счету можно с помощью отчета «Выписка по лицевому счету». Данный отчет отображает общую информацию по лицевому счету, список собственников и взаиморасчеты по услугам за указанный период. Для вызова отчета необходимо нажать кнопку «Выписка по лицевому счету» в командной панели табличной части «Сведения о плательщике жилищно-коммунальные и прочих услуг (файл I)». Отчет формируется автоматически по лицевому счету, выделенному в табличной части «Сведения о плательщике жилищно-коммунальные и прочих услуг (файл I)».

 

Из документа можно сформировать список плательщиков и список жильцов, которые учитываются при расчете платы за жилищно-коммунальные и прочие услуги. Для этого необходимо нажать на кнопку «Печать» и выбрать одну из доступных печатных форм.

 

Для выгрузки подготовленных данных необходимо воспользоваться кнопкой командной панели «Сохранить на диск», после чего выбрать каталог сохранения файлов. Если все данные в документе заполнены корректно, то в указанный каталог будут сохранены 3 файла в формате *.txt. Наименование файлов удовлетворяют следующим условиям:

  1. «P1_NNNNN_ММГГГГГ» - файл, содержащий сведения о плательщике за жилищно-коммунальные и прочие услуги.
  2. «P2_NNNNN_ММГГГГГ» - файл, содержащий сведения о начислениях за жилые помещения, коммунальные и прочие услуги.
  3. «P2_NNNNN_ММГГГГГ» - файл, содержащий сведения о жильцах, которые учитываются при расчете платы за жилищно-коммунальные и прочие услуги.

Где:

  • «NNNNN» - код организации.
  • «ММГГГГГ» - месяц и год, за которые предоставлены сведения по начислениям.

По завершении процедуры сохранения файлов программой будет предложено сформировать итоговый отчет и сводную ведомость по выгруженным данным.

Принятый в ГКУ "ГЦЖС" отчет можно пометить флажком "Отчет сдан", при этом поля на форме документа станут недоступными для редактирования.


Документ можно открыть через меню «Учет в ЖКХ / Сервис / Выгрузки в ГЦЖС / Отчет о начислениях в ЖКХ».
