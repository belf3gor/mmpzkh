Документ «Установки процентов вознаграждения по услугам»

Документ предназначен для установки процентов вознаграждения по услугам и процентов штрафа в случае недовыполнения сбора по услугам на сумму недоплаты, если сумма сбора менее 95% от суммы начислений, по определенному поставщику услуг. Для каждого поставщика вводится отдельный документ установки процентов.

При вводе нового документа необходимо указать:

  • «Дата» – дата документа;
  • «Номер» – номер документа;
  • «Организация» – организация, к которой относится данный документ;
  • «Управляющая компания» – управляющая компания выбирается из справочника «Контрагенты»;
  • «Поставщик услуг» – поставщик услуг выбирается из справочника «Контрагенты»;
  • «Способ хранения процента» – способ хранения процента выбирается из списка: без НДС или НДС в том числе;
  • в табличной части задаются:
    • «N» номер строки табличной части устанавливается автоматически;
    • «Виды услуг» вид услуги из справочника «Виды услуг»;
    • «Услуга» услуга из справочника «Услуги»;
    • «Процент» процент вознаграждения;
    • «Процент штрафа» процент штрафа;
    • «Способ расчета» способ расчета.

В поле «Комментарий» – указывается комментарий к документу.

Внизу формы документа расположены кнопки:

  • «ОК» – записать и провести документ;
  • «Записать» – записать документ;
  • «Закрыть» – закрыть форму документа.

Документ можно открыть с помощью пункта меню «Учет в ЖКХ / Документы / Учет вознаграждений / Установка процентов вознаграждения по услугам».
