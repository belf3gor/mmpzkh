Документ «Ввод показаний счетчика»

Документ предназначен для ввода показаний Приборов учета, установленных на Здания, Подъезды, Помещения и/или Лицевые счета.

Документ доступен в разделе «Приборы учета» интерфейса программы.

Контроль вводимых показаний

Если в Настройках параметров учета ЖКХ установлена настройка контроля показаний, то в документе используется контроль вводимых показаний, а именно:

  • Проверка превышения введенного расхода над средним расходом по прибору учета за последние 6 месяцев:
    • Определяются средний расход за последние 6 месяцев. В случае, если прибор учета был установлен в течение последних 6 месяцев, то средний расход определяется за срок его действия.
    • Текущий введенный расход по прибору учета приводится к расходу за один месяц по следующей формуле: [Введенный расход по тарифу] / [Количество месяцев отсутствия показаний по тарифу], где [Количество месяцев отсутствия показаний по тарифу] – количество месяцев, прошедших с месяца ввода предыдущих показаний по тарифу. Например, если предыдущие показания по прибору учета и тарифу были введены в прошлом месяце, то значение данного параметра равно 1, в позапрошлом месяце – 2, и т.д., если показания вводятся впервые и не являются начальными, то 1.
    • Определяется процент превышения текущего расхода над среднемесячным в % по формуле 100 * (ТР - СР) / СР, где ТР - текущий расход, СР - средний расход.
    • В случае, если полученный процент превышения больше допустимого, установленного в Настройках параметров учета ЖКХ, то введенные показания считаются ошибочными.
  • Проверка ввода отрицательного расхода. В случае, если текущие показания меньше предыдущих, и это не переход показаний прибора учета через нулевые показания, то считается, что введеные показания ошибочные.

Контроль показаний производится по каждому виду тарифа отдельно.

Информация о результатах проверки введенных показаний отображается непосредственно в табличной части документа, а также в информационном баннере, расположенном в нижней части формы документа.

Формирование корректировки начислений по отрицательному расходу приборов учета

Если по некоторым приборам учета, установленным на лицевые счета или помещения, введен отрицательный расход, то по таким приборам учета можно сформировать Корректировку
начислений прошлых периодов, в которых передавались ошибочные показания. При этом отрицательный расход прибора учета не будет учтен в текущих начислениях.

Перед началом формирования корректировки начислений пользователь должен подтвердить корректность введенного отрицательного расхода, установив флажок "Показания корректны" в строке показаний документа.

Формирование Корректировки начислений производится с помощью команды "Сформировать", расположенной в нижней части формы документа.

Корректировка начислений производится по методу LIFO: сначала корректируются более поздние начисления, которые ближе к дате ввода текущего показания, а затем более ранние, и т.д.

Если документ Корректировка начислений уже был сформирован, то пользователь может перезаполнить данный документ с помощью команды "Перезаполнить".

Примечание:
Если по лицевому счету по одной услуге установлено несколько приборов учета, по ним введен отрицательный расход, а даты показания различны, то корректировка суммарного отрицательного расхода по таким приборам учета будет производиться, начиная с месяца, предшествующего месяцу более поздней даты ввода показаний этих приборов учета. Если отрицательный расход полностью скорректировать не удалось, то корректировка будет производиться за счет начислений периода соответствующего более поздней дате ввода показаний этих приборов учета.

Ввод показаний зависимых приборов учета

Расход по показаниям приборов учета записывается с учетом зависимостей приборов учета, установленных в регистре сведений Зависимые приборы учета.

Например:

Пусть в регистре сведений Зависимые приборы учета настроена следующая зависимость приборов учета:

"Выходной" прибор учета

"Входной" прибор учета

ИПУ3 ИПУ2
ИПУ2 ИПУ1

В документе ввода показаний прибора учета введены следующие показания приборов учета:

№
строки

"Выходной" прибор учета

Текущее показание

1

 ИПУ1 1000
2 ИПУ2 350
3 ИПУ3 200

где:

ИПУ1, ИПУ2, ИПУ3 - наименования индивидуальных приборов учета, которые имеют способ регистрации показаний "Расход за период расчета" и коэффициент трансформации "1".

Тогда расход по приборам учета будет расчитан следующим образом:

По ИПУ1 расход составит 1000 - 350 = 650.

По ИПУ2 расход составит 350 - 200 = 150.

По ИПУ3 расход составит 200.
