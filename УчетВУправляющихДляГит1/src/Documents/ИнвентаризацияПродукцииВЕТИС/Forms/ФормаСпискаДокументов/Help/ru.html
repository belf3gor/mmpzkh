Форма списка документов

Единое рабочее место, предусмотренное для формирования, контроля и обработки документов Инвентаризация продукции ВЕТИС. Рабочее место организовано в виде двух закладок:

  • К оформлению – показываются распоряжения на оформление документов Инвентаризации продукции ВетИС. Распоряжениями могут быть такие документы, как Оприходования излишков товаров, Списание недостач товаров, Пересортица товаров, Порча товаров. Распоряжениям, по которым не оформлен документ Инвентаризация продукции ВетИС, соответствует статус Не оформлено. Оформление заявок на получение сертификата выполняется по гиперссылке Оформить.
  • Инвентаризации продукции ВЕТИС – показываются оформленные документы.

Предусмотрена возможность отбора документов в соответствии с их статусами и действиями, которые необходимо выполнить при работе с этими документами.

Для передачи данных по входящей транспортной операции ВЕТИС в систему ВЕТИС предусмотрена команда Выполнить обмен. Список всех выполненных и предстоящих операций можно открыть по команде Протокол обмена. По каждой выполненной операции можно просмотреть отправленные XML-сообщения.
