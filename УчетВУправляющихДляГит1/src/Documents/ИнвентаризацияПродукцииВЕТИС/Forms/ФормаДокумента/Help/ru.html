Инвентаризация продукции ВетИС

Документ предназначен для выгрузки в ВЕТИС информации об изменениях остатков продукции по итогам инвентаризации и позволяет выполнить следующие операции:

  • оприходовать продукцию в ВЕТИС с формированием новой записи складского журнала (операция Добавление),
  • полностью списать остаток продукции, которая числится на учете в ВЕТИС с указанной записью журнала (операция Удаление),
  • изменить количество в учете ВЕИТС по продукции с указанной записью журнала (операция Изменение).

Заполнение основной информации

На закладке Основное указывается Хозяйствующий субъект и Предприятие, на склада которого отржаеется инвентаризация.

Формирование документа инвентаризации выполняется на основании распоряжений, в качестве которых могут быть такие документы, как Оприходования излишков товаров, Списание недостач товаров, Пересортица товаров, Порча товаров.

Документ рекомендуется оформлять на основании уже отраженных в информационной базе складских актов. Кроме того, документ-основание можно заполнить следующими способами:

  • оформить новый из формы документа исходящей транспортной операции ВЕТИС (гиперссылка Оформить). По результату оформления нового документа поле Документ-основание будет заполнено автоматически созданным документом.
  • выбрать уже оформленный документ из списка документов (гиперссылка Выбрать).

Если Инвентаризация продукции ВетИС введена на основании распоряжения, то документ-основание заполняется автоматически.

Заполнение акта несоответствия

В документе необходимо указать реквизиты акта о несоответствии - документа, которым отражены результаты инвентаризации на предприятии (гиперссылка заполните акт несоответствия на форме документа). При создании документа на основании складских актов, дата и номер складского акта автоматически указывается в качестве даты и номера Акта о несоответствии. При создании документа вручную необходимо перейти к форме заполнения этих реквизитов по ссылке заполните акт несоответствия. Для документа предусмотрена печатная форма Акт о несоответствии.

Заполнение информации о товарах

В табличной части указывается отржаемая операция номенклатура и соответствующая ей продукция ВЕТИС.
Заполнить список продукции можно следующими способами:

Для продукции заполняются следующие сведения:

  • поле Количество (старое, изменение, новое) – количество номенклатуры, в зарегистрированной в справочнике Номенклатура. Вводится вручную или заполняется из документа-основания, используемого для оформления исходящей транспортной операции ВЕТИС.
  • поле Ед.изм. (ВЕТИС) – заполняется единицей измерения ВЕТИС.
  • поле Количество (ВЕТИС) (старое, изменение, новое) – фактическое количество продукции ВЕТИС, отгружаемой со склада грузоотправителя. Заполняется по данным складских остатков из журнала продукции ВЕТИС.
  • в поле Запись складского журнала заполняется, если продукция уже учтена в ВЕТИС. Запись можно подобрать командой Заполнить - Заполнить записи складского журнала. Если продукция еще не учтена в ВетИС необходимо заполнить реквизиты для формирования новой записи складского журнала. Запись будет сформирована после передачи документа в ВЕТИС и поставлена в документ автоматически. Реквизиты записи складского журнала заполняются на отдельной панели. К ним относится:
    • Признак скоропортящейся продукции,
    • Признак низкокачественно продукции,
    • Дата производства,
    • Срок годности,
    • Упаковки и штрихкоды,
    • Производители,
    • Производственные партии,
    • Страна происхождения.
  • путем подбора из справочника Номенклатура (команда Заполнить – Открыть подбор). При выборе номенклатуры система находит список продукции, соответствующий указанной номенклатуре.
  • путем подбора данных по остаткам продукции ВЕТИС на складе грузоотправителя (команда Заполнить - По записям журнала продукции ВЕТИС).
  • перезаполнить по документу основания (Заполнить – Перезаполнить по основанию). Для такой возможности продукция ВЕТИС должна быть сопоставлена с номенклатурой, указанной в документе-основания.
  • автоматически по данным остатков продукции ВЕТИС (если используется учет по сериям) (команда Заполнить записи складского журнала).

Этапы работы по инвентаризации продукции ВЕТИС

На каждом этапе работы документу присваивается статус и система автоматически выдает рекомендации по дальнейшим действиям при работе с этим документом. В процессе подготовки и обмена данными инвентаризация продукции ВЕТИС проходит следующие этапы (на примере одного из сценариев):

  • Подготовка документа в статусе Черновик.
  • Передача данных. При передачи данных ВЕТИС документу могут быть присвоены статусы:
    • К передаче – данные ВЕТИС переданы в сервис.
    • Обрабатывается – данные ВЕТИС проходят обработку в сервисе, ожидаем завершения обработки данных.
    • Отклонен ВетИС – по результату обработки в сервисе произошла ошибка, проведение инвентаризации отклонено с указанием причины. После того как будет устранена причина отклонения документ можно вновь оформить по соответствующей гиперссылке на форме документа.
    • Выполнен - при успешном проведении инвентаризации.
    • Выполнен через web - присваивается документу, созданному через web. Потребуется подтвердить корректность документа, созданного через web по соответствующей гиперссылке на форме документа.
