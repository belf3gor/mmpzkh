Документ «Отчет о выпадающих доходах»

Данный документ предназначен для подготовки сведений и формирования отчетов по выпадающим доходам для передачи в государственное казенное учреждение города Москвы "Городской центр жилищных субсидий" (ГКУ "ГЦЖС").

ГУ ГЦЖС производит компенсацию льгот на основании постановления Правительства Москвы от 04.06.2002 г. № 411-ПП, а также в соответствии с постановлением Правительства Москвы от 07.12.2004 г. № 850-ПП. Сведения выгружаются по формату в соответствии с письмом ГКУ "ГЦЖС" г. Москвы от 05.02.2010 № И - 127/0.

При вводе документа необходимо указать:

  • «Номер» – номер документа устанавливается автоматически;
  • «Дата» – дата документа устанавливается автоматически;
  • «Организация» – ТСЖ выбирается из предложенного списка;
  • «Режим работы» – выбирается режим, в котором будет работать документ.
    • Основной - данный режим предназначен для подготовки данных для передачи в ГКУ "ГЦЖС";
    • Корректирующий - данный режим предназначен для подготовки корректировочных данных ранее сданного отчета, если в течение отчетного периода изменилась сумма, подлежащая компенсации за период, предшествующий отчетному.

На форме данного документа расположены 3 табличные части:

  • «Плательщики» - содержит детальную информацию о льготниках;
  • «Услуга» - услуга или группа услуг справочника «Услуги», по которой необходимо предоставить сведения о выпадающих доходах. Поле не обязательно для заполнения. Если поле будет не заполнено, то при автоматическом заполнении данные будут собраны по всем услугам;
  • «Документы, удостоверяющие льготы» - содержит детальную информацию об одной или нескольких льготных категориях выделенного в табличной части «Плательщики» льготника;
  • «Скидки» - содержит детальную информацию о предоставленных по льготной категории скидках на оплату ЖКУ. Кнопка "Все услуги" отключает отбор по льготной категории, при этом отбор будет произведен только по выделенному в табличной части «Плательщики» льготнику.

Перед началом работы с документом необходимо выполнить настройку выгрузки. Для этого:

  • Необходимо удостовериться в наличии корректной настройки выгрузки. Настройка выгрузки должна быть произведена по времени раньше, чем создан документ;
  • Указать необходимые коды в справочнике «Виды документов, удостоверяющих льготы».

Для автоматического заполнения документа необходимо на командной панели нажать кнопку "Заполнить".

По заполненным данным можно сформировать отчет о выпадающих доходах. Для этого необходимо воспользоваться кнопкой командной панели "Печать".

Для выгрузки данных необходимо воспользоваться кнопкой командной панели "Сохранить на диск", после чего выбрать каталог сохранения файла и название файла. По умолчанию название файла представляется в формате: [номер договора организации]_[дата выгрузки].txt, где "дата выгрузки" представлена в формате "ММГГГГ". Если все данные заполнены корректно, то сформируется файл в формате "TXT" и можно будет сформировать итоговый отчет и сводную ведомость по выгруженным данным.

Принятый в ГЦЖС отчет можно пометить флажком "Отчет сдан", при этом документ станет недоступным для редактирования.


Документ можно открыть через меню «Учет в ЖКХ / Сервис / Выгрузки в ГЦЖС / Отчет о выпадающих доходах».
