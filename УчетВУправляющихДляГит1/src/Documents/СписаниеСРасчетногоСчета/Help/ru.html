Документ "Списание с расчетного счета"

Документ предназначен для учета списания безналичных денежных средств с расчетного счета организации.

Документ можно ввести на основании документов: Счет от поставщика, Поступление (акт, накладная), Поступление доп. расходов, Поступление НМА, Отчет комитенту (принципалу) о продажах, Отчет комиссионера (агента) о продажах, Возврат товаров от покупателя, Платежное поручение, Ведомость на выплату зарплаты.

Ввод на основании

При вводе документа в шапке нужно указать следующие реквизиты:

  • Банковский счет - расчетный счет организации, с которого были списаны денежные средства.
  • Счет учета - счет учета денежных средств.
  • Получатель - контрагент, который является получателем денежных средств.
  • Сумма - сумма списанных с расчетного счета денежных средств.

Оплата поставщику

Для отражения оплаты поставщику за товары, работы и услуги необходимо выбрать в поле Вид операции операцию Оплата поставщику. После выбора операции нужно заполнить реквизиты расшифровки платежа:

  • Договор - договор с контрагентом. Должен иметь вид "С поставщиком", "С комиссионером (агентом)" или "С комитентом (принципалом)".
  • Погашение задолженности - способ погашения задолженности перед поставщиком в разрезе документов расчетов. Необходимо выбрать один из возможных способов: Автоматически, По документу или Не погашать.
  • Документ расчетов - указывается только при выборе способа погашения задолженности По документу. В этом случае при проведении будет погашена задолженность только по указанному документу расчетов.
  • Счет расчетов - счет учета, на котором при проведении будет погашаться остаток задолженности. Не указывается при выборе способа погашения задолженности Не погашать.
  • Счет авансов - счет учета, на который относится часть оплаты, оставшаяся нераспределенной после погашения задолженности контрагента.

Общую сумму произведенной оплаты можно распределить для отражения в учете по нескольким договорам или документам расчетов.

Возврат покупателю

Для отражения возврата покупателю ранее полученной оплаты необходимо выбрать в поле Вид операции операцию Возврат покупателю. После выбора операции нужно заполнить реквизиты расшифровки платежа:

  • Договор - договор с контрагентом. Должен иметь вид "С покупателем", "С комиссионером (агентом)" или "С комитентом (принципалом)".
  • Погашение задолженности - способ погашения задолженности перед покупателем по ранее полученным авансам в разрезе документов расчетов. Необходимо выбрать один из возможных способов: Автоматически, По документу или Не погашать.
  • Документ расчетов - указывается только при выборе способа погашения задолженности По документу. В этом случае при проведении будет погашена задолженность по авансу, полученному от покупателя только по указанному документу расчетов.
  • Счет авансов - счет учета, на котором при проведении будет погашаться остаток задолженности по ранее полученным авансам.
  • Счет расчетов - счет учета, на который относится часть возвращаемой суммы, оставшаяся нераспределенной после погашения задолженности контрагента.

Общую сумму произведенного возврата можно распределить для отражения в учете по нескольким договорам или документам расчетов.

Перечисление налога

Для отражения перечисления налога (сбора, взноса) в бюджет необходимо выбрать в поле Вид операции операцию Перечисление налога. После выбора операции нужно заполнить реквизиты расшифровки платежа:

  • Счет дебета - счет учета, на котором учитываются расчеты по данному налогу (сбору).

Возврат займа контрагенту

Для отражения погашения задолженности нашей организации по полученным займам от контрагентов (за исключением банков) необходимо выбрать в поле Вид операции операцию Расчеты по кредитам и займам. После выбора операции нужно заполнить реквизиты расшифровки платежа:

  • Вид платежа - определяет характер платежа по строке расшифровки. Может принимать значения "Погашение долга" и "Уплата процентов".
  • Договор - договор с контрагентом. Должен иметь вид "Прочее".
  • Счет расчетов - счет учета, на котором учитываются расчеты по полученному займу.

Возврат кредита банку

Для отражения погашения задолженности нашей организации по кредитам в банках, необходимо выбрать в поле Вид операции операцию Возврат кредита банку. После выбора операции нужно заполнить реквизиты расшифровки платежа:

  • Вид платежа - определяет характер платежа по строке расшифровки. Может принимать значения "Погашение долга" и "Уплата процентов".
  • Договор - договор с контрагентом. Должен иметь вид "Прочее".
  • Счет расчетов - счет учета, на котором учитываются расчеты по полученному кредиту.

Выдача займа контрагенту

Для отражения выдачи займов другой организации, необходимо выбрать в поле Вид операции операцию Выдача займа контрагенту. После выбора операции нужно заполнить реквизиты расшифровки платежа:

  • Договор - договор с контрагентом. Должен иметь вид "Прочее".
  • Счет расчетов - счет учета, на котором учитываются расчеты по выданному займу.

Прочие расчеты с контрагентами

Для отражения прочих операций, связанных с расчетами с прочими дебиторами и кредиторами, за исключением операций по оплате товаров, работ и услуг, а также за исключением операций по кредитам и займам, необходимо выбрать в поле Вид операции операцию Прочие расчеты с контрагентами. После выбора операции нужно заполнить реквизиты расшифровки платежа:

  • Договор - договор с контрагентом. Должен иметь вид "Прочее".
  • Счет расчетов - счет учета, на котором учитываются расчеты с контрагентом.

Перевод на другой счет организации

Для отражения перемещения денежных средств с одного расчетного счета нашей организации на другой необходимо выбрать в поле Вид операции операцию Перевод на другой счет организации.

После выбора операции нужно заполнить реквизиты шапки:

  • Счет получателя - расчетный счет нашей организации, на который были переведены денежные средства.

Также нужно заполнить реквизиты расшифровки платежа:

  • Счет дебета - счет учета денежных средств получателя.  Заполняется только в том случае, если в учетной политике не выбрана опция "Использовать 57 счет для перемещения денежных средств". Если опция установлена, то выбор счета дебета не доступен, и в качестве корр.счета всегда будет использован счет 57

Перечисление заработной платы

Для отражения безналичного перечисления заработной платы на банковские счета сотрудников необходимо выбрать в поле Вид операции операцию Перечисление заработной платы. После выбора операции нужно заполнить закладки:

  • Перечисление ЗП
  • Счета затрат на компенсацию за задержку зарплаты

На закладке Перечисление ЗП указываются суммы, перечисленные сотрудникам. При этом заполняются реквизиты: 

  • Ведомость - ведомость на выплату заработной платы. В ведомости должен быть указан способ выплаты Через банк.
  • Сумма платежа - сумма перечисленной зарплаты.
  • Физ.лицо - сотрудник организации, которому перечисляется зарплата.

На закладке Счета затрат на компенсацию за задержку зарплаты указывается Счет затрат, на который будут отнесены суммы, выплаченные сотрудникам в качестве компенсации, предусмотренной законодательством в случае задержки выплаты зарплаты.

Чтобы перенести в табличную часть Перечисление ЗП выплаты сразу всем сотрудникам по ведомости, в меню Заполнить нужно выбрать Заполнить по ведомости на выплату зарплаты и указать нужную ведомость.

Одним платежным документом может перечисляться зарплата, начисленная по нескольким ведомостям. Для переноса в табличную часть Перечисление ЗП всех выплат по второй и последующим ведомостям нужно в меню Заполнить выбрать Добавить из ведомости на выплату зарплаты.

Перечисление подотчетному лицу

Для отражения перечисления средств на банковскую карту подотчетного лица необходимо выбрать в поле Вид операции операцию Перечисление подотчетному лицу. После выбора операции нужно заполнить реквизиты расшифровки платежа:

  • Подотчетное лицо - сотрудник организации, которому перечислены денежные средства под отчет.

Перечисление сотруднику по договору подряда

Для отражения перечисления средств на банковскую карту сотрудника дохода в рамках исполнения гражданско-правового договора, предметом которого является выполнение работ, оказание услуг (договора подряда) необходимо выбрать в поле Вид операции операцию Перечисление сотруднику по договору подряда. После выбора операции нужно заполнить реквизиты расшифровки платежа:

  • Получатель - физическое лицо, которому перечислен доход.

Личные средства предпринимателя

Для отражения операции перевода денежных средств со счета для ведения предпринимательской деятельности на личный счет индивидуального предпринимателя, необходимо выбрать в поле Вид операции операцию Личные средства предпринимателя. Операция доступна только для индивидуальных предпринимателей.

Снятие наличных

Для отражения операции снятия наличных денежных средств с расчетного счета, необходимо выбрать в поле Вид операции операцию Снятие наличных. Ввод платежного поручения для данного вида операции не доступен.

Если в учетной политике не выбрана опция "Использовать 57 счет для перемещения денежных средств", то документ не формирует проводок. Для отражения списания денежных средств с расчетного счета в бухгалтерском учете необходимо использовать документ Поступление наличных с видом операции Получение наличных в банке.

Если опция в учетной политике установлена, будет сформирована проводка по банку в корреспонденции со счетом 57.01.

Уплата налога за третьих лиц

Для отражения  перечисления в бюджет налога за других юридических или физических лиц необходимо выбрать в поле Вид операции операцию Уплата налога за третьих лиц. В поле Налогоплательщик необходимо указать контрагента, за которого уплачивается налог. В поле Получатель указывается налоговый орган - получатель платежа. Нужно заполнить реквизиты расшифровки платежа:

  • Договор - договор с контрагентом - налогоплательщиком. Должен иметь вид "Прочее".
  • Счет расчетов - счет учета, на котором учитываются расчеты с контрагентом - налогоплательщиком.

Прочее списание

Для отражения всех других операций, помимо перечисленных выше, необходимо выбрать в поле Вид операции операцию Прочее списание. После выбора операции нужно заполнить реквизиты расшифровки платежа:

  • Счет дебета - счет учета для проводки в корреспонденции со счетом учета денежных средств.

Ввод на основании

На основании Списания с расчетного счета можно ввести следующие документы:

  • Счет-фактура полученный
