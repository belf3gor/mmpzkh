Документ «Установка собственников помещений»

Документ предназначен для установки собственников помещений.

При вводе нового документа необходимо указать:

  • «Номер» – номер документа устанавливается автоматически; 
  • «Дата» – дата документа устанавливается автоматически;
  • «Вид операции» - вид операции документа: «Регистрирование прав собственности» или «Аннулирование прав собственности»;
  • «Здание» - из справочника «Здания, сооружения» выбирается здание;
  • «Помещение» - из справочника «Помещения» выбирается помещение, для которого необходимо зарегистрировать или аннулировать права собственности;
  • «Вид собственности» - вид собственности на помещение: «Государственная», «Муниципальная», «Частная» или «Ведомственная»;

Если выбирается вид собственности «Частный», то появляется флажок «Несколько собственников». При установке данного флажка поля собственника, реквизиты документов и основание исчезают и появляется таблица с такими же колонками. Таким образом можно указать несколько собственников на данное помещение.

Также для вида собственности «Частный» доступен флажок «Определить доли собственности». Он устанавливается, если частная собственность долевая и необходимо указать доли собственников.

При установке данного флажка в таблице появляется колонка «Доля». Доли собственников указываются в виде простой дроби.

Если уже были ранее введены собственники по данному помещению, то при проведении эта информация анализируется и отменяется по собственникам, не указанным в данном документе.

Можно заполнить документ имеющимися сведениями о собственности автоматически. Для этого необходимо нажать на кнопку «Заполнить».

Информация о собственности отображается в справочниках:

  • «Помещения»
    В справочнике отображается информация по собственности на данное помещение. При просмотре вида собственности открывается документ, которым введена собственность.
  • «Лицевые счета»
    В справочнике отображается информация по собственности для ответственного собственника.
  • «Жильцы»
    В справочнике отображается информация по собственности для жильца. По кнопке изменить открывается документ, которым введена собственность.
  • «Здания, сооружения»
    Тип собственности здания не задается интерактивно, а определяется по виду собственности помещений данного здания:
    • если в квартирах здания все квартиры муниципальные – дом муниципальный;
    • если в квартирах здания все квартиры государственные – дом государственный;
    • в противном случае дом общий.

При необходимости можно заполнить поле «Комментарий», которое будет отображаться в списке документов.

Документ доступен в разделе «Объекты учета» - «Учет собственности».
