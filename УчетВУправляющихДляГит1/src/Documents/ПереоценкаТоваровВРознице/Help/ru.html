Документ "Переоценка товаров в рознице" 

Документ предназначен для переоценки остатков товаров, учитываемых по продажным ценам, при изменении цен.

Переоценка на складах с видом "Розничный"
Переоценка на складах с видом "Неавтоматизированная торговая точка" (сводно)
Переоценка на складах с видом "Неавтоматизированная торговая точка" (по номенклатуре)

Печатные формы

При вводе документа в шапке обязательно нужно указать Склад, на котором выполняется переоценка.

Переоценка на складах с видом "Розничный"

В табличной части документа указывается:

  • Номенклатура товара, остаток которого переоценивается,
  • Количество (остаток товара),
  • Старая и новая розничная цена.

Для заполнения табличной части документа предназначена кнопка Заполнить. Предусмотрены следующие варианты:  

  • Заполнить по остаткам - табличная часть заполняется остатками товаров на указанном складе. Действующая цена (до переоценки) определяется по остаткам товаров в бухгалтерском учете, новая цена берется из регистра Цены номенклатуры.
  • Заполнить по измененным ценам - табличная часть заполняется остатками товаров, на которые изменилась цена. Товары, на которые цена не изменилась, в документе не заполняются. Действующая цена (до переоценки) определяется по остаткам товаров в бухгалтерском учете, новая цена берется из регистра Цены номенклатуры.

Переоценка на складах с видом "Неавтоматизированная торговая точка" (сводно)

В связи с отсутствием учета по номенклатуре, для неавтоматизированной торговой точки автоматическое заполнение документа не предусмотрено, но можно зарегистрировать изменение общей покупной стоимости товаров, имеющихся в торговой точке.

При выборе склада с видом "Неавтоматизированная торговая точка" необходимо согласится с предложением свернуть табличную часть по номенклатуре. После свертки табличной части нужно указать сумму переоценки - сумму, на которую изменилась общая покупная стоимость товаров, имеющихся в торговой точке.

Переоценка на складах с видом "Неавтоматизированная торговая точка" (по номенклатуре)

В связи с отсутствием учета по номенклатуре, для неавтоматизированной торговой точки автоматическое заполнение документа не предусмотрено, но можно заполнить данные о номенклатуре вручную. Для этого при выборе склада с видом "Неавтоматизированная торговая точка" необходимо отказаться от предложения свернуть табличную часть по номенклатуре.

В табличной части документа указывается:

  • Номенклатура товара, остаток которого переоценивается,
  • Количество (остаток товара),
  • Старая и новая розничная цена.
