Документ "Поступление на расчетный счет"

Документ предназначен для учета поступления безналичных денежных средств на расчетный счет организации по выписке банка.

Документ можно ввести на основании документов: Реализация (акт, накладная), Оказание производственных услуг, Передача НМА, Передача ОС, Отчет комитенту (принципалу) о продажах, Отчет комиссионера (агента) о продажах, Выдача наличных, Платежное требование, Счет покупателю.

Ввод на основании

При вводе документа в шапке нужно указать следующие реквизиты:

  • Банковский счет - расчетный счет организации, на который были получены денежные средства.
  • Счет учета - счет учета денежных средств.
  • Плательщик - контрагент, который является плательщиком денежных средств.
  • Сумма - сумма полученных на расчетный счет денежных средств.

Оплата от покупателя

Для отражения полученной от покупателя оплаты за товары, работы и услуги необходимо выбрать в поле Вид операции операцию Оплата от покупателя. После выбора операции нужно заполнить реквизиты расшифровки платежа:

  • Договор - договор с контрагентом. Должен иметь вид "С покупателем", "С комиссионером (агентом)" или "С комитентом (принципалом)".
  • Погашение задолженности - способ погашения задолженности покупателя в разрезе документов расчетов. Необходимо выбрать один из возможных способов: Автоматически, По документу или Не погашать.
  • Документ расчетов - указывается только при выборе способа погашения задолженности По документу. В этом случае при проведении будет погашена задолженность только по указанному документу расчетов.
  • Счет расчетов - счет учета, на котором при проведении будет погашаться остаток задолженности. Не указывается при выборе способа погашения задолженности Не погашать.
  • Счет авансов - счет учета, на который относится часть оплаты, оставшаяся нераспределенной после погашения задолженности контрагента.

Общую сумму полученной оплаты можно распределить для отражения в учете по нескольким договорам или документам расчетов.

Возврат от поставщика

Для отражения возврата от поставщика ранее произведенной нашей организацией оплаты необходимо выбрать в поле Вид операции операцию Возврат от поставщика. После выбора операции нужно заполнить реквизиты расшифровки платежа:

  • Договор - договор с контрагентом. Должен иметь вид "С поставщиком", "С комиссионером (агентом)" или "С комитентом (принципалом)".
  • Погашение задолженности - способ погашения задолженности перед покупателем по выданным ранее авансам в разрезе документов расчетов. Необходимо выбрать один из возможных способов: Автоматически, По документу или Не погашать.
  • Документ расчетов - указывается только при выборе способа погашения задолженности По документу. В этом случае при проведении будет погашена задолженность по авансу, выданному поставщику только по указанному документу расчетов.
  • Счет авансов - счет учета, на котором при проведении будет погашаться остаток задолженности по ранее выданным авансам.
  • Счет расчетов - счет учета, на который относится часть возвращаемой суммы, оставшаяся нераспределенной после погашения задолженности контрагента.

Общую сумму полученного возврата можно распределить для отражения в учете по нескольким договорам или документам расчетов.

Получение займа от контрагента

Для отражения получения нашей организацией займов от юридических и физических лиц (за исключением банков), необходимо выбрать в поле Вид операции операцию Получение займа от контрагента. После выбора операции нужно заполнить реквизиты расшифровки платежа:

  • Договор - договор с контрагентом. Должен иметь вид "Прочее".
  • Счета расчетов - счет учета, на котором учитываются расчеты по полученному займу.

Получение кредита в банке

Для отражения получения нашей организацией кредитов в банках, необходимо выбрать в поле Вид операции операцию Получение кредита в банке. После выбора операции нужно заполнить реквизиты расшифровки платежа:

  • Договор - договор с контрагентом. Должен иметь вид "Прочее".
  • Счета расчетов - счет учета, на котором учитываются расчеты по полученному кредиту.

Возврат займа контрагентом

Для отражения возврата займов, выданных ранее другой организации, необходимо выбрать в поле Вид операции операцию Возврат займа контрагентом. После выбора операции нужно заполнить реквизиты расшифровки платежа:

  • Договор - договор с контрагентом. Должен иметь вид "Прочее".
  • Счета расчетов - счет учета, на котором учитываются расчеты по выданному займу.

Прочие расчеты с контрагентами

Для отражения прочих операций, связанных с расчетами с прочими дебиторами и кредиторами, за исключением операций, связанных с оплатой товаров, работ и услуг, а также за исключением операций по кредитам и займам, необходимо выбрать в поле Вид операции операцию Прочие расчеты с контрагентами. После выбора операции нужно заполнить реквизиты расшифровки платежа:

  • Договор - договор с контрагентом. Должен иметь вид "Прочее".
  • Счет расчетов - счет учета, на котором учитываются расчеты с контрагентом.

Инкассация

Для зачисления на расчетный счет денежных средств, ранее сданных на инкассацию в торговых точках организации, необходимо выбрать в поле Вид операции операцию Инкассация. После выбора операции нужно заполнить реквизиты расшифровки платежа:

  • Счет кредита - счет учета, на котором учитываются денежные средства, сданные на инкассацию.

Если одним банковским документом на расчетный счет зачисляются средства, инкассированные из нескольких торговых точек, то можно разбить общую сумму по нескольким торговым точкам.

Приобретение иностранной валюты

Для зачисления приобретенной иностранной валюты на валютный счет организации необходимо выбрать в поле Вид операции операцию Приобретение иностранной валюты. После выбора операции нужно заполнить реквизиты расшифровки платежа:

  • Договор - договор с контрагентом (банком) на приобретение валюты. Должен иметь вид "Прочее", валютой взаиморасчетов по договору должны быть рубли.
  • Курс приобретения валюты - курс, по которому в соответствие с договором с банком приобретается валюта.
  • Курс ЦБ РФ на дату приобретения валюты - официальный курс Центрального Банка на дату приобретения.
  • Сумма в рублях по курсу приобретения - сумма, потраченная банком на приобретение валюты по договору с нашей организацией. Рассчитывается автоматически при вводе суммы зачисленной на счет валюты и курса приобретения.
  • Счет расчетов с банком - счет учета, на котором учитываются расчеты с банком по приобретению валюты.
  • Отражать разницу в курсе в составе расходов - определяет порядок отражения в учете суммы разницы между курсом ЦБ РФ и курсом приобретения валюты банком. Как правило, эта разница учитывается в составе прочих расходов организации. Флажок Отражать разницу в курсе в составе расходов при этом должен быть установлен. Если разницу требуется отразить в учете каким-либо другим образом -  флажок необходимо снять, и тогда разница останется на счете расчетов с банком, отнести ее на нужный счет бухгалтерского учета нужно будет вручную документом Операция (бухгалтерский и налоговый учет).

Поступления от продажи иностранной валюты

Для зачисления рублевых средств, вырученных от продажи иностранной валюты, необходимо выбрать в поле Вид операции операцию Поступления от продажи иностранной валюты. После выбора операции нужно заполнить закладки:

  • Договор - договор с контрагентом (банком) на продажу валюты. Должен иметь вид "Прочее", валютой взаиморасчетов по договору должна быть та валюта, продажа которой производится.
  • Курс реализации валюты - курс, по которому в соответствие с договором с банком реализуется валюта.
  • Курс ЦБ РФ на дату реализации валюты - официальный курс Центрального Банка на дату реализации.
  • Сумма реализованной валюты - сумма валюты, которая была реализована банком по договору с нашей организацией. Рассчитывается автоматически при вводе суммы зачисленной на счет рублевой суммы и курса реализации.
  • Счет расчетов с банком - счет учета, на котором учитываются расчеты с банком по реализации валюты.

Поступление по платежным картам

При поступлении денежных средств от банка по розничным продажам, совершенным по платежным картам или по банковским кредитам, предоставленным банком нашим розничным покупателям, необходимо выбрать в поле Вид операции операцию Поступление по платежным картам. После выбора операции нужно заполнить закладки:

  • Расшифровка платежа
  • Учет услуг банка

На закладке Расшифровка платежа заполняются реквизиты: 

  • Договор - договор с банком. Должен иметь вид "Прочее".
  • Счет расчетов - счет учета, на котором учитываются расчеты с банком.

Закладка Учет услуг банка заполняется, если полученный банковский документ содержит информацию о суммах, удержанных банком за свои услуги. В этом случае заполняются реквизиты:

  • Комиссия - комиссия, удержанная банком.
  • Счет затрат - счет учета, на который будут отнесены услуги банка.

Личные средства предпринимателя

Для отражения поступления на расчетный счет используемый для ведения предпринимательской деятельности личных денежных средства индивидуального предпринимателя, необходимо выбрать в поле Вид операции операцию Личные средства предпринимателя. Операция доступна только для индивидуальных предпринимателей.

Перевод с другого счета

Для отражения перемещения денежных средств с другого расчетного счета нашей организации, необходимо выбрать в поле Вид операции операцию Перевод с другого счета.

После выбора операции нужно заполнить реквизиты шапки:

  • Счет плательщика- расчетный счет нашей организации, с которого были переведены денежные средства.

Если в учетной политике не выбрана опция "Использовать 57 счет для перемещения денежных средств", то документ с видом операции Перевод с другого счета не формирует проводки. Для отражения операции поступления денежных средств на расчетный счет в бухгалтерском учете необходимо ввести документ Списание с расчетного счета с видом операции Перевод на другой счет организации.

Если опция в учетной политике установлена, будет сформирована проводка по банку в корреспонденции со счетом 57.01.

Взнос наличными

Для отражения внесения наличных денежных средств на расчетный счет, необходимо выбрать в поле Вид операции операцию Взнос наличными.

Если в учетной политике не выбрана опция "Использовать 57 счет для перемещения денежных средств", то документ с видом операции Взнос наличными не формирует проводки. Для отражения операции поступления денежных средств на расчетный счет в бухгалтерском учете необходимо ввести документ Выдача наличных с видом операции Взнос наличными в банк.

Если опция в учетной политике установлена, будет сформирована проводка по банку в корреспонденции со счетом 57.01.

Прочее поступление

Для отражения всех других операций, помимо перечисленных выше, необходимо выбрать в поле Вид операции операцию Прочее поступление. После выбора операции нужно заполнить реквизиты расшифровки платежа:

  • Счет кредита - счет учета для проводки в корреспонденции со счетом учета денежных средств.

Ввод на основании

На основании Поступления на расчетный счет можно ввести следующие документы:

  • Счет-фактура выданный
