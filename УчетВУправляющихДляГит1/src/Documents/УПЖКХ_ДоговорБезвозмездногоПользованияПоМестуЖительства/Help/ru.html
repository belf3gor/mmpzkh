Документ «Договор безвозмездного пользования по месту жительства»

Документ предназначен для регистрации договоров безвозмездного пользования по месту жительства.

При вводе нового документа необходимо указать:

  • «Дата» – дата документа;
  • «Номер» – номер документа;
  • «Организация» – организация, к которой относится данный документ;
  • «Лицевой счет» – лицевой счет выбирается из справочника «Лицевые счета»;
  • «Проживающий» – проживающий выбирается из справочника «Проживающие».

В поле «Комментарий» – указывается комментарий к документу.

Внизу формы документа расположены кнопки:

  • «ОК» – записать и провести документ;
  • «Записать» – записать документ;
  • «Закрыть» – закрыть форму документа.

Из формы документа можно распечатать форму договора передачи в безвозмездное временное пользование квартиры с правом регистрации по месту жительства.

Документ можно открыть с помощью пункта меню «Учет в ЖКХ / Паспортный учет / Журнал учета документов по месту жительства».
