Документ «Голосование»

Документ предназначен для регистрации тем голосования и последующего подведения итогов.

При вводе нового документа указываются:

  • «Номер» – номер документа. Устанавливается автоматически;
  • «Дата» – дата документа. Указывается дата, на которую вводятся данные о суммах фактических затрат;
  • «Организация» – из справочника «Организации» выбирается организация;
  • «Реестр участников» – из списка документов «Реестр участников голосования».

Ниже на форме располагаются закладки:

  • «Список тем голосования». На этой закладке расположены 2 табличные части:
    • В первой табличной части указываются список тем голосования. Данная табличная часть состоит из колонок:
      • «N» - порядковый номер строки табличной части устанавливается автоматически;
      • «Тема» - из справочника «Темы голосования» выбирается тема голосования.
    • Во второй табличной части указываются списки ответов для выбранной темы голосования. Данная табличная часть состоит из колонок:
      • «Код» - код элемента справочника «Варианты выбора голосования» устанавливается автоматически при заполнении табличной части с темами голосования;
      • «Наименование» - из справочника «Варианты выбора голосования» устанавливается автоматически при заполнении табличной части с темами голосования.
  • «Подведение итогов». Закладка предназначена для заполнения результатов голосования. На закладке заполняются:
    • «Поле группировки» – выбирается из списка вариант заполнения табличной части расположенной ниже
    • Ниже располагается табличная часть, состоящая из колонок:
      • «Тема» - элемента справочника «Темы голосования» заполняется автоматически по табличной части на закладке «Список тем голосования»;
      • «Лицевой счет» - из справочника «Лицевые счета» заполняется автоматически по указанному реестру участников «Реестр участников голосования»;
      • «Владелец» - из справочника «Жильцы» устанавливается автоматически при заполнении колонки «Лицевой счет»;
      • «Квартира» - из справочника «Помещения» устанавливается автоматически при заполнении колонки «Лицевой счет»;
      • «Вариант» - из справочника «Варианты выбора голосования» выбирается ответ по текущей теме голосования.

В поле «Комментарий» – указывается комментарий к документу.

Документ не формирует движения.

Внизу формы документа расположены кнопки:

  • «ОК» – записать, провести документ и закрыть форму документа;
  • «Записать» – записать документ;
  • «Закрыть» – закрыть форму документа.

Документ позволяет формировать печатные формы:

  • «Бланк голосования (вертикально)» - для каждого лицевого счета, указанного в реестре голосования, на отдельных листах отображаются списки тем и, в один столбец, варианты ответов ;
  • «Бланк голосования (горизонтально)» - для каждого лицевого счета, указанного в реестре голосования, на отдельных листах отображаются списки тем и, в три столбца, варианты ответов;
  • «Результаты голосования» - сводная информация по результатам голосования;
  • «Список тем голосования» - содержит списки тем и варианты голосования.

Документ можно открыть через меню «Учет в ЖКХ / Документы / Голосование / Голосование».
