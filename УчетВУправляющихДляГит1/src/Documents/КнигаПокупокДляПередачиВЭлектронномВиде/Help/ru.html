Документ "Книга покупок для передачи в электронном виде"

Документ предназначен для хранения книги покупок в информационной базе для последующей выгрузки в электронном виде. По кнопке Заполнить документ автоматически заполняется записями, зарегистрированными в информационной базе за отчетный период документа.
