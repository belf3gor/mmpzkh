Документ «Закрытие периода по начислениям ЖКХ»

Документ предназначен для распределения авансов на долги по услугам в пределах одного договора лицевого счета.

Данный документ необходимо проводить в конце месяца, после всех начислений.

В документе можно указать «Объект», по которому будет производиться закрытие периода ЖКХ. В качестве объекта можно указать лицевой счет, здание или группу зданий или лицевых счетов. Если объект не указан, то закрытие периода ЖКХ будет производиться на все объекты.

Документ может производить распределение авансов в автоматическом режиме. Для этого флажок «Корректировка расчета» должен быть снят. Если требуется ручная корректировка, то необходимо установить флажок «Корректировка расчета».

При установленном флажке «Корректировка расчета» в документе становятся доступными табличные части:

  • «Лицевые счета» – выводятся лицевые счета, у которых имеются авансы или задолженность по услугам на конец периода, указанного в документе;
  • «Переплаты» – выводятся авансы в разрезе услуг, видов начисления (начисление или пени) и месяцев начисления;
  • «Долги» – выводится задолженность по услугам в разрезе видов начисления и месяцев начисления.

Табличные части могут быть заполнены автоматически при нажатии кнопки «Заполнить». Для выделенной строки в таблице лицевых счетов отображаются данные о переплатах и долгах в двух других табличных частях.

При нажатии кнопки «Распределить» происходит автоматическое распределение авансов по задолженностям услуг в порядке, описанном ниже. Сумма корректировки отображается в колонках «Сумма корректировки» и при необходимости может быть исправлена вручную. Сумма корректировок в пределах одной строки (лицевого счета, договора) должна быть равна 0.

Распределение авансов по задолженностям услуг зависит от настроек, установленных в Учетной политике ЖКХ на закладке «Закрытие периода», и производится в следующем порядке:

  1. Сначала аванс по услуге распределяется по долгам этой же услуги по всем месяцам задолженности (авансы берутся отдельно по начислениям, отдельно по пени). Если остается нераспределенная сумма, то она запоминается.
  2. Если в Учетной политике ЖКХ установлен флаг «Разрешить перенос аванса по начислениям на погашение задолженности по пени» и если остается нераспределенная сумма аванса по начислению по услуге, то она распределяется по долгам пени этой же услуги по всем месяцам задолженности.
  3. Далее если флаг «Разрешить перенос авансов с одной услуги на долги по другим услугам» в Учетной политике ЖКХ снят, то на этом распределение заканчивается. Если флаг установлен, то нераспределенные остатки авансов по начислениям всех услуг суммируются и распределяются дальше.
  4. Если в Учетной политике ЖКХ установлен флаг «Разрешить перенос аванса по начислениям на погашение задолженности по пени», то нераспределенные остатки авансов по начислениям услуг распределяются на задолженность по начислениям и/или пени других услуг согласно варианту оплаты пени (вариант оплаты пени указывается в настройке расчета пени на закладке «Пени»). В противном случае задолженность по пени других услуг погашаться не будет.
  5. При этом, если в Учетной политике ЖКХ установлен флаг «Выполнять перенос в рамках одного месяца начисления», то общий нераспределенный остаток аванса распределяется по долгам услуг только в пределах одного месяца начисления. В противном случае распределение производится по всем месяцам задолженности по методу ФИФО. А уже внутри каждого месяца распределяется в первую очередь по услугам, указанным в списке услуг для распределения оплаты по их порядку (порядок услуг указывается в настройках Учетной политике ЖКХ на закладке «Регистрация оплаты»), а затем - по всем остальным услугам пропорционально суммам задолженности.
