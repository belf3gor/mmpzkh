Документ «Распределение услуг сторонних организаций»

Данный документ позволяет распределить затраты на услуги сторонних организаций по домам.

При вводе нового документа необходимо указать:

  • «Дата» – дата документа;
  • «Номер» – номер документа;
  • «Организация» – организация, к которой относится данный документ.

В верхней части формы в табличной части указываются либо документы поступления товаров и услуг текущей организации, либо услуги в зависимости от выбора операции «По документам» и «Прочее» соответственно.

  • При выборе операции «По документам» табличная часть «Документы списания услуг» состоит из следующих колонок:
    • «N» – порядковый номер строки;
    • «Документ поступления услуг» – из списка документов «Поступление товаров и услуг» выбирается документ, по которому будет произведено распределение;
    • «Контрагент» – контрагент, соответствует контрагенту из документа поступления услуг;
    • «Сумма» – сумма для распределения, соответствует сумме из документа поступления услуг.
  • При выборе операции «Прочее» табличная часть «Услуги» состоит из следующих колонок:
    • «N» – порядковый номер строки;
    • «Номенклатура» – из справочника «Номенклатура» выбирается номенклатура, по которой будет произведено распределение;
    • «Количество» – количество номенклатуры для распределения;
    • «Цена» – цена номенклатуры для распределения;  
    • «Сумма» – сумма распределения.
    Для заполнения табличной части предусмотрены кнопки:
    • «Подбор» открывает справочник «Номенклатура» в режиме подбора;
    • «Заполнить» позволяет заполнить номенклатуру и сумму в соответствии с выбранным документом «Поступление товаров и услуг».

    В нижней части формы в табличной части указываются дома, по которым необходимо распределить затраты на купленные услуги:

    • «N» – порядковый номер строки;
    • «Объект» – из справочника «Здания» выбирается объект, по которому необходимо распределить затраты;
    • «Номенклатура» – наименование услуги для распределения;
    • «Единица измерения» – базовая единица измерения услуги;
    • «Количество» – количество услуги распределения;
    • «Сумма» – сумма распределения.

    Заполнение списка домов может быть произведено с помощью обработки «Заполнение получателями услуг», которая вызывается при нажатии кнопки «Заполнить» командной панели.

    После заполнения списка домов нужно в поле «Распределение сумм» указать способ распределения затрат по домам: «по площади», «пропорционально количеству домов», «пропорционально количеству проживающих».

    После выбора способа распределения нужно нажать кнопку «Распределить» командной панели. После этого услуги и суммы, указанные в верхней табличной части документа, распределятся по домам.

    Кнопка «Сбросить» командной панели предназначена для сброса текущего распределения. При этом табличная часть приводится в состояние до нажатия кнопки «Распределить».

    В поле «Комментарий» – указывается комментарий к документу.

    Внизу формы документа расположены кнопки:

    • «ОК» – записать и провести документ;
    • «Записать» – записать документ;
    • «Закрыть» – закрыть форму документа.

    Документ можно открыть с помощью пункта меню «Учет в ЖКХ / Документы / Распределение услуг сторонних организаций».
